# t检验笔记

## 什么是t检验？

t检验（t-test）是一种统计假设检验方法，用于确定两组数据的平均值是否存在显著差异。它由威廉·戈塞于1908年提出，以学生笔名（Student）发表，因此也称为Student's t-test。

## 什么时候使用t检验？

- 样本量较小（通常n < 30）
- 总体标准差未知
- 数据呈近似正态分布

## t检验的三种主要类型

### 1. 单样本t检验
- **目的**：检验样本均值与已知总体均值是否有显著差异
- **公式**：t = (x̄ - μ) / (s / √n)
- **例子**：某地区声称平均身高170cm，抽取20人测量后检验是否显著偏离170cm

### 2. 独立样本t检验
- **目的**：检验两个独立样本的均值是否有显著差异
- **公式**：t = (x̄₁ - x̄₂) / √(s²₁/n₁ + s²₂/n₂)
- **例子**：比较使用新教学方法与旧教学方法的两组学生的成绩差异

### 3. 配对样本t检验
- **目的**：检验同一组受试者在不同条件下的均值差异（前后测量）
- **公式**：t = x̄ₑ / (sₑ / √n)，其中x̄ₑ是差值的均值
- **例子**：同一组人减肥前后的体重变化

## 详细例子：独立样本t检验

### 问题背景

某制药公司开发了一种新药，声称可以降低血压。为验证药效，进行了一项对照实验：

- **实验组**（使用新药）：10名患者
- **对照组**（使用安慰剂）：10名患者

### 原始数据（收缩压 mmHg）

| 实验组 | 对照组 |
|--------|--------|
| 135 | 142 |
| 128 | 138 |
| 130 | 145 |
| 125 | 140 |
| 132 | 139 |
| 129 | 143 |
| 127 | 141 |
| 131 | 144 |
| 126 | 137 |
| 130 | 140 |

### 计算过程

**第一步：计算基本统计量**

实验组（x₁）：
- 样本量 n₁ = 10
- 均值 x̄₁ = (135+128+130+125+132+129+127+131+126+130)/10 = **129.3**
- 方差 s²₁ = 8.9

对照组（x₂）：
- 样本量 n₂ = 10
- 均值 x̄₂ = (142+138+145+140+139+143+141+144+137+140)/10 = **140.9**
- 方差 s²₂ = 7.2

**第二步：建立假设**

- H₀：μ₁ = μ₂（新药与安慰剂效果相同）
- H₁：μ₁ ≠ μ₂（新药与安慰剂效果不同）

**第三步：选择显著性水平**

α = 0.05

**第四步：计算t值**

t = (x̄₁ - x̄₂) / √(s²₁/n₁ + s²₂/n₂)

t = (129.3 - 140.9) / √(8.9/10 + 7.2/10)

t = -11.6 / √(0.89 + 0.72)

t = -11.6 / √1.61

t = -11.6 / 1.27

**t ≈ -9.13**

**第五步：确定自由度**

df = n₁ + n₂ - 2 = 10 + 10 - 2 = 18

**第六步：查找临界值或计算p值**

对于df = 18，α = 0.05（双尾检验），临界值 t₍₀.₀₂₅,₁₈₎ = ±2.101

由于 |t| = 9.13 > 2.101，p < 0.001

**第七步：做出结论**

由于 p < 0.001 < 0.05，我们**拒绝零假设**。

### 结果解释

新药组的收缩压平均值（129.3 mmHg）显著低于对照组（140.9 mmHg），差异具有统计学显著性（p < 0.001）。这表明该新药确实具有降低血压的效果。

## 假设检验步骤

1. **建立假设**
   - H₀（零假设）：两组均值相等
   - H₁（备择假设）：两组均值不相等

2. **选择显著性水平**
   - 通常α = 0.05（5%）

3. **计算t值**

4. **确定临界值或计算p值**

5. **做出结论**
   - p < α：拒绝H₀，差异显著
   - p ≥ α：无法拒绝H₀，差异不显著

## 自由度

- 单样本：df = n - 1
- 独立样本：df = n₁ + n₂ - 2
- 配对样本：df = n - 1

## 注意事项

- 数据需满足正态性假设（小样本时尤为重要）
- 方差齐性（独立样本t检验需满足）
- 数据为连续型变量
- 样本间相互独立（配对检验除外）
