# 统计学习笔记

本仓库包含统计学习入门笔记，涵盖从基础概念到实际应用的完整学习路径。通过通俗易懂的语言、大量生活类比和实例，帮助学习者理解统计学习的核心概念和应用。

## 项目简介

这是一个面向统计学习初学者的完整学习笔记库，包含了从基础统计概念到高级机器学习算法的系统性讲解。每个主题都配有详细的理论解释、可视化图表、代码示例和实践项目，帮助读者在理论和实践中建立扎实的统计学习基础。

### 主要特点

- **通俗易懂**：用简单的语言解释复杂概念，配合大量生活类比
- **理论实践结合**：每个概念都有理论解释和代码示例
- **可视化丰富**：包含大量图表帮助理解抽象概念
- **循序渐进**：按照学习难度组织内容，适合不同水平的学习者
- **完整项目**：提供完整的实践项目，巩固理论知识

### 适用人群

- 统计学习初学者
- 想系统学习机器学习的程序员
- 需要补充统计知识的数据分析师
- 对机器学习感兴趣的学生和研究人员

### 学习建议

1. **按顺序学习**：建议按照目录顺序逐步学习，每个主题建立在前一个主题的基础上
2. **理论结合实践**：理解理论后，运行代码示例加深理解
3. **动手实践**：完成每个章节的练习项目
4. **回顾总结**：定期回顾已学内容，巩固知识
5. **拓展阅读**：参考推荐资源，深入学习

## 快速入门

### 1. 克隆仓库

```bash

cd test_gh
```

### 2. 创建虚拟环境

```bash

python -m venv venv

# Windows 激活
venv\Scripts\activate

source venv/bin/activate
```

从 [01-基础概念](./statistical-learning-notes/01-基础概念/) 开始学习，按照学习路径逐步深入。

## 学习目录

### 基础部分 (难度: ⭐)

- [01-基础概念](./statistical-learning-notes/01-基础概念/) - 统计学习的定义、监督学习与无监督学习、训练集划分、偏差-方差权衡
  - 预计学习时间：2-3 小时

### 核心算法 (难度: ⭐⭐)

- [02-线性模型](./statistical-learning-notes/02-线性模型/) - 线性回归、逻辑回归、梯度下降、正则化方法
  - 预计学习时间：4-5 小时
- [03-判别分析](./statistical-learning-notes/03-判别分析/) - 线性判别分析(LDA)、二次判别分析(QDA)
  - 预计学习时间：2-3 小时

### 进阶算法 (难度: ⭐⭐⭐)

- [04-非线性模型](./statistical-learning-notes/04-非线性模型/) - 决策树、随机森林、梯度提升树
  - 预计学习时间：3-4 小时
- [05-支持向量机](./statistical-learning-notes/05-支持向量机/) - 最大间隔分类器、核函数、软间隔
  - 预计学习时间：2-3 小时
- [06-无监督学习](./statistical-learning-notes/06-无监督学习/) - K-Means、层次聚类、PCA
  - 预计学习时间：3-4 小时

### 实践应用 (难度: ⭐⭐)

- [07-模型评估](./statistical-learning-notes/07-模型评估/) - 交叉验证、评估指标
  - 预计学习时间：2-3 小时
- [08-实践项目](./statistical-learning-notes/08-实践项目/) - 线性回归实现、分类任务、可视化等
  - 预计学习时间：4-5 小时

## 常见问题

### Q: 我需要什么数学基础？

A: 建议掌握：
- 基础代数（线性方程、矩阵）
- 微积分基础（导数、极值）
- 概率论基础（概率分布、期望、方差）

如果数学基础较弱，建议先复习这些基础知识。

### Q: 每个章节大概需要多长时间？

A: 根据难度不同，预计学习时间从 2 小时到 6 小时不等。详细的时间安排请参考 [学习路径文档](./LEARNING_PATH.md)。

### Q: 代码报错怎么办？

A: 常见问题解决：
1. 确认已安装所有依赖包：`pip install -r requirements.txt`
2. 检查 Python 版本是否为 3.8+
3. 查看错误信息，确认是库版本问题还是代码问题
4. 在 GitHub Issues 中搜索是否有类似问题
5. 创建新的 issue 提供详细的错误信息

### Q: 可以跳过某些章节吗？

A: 基础概念章节建议按顺序学习，因为后续内容建立在前期知识基础上。如果您有一定基础，可以：
- 快速浏览基础章节
- 重点学习不熟悉的内容
- 直接跳到感兴趣的实践项目

### Q: 如何贡献内容？

A: 欢迎任何形式的贡献！请参考 [贡献指南](./CONTRIBUTING.md) 了解详细信息。

### Q: 推荐的学习顺序是什么？

A: 推荐的学习顺序请参考 [学习路径文档](./LEARNING_PATH.md)，其中详细说明了循序渐进的学习计划。
