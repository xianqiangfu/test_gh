# 多重比较校正方法

## 多重比较问题

### 定义

多重比较（Multiple Comparisons）是指在同一个研究中进行多个统计检验的情况。当进行多次假设检验时，第一类错误率会增加，这就是多重比较问题。

### 问题本质

当我们进行一次假设检验时，如果设定显著性水平为 α = 0.05，那么在原假设为真的情况下，错误拒绝原假设的概率是 5%。

但当进行 k 次独立的假设检验时，**至少出现一次假阳性**（第一类错误）的概率为：

$$P(\text{至少一次错误}) = 1 - (1 - \alpha)^k$$

### 问题演示

| 检验次数 k | 至少一次假阳性的概率 (α = 0.05) |
|-----------|----------------------------------|
| 1 | 5.00% |
| 2 | 9.75% |
| 3 | 14.26% |
| 5 | 22.62% |
| 10 | 40.13% |
| 20 | 64.15% |
| 50 | 92.31% |
| 100 | 99.41% |

**问题严重性：** 做 20 次检验时，至少一次假阳性的概率高达 64%！

### 多重比较的常见场景

1. **ANOVA事后检验：** ANOVA检验显著后，进行多组两两比较
2. **多个结局指标：** 同一研究测量多个结局变量
3. **亚组分析：** 在不同亚组中检验同一假设
4. **多个时间点：** 在不同时间点进行重复检验
5. **多变量分析：** 检验多个变量与结果的关联
6. **基因表达分析：** 同时检验成千上万个基因的差异表达
7. **神经影像分析：** 检验大脑多个体素的活动差异

## 错误率指标

### 1. 比较错误率（Per-Comparison Error Rate, PCER）

**定义：** 单个检验的第一类错误率

$$PCER = \frac{\text{第一类错误次数}}{\text{检验次数}}$$

**特点：**
- 不随检验次数增加而累积
- 但无法控制整体风险

### 2. 家族错误率（Family-wise Error Rate, FWER）

**定义：** 在所有检验中，至少出现一次第一类错误的概率

$$FWER = P(\text{至少拒绝一个正确的H₀})$$

**数学表达：**

$$FWER = 1 - P(\text{所有正确的H₀都未被拒绝})$$

**特点：**
- 最严格的控制标准
- 适用于错误率非常敏感的情况
- 控制策略：使 FWER ≤ α

**示例：** 做 10 次检验，控制 FWER = 0.05，意味着即使做 10 次检验，至少一次假阳性的概率也不超过 5%。

### 3. 错误发现率（False Discovery Rate, FDR）

**定义：** 在所有被拒绝的假设中，错误拒绝的比例（期望值）

$$FDR = E\left[\frac{V}{R}\right]$$

其中：
- V = 被错误拒绝的假设数（假阳性）
- R = 被拒绝的假设总数（阳性发现）
- 当 R = 0 时，定义 FDR = 0

**关系：**

$$FWER = P(V \geq 1) = P(V/R \geq 1)$$

**特点：**
- 比 FWER 宽松，但有功效
- 适用于探索性研究
- 控制策略：使 FDR ≤ q（通常 q = 0.05 或 0.10）

### 4. 假阴性率（False Negative Rate, FNR）

**定义：** 实际为真的假设中，未被拒绝的比例

$$FNR = \frac{T}{m_0}$$

其中：
- T = 错误不拒绝的假设数（假阴性）
- m₀ = 原假设为真的总数

### 错误率总结表

| 假设状态 | 不拒绝 H₀ | 拒绝 H₀ | 总计 |
|---------|-----------|---------|------|
| H₀ 为真 | U (正确) | V (第一类错误) | m₀ |
| H₀ 为假 | T (第二类错误) | S (正确) | m₁ |
| **总计** | m - R | R | m |

其中：
- m = 总检验次数
- m₀ = 原假设为真的次数
- m₁ = 备择假设为真的次数
- U = 正确不拒绝的次数
- V = 第一类错误次数
- T = 第二类错误次数
- S = 正确拒绝的次数
- R = 拒绝总数 = V + S
- m - R = 不拒绝总数 = U + T

## Bonferroni校正

### 原理

Bonferroni校正是最简单、最常用的多重比较校正方法。

**核心思想：** 将显著性水平 α 除以检验次数 k

$$\alpha' = \frac{\alpha}{k}$$

### 决策规则

对于每个检验：

- 如果 p < α'，拒绝 H₀
- 如果 p ≥ α'，不拒绝 H₀

**等价形式：** 也可以将每个 p 值乘以 k

$$p_{adjusted} = p \times k$$

- 如果 p_adjusted < α，拒绝 H₀
- 如果 p_adjusted ≥ α，不拒绝 H₀

### 优点

1. **简单易懂：** 计算非常简单
2. **严格控制：** 严格控制 FWER ≤ α
3. **通用性强：** 不需要假设检验之间的独立性

### 缺点

1. **过于保守：** 当检验次数多时，过于严格
2. **功效低：** 增加第二类错误率
3. **不适合大规模检验：** 如基因表达分析中检验上万基因

### 示例

进行 5 次检验，设定 α = 0.05：

**原始 p 值：** [0.002, 0.015, 0.030, 0.045, 0.080]

**Bonferroni校正：**

- 调整后 α' = 0.05 / 5 = 0.01
- 或调整后 p 值 = p × 5

| 检验 | 原始 p 值 | 调整后 p 值 | 是否显著 (α=0.05) |
|------|-----------|------------|-------------------|
| 1 | 0.002 | 0.010 | **是** |
| 2 | 0.015 | 0.075 | 否 |
| 3 | 0.030 | 0.150 | 否 |
| 4 | 0.045 | 0.225 | 否 |
| 5 | 0.080 | 0.400 | 否 |

**结果：** 只有第 1 个检验显著（p_adjusted = 0.010 < 0.05）

### Bonferroni的数学证明

Bonferroni不等式：

$$P\left(\bigcup_{i=1}^k A_i\right) \leq \sum_{i=1}^k P(A_i)$$

其中 A_i 表示第 i 次检验的第一类错误事件。

因此：

$$FWER = P\left(\bigcup_{i=1}^k V_i = 1\right) \leq \sum_{i=1}^k P(V_i = 1) = \sum_{i=1}^k \alpha_i = k \cdot \alpha' = k \cdot \frac{\alpha}{k} = \alpha$$

## Holm-Bonferroni方法（逐步向下法）

### 原理

Holm-Bonferroni方法是对Bonferroni校正的改进，属于**逐步逐步向下法**（Step-down Procedure），在严格控制FWER的同时提高统计功效。

### 算法步骤

1. **排序：** 将 k 个 p 值从小到大排序：p₁ ≤ p₂ ≤ ... ≤ p_k

2. **逐步检验：**
   - 第 1 步：如果 p₁ < α/k，拒绝 H₁(1)，继续下一步
   - 第 2 步：如果 p₂ < α/(k-1)，拒绝 H₁(2)，继续下一步
   - ...
   - 第 i 步：如果 pᵢ < α/(k-i+1)，拒绝 H₁(i)，继续下一步
   - ...
   - 直到某步 pᵢ ≥ α/(k-i+1)，停止检验，保留该步及之后的所有假设

3. **停止规则：** 一旦某个假设不被拒绝，保留该步及之后的所有假设

### 决策规则

$$\text{拒绝 } H_{(i)} \text{ 如果 } p_{(i)} \leq \frac{\alpha}{k - i + 1}$$

### 示例

进行 5 次检验，设定 α = 0.05：

**原始 p 值：** [0.002, 0.015, 0.030, 0.045, 0.080]

**排序后的 p 值：** p₁=0.002, p₂=0.015, p₃=0.030, p₄=0.045, p₅=0.080

**逐步检验：**

| 步骤 i | 排序后的 p 值 | 阈值 α/(5-i+1) | 是否拒绝 |
|--------|--------------|---------------|---------|
| 1 | p₁ = 0.002 | 0.05/5 = 0.010 | **拒绝** |
| 2 | p₂ = 0.015 | 0.05/4 = 0.0125 | 保留 |
| 3 | p₃ = 0.030 | 0.05/3 = 0.0167 | 保留 |
| 4 | p₄ = 0.045 | 0.05/2 = 0.025 | 保留 |
| 5 | p₅ = 0.080 | 0.05/1 = 0.050 | 保留 |

**结果：** 只有第 1 个假设被拒绝

与 Bonferroni 比较：
- Bonferroni: 阈值统一为 0.01
- Holm: 第 1 个阈值 0.01，第 2 个阈值 0.0125，...

### Holm与Bonferroni的比较

| 特性 | Bonferroni | Holm-Bonferroni |
|------|------------|-----------------|
| FWER控制 | 严格控制 | 严格控制 |
| 统计功效 | 较低 | 较高 |
| 计算复杂度 | 简单 | 中等 |
| 适用性 | 通用 | 通用 |
| 检验独立性 | 不需要 | 不需要 |

**Holm的优势：** 在严格控制FWER的前提下，比Bonferroni有更高的统计功效。

## False Discovery Rate (FDR)方法

### Benjamini-Hochberg (BH)方法

Benjamini-Hochberg (1995) 提出了控制FDR的方法，是FDR控制的经典方法。

### 算法步骤

1. **排序：** 将 k 个 p 值从小到大排序：p₁ ≤ p₂ ≤ ... ≤ p_k

2. **计算临界值：** 对于每个 i，计算 cᵢ = (i/k) × q

   其中 q 是目标FDR水平（通常 q = 0.05）

3. **寻找最大 i：** 找到最大的 i，使得 pᵢ ≤ (i/k) × q

4. **拒绝假设：** 拒绝 H₁(1), H₁(2), ..., H₁(i)

### 决策规则

$$\text{拒绝 } H_{(i)} \text{ 如果 } p_{(i)} \leq \frac{i}{k} \cdot q$$

### 示例

进行 10 次检验，设定 q = 0.05：

**原始 p 值：** [0.001, 0.003, 0.007, 0.012, 0.020, 0.035, 0.050, 0.080, 0.120, 0.250]

**排序和检验：**

| i | pᵢ | (i/k)×q | pᵢ ≤ (i/k)×q? |
|---|-----|---------|---------------|
| 1 | 0.001 | 0.005 | **是** |
| 2 | 0.003 | 0.010 | **是** |
| 3 | 0.007 | 0.015 | **是** |
| 4 | 0.012 | 0.020 | **是** |
| 5 | 0.020 | 0.025 | **是** |
| 6 | 0.035 | 0.030 | 否 |
| 7 | 0.050 | 0.035 | 否 |
| 8 | 0.080 | 0.040 | 否 |
| 9 | 0.120 | 0.045 | 否 |
| 10 | 0.250 | 0.050 | 否 |

**结果：** 拒绝前 5 个假设（i = 1 到 5）

### BH方法的优点

1. **平衡错误控制与功效：** 既控制错误率，又保持较高的功效
2. **适合大规模检验：** 特别适合基因组学、神经影像等大量检验
3. **探索性研究友好：** 适合发现性研究

### BH方法的假设

BH方法假设检验之间是**独立**或**正相关**的。

如果检验之间是负相关的，需要使用**BH-Y方法**（Benjamini-Yekutieli方法）：

$$c_i = \frac{i}{k} \cdot \frac{q}{\sum_{j=1}^k 1/j}$$

其中 $\sum_{j=1}^k 1/j$ 是调和数。

### Storey's q-value 方法

Storey提出了一种更灵活的FDR控制方法，称为q-value。

**q-value定义：** 对于给定检验，q-value是拒绝该假设时，FDR的估计值。

**优势：**
- 提供每个检验的FDR估计，而不仅仅是二元的拒绝/保留决策
- 可以根据不同的q阈值灵活调整

## 其他校正方法

### 1. Šidák校正

**原理：** 假设检验之间相互独立时，使用精确校正

$$\alpha' = 1 - (1 - \alpha)^{1/k}$$

**特点：**
- 比Bonferroni略不保守
- 要求检验之间相互独立
- 当k较大时，与Bonferroni接近

### 2. Hochberg方法（逐步向上法）

**原理：** Holm方法的反向，从最大的p值开始检验

**算法：**
1. 将p值从小到大排序：p₁ ≤ p₂ ≤ ... ≤ p_k
2. 从最大的p_k开始：
   - 如果 p_k ≤ α，拒绝所有假设
   - 如果 p_k > α，检验 p_{k-1} ≤ α?
   - 如果 p_{k-1} ≤ α，拒绝 H₁(1), ..., H₁(k-1)
   - 继续直到找到第一个不显著的检验

**特点：**
- 比Holm更有功效
- 要求检验之间相互独立或正相关

### 3. Hommel方法

**原理：** 基于拒绝域的封闭检验原则

**特点：**
- 比Hochberg更有功效
- 计算较复杂
- 严格控制FWER

### 4. Scheffé方法

**原理：** 适用于ANOVA中所有可能的线性比较

**检验统计量：**

$$F = \frac{SS_{comparison}/df_{comparison}}{MS_{error}}$$

临界值：$F_{\alpha}(df_{comparison}, df_{error})$

**特点：**
- 非常保守
- 适用于复杂的线性比较
- 适用于ANOVA事后检验

### 5. Tukey HSD (Honestly Significant Difference)

**原理：** 基于学生化范围分布

**检验统计量：**

$$q = \frac{\bar{y}_i - \bar{y}_j}{\sqrt{MS_{error}/n}}$$

**特点：**
- 专门用于ANOVA事后检验
- 控制FWER
- 适用于所有两两比较
- 比Bonferroni有功效

### 6. Dunnett检验

**原理：** 专门用于多个实验组与对照组的比较

**特点：**
- 比两两比较更有功效
- 适用于有明确对照组的设计

## 校正方法的选择

### 选择原则

| 场景 | 推荐方法 | 原因 |
|------|----------|------|
| 检验次数少（< 10） | Bonferroni 或 Holm | 简单有效，FWER控制严格 |
| 检验次数中等（10-50） | Holm | 平衡FWER和功效 |
| 检验次数多（> 50） | FDR (BH) | 功效高，适合探索性研究 |
| 基因组/神经影像 | FDR (BH 或 q-value) | 大规模检验，发现性研究 |
| ANOVA事后检验 | Tukey HSD | 专为两两比较设计 |
| 有对照组 | Dunnett | 更有功效 |
| 严格控制错误率 | Bonferroni 或 Holm | FWER控制最严格 |
| 探索性研究 | FDR | 平衡发现和错误 |
| 独立检验 | Šidák 或 Hochberg | 更有功效 |

### FWER vs FDR 的选择

| 特性 | FWER控制 | FDR控制 |
|------|----------|---------|
| 严格控制 | 非常严格 | 宽松 |
| 统计功效 | 低 | 高 |
| 适用研究 | 确证性研究 | 探索性研究 |
| 发现率 | 低 | 高 |
| 假阳性控制 | 优秀 | 良好 |
| 大规模检验 | 不适合 | 适合 |

**选择指南：**

- **选择FWER：**
  - 确证性研究
  - 错误率极其敏感（如临床试验）
  - 检验次数较少

- **选择FDR：**
  - 探索性研究
  - 发现新现象
  - 大规模检验（基因组学、神经影像）
  - 可以容忍一定比例的假阳性

### 方法对比总结

| 方法 | 控制指标 | 保守程度 | 功效 | 假设要求 | 适用场景 |
|------|----------|----------|------|----------|----------|
| Bonferroni | FWER | 非常保守 | 低 | 无 | 检验次数少 |
| Holm | FWER | 保守 | 中等 | 无 | 通用 |
| Hochberg | FWER | 中等 | 较高 | 独立/正相关 | 通用 |
| Hommel | FWER | 较不保守 | 高 | 无 | 通用 |
| BH | FDR | 宽松 | 高 | 独立/正相关 | 大规模检验 |
| Šidák | FWER | 保守 | 中等 | 独立 | 独立检验 |
| Tukey HSD | FWER | 中等 | 较高 | ANOVA | ANOVA事后检验 |
| Dunnett | FWER | 中等 | 高 | ANOVA | 对照比较 |

## Python实现示例

### 1. 基本的多重比较校正

```python
import numpy as np
import pandas as pd
from scipy import stats
from statsmodels.stats.multitest import multipletests
import matplotlib.pyplot as plt
import seaborn as sns

# 示例：假设进行了10次检验，得到以下p值
p_values = np.array([0.001, 0.003, 0.015, 0.020, 0.035, 0.050, 0.080, 0.120, 0.250, 0.450])

# 1. Bonferroni校正
reject_bonf, p_bonf, _, _ = multipletests(p_values, method='bonferroni')

print("=" * 60)
print("Bonferroni校正结果")
print("=" * 60)
results_df = pd.DataFrame({
    '原始p值': p_values,
    'Bonferroni校正后p值': p_bonf,
    '是否显著': reject_bonf
})
print(results_df.to_string())
print(f"\n显著检验数量: {sum(reject_bonf)} / {len(p_values)}")

# 2. Holm-Bonferroni校正（逐步向下法）
reject_holm, p_holm, _, _ = multipletests(p_values, method='holm')

print("\n" + "=" * 60)
print("Holm-Bonferroni校正结果")
print("=" * 60)
results_df['Holm校正后p值'] = p_holm
results_df['Holm是否显著'] = reject_holm
print(results_df.to_string())
print(f"\n显著检验数量: {sum(reject_holm)} / {len(p_values)}")

# 3. Benjamini-Hochberg FDR校正
reject_bh, p_bh, _, _ = multipletests(p_values, method='fdr_bh')

print("\n" + "=" * 60)
print("Benjamini-Hochberg FDR校正结果")
print("=" * 60)
results_df['BH校正后p值'] = p_bh
results_df['BH是否显著'] = reject_bh
print(results_df.to_string())
print(f"\n显著检验数量: {sum(reject_bh)} / {len(p_values)}")

# 4. Šidák校正
reject_sidak, p_sidak, _, _ = multipletests(p_values, method='sidak')

print("\n" + "=" * 60)
print("Šidák校正结果")
print("=" * 60)
results_df['Šidák校正后p值'] = p_sidak
results_df['Šidák是否显著'] = reject_sidak
print(results_df.to_string())
print(f"\n显著检验数量: {sum(reject_sidak)} / {len(p_values)}")

# 5. Hochberg校正
reject_hochberg, p_hochberg, _, _ = multipletests(p_values, method='hochberg')

print("\n" + "=" * 60)
print("Hochberg校正结果")
print("=" * 60)
results_df['Hochberg校正后p值'] = p_hochberg
results_df['Hochberg是否显著'] = reject_hochberg
print(results_df.to_string())
print(f"\n显著检验数量: {sum(reject_hochberg)} / {len(p_values)}")
```

### 2. FWER累积演示

```python
# 演示多重比较导致的FWER累积
def simulate_fwer(n_tests, alpha=0.05, n_sim=10000):
    """模拟多次检验的FWER"""
    fwer_list = []
    for _ in range(n_sim):
        # 生成n_tests个p值（H0为真，均匀分布）
        p_values = np.random.uniform(0, 1, n_tests)
        # 检查至少有一个p值小于alpha
        fwer_list.append(any(p_values < alpha))
    return np.mean(fwer_list)

test_numbers = [1, 2, 3, 5, 10, 20, 50, 100]
fwer_values = [simulate_fwer(n) for n in test_numbers]

print("FWER累积演示:")
print("=" * 60)
for n, fwer in zip(test_numbers, fwer_values):
    print(f"检验次数 {n:3d}: FWER = {fwer:.4f}")

# 可视化
plt.figure(figsize=(10, 6))
plt.plot(test_numbers, fwer_values, 'bo-', linewidth=2, markersize=8)
plt.axhline(y=0.05, color='r', linestyle='--', label='α = 0.05')
plt.xlabel('检验次数', fontsize=12)
plt.ylabel('至少一次假阳性的概率 (FWER)', fontsize=12)
plt.title('多重比较的FWER累积效应', fontsize=14)
plt.legend(fontsize=12)
plt.grid(True, alpha=0.3)
plt.show()
```

```python
# q-value计算
def calculate_q_values(p_values):
    """计算q-value（简化版）"""
    # 排序
    sorted_indices = np.argsort(p_values)
    sorted_p = p_values[sorted_indices]

    n = len(p_values)
    q_values = np.zeros(n)

    # 计算累积估计
    for i in range(n):
        q_values[i] = (n * sorted_p[i]) / (i + 1)

    # 修正单调性
    for i in range(n-2, -1, -1):
        q_values[i] = min(q_values[i], q_values[i+1])

    # 限制在[0, 1]
    q_values = np.clip(q_values, 0, 1)

    # 返回原始顺序
    original_order = np.argsort(sorted_indices)
    return q_values[original_order]
```

### 5. 校正方法功效比较

```python
# 比较不同方法的功效
def compare_methods_power(n_tests=20, true_effects=5, effect_size=0.8, alpha=0.05, n_sim=1000):
    """比较不同方法的功效"""
    methods = ['bonferroni', 'holm', 'fdr_bh', 'sidak']
    power_results = {method: [] for method in methods}

    for _ in range(n_sim):
        # 生成p值
        p_values = np.random.uniform(0, 1, n_tests)

        # 对有真实效应的检验添加效应
        for i in range(true_effects):
            # t检验p值计算
            n = 30
            t_stat = np.random.normal(effect_size * np.sqrt(n), 1)
            p_val = 2 * (1 - stats.t.cdf(abs(t_stat), df=2*n-2))
            p_values[i] = p_val

        # 应用不同方法
        for method in methods:
            reject, _, _, _ = multipletests(p_values, method=method)
            # 计算功效：在true_effects中拒绝的比例
            power = sum(reject[:true_effects]) / true_effects
            power_results[method].append(power)

    return {method: np.mean(powers) for method, powers in power_results.items()}

effect_sizes = [0.5, 0.8, 1.2, 1.5]
power_comparison = {}

for es in effect_sizes:
    powers = compare_methods_power(effect_size=es)
    power_comparison[es] = powers

print("\n不同效应大小下的功效比较:")
print("=" * 70)
print(f"{'效应大小':<10} {'Bonferroni':<12} {'Holm':<12} {'BH-FDR':<12} {'Šidák':<12}")
print("-" * 70)
for es, powers in power_comparison.items():
    print(f"{es:<10} {powers['bonferroni']:<12.4f} {powers['holm']:<12.4f} "
          f"{powers['fdr_bh']:<12.4f} {powers['sidak']:<12.4f}")

# 可视化

for method in ['bonferroni', 'holm', 'fdr_bh', 'sidak']:
    powers = [power_comparison[es][method] for es in effect_sizes]
```

### 6. 实际应用示例：临床试验

```python
# 临床试验示例：多个终点指标
np.random.seed(42)

# 模拟临床试验数据
n_patients = 100
n_endpoints = 6  # 6个终点指标

# 生成原始p值
p_values = np.array([0.008, 0.015, 0.025, 0.040, 0.065, 0.120])

endpoints = ['主要终点', '次要终点1', '次要终点2', '探索性终点1', '探索性终点2', '探索性终点3']

# 应用不同的校正策略
print("\n临床试验多重比较校正示例:")
print("=" * 90)

# 1. 层级策略：主要终点不校正，次要终点校正
strategy_results = []

for i, (p, endpoint) in enumerate(zip(p_values, endpoints)):
    if '主要' in endpoint:
        # 主要终点不校正
        reject = p < 0.05
        p_corr = p
        method = '不校正'
    else:
        # 其他终点用Bonferroni校正
        p_corr = p * 5  # 5个次要/探索性终点
        reject = p_corr < 0.05
        method = 'Bonferroni'

    strategy_results.append({
        '终点': endpoint,
        '原始p值': p,
        '校正方法': method,
        '校正后p值': p_corr,
        '是否显著': reject
    })

results_df = pd.DataFrame(strategy_results)
print(results_df.to_string(index=False))

# 2. 统一FDR校正
reject_bh, p_bh, _, _ = multipletests(p_values, method='fdr_bh')

print("\n" + "=" * 90)
print("统一FDR校正 (Benjamini-Hochberg):")
print("=" * 90)
fdr_results = pd.DataFrame({
    '终点': endpoints,
    '原始p值': p_values,
    'FDR校正后p值': p_bh,
    '是否显著': reject_bh
})
print(fdr_results.to_string(index=False))
print(f"\n显著终点数量: {sum(reject_bh)} / {len(p_values)}")
```

## 总结

多重比较校正是统计分析中的重要概念，尤其在进行大量假设检验时至关重要。

### 核心要点

1. **多重比较问题：**
   - 多次检验导致第一类错误率累积
   - FWER = 1 - (1-α)^k

2. **错误率指标：**
   - FWER：至少一次假阳性的概率（严格）
   - FDR：假阳性占所有阳性的比例（宽松但有用）

3. **主要校正方法：**
   - Bonferroni：最简单、最保守
   - Holm：逐步向下法，比Bonferroni有功效
   - BH-FDR：控制FDR，适合大规模检验
   - Tukey HSD：ANOVA事后检验专用

4. **方法选择：**
   - 检验次数少：Bonferroni或Holm
   - 检验次数多：FDR
   - ANOVA事后检验：Tukey HSD
   - 严格控制错误：FWER方法
   - 探索性研究：FDR方法

5. **Python实现：**
   - `statsmodels.stats.multitest.multipletests()` 函数提供多种方法
   - `pairwise_tukeyhsd()` 用于ANOVA事后检验

### 最佳实践

1. **预先规划：**
   - 研究设计阶段确定检验策略
   - 区分主要终点和次要终点

2. **合理选择：**
   - 根据研究类型（确证性 vs 探索性）选择方法
   - 根据检验次数选择合适的方法

3. **报告完整：**
   - 报告原始p值和校正后p值
   - 说明使用的校正方法
   - 报告效应量和置信区间

4. **避免过度校正：**
   - 不需要对所有检验都校正
   - 使用层级策略

5. **关注功效：**
   - 校正会降低功效
   - 合理的样本量设计

### 常见误区

1. **误区1：所有检验都需要校正**
   - 预先指定的主要终点不需要校正
   - 事后探索性分析才需要校正

2. **误区2：Bonferroni是唯一方法**
   - Holm更有功效
   - 大规模检验应使用FDR

3. **误区3：p值小于0.05就显著**
   - 多重比较后，阈值会变化
   - 需要考虑校正

4. **误区4：过度校正导致假阴性**
   - 合理的校正策略
   - 层级检验

理解多重比较校正的原理和应用，对于进行科学、严谨的统计分析至关重要。选择合适的校正方法，既能控制错误率，又能保持合理的统计功效。
