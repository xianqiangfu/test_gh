# 自助法（Bootstrap）

## 定义

自助法（Bootstrap）是一种基于重采样（resampling）的统计推断方法，由Bradley Efron于1979年提出。它通过从原始数据中有放回地重复抽样，估计统计量的抽样分布，从而进行参数估计、置信区间构造和假设检验。

## 核心思想

传统统计推断依赖于理论分布（如正态分布、t分布），而自助法通过数据驱动的重采样来获得统计量的经验分布。

**优势：**
- 不需要知道总体分布的具体形式
- 适用于复杂的统计量
- 适用于小样本情况
- 计算机技术进步使其变得实用

## 基本原理

### 重采样（Resampling）

重采样是从原始样本中创建新的样本集的过程。自助法使用**有放回抽样**（sampling with replacement）：

1. 原始样本：{x₁, x₂, ..., xₙ}
2. 有放回抽样n次，得到Bootstrap样本
3. 每个原始观测值在Bootstrap样本中可能出现0次、1次或多次

**重要性质：**
- 每个Bootstrap样本的大小与原始样本相同
- 原始样本中每个观测值被选中的概率约为 1 - (1-1/n)ⁿ ≈ 63.2%
- 平均约36.8%的观测值不会出现在特定Bootstrap样本中

## 自助法的基本步骤

### 单参数估计

1. **收集原始数据：** 获得大小为n的原始样本
2. **有放回抽样：** 从原始样本中有放回地抽取n个观测值，形成第一个Bootstrap样本
3. **计算统计量：** 计算Bootstrap样本的统计量（如均值、中位数）
4. **重复抽样：** 重复步骤2-3 B次（B通常为1000-10000）
5. **构建分布：** 使用B个Bootstrap统计量构建经验分布
6. **统计推断：** 基于Bootstrap分布进行推断

### 置信区间构造

基于Bootstrap分布，可以构造多种类型的置信区间：

#### 1. 百分位法（Percentile Method）

直接使用Bootstrap统计量的百分位数：

$$[\hat{\theta}^{*(\alpha/2)}, \hat{\theta}^{*(1-\alpha/2)}]$$

其中$\hat{\theta}^{*(p)}$表示Bootstrap统计量的第p百分位数。

**优点：** 简单直观
**缺点：** 当统计量有偏倚时不准确

#### 2. 百分位t法（Bootstrap-t）

1. 对每个Bootstrap样本计算t统计量：
   $$t^* = \frac{\hat{\theta}^* - \hat{\theta}}{SE^*}$$
2. 使用t*分布的百分位数构造区间

**优点：** 考虑了估计的标准误
**缺点：** 对每个Bootstrap样本都需要估计标准误

#### 3. 偏差校正加速法（BCa）

通过校正偏差和加速因子改善置信区间：

$$[\hat{\theta}^{*(\alpha_1)}, \hat{\theta}^{*(\alpha_2)}]$$

其中$\alpha_1$和$\alpha_2$通过偏差校正和加速因子调整。

**优点：** 精度最高，适用于大多数情况
**缺点：** 计算较复杂

## 应用场景

### 1. 标准误估计

当统计量的标准误没有解析解时：

```python
# 中位数的标准误
import numpy as np
from sklearn.utils import resample

data = np.array([...])
boot_medians = []
for _ in range(1000):
    boot_sample = resample(data, replace=True)
    boot_medians.append(np.median(boot_sample))
se = np.std(boot_medians)
```

### 2. 置信区间构造

适用于复杂统计量或非正态分布：

```python
# 偏度的置信区间
boot_skews = []
for _ in range(1000):
    boot_sample = resample(data, replace=True)
    boot_skews.append(stats.skew(boot_sample))
ci = np.percentile(boot_skews, [2.5, 97.5])
```

### 3. 假设检验

#### Permutation Test（置换检验）

检验两组样本是否来自同一分布：

1. 将两组数据合并
2. 随机打乱标签，分配到两组
3. 计算两组统计量差异
4. 重复多次，得到零分布
5. 比较观测值与零分布

```python
# 检验两组均值差异
group1 = np.array([...])
group2 = np.array([...])
observed_diff = np.mean(group1) - np.mean(group2)

combined = np.concatenate([group1, group2])
boot_diffs = []
for _ in range(1000):
    np.random.shuffle(combined)
    boot_diffs.append(np.mean(combined[:len(group1)]) - 
                      np.mean(combined[len(group1):]))

p_value = np.mean(np.abs(boot_diffs) >= np.abs(observed_diff))
```

### 4. 回归模型推断

检验回归系数的显著性：

```python
# OLS回归的Bootstrap
boot_coefs = []
for _ in range(1000):
    boot_sample = resample(X_y, replace=True)
    X_boot = boot_sample[:, :-1]
    y_boot = boot_sample[:, -1]
    model = sm.OLS(y_boot, sm.add_constant(X_boot)).fit()
    boot_coefs.append(model.params)
```

## 自助法的类型

### 1. 非参数自助法（Nonparametric Bootstrap）

- 不对总体分布做任何假设
- 直接从原始数据重采样
- 最常用的方法

### 2. 参数自助法（Parametric Bootstrap）

- 假设数据来自特定分布族
- 估计参数后从该分布生成新样本

```python
# 参数自助法示例
# 假设数据来自正态分布
mu_hat = np.mean(data)
sigma_hat = np.std(data)
boot_samples = [np.random.normal(mu_hat, sigma_hat, len(data)) 
                for _ in range(1000)]
```

### 3. 残差自助法（Residual Bootstrap）

用于回归分析，重采样残差：

1. 拟合模型，得到残差
2. 对残差重采样
3. 构造新响应变量
4. 重新拟合模型

## Bootstrap重复次数的选择

| 目的 | 最少重复次数 | 推荐次数 |
|------|--------------|----------|
| 标准误估计 | 50-100 | 200-500 |
| 置信区间 | 1000 | 1000-2000 |
| 精密推断 | 2000 | 5000+ |

**考虑因素：**
- 计算资源
- 所需精度
- 统计量的复杂性

## 自助法的优缺点

### 优点

1. **通用性强：** 适用于任何统计量
2. **无需分布假设：** 不依赖特定分布
3. **适用于小样本：** 传统方法假设的大样本条件
4. **直观易懂：** 容易解释和实现
5. **自动化：** 容易编程实现

### 缺点

1. **计算成本高：** 需要大量重复计算
2. **依赖原始样本：** 原始样本有偏时Bootstrap也有偏
3. **非独立数据：** 时间序列、空间数据需要特殊处理
4. **极端值敏感：** 异常值会显著影响结果
5. **重采样方法选择：** 不同重采样方法可能产生不同结果

## 自助法的适用条件

1. **独立性：** 观测值之间相互独立
   - 对于相关数据（如时间序列），需使用Block Bootstrap

2. **代表性：** 原始样本能代表总体

3. **样本量：** 虽然适用于小样本，但样本过小时结果不稳定

## Block Bootstrap（分块自助法）

对于时间序列或空间相关数据，使用Block Bootstrap保持数据结构：

### Moving Block Bootstrap（移动分块）

1. 将数据分成大小为l的块
2. 随机选择块（有放回）
3. 拼接形成Bootstrap样本

```python
# Moving Block Bootstrap示例
def block_bootstrap(data, block_size, n_bootstrap=1000):
    n = len(data)
    n_blocks = int(np.ceil(n / block_size))
    boot_samples = []
    
    for _ in range(n_bootstrap):
        blocks = []
        for _ in range(n_blocks):
            start_idx = np.random.randint(0, n - block_size + 1)
            blocks.append(data[start_idx:start_idx + block_size])
        boot_sample = np.concatenate(blocks)[:n]
        boot_samples.append(boot_sample)
    
    return boot_samples
```

### Stationary Bootstrap（平稳自助法）

每个块的长度服从几何分布，更适合非平稳数据。

## Python实现

### 基础Bootstrap实现

```python
import numpy as np
from scipy import stats

def bootstrap(data, stat_func=np.mean, n_bootstrap=1000, 
              alpha=0.05, method='percentile'):
    """
    Bootstrap统计推断
    
    参数:
        data: 输入数据
        stat_func: 统计量函数
        n_bootstrap: Bootstrap重复次数
        alpha: 显著性水平
        method: 置信区间方法 ('percentile', 'bca')
    """
    n = len(data)
    boot_stats = []
    
    # Bootstrap重采样
    for _ in range(n_bootstrap):
        boot_sample = np.random.choice(data, size=n, replace=True)
        boot_stats.append(stat_func(boot_sample))
    
    boot_stats = np.array(boot_stats)
    
    # 点估计
    estimate = stat_func(data)
    
    # 标准误
    se = np.std(boot_stats, ddof=1)
    
    # 置信区间
    if method == 'percentile':
        ci = np.percentile(boot_stats, [alpha/2*100, (1-alpha/2)*100])
    
    return {
        'estimate': estimate,
        'se': se,
        'ci': ci,
        'bootstrap_samples': boot_stats
    }

# 使用示例
data = np.random.normal(10, 2, 50)
result = bootstrap(data, stat_func=np.mean, n_bootstrap=10000)
print(f"均值估计: {result['estimate']:.2f}")
print(f"标准误: {result['se']:.2f}")
print(f"95% CI: ({result['ci'][0]:.2f}, {result['ci'][1]:.2f})")
```

### 回归Bootstrap

```python
import statsmodels.api as sm

def bootstrap_regression(X, y, n_bootstrap=1000):
    """
    回归系数的Bootstrap推断
    """
    X = sm.add_constant(X)
    n = len(y)
    boot_coefs = []
    
    for _ in range(n_bootstrap):
        # 重采样索引
        indices = np.random.choice(n, size=n, replace=True)
        X_boot = X[indices]
        y_boot = y[indices]
        
        # 拟合模型
        model = sm.OLS(y_boot, X_boot).fit()
        boot_coefs.append(model.params)
    
    boot_coefs = np.array(boot_coefs)
    
    # 计算置信区间
    ci_low = np.percentile(boot_coefs, 2.5, axis=0)
    ci_high = np.percentile(boot_coefs, 97.5, axis=0)
    
    return {
        'coefficients': boot_coefs,
        'ci_low': ci_low,
        'ci_high': ci_high
    }
```

## Bootstrap的失效情况

1. **有限总体：** 当样本占总体比例较大时（>10%），需要有限总体校正
2. **极端值：** 单个极端值主导统计量时
3. **异常分布：** 数据分布极不规则时
4. **小样本：** 样本量极小（n < 5）时结果不稳定

## 与其他方法比较

| 方法 | 适用条件 | 优点 | 缺点 |
|------|----------|------|------|
| 参数方法 | 已知分布 | 精确、高效 | 假设可能不满足 |
| 自助法 | 无分布假设 | 通用、灵活 | 计算成本高 |
| 置换检验 | 独立性假设 | 精确p值 | 适用于特定假设 |
| Jackknife | 独立性假设 | 计算量小 | 不适用于光滑统计量 |

## 总结

自助法是现代统计推断的重要工具，其核心优势在于不依赖分布假设和适用于复杂统计量。随着计算能力的发展，Bootstrap已成为统计学习和数据分析中的标准方法。正确理解其原理、适用条件和局限性，才能有效应用这一强大工具。

**关键要点：**
1. 自助法通过有放回重采样估计统计量分布
2. 适用于无解析解或分布未知的统计推断
3. BCa方法通常提供最准确的置信区间
4. 计算成本是主要限制
5. 对相关数据需要使用特殊方法（如Block Bootstrap）
