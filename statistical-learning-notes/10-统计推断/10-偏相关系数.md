# 偏相关系数

## 定义

偏相关系数衡量在控制一个或多个其他变量的影响后，两个变量之间的净相关关系。它反映了当排除其他变量作用时，两个变量之间的真实关联程度。

## 数学定义

### 二变量情况（控制一个变量）

对于三个变量X、Y、Z，X和Y的一阶偏相关系数（控制Z）定义为：

$$r_{XY|Z} = \frac{r_{XY} - r_{XZ}r_{YZ}}{\sqrt{(1-r_{XZ}^2)(1-r_{YZ}^2)}}$$

其中：
- $r_{XY}$ 是X和Y的简单相关系数
- $r_{XZ}$ 是X和Z的简单相关系数  
- $r_{YZ}$ 是Y和Z的简单相关系数

### 多变量情况（控制多个变量）

控制变量集合 $\mathbf{C} = \{Z_1, Z_2, ..., Z_k\}$ 时，X和Y的偏相关系数可以通过逆矩阵计算：

$$r_{XY|\mathbf{C}} = \frac{-P_{XY}}{\sqrt{P_{XX}P_{YY}}}$$

其中 $P_{ij}$ 是相关矩阵的逆矩阵（精度矩阵）中第i行第j列的元素。

## 偏相关系数的性质

### 1. 取值范围

$$-1 \leq r_{XY|\mathbf{C}} \leq 1$$

### 2. 对称性

$$r_{XY|\mathbf{C}} = r_{YX|\mathbf{C}}$$

### 3. 与简单相关的关系

- 当控制变量与X、Y都不相关时，偏相关 = 简单相关
- 当控制变量解释了X和Y的共变时，偏相关可能远小于简单相关
- 偏相关系数的绝对值通常小于或等于简单相关系数

### 4. 几何解释

偏相关系数等于X和Y在去除所有控制变量线性影响后的残差之间的简单相关系数。

## 一阶偏相关与高阶偏相关

### 一阶偏相关

控制一个变量的偏相关，记作 $r_{XY|Z}$。

**示例：** 研究收入（X）与幸福感（Y）的关系，控制年龄（Z）的影响。

### 高阶偏相关

控制两个或多个变量的偏相关：
- 二阶偏相关：$r_{XY|ZW}$（控制Z和W）
- 三阶偏相关：$r_{XY|ZWT}$（控制Z、W、T）

**递推公式：**

$$r_{XY|ZW} = \frac{r_{XY|Z} - r_{XW|Z}r_{YW|Z}}{\sqrt{(1-r_{XW|Z}^2)(1-r_{YW|Z}^2)}}$$

## 偏相关与简单相关的区别

### 简单相关系数

$$r_{XY} = \frac{\text{Cov}(X,Y)}{\sigma_X \sigma_Y}$$

- 考虑X和Y的总相关
- 包含通过其他变量的间接相关
- 可能产生虚假相关

### 偏相关系数

$$r_{XY|\mathbf{C}}$$

- 考虑X和Y的净相关
- 排除控制变量的影响
- 反映直接关系

### 典型案例

**示例1：虚假相关**

| 变量 | 含义 |
|------|------|
| X | 冰淇淋销量 |
| Y | 游泳溺水事故数 |
| Z | 气温 |

简单相关 $r_{XY}$ 可能很高（正相关），但这是由于气温Z的共同影响。控制Z后，偏相关 $r_{XY|Z}$ 应接近0。

**示例2：抑制变量**

| 变量 | 含义 |
|------|------|
| X | 数学能力 |
| Y | 物理成绩 |
| Z | 学习动机 |

可能 $r_{XY}$ 很小（甚至不显著），但 $r_{XY|Z}$ 显著为正。学习动机是抑制变量，控制后才能看到数学和物理的真实关系。

## 半偏相关

### 定义

半偏相关衡量一个变量去除控制变量影响后，与另一个变量的相关。

### 半偏相关系数（X相对于Y，控制Z）

$$r_{X(Y|Z)} = r_{XY} - r_{XZ}r_{YZ}$$

或标准化的形式：

$$r_{X(Y|Z)} = \frac{r_{XY} - r_{XZ}r_{YZ}}{\sqrt{1-r_{XZ}^2}}$$

### 与偏相关的区别

| 类型 | 控制变量影响 | 解释 |
|------|------------|------|
| 偏相关 $r_{XY|Z}$ | 同时从X和Y中去除Z的影响 | X和Y的净相关 |
| 半偏相关 $r_{X(Y|Z)}$ | 只从X中去除Z的影响 | X（净）与Y（总）的相关 |

### 应用场景

半偏相关常用于：
- 变量重要性排序
- 确定每个预测变量的独特贡献
- 回归分析中的半偏R²

## 显著性检验

### t检验

原假设：$H_0: \rho_{XY|\mathbf{C}} = 0$

检验统计量：

$$t = \frac{r_{XY|\mathbf{C}}\sqrt{n-k-2}}{\sqrt{1-r_{XY|\mathbf{C}}^2}}$$

其中：
- $n$ 是样本量
- $k$ 是控制变量个数
- 自由度 = $n - k - 2$

### Fisher's z变换

对于大样本，使用Fisher's z变换：

$$z = \frac{1}{2}\ln\left(\frac{1+r_{XY|\mathbf{C}}}{1-r_{XY|\mathbf{C}}}\right)$$

$$z \sim N\left(0, \frac{1}{n-k-3}\right)$$

## 偏相关矩阵

### 定义

包含所有变量对之间偏相关系数的矩阵。

### 用途

- 识别变量间的直接关系
- 构建因果网络
- 评估多重共线性

### 精度矩阵与偏相关

对于随机向量 $\mathbf{X} = (X_1, X_2, ..., X_p)^T$，其精度矩阵 $\mathbf{P} = \mathbf{\Sigma}^{-1}$ 的元素与偏相关系数的关系：

$$\rho_{ij|\mathbf{C}} = \frac{-P_{ij}}{\sqrt{P_{ii}P_{jj}}}$$

其中 $\mathbf{C}$ 是除了 $X_i$ 和 $X_j$ 之外的所有变量。

## 偏相关在实际中的应用

### 1. 社会科学

研究教育程度（X）与收入（Y）的关系，控制：
- 工作经验（Z₁）
- 行业类型（Z₂）
- 地理位置（Z₃）

### 2. 医学研究

研究吸烟（X）与肺癌风险（Y）的关系，控制：
- 年龄（Z₁）
- 饮食习惯（Z₂）
- 家族史（Z₃）

### 3. 心理学

研究压力（X）与睡眠质量（Y）的关系，控制：
- 运动量（Z₁）
- 咖啡因摄入（Z₂）
- 心理健康状况（Z₃）

### 4. 经济学

研究利率（X）与通货膨胀（Y）的关系，控制：
- GDP增长率（Z₁）
- 货币供应量（Z₂）
- 失业率（Z₃）

## 偏相关的局限性与注意事项

### 1. 控制变量的选择

- 控制过多变量可能引入偏差
- 控制过少变量可能无法去除混淆
- 需要基于理论选择控制变量

### 2. 因果推断的局限

偏相关不等于因果关系：
- 即使控制变量，仍可能有未观测的混淆
- 偏相关只反映统计关联

### 3. 样本量要求

对于k个控制变量，至少需要：
$$n \geq k + 3 + 50 = k + 53$$

更严格的要求：每个变量至少10-20个观测值。

### 4. 多重共线性

当控制变量高度相关时：
- 偏相关系数估计不稳定
- 标准误增大
- 需要进行共线性诊断

### 5. 非线性关系

偏相关系数只衡量线性关系：
- 非线性关系需要其他方法
- 可能遗漏重要的关联

## Python实现示例

```python
import numpy as np
import pandas as pd
from scipy import stats
import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.preprocessing import StandardScaler

# ============================================
# 1. 计算一阶偏相关系数
# ============================================

def partial_correlation(x, y, z):
    """
    计算控制z时，x和y的一阶偏相关系数
    
    参数:
        x, y, z: 数组，长度相同
    
    返回:
        r_xy_z: 偏相关系数
        p_value: 显著性检验的p值
    """
    # 计算简单相关系数
    r_xy = np.corrcoef(x, y)[0, 1]
    r_xz = np.corrcoef(x, z)[0, 1]
    r_yz = np.corrcoef(y, z)[0, 1]
    
    # 计算偏相关系数
    r_xy_z = (r_xy - r_xz * r_yz) / np.sqrt((1 - r_xz**2) * (1 - r_yz**2))
    
    # t检验
    n = len(x)
    t_stat = r_xy_z * np.sqrt(n - 3) / np.sqrt(1 - r_xy_z**2)
    p_value = 2 * (1 - stats.t.cdf(abs(t_stat), n - 3))
    
    return r_xy_z, p_value

# 示例数据
np.random.seed(42)
n = 100

# 生成相关数据：X和Y都受Z影响
Z = np.random.normal(0, 1, n)
X = 0.5 * Z + np.random.normal(0, 0.5, n)
Y = 0.5 * Z + np.random.normal(0, 0.5, n)

# 计算简单相关和偏相关
r_xy = np.corrcoef(X, Y)[0, 1]
r_xy_z, p_val = partial_correlation(X, Y, Z)

print(f"简单相关系数 r_XY = {r_xy:.4f}")
print(f"偏相关系数 r_XY|Z = {r_xy_z:.4f}")
print(f"显著性检验 p值 = {p_val:.4f}")

# ============================================
# 2. 计算高阶偏相关系数（控制多个变量）
# ============================================

def partial_correlation_multiple(x, y, controls):
    """
    计算控制多个变量时，x和y的偏相关系数
    
    参数:
        x, y: 目标变量
        controls: 控制变量矩阵 (n x k)
    
    返回:
        r_xy_c: 偏相关系数
        p_value: 显著性检验的p值
    """
    # 合并所有变量
    data = np.column_stack([x, y, controls])
    df = pd.DataFrame(data)
    
    # 计算相关矩阵
    corr_matrix = df.corr().values
    
    # 计算精度矩阵（相关矩阵的逆）
    precision_matrix = np.linalg.inv(corr_matrix)
    
    # 提取偏相关系数
    r_xy_c = -precision_matrix[0, 1] / np.sqrt(precision_matrix[0, 0] * precision_matrix[1, 1])
    
    # t检验
    n = len(x)
    k = controls.shape[1] if len(controls.shape) > 1 else 1
    t_stat = r_xy_c * np.sqrt(n - k - 2) / np.sqrt(1 - r_xy_c**2)
    p_value = 2 * (1 - stats.t.cdf(abs(t_stat), n - k - 2))
    
    return r_xy_c, p_value

# 示例：控制两个变量
W = np.random.normal(0, 1, n)
r_xy_zw, p_val2 = partial_correlation_multiple(X, Y, np.column_stack([Z, W]))

print(f"\n控制Z和W的偏相关系数 r_XY|ZW = {r_xy_zw:.4f}")
print(f"显著性检验 p值 = {p_val2:.4f}")

# ============================================
# 3. 计算偏相关矩阵
# ============================================

def partial_correlation_matrix(df):
    """
    计算偏相关矩阵
    
    参数:
        df: DataFrame，包含所有变量
    
    返回:
        partial_corr_matrix: 偏相关系数矩阵
        p_value_matrix: p值矩阵
    """
    # 标准化数据
    scaler = StandardScaler()
    df_scaled = pd.DataFrame(scaler.fit_transform(df), columns=df.columns)
    
    # 计算相关矩阵
    corr_matrix = df_scaled.corr().values
    
    # 计算精度矩阵
    precision_matrix = np.linalg.inv(corr_matrix)
    
    # 计算偏相关矩阵
    n = len(df.columns)
    partial_corr_matrix = np.zeros((n, n))
    p_value_matrix = np.ones((n, n))
    
    for i in range(n):
        for j in range(n):
            if i != j:
                # 偏相关系数
                partial_corr_matrix[i, j] = -precision_matrix[i, j] / np.sqrt(
                    precision_matrix[i, i] * precision_matrix[j, j]
                )
                
                # t检验
                k = n - 2  # 控制变量个数
                t_stat = partial_corr_matrix[i, j] * np.sqrt(len(df) - k - 2) / np.sqrt(
                    1 - partial_corr_matrix[i, j]**2
                )
                p_value_matrix[i, j] = 2 * (1 - stats.t.cdf(abs(t_stat), len(df) - k - 2))
    
    return pd.DataFrame(partial_corr_matrix, 
                       index=df.columns, 
                       columns=df.columns), \
           pd.DataFrame(p_value_matrix, 
                       index=df.columns, 
                       columns=df.columns)

# 创建示例数据集
np.random.seed(42)
data = {
    '数学': np.random.normal(75, 10, 100),
    '物理': np.random.normal(70, 12, 100),
    '化学': np.random.normal(72, 11, 100),
    '学习时间': np.random.normal(20, 5, 100),
    '智商': np.random.normal(100, 15, 100)
}

# 添加一些真实关系
data['物理'] = 0.5 * data['数学'] + 0.3 * data['智商'] + np.random.normal(0, 5, 100)
data['化学'] = 0.4 * data['物理'] + 0.2 * data['学习时间'] + np.random.normal(0, 5, 100)

df = pd.DataFrame(data)

# 计算偏相关矩阵
partial_corr, p_values = partial_correlation_matrix(df)

print("\n偏相关矩阵:")
print(partial_corr.round(3))

print("\n偏相关系数的p值矩阵:")
print(p_values.round(3))

# ============================================
# 4. 计算半偏相关系数
# ============================================

def semipartial_correlation(x, y, z):
    """
    计算从x中去除z影响后，与y的半偏相关系数
    
    参数:
        x, y, z: 数组，长度相同
    
    返回:
        r_x_y_z: 半偏相关系数
    """
    # 计算简单相关系数
    r_xy = np.corrcoef(x, y)[0, 1]
    r_xz = np.corrcoef(x, z)[0, 1]
    r_yz = np.corrcoef(y, z)[0, 1]
    
    # 计算半偏相关系数
    r_x_y_z = (r_xy - r_xz * r_yz) / np.sqrt(1 - r_xz**2)
    
    return r_x_y_z

# 示例
r_x_y_z = semipartial_correlation(X, Y, Z)
print(f"\n半偏相关系数 r_X(Y|Z) = {r_x_y_z:.4f}")

# ============================================
# 5. 可视化偏相关关系
# ============================================

def plot_correlation_comparison(df, var1, var2, control_vars):
    """
    可视化简单相关与偏相关的比较
    
    参数:
        df: DataFrame
        var1, var2: 要比较的两个变量
        control_vars: 控制变量列表
    """
    fig, axes = plt.subplots(1, 3, figsize=(15, 5))
    
    # 简单相关
    sns.scatterplot(data=df, x=var1, y=var2, ax=axes[0])
    axes[0].set_title(f'简单相关: r = {np.corrcoef(df[var1], df[var2])[0,1]:.3f}')
    axes[0].set_xlabel(var1)
    axes[0].set_ylabel(var2)
    
    # 去除控制变量影响后的变量
    from sklearn.linear_model import LinearRegression
    
    # 从var1中去除控制变量的影响
    X_controls = df[control_vars].values
    y_var1 = df[var1].values
    
    reg1 = LinearRegression()
    reg1.fit(X_controls, y_var1)
    var1_residual = y_var1 - reg1.predict(X_controls)
    
    # 从var2中去除控制变量的影响
    y_var2 = df[var2].values
    reg2 = LinearRegression()
    reg2.fit(X_controls, y_var2)
    var2_residual = y_var2 - reg2.predict(X_controls)
    
    # 偏相关
    r_partial = np.corrcoef(var1_residual, var2_residual)[0, 1]
    
    sns.scatterplot(x=var1_residual, y=var2_residual, ax=axes[1])
    axes[1].set_title(f'偏相关 (控制{len(control_vars)}个变量): r = {r_partial:.3f}')
    axes[1].set_xlabel(f'{var1} (残差)')
    axes[1].set_ylabel(f'{var2} (残差)')
    axes[1].axhline(y=0, color='r', linestyle='--', alpha=0.3)
    axes[1].axvline(x=0, color='r', linestyle='--', alpha=0.3)
    
    # 比较条形图
    correlations = [
        np.corrcoef(df[var1], df[var2])[0, 1],
        r_partial
    ]
    labels = ['简单相关', '偏相关']
    
    axes[2].bar(labels, correlations, color=['blue', 'orange'])
    axes[2].set_title('相关系数比较')
    axes[2].set_ylabel('相关系数')
    axes[2].axhline(y=0, color='black', linestyle='-', linewidth=0.5)
    axes[2].set_ylim([-1, 1])
    
    # 添加数值标签
    for i, v in enumerate(correlations):
        axes[2].text(i, v + 0.05 if v >= 0 else v - 0.15, 
                    f'{v:.3f}', ha='center', va='bottom' if v >= 0 else 'top')
    
    plt.tight_layout()
    plt.show()

# 使用示例
plot_correlation_comparison(df, '数学', '物理', ['智商', '学习时间'])

# 半偏R²（每个变量解释的变异比例）
def semipartial_r2(x, y, controls):
    """计算半偏R²"""
    from sklearn.linear_model import LinearRegression
    from sklearn.metrics import r2_score
    
    # 完整模型
    X_full = np.column_stack([x, controls])
    model_full = LinearRegression().fit(X_full, y)
    r2_full = r2_score(y, model_full.predict(X_full))
    
    # 简化模型（不含x）
    model_reduced = LinearRegression().fit(controls, y)
    r2_reduced = r2_score(y, model_reduced.predict(controls))
    
    return r2_full - r2_reduced

print("\n各变量的半偏R²（独特解释变异比例）:")
for var in ['数学', '学习时间', '智商']:
    controls = np.column_stack([df[v].values for v in ['数学', '学习时间', '智商'] if v != var])
    sr2 = semipartial_r2(df[var].values, y_reg.values, controls)
    print(f"{var}: {sr2:.4f} ({sr2*100:.2f}%)")
```

## 偏相关与因果推断

### 因果图与偏相关

在因果推断中，偏相关系数用于：
- 识别因果结构
- 检验条件独立性
- 构建因果网络

### 条件独立性检验

如果 $r_{XY|\mathbf{C}} = 0$（统计上不显著），则在给定控制变量C的条件下，X和Y条件独立。

### 示例：因果发现

```python
# PC算法（简化版）示例
def pc_algorithm(df, alpha=0.05):
    """
    简化的PC算法，用于因果发现
    
    参数:
        df: DataFrame
        alpha: 显著性水平
    
    返回:
        graph: 因果图（邻接矩阵）
    """
    n_vars = len(df.columns)
    graph = np.ones((n_vars, n_vars)) - np.eye(n_vars)
    
    # 计算偏相关矩阵
    partial_corr, p_values = partial_correlation_matrix(df)
    
    # 移除不显著的边
    for i in range(n_vars):
        for j in range(i+1, n_vars):
            if p_values.iloc[i, j] > alpha:
                graph[i, j] = 0
                graph[j, i] = 0
    
    return graph

# 应用PC算法
graph = pc_algorithm(df, alpha=0.05)
print("\n因果图（邻接矩阵）:")
print(pd.DataFrame(graph, index=df.columns, columns=df.columns))
```

## 总结

偏相关系数是统计分析中的重要工具，用于：

**关键要点：**

1. **定义与意义：**
   - 衡量控制其他变量后两变量的净相关
   - 区分直接关系和间接关系

2. **计算方法：**
   - 一阶偏相关：控制一个变量
   - 高阶偏相关：控制多个变量
   - 使用精度矩阵或递推公式

3. **与简单相关的区别：**
   - 简单相关包含间接相关
   - 偏相关反映直接关系
   - 可能揭示被掩盖的真实关系

4. **半偏相关：**
   - 只从一侧变量去除控制变量影响
   - 用于评估变量独特贡献
   - 与半偏R²相关

5. **应用场景：**
   - 社会科学研究
   - 医学研究
   - 心理学研究
   - 经济分析

6. **注意事项：**
   - 控制变量的选择很重要
   - 偏相关不等于因果关系
   - 需要足够的样本量
   - 注意多重共线性

7. **Python实现：**
   - 可以使用numpy、scipy手动计算
   - pingouin库提供便捷函数
   - 可视化有助于理解

偏相关系数帮助研究者透过现象看本质，识别变量间的真实关系，是多元统计分析不可或缺的工具。
