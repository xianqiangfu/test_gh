# t检验（t-test）

## 定义

t检验是一种统计假设检验方法，用于判断样本均值与总体均值或两个样本均值之间是否存在显著差异。当总体标准差未知且样本量较小（通常n < 30）时使用t检验。

## t统计量

t统计量的基本形式为：

$$t = \frac{\bar{x} - \mu_0}{s/\sqrt{n}}$$

其中：
- $\bar{x}$ 是样本均值
- $\mu_0$ 是假设的总体均值
- $s$ 是样本标准差
- $n$ 是样本大小
- 自由度 df = n - 1

## 三种t检验类型

### 1. 单样本t检验

**用途：** 检验样本均值是否等于某个已知值

**假设：**
- H₀: μ = μ₀
- H₁: μ ≠ μ₀（双侧）或 μ > μ₀ / μ < μ₀（单侧）

**统计量：**
$$t = \frac{\bar{x} - \mu_0}{s/\sqrt{n}}$$

**应用实例：**
某工厂声称其生产的电池平均寿命为100小时。随机抽取25个电池，测得平均寿命为98小时，标准差为5小时。检验工厂声称是否正确。

### 2. 独立双样本t检验

**用途：** 比较两个独立样本的均值

**假设：**
- H₀: μ₁ = μ₂
- H₁: μ₁ ≠ μ₂

#### 2.1 等方差假设（合并方差t检验）

当两个总体方差相等（σ₁² = σ₂²）时：

$$t = \frac{\bar{x}_1 - \bar{x}_2}{s_p\sqrt{\frac{1}{n_1} + \frac{1}{n_2}}}$$

其中合并方差：
$$s_p^2 = \frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1 + n_2 - 2}$$

自由度：df = n₁ + n₂ - 2

#### 2.2 异方差假设（Welch's t-test）

当两个总体方差不相等时：

$$t = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}}$$

自由度（Welch-Satterthwaite近似）：
$$df \approx \frac{\left(\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}\right)^2}{\frac{(s_1^2/n_1)^2}{n_1-1} + \frac{(s_2^2/n_2)^2}{n_2-1}}$$

**应用实例：**
比较两种教学方法的效果。A组30人，平均分85，标准差6；B组35人，平均分82，标准差7。检验两种方法是否存在显著差异。

### 3. 配对样本t检验

**用途：** 比较同一组对象在不同条件下的差异

**假设：**
- H₀: μ_d = 0
- H₁: μ_d ≠ 0

其中d表示配对差值。

**统计量：**
$$t = \frac{\bar{d}}{s_d/\sqrt{n}}$$

其中：
- $\bar{d}$ 是差值均值
- $s_d$ 是差值标准差
- n 是配对数量

自由度：df = n - 1

**应用实例：**
测量20人减肥前后的体重变化，检验减肥效果是否显著。

## 前提条件

1. **正态性：** 样本来自正态分布或近似正态分布
   - 小样本时（n < 30）正态性假设更重要
   - 可用Q-Q图、Shapiro-Wilk检验检验正态性

2. **独立性：**
   - 单样本和双样本检验要求观测值相互独立
   - 配对检验要求配对内相关，配对间独立

3. **齐方差（双样本检验）：**
   - 等方差t检验要求两总体方差相等
   - 可用Levene检验或Bartlett检验检验方差齐性

## t分布

t分布与标准正态分布类似，但尾部更厚，对小样本更保守。

### t分布性质

- 当样本量n → ∞时，t分布趋近于标准正态分布
- 自由度越小，t分布尾部越厚
- t分布的形状由自由度决定

### 临界值

常用t分布临界值（双侧，α = 0.05）：

| 自由度 | t临界值 |
|--------|---------|
| 5      | 2.571   |
| 10     | 2.228   |
| 15     | 2.131   |
| 20     | 2.086   |
| 30     | 2.042   |
| 40     | 2.021   |
| 60     | 2.000   |
| 120    | 1.980   |
| ∞      | 1.960   |

## 假设检验步骤

1. **建立假设：** 明确原假设H₀和备择假设H₁
2. **选择显著性水平：** 通常α = 0.05
3. **计算检验统计量：** 根据t检验类型计算t值
4. **确定临界值或p值：** 查t分布表或计算p值
5. **做出决策：**
   - 如果 |t| > t_{α/2} 或 p < α，拒绝H₀
   - 否则，不拒绝H₀
6. **得出结论：** 根据决策解释实际意义

## Python实现示例

```python
import numpy as np
from scipy import stats

# 单样本t检验
sample = np.array([25, 27, 24, 26, 25, 26, 28, 24, 25, 27])
mu0 = 25
t_stat, p_value = stats.ttest_1samp(sample, mu0)

# 独立双样本t检验（等方差）
group1 = np.array([85, 82, 88, 79, 86, 84, 87, 83, 85, 81])
group2 = np.array([82, 80, 85, 78, 81, 83, 80, 82, 79, 84])
t_stat, p_value = stats.ttest_ind(group1, group2)

# 独立双样本t检验（异方差）
t_stat, p_value = stats.ttest_ind(group1, group2, equal_var=False)

# 配对样本t检验
before = np.array([70, 72, 68, 75, 71, 69, 73, 70, 72, 71])
after = np.array([68, 70, 66, 72, 69, 67, 71, 68, 70, 69])
t_stat, p_value = stats.ttest_rel(after, before)
```

## 注意事项

1. **样本量与正态性：**
   - 大样本时（n > 30），t检验对正态性偏离有一定鲁棒性
   - 严重偏离正态时，考虑非参数检验（如Wilcoxon检验）

2. **异常值影响：**
   - 异常值会显著影响t检验结果
   - 检查数据是否存在异常值

3. **效应量：**
   - 显著性不等于实际重要性
   - 计算Cohen's d评估效应量：
     $$d = \frac{\bar{x}_1 - \bar{x}_2}{s_{pooled}}$$
   - d < 0.2: 小效应；0.2 < d < 0.8: 中等效应；d > 0.8: 大效应

4. **多重比较：**
   - 进行多次t检验时，考虑Bonferroni校正或其他多重比较校正

## 与Z检验的区别

| 特征 | Z检验 | t检验 |
|------|-------|-------|
| 总体标准差 | 已知 | 未知，用样本估计 |
| 样本量 | 大样本 | 小样本更适用 |
| 分布 | 标准正态分布 | t分布 |
| 自由度 | 无 | n - 1 |

## 总结

t检验是统计学中最基础且最重要的假设检验方法之一，适用于小样本、总体标准差未知的情况。理解不同类型的t检验及其适用条件对于正确进行统计分析至关重要。
