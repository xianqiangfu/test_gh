# 贝叶斯估计（Bayesian Estimation）

## 定义

贝叶斯估计是一种基于贝叶斯定理的统计推断方法，利用先验信息和样本数据，通过后验分布对未知参数进行估计。与频率学派方法不同，贝叶斯估计将参数视为随机变量，具有概率分布。

## 贝叶斯定理

贝叶斯定理是贝叶斯估计的理论基础：

$$P(\theta|X) = \frac{P(X|\theta) \cdot P(\theta)}{P(X)} = \frac{P(X|\theta) \cdot P(\theta)}{\int P(X|\theta) \cdot P(\theta) \, d\theta}$$

其中：
- $P(\theta|X)$：后验分布（Posterior）
- $P(X|\theta)$：似然函数（Likelihood）
- $P(\theta)$：先验分布（Prior）
- $P(X)$：边际似然（Marginal Likelihood），也称为证据（Evidence）

## 贝叶斯估计的要素

### 1. 先验分布 $P(\theta)$

先验分布代表了在观察数据之前对参数的认知。

**常见先验类型：**

- **无信息先验（Non-informative Prior）**：当对参数没有先验信息时使用
  - 均匀先验：$P(\theta) \propto 1$
  - Jeffreys先验：$P(\theta) \propto \sqrt{I(\theta)}$，其中$I(\theta)$是Fisher信息

- **共轭先验（Conjugate Prior）**：选择先验使得后验分布与先验分布属于同一族
  - Beta-Binomial模型：Beta先验，二项似然
  - Normal-Normal模型：Normal先验，正态似然
  - Gamma-Poisson模型：Gamma先验，泊松似然

- **信息先验（Informative Prior）**：基于历史信息或专家知识确定

### 2. 似然函数 $P(X|\theta)$

似然函数描述了在给定参数下数据的概率分布。

### 3. 后验分布 $P(\theta|X)$

后验分布结合了先验信息和数据信息，是参数估计的基础。

## 点估计

贝叶斯点估计从后验分布中提取单一值作为参数估计，常用的有：

### 1. 后验均值

$$\hat{\theta}_{mean} = E[\theta|X] = \int \theta \cdot P(\theta|X) \, d\theta$$

**特点：**
- 使均方误差（MSE）最小的估计
- 对称后验分布时最常用
- 受异常值影响较大

### 2. 后验中位数

$$\hat{\theta}_{median} = \text{median}(P(\theta|X))$$

**特点：**
- 使平均绝对误差最小的估计
- 对异常值不敏感
- 适用于非对称后验分布

### 3. 后验众数（MAP估计）

$$\hat{\theta}_{MAP} = \arg\max_{\theta} P(\theta|X)$$

等价于：
$$\hat{\theta}_{MAP} = \arg\max_{\theta} \left[ \log P(X|\theta) + \log P(\theta) \right]$$

**特点：**
- 后验分布的峰值点
- 计算相对简单（只需优化）
- 正则化最大似然估计的推广

## 区间估计

贝叶斯区间估计称为**可信区间（Credible Interval）**，与频率学派的置信区间有本质区别。

### 可信区间定义

一个$(1-\alpha)$可信区间$[a, b]$满足：

$$P(a \leq \theta \leq b | X) = 1 - \alpha$$

### 可信区间类型

#### 1. 等尾可信区间

$$P(\theta < a | X) = \alpha/2$$
$$P(\theta > b | X) = \alpha/2$$

#### 2. 最高后验密度区间（HPD Interval）

后验密度最高的区域，满足：
- $P(\theta \in \text{HPD} | X) = 1 - \alpha$
- HPD内任意点的后验密度高于HPD外任意点

**特点：**
- 对于单峰对称后验，HPD区间与等尾区间相同
- 对于非对称后验，HPD区间更短（更精确）

### 与置信区间的区别

| 特征 | 贝叶斯可信区间 | 频率学派置信区间 |
|------|----------------|------------------|
| 解释 | 参数有概率落在区间内 | 区间以特定概率覆盖参数 |
| 计算 | 基于后验分布 | 基于抽样分布 |
| 信息 | 融入先验信息 | 仅使用样本信息 |
| 含义 | 更直观 | 易被误解 |

## 贝叶斯预测

贝叶斯预测不仅考虑参数的不确定性，还考虑未来的观测不确定性。

### 预测分布

对于新观测$X_{new}$，预测分布为：

$$P(X_{new}|X) = \int P(X_{new}|\theta) \cdot P(\theta|X) \, d\theta$$

### 点预测

- 后验均值预测：$\hat{X}_{new} = E[X_{new}|X]$
- 后验中位数预测

### 预测区间

类似可信区间，可以构造预测的区间估计。

## 与频率学派估计的比较

### 哲学基础

| 方面 | 贝叶斯学派 | 频率学派 |
|------|-----------|----------|
| 参数性质 | 随机变量 | 固定未知常数 |
| 信息来源 | 先验 + 数据 | 仅数据 |
| 推断基础 | 后验分布 | 抽样分布 |
| 解释方式 | 概率是信念度 | 概率是长期频率 |

### 估计方法比较

**贝叶斯估计优势：**
- 自然融入先验信息
- 提供完整的后验分布
- 小样本情况下表现更好
- 推断结果更直观

**频率学派估计优势：**
- 不依赖主观先验
- 计算相对简单
- 大样本性质良好
- 传统应用广泛

### 选择考虑

- **有先验信息**：优先贝叶斯方法
- **无先验信息**：两者相近
- **小样本**：贝叶斯方法更有优势
- **大样本**：两者结果趋于一致
- **计算资源**：频率学派计算更简单

## 常见贝叶斯估计模型

### 1. Beta-Binomial模型

**场景：** 估计二项分布的成功概率p

**似然函数：** $P(X|p) = \binom{n}{x} p^x (1-p)^{n-x}$

**先验分布：** $p \sim Beta(\alpha, \beta)$

**后验分布：** $p|X \sim Beta(\alpha + x, \beta + n - x)$

**后验均值：** $\frac{\alpha + x}{\alpha + \beta + n}$

### 2. Normal-Normal模型（已知方差）

**场景：** 估计正态分布均值μ（已知方差σ²）

**似然函数：** $P(X|\mu) \propto \exp\left[-\frac{1}{2\sigma^2}\sum (x_i - \mu)^2\right]$

**先验分布：** $\mu \sim N(\mu_0, \tau^2)$

**后验分布：** $\mu|X \sim N(\mu_n, \tau_n^2)$

其中：
$$\mu_n = \frac{\frac{n\bar{x}}{\sigma^2} + \frac{\mu_0}{\tau^2}}{\frac{n}{\sigma^2} + \frac{1}{\tau^2}}$$
$$\tau_n^2 = \left(\frac{n}{\sigma^2} + \frac{1}{\tau^2}\right)^{-1}$$

### 3. Gamma-Poisson模型

**场景：** 估计泊松分布的率参数λ

**似然函数：** $P(X|\lambda) \propto \lambda^{\sum x_i} e^{-n\lambda}$

**先验分布：** $\lambda \sim Gamma(\alpha, \beta)$

**后验分布：** $\lambda|X \sim Gamma(\alpha + \sum x_i, \beta + n)$

## 贝叶斯计算方法

### 1. 解析方法

对于共轭先验，后验分布有解析形式。

### 2. 数值积分

对于简单问题，可以使用数值积分方法。

### 3. MCMC方法（马尔可夫链蒙特卡洛）

对于复杂问题，使用MCMC方法采样后验分布：

- **Metropolis-Hastings算法**
- **Gibbs采样**
- **Hamiltonian Monte Carlo（HMC）**

### 4. 变分推断

近似推断方法，用简单分布近似复杂后验分布。

## Python实现示例

### 1. Beta-Binomial模型

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# Beta-Binomial模型
x = 30  # 成功次数
n = 100  # 总试验次数

# 先验参数（弱先验）
alpha_prior = 1
beta_prior = 1

# 后验分布
alpha_post = alpha_prior + x
beta_post = beta_prior + n - x

# 后验均值
mean_post = alpha_post / (alpha_post + beta_post)
print(f"后验均值: {mean_post:.4f}")

# 后验中位数
median_post = stats.beta.median(alpha_post, beta_post)
print(f"后验中位数: {median_post:.4f}")

# 后验众数（MAP）
mode_post = (alpha_post - 1) / (alpha_post + beta_post - 2)
print(f"后验众数: {mode_post:.4f}")

# 95%可信区间
ci_lower = stats.beta.ppf(0.025, alpha_post, beta_post)
ci_upper = stats.beta.ppf(0.975, alpha_post, beta_post)
print(f"95%可信区间: [{ci_lower:.4f}, {ci_upper:.4f}]")

# 绘制后验分布
theta = np.linspace(0.1, 0.6, 500)
posterior = stats.beta.pdf(theta, alpha_post, beta_post)

plt.figure(figsize=(10, 6))
plt.plot(theta, posterior, 'b-', linewidth=2, label='后验分布')
plt.axvline(mean_post, color='r', linestyle='--', label='后验均值')
plt.axvline(ci_lower, color='g', linestyle=':', label='95%可信区间')
plt.axvline(ci_upper, color='g', linestyle=':')
plt.xlabel('参数 p')
plt.ylabel('概率密度')
plt.title('Beta-Binomial模型的后验分布')
plt.legend()
plt.grid(True)
plt.show()

# 预测分布
# 对于新的一次试验，预测P(X_new=1|X) = E[p|X] = 后验均值
p_success = mean_post
print(f"新一次试验成功的预测概率: {p_success:.4f}")
```

### 2. Normal-Normal模型

```python
import numpy as np
from scipy import stats

# Normal-Normal模型
data = np.array([25, 27, 24, 26, 25, 26, 28, 24, 25, 27])
n = len(data)
x_bar = np.mean(data)
sigma_squared = 4  # 已知方差
sigma = np.sqrt(sigma_squared)

# 先验参数
mu_0 = 25  # 先验均值
tau_squared = 9  # 先验方差

# 后验参数
tau_n_squared = 1 / (n/sigma_squared + 1/tau_squared)
mu_n = tau_n_squared * (n*x_bar/sigma_squared + mu_0/tau_squared)

print(f"后验均值: {mu_n:.4f}")
print(f"后验方差: {tau_n_squared:.4f}")

# 95%可信区间
ci_lower = stats.norm.ppf(0.025, mu_n, np.sqrt(tau_n_squared))
ci_upper = stats.norm.ppf(0.975, mu_n, np.sqrt(tau_n_squared))
print(f"95%可信区间: [{ci_lower:.4f}, {ci_upper:.4f}]")

# 预测新观测
x_new_pred_mean = mu_n
x_new_pred_var = tau_n_squared + sigma_squared
print(f"新观测预测均值: {x_new_pred_mean:.4f}")
print(f"新观测预测方差: {x_new_pred_var:.4f}")
```

### 3. 使用PyMC3进行贝叶斯推断

```python
import numpy as np
import pymc3 as pm
import arviz as az

# 准备数据
data = np.array([25, 27, 24, 26, 25, 26, 28, 24, 25, 27])

with pm.Model() as model:
    # 先验分布
    mu = pm.Normal('mu', mu=25, sigma=3)
    sigma = pm.HalfNormal('sigma', sigma=2)

    # 似然函数
    likelihood = pm.Normal('likelihood', mu=mu, sigma=sigma, observed=data)

    # 采样后验分布
    trace = pm.sample(2000, tune=1000, chains=2)

# 查看结果
print(az.summary(trace, hdi_prob=0.95))

# 绘制后验分布
az.plot_posterior(trace, hdi_prob=0.95)
```

## 注意事项

### 1. 先验选择

- **先验敏感性**：小样本时后验对先验更敏感
- **先验评估**：考虑先验的合理性，进行敏感性分析
- **无信息先验**：当没有明确先验信息时使用

### 2. 计算挑战

- **高维问题**：参数维度高时计算困难
- **收敛诊断**：MCMC采样需要检查收敛性
- **计算成本**：贝叶斯方法通常计算量更大

### 3. 模型选择

- **模型比较**：使用边缘似然、WAIC、LOO-CV等指标
- **模型平均**：考虑模型不确定性
- **模型验证**：使用后验预测检验

## 应用场景

贝叶斯估计广泛应用于：

1. **医学研究**：临床试验、疾病诊断
2. **机器学习**：贝叶斯神经网络、高斯过程
3. **金融建模**：风险评估、投资组合优化
4. **质量控制**：过程监控、缺陷检测
5. **环境科学**：气候建模、污染评估
6. **社会科学**：调查分析、政策评估

## 总结

贝叶斯估计提供了一种自然融入先验信息的统计推断框架：

**核心优势：**
- 系统结合先验信息和样本数据
- 提供完整的后验分布
- 推断结果直观易懂
- 小样本情况下表现优异

**关键概念：**
- 贝叶斯定理连接先验、似然和后验
- 点估计：后验均值、中位数、众数
- 区间估计：可信区间，参数的概率陈述
- 贝叶斯预测：考虑参数和观测的双重不确定性

**实践要点：**
- 合理选择先验分布
- 理解与频率学派方法的区别
- 掌握计算方法和工具
- 进行模型诊断和验证

理解贝叶斯估计的基本原理和应用方法，将为复杂统计问题提供强大的分析工具。
