# 平稳性检验（Stationarity Test）

## 定义

平稳性是时间序列分析中的一个核心概念。一个平稳时间序列的统计性质（如均值、方差、自协方差）不随时间变化。平稳性检验用于判断时间序列是否满足平稳性假设，这对选择合适的模型（如ARMA、ARIMA）至关重要。

## 平稳性的类型

### 1. 严平稳（Strict Stationarity）

**定义：** 时间序列{Xₜ}是严平稳的，如果对于任意的k、n和时刻点t₁, t₂, ..., tₖ，其联合分布满足：

$$F(X_{t_1+h}, X_{t_2+h}, ..., X_{t_k+h}) = F(X_{t_1}, X_{t_2}, ..., X_{t_k})$$

其中h是任意的时间平移量。

**特点：**
- 要求所有统计性质都不随时间变化
- 理论上强，但在实际中难以验证
- 要求太严格，实际应用中很少使用

### 2. 弱平稳（Weak Stationarity / Covariance Stationarity）

**定义：** 时间序列{Xₜ}是弱平稳的，如果满足以下三个条件：

1. **均值为常数：**
   $$E[Xₜ] = μ \quad \text{对所有} \ t$$

2. **方差有限且为常数：**
   $$Var(Xₜ) = σ² < \infty \quad \text{对所有} \ t$$

3. **自协方差只依赖于时间滞后：**
   $$Cov(Xₜ, X_{t+k}) = γ(k) \quad \text{只依赖于} \ k$$

**特点：**
- 只要求一阶和二阶矩的性质不变
- 是时间序列分析中最常用的平稳性定义
- 如果序列是正态分布的，则严平稳和弱平稳等价

### 3. 趋势平稳（Trend Stationary）

**定义：** 序列的确定性趋势去除后是平稳的。

**模型形式：**
$$Xₜ = f(t) + εₜ$$

其中f(t)是确定性趋势函数，εₜ是平稳噪声。

**例子：**
- 线性趋势：Xₜ = a + bt + εₜ
- 二次趋势：Xₜ = a + bt + ct² + εₜ

### 4. 差分平稳（Difference Stationary）

**定义：** 序列经过适当阶数的差分后是平稳的。

**单位根过程：**
$$Xₜ = X_{t-1} + εₜ$$

其中εₜ是平稳噪声。

**差分后：**
$$ΔXₜ = Xₜ - X_{t-1} = εₜ$$

## 平稳性的图形检验

### 1. 时序图（Time Series Plot）

**非平稳的特征：**
- 明显的趋势（上升或下降）
- 方差随时间变化（异方差）
- 周期性模式
- 结构性变化

**平稳的特征：**
- 围绕常数均值波动
- 波动幅度相对稳定
- 没有明显趋势

**Python示例：**
```python
import numpy as np
import matplotlib.pyplot as plt

# 生成平稳序列
np.random.seed(42)
n = 200
stationary = np.random.normal(0, 1, n)

# 生成非平稳序列（带趋势）
trend = np.linspace(0, 10, n)
non_stationary = trend + np.random.normal(0, 1, n)

# 绘图
fig, axes = plt.subplots(1, 2, figsize=(12, 4))

axes[0].plot(stationary)
axes[0].set_title('平稳序列')
axes[0].set_xlabel('时间')
axes[0].set_ylabel('值')

axes[1].plot(non_stationary)
axes[1].set_title('非平稳序列（带趋势）')
axes[1].set_xlabel('时间')
axes[1].set_ylabel('值')

plt.tight_layout()
plt.show()
```

### 2. 自相关函数图（ACF Plot）

**平稳序列的ACF特征：**
- ACF值快速衰减到0
- 通常在几阶滞后后就变得不显著
- 呈现指数衰减或阻尼正弦波形式

**非平稳序列的ACF特征：**
- ACF值缓慢衰减
- 在很多阶滞后后仍然显著
- 呈现线性缓慢下降

**Python示例：**
```python
from statsmodels.graphics.tsaplots import plot_acf

fig, axes = plt.subplots(1, 2, figsize=(12, 4))

plot_acf(stationary, lags=20, ax=axes[0])
axes[0].set_title('平稳序列的ACF')

plot_acf(non_stationary, lags=20, ax=axes[1])
axes[1].set_title('非平稳序列的ACF')

plt.tight_layout()
plt.show()
```

## 单位根检验

单位根检验是检验时间序列平稳性的统计方法，其原假设通常是"序列存在单位根"（即非平稳）。

### 1. ADF检验（Augmented Dickey-Fuller Test）

**原理：** 检验回归模型中单位根的存在性

**检验模型（ADF回归）：**
$$ΔXₜ = α + βt + γX_{t-1} + δ₁ΔX_{t-1} + ... + δ_{p-1}ΔX_{t-p+1} + εₜ$$

其中：
- Δ 是差分算子：ΔXₜ = Xₜ - Xₜ₋₁
- p 是滞后阶数
- εₜ 是白噪声

**假设检验：**
- H₀: γ = 0（存在单位根，序列非平稳）
- H₁: γ < 0（不存在单位根，序列平稳）

**检验统计量：**
$$ADF = \frac{\hat{γ}}{SE(\hat{γ})}$$

**决策规则：**
- 如果 ADF统计量 < 临界值，拒绝H₀，序列平稳
- 如果 p值 < 显著性水平（如0.05），拒绝H₀

**Python实现：**
```python
from statsmodels.tsa.stattools import adfuller

def adf_test(series, title=''):
    """执行ADF检验并输出结果"""
    print(f'ADF检验结果 - {title}')
    print('-' * 40)

    result = adfuller(series, autolag='AIC')

    print(f'ADF统计量: {result[0]:.4f}')
    print(f'p值: {result[1]:.4f}')
    print(f'临界值:')
    for key, value in result[4].items():
        print(f'  {key}: {value:.4f}')

    print(f'滞后阶数: {result[2]}')
    print(f'观测值数量: {result[3]}')
    print(f'临界信息准则: {result[5]}')

    # 解释结果
    if result[1] < 0.05:
        print('结论: 拒绝原假设，序列是平稳的 ✓')
    else:
        print('结论: 不能拒绝原假设，序列非平稳 ✗')
    print()

# 示例使用
adf_test(stationary, '平稳序列')
adf_test(non_stationary, '非平稳序列')
```

**ADF检验的三种变体：**

1. **无常数、无趋势：** ΔXₜ = γXₜ₋₁ + ΣδᵢΔXₜ₋ᵢ + εₜ
2. **有常数、无趋势：** ΔXₜ = α + γXₜ₋₁ + ΣδᵢΔXₜ₋ᵢ + εₜ
3. **有常数、有趋势：** ΔXₜ = α + βt + γXₜ₋₁ + ΣδᵢΔXₜ₋ᵢ + εₜ

### 2. PP检验（Phillips-Perron Test）

**特点：**
- 与ADF检验类似，但不需要指定滞后阶数
- 对误差项的序列相关和异方差更稳健
- 使用非参数方法修正检验统计量

**假设检验：**
- H₀: 存在单位根（非平稳）
- H₁: 不存在单位根（平稳）

**Python实现：**
```python
from statsmodels.tsa.stattools import pp

def pp_test(series, title=''):
    """执行PP检验并输出结果"""
    print(f'PP检验结果 - {title}')
    print('-' * 40)

    result = pp(series)

    print(f'PP统计量: {result[0]:.4f}')
    print(f'p值: {result[1]:.4f}')
    print(f'临界值:')
    for key, value in result[2].items():
        print(f'  {key}: {value:.4f}')

    # 解释结果
    if result[1] < 0.05:
        print('结论: 拒绝原假设，序列是平稳的 ✓')
    else:
        print('结论: 不能拒绝原假设，序列非平稳 ✗')
    print()

# 示例使用
pp_test(stationary, '平稳序列')
pp_test(non_stationary, '非平稳序列')
```

### 3. KPSS检验（Kwiatkowski-Phillips-Schmidt-Shin Test）

**特点：**
- 与ADF/PP检验的原假设相反
- 原假设是序列是平稳的
- 备择假设是序列存在单位根（非平稳）

**假设检验：**
- H₀: 序列是平稳的（或趋势平稳的）
- H₁: 序列存在单位根（非平稳）

**检验类型：**
1. **常数检验：** 检验序列是否围绕常数平稳
2. **趋势检验：** 检验序列是否围绕趋势平稳

**Python实现：**
```python
from statsmodels.tsa.stattools import kpss

def kpss_test(series, regression='c', title=''):
    """执行KPSS检验并输出结果"""
    print(f'KPSS检验结果 - {title}')
    print('-' * 40)

    # regression: 'c' for constant, 'ct' for constant and trend
    result = kpss(series, regression=regression)

    print(f'KPSS统计量: {result[0]:.4f}')
    print(f'p值: {result[1]:.4f}')
    print(f'临界值:')
    for key, value in result[3].items():
        print(f'  {key}: {value:.4f}')

    # 解释结果
    if result[1] > 0.05:
        print('结论: 不能拒绝原假设，序列是平稳的 ✓')
    else:
        print('结论: 拒绝原假设，序列非平稳 ✗')
    print()

# 示例使用
kpss_test(stationary, '平稳序列')
kpss_test(non_stationary, '非平稳序列')
```

**对比ADF和KPSS：**
- ADF: H₀ = 非平稳，H₁ = 平稳
- KPSS: H₀ = 平稳，H₁ = 非平稳
- 同时使用两种检验可以相互验证

### 4. DF-GLS检验（Elliott-Rothenberg-Stock Test）

**特点：**
- 比ADF检验功效更高（power）
- 对确定性趋势的处理更优
- 适合小样本

**Python实现：**
```python
from statsmodels.tsa.stattools import dfuller
from arch.unitroot import DFGLS

def dfgls_test(series, title=''):
    """执行DF-GLS检验"""
    print(f'DF-GLS检验结果 - {title}')
    print('-' * 40)

    result = DFGLS(series)

    print(f'DF-GLS统计量: {result.stat:.4f}')
    print(f'p值: {result.pvalue:.4f}')
    print(f'临界值:')
    for key, value in result.critical_values.items():
        print(f'  {key}: {value:.4f}')

    if result.pvalue < 0.05:
        print('结论: 拒绝原假设，序列是平稳的 ✓')
    else:
        print('结论: 不能拒绝原假设，序列非平稳 ✗')
    print()
```

## 差分变换

差分是使非平稳序列平稳化的常用方法。

### 1. 一阶差分

**定义：**
$$ΔXₜ = Xₜ - X_{t-1}$$

**用途：** 消除线性趋势

**Python示例：**
```python
# 一阶差分
diff_1 = np.diff(non_stationary, n=1)

# 绘图对比
fig, axes = plt.subplots(1, 2, figsize=(12, 4))

axes[0].plot(non_stationary)
axes[0].set_title('原始序列（非平稳）')

axes[1].plot(diff_1)
axes[1].set_title('一阶差分后')

plt.tight_layout()
plt.show()
```

### 2. 二阶差分

**定义：**
$$Δ²Xₜ = Δ(ΔXₜ) = Xₜ - 2X_{t-1} + X_{t-2}$$

**用途：** 消除二次趋势

**Python示例：**
```python
# 二阶差分
diff_2 = np.diff(non_stationary, n=2)

# 绘图
plt.figure(figsize=(10, 4))
plt.plot(diff_2)
plt.title('二阶差分后')
plt.xlabel('时间')
plt.ylabel('值')
plt.show()
```

### 3. 季节性差分

**定义：**
$$ΔₛXₜ = Xₜ - X_{t-s}$$

其中s是季节周期（如12代表月度数据的年季节性）。

**用途：** 消除季节性

**Python示例：**
```python
# 生成带季节性的序列
seasonal = np.sin(np.linspace(0, 20*np.pi, n)) + np.random.normal(0, 0.5, n)

# 季节性差分（假设周期为12）
seasonal_diff = seasonal[12:] - seasonal[:-12]

# 绘图
fig, axes = plt.subplots(1, 2, figsize=(12, 4))

axes[0].plot(seasonal)
axes[0].set_title('原始序列（带季节性）')

axes[1].plot(seasonal_diff)
axes[1].set_title('季节性差分后')

plt.tight_layout()
plt.show()
```

### 4. 组合差分

**定义：**
$$ΔₛΔXₜ = Δₛ(Xₜ - X_{t-1}) = Xₜ - X_{t-1} - X_{t-s} + X_{t-s-1}$$

**用途：** 同时消除趋势和季节性

**Python示例：**
```python
# 先一阶差分，再季节性差分
diff_1 = np.diff(seasonal, n=1)
diff_1_seasonal = diff_1[12:] - diff_1[:-12]

plt.figure(figsize=(10, 4))
plt.plot(diff_1_seasonal)
plt.title('组合差分后（一阶+季节性）')
plt.xlabel('时间')
plt.ylabel('值')
plt.show()
```

## 平稳化方法

### 1. 对数变换

**用途：**
- 稳定方差（消除异方差）
- 处理指数增长趋势

**公式：**
$$Yₜ = ln(Xₜ)$$

**适用条件：**
- 数据为正数
- 方差与均值相关

**Python示例：**
```python
# 生成方差随时间增大的序列
np.random.seed(42)
heteroscedastic = np.cumsum(np.random.normal(0, 0.1, n)) * np.linspace(0.5, 2, n)

# 对数变换（加常数确保为正）
log_transformed = np.log(heteroscedastic - np.min(heteroscedastic) + 1)

# 绘图对比
fig, axes = plt.subplots(1, 2, figsize=(12, 4))

axes[0].plot(heteroscedastic)
axes[0].set_title('原始序列（异方差）')

axes[1].plot(log_transformed)
axes[1].set_title('对数变换后')

plt.tight_layout()
plt.show()
```

### 2. Box-Cox变换

**定义：**
$$Yₜ = \begin{cases}
\frac{Xₜ^λ - 1}{λ}, & λ \neq 0 \\
\ln(Xₜ), & λ = 0
\end{cases}$$

**特点：**
- 对数变换的推广
- 包含参数λ，通过数据估计最优λ值
- λ = 1 时为恒等变换，λ = 0 时为对数变换

**Python示例：**
```python
from scipy import stats

# Box-Cox变换
transformed, lambda_ = stats.boxcox(heteroscedastic - np.min(heteroscedastic) + 1)

print(f'最优λ值: {lambda_:.4f}')

# 绘图
plt.figure(figsize=(10, 4))
plt.plot(transformed)
plt.title(f'Box-Cox变换 (λ = {lambda_:.4f})')
plt.xlabel('时间')
plt.ylabel('值')
plt.show()
```

**方法1：减去趋势线**

## 完整的平稳性检验流程

### 标准流程

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from statsmodels.tsa.stattools import adfuller, kpss
from statsmodels.graphics.tsaplots import plot_acf, plot_pacf

def test_stationarity(series, title=''):
    """完整的平稳性检验流程"""

    print(f'平稳性检验 - {title}')
    print('=' * 50)

    # 1. 绘制时序图
    plt.figure(figsize=(12, 4))
    plt.plot(series)
    plt.title(f'时序图 - {title}')
    plt.xlabel('时间')
    plt.ylabel('值')
    plt.grid(True)
    plt.show()

    # 2. 计算滚动统计量
    rolling_mean = pd.Series(series).rolling(window=12).mean()
    rolling_std = pd.Series(series).rolling(window=12).std()

    plt.figure(figsize=(12, 4))
    plt.plot(series, label='原始序列')
    plt.plot(rolling_mean, 'r-', label='滚动均值')
    plt.plot(rolling_std, 'g-', label='滚动标准差')
    plt.title(f'滚动统计量 - {title}')
    plt.legend()
    plt.grid(True)
    plt.show()

    # 3. ADF检验
    print('ADF检验:')
    result_adf = adfuller(series.dropna(), autolag='AIC')
    print(f'  ADF统计量: {result_adf[0]:.4f}')
    print(f'  p值: {result_adf[1]:.4f}')
    adf_conclusion = '平稳' if result_adf[1] < 0.05 else '非平稳'
    print(f'  结论: {adf_conclusion}')

    # 4. KPSS检验
    print('\nKPSS检验:')
    result_kpss = kpss(series.dropna(), regression='c')
    print(f'  KPSS统计量: {result_kpss[0]:.4f}')
    print(f'  p值: {result_kpss[1]:.4f}')
    kpss_conclusion = '平稳' if result_kpss[1] > 0.05 else '非平稳'
    print(f'  结论: {kpss_conclusion}')

    # 5. ACF图
    fig, axes = plt.subplots(1, 2, figsize=(12, 4))
    plot_acf(series.dropna(), lags=20, ax=axes[0])
    plot_pacf(series.dropna(), lags=20, ax=axes[1])
    axes[0].set_title('ACF')
    axes[1].set_title('PACF')
    plt.tight_layout()
    plt.show()

    # 6. 综合判断
    print('\n综合判断:')
    if adf_conclusion == '平稳' and kpss_conclusion == '平稳':
        print('  序列是平稳的 ✓')
        return True
    else:
        print('  序列可能非平稳，建议进行变换')
        return False

# 示例使用
np.random.seed(42)
n = 200

# 测试平稳序列
stationary_series = np.random.normal(0, 1, n)
is_stationary = test_stationarity(stationary_series, '平稳序列')

# 测试非平稳序列
non_stationary_series = np.cumsum(np.random.normal(0, 1, n))
is_stationary = test_stationarity(non_stationary_series, '非平稳序列（随机游走）')
```

### 自动平稳化流程

```python
def make_stationary(series, max_diff=3, alpha=0.05):
    """自动寻找合适的平稳化方法"""

    original_series = series.copy()
    diff_count = 0

    for d in range(max_diff + 1):
        if d == 0:
            current_series = original_series
        else:
            current_series = pd.Series(current_series).diff().dropna()
            diff_count = d

        # ADF检验
        result_adf = adfuller(current_series.dropna(), autolag='AIC')

        # KPSS检验
        result_kpss = kpss(current_series.dropna(), regression='c')

        # 判断是否平稳
        if result_adf[1] < alpha and result_kpss[1] > alpha:
            print(f'序列在{d}阶差分后平稳')
            print(f'ADF p值: {result_adf[1]:.4f}')
            print(f'KPSS p值: {result_kpss[1]:.4f}')
            return current_series, diff_count

    print(f'在{max_diff}阶差分内未找到平稳序列')
    return current_series, diff_count

# 示例使用
# 生成非平稳序列
np.random.seed(42)
trend_series = np.cumsum(np.random.normal(0, 1, 200)) + np.linspace(0, 50, 200)

# 自动平稳化
stationary_result, d = make_stationary(trend_series, max_diff=2)

# 绘图
fig, axes = plt.subplots(2, 1, figsize=(12, 8))

axes[0].plot(trend_series)
axes[0].set_title('原始序列')
axes[0].grid(True)

axes[1].plot(stationary_result)
axes[1].set_title(f'{d}阶差分后')
axes[1].grid(True)

plt.tight_layout()
plt.show()
```

## 注意事项

### 1. 检验方法的局限性

- **ADF检验：** 功率可能不足，对小样本不敏感
- **KPSS检验：** 对滞后长度选择敏感
- **所有检验：** 都假设一定的数据生成过程，实际数据可能违反

### 2. 多重检验问题

同时使用多种检验时，考虑多重比较校正。

### 3. 结构性变化

如果数据存在结构性变化（如政策变化、金融危机），标准检验可能失效。

### 4. 小样本问题

小样本情况下，检验统计量的分布可能偏离理论分布。

### 5. 过度差分

过度差分会导致：
- 信息损失
- 引入不必要的负相关
- 增加模型复杂度

```python
# 过度差分示例
np.random.seed(42)
white_noise = np.random.normal(0, 1, 200)

# 白噪声已经是平稳的，不需要差分
print('白噪声（原始）ADF检验:')
result = adfuller(white_noise)
print(f'ADF统计量: {result[0]:.4f}')
print(f'p值: {result[1]:.4f}')

# 过度差分
over_diffed = np.diff(white_noise, n=1)
print('\n过度差分后ADF检验:')
result = adfuller(over_diffed)
print(f'ADF统计量: {result[0]:.4f}')
print(f'p值: {result[1]:.4f}')
print('注意: 过度差分会引入不必要的负相关')
```

## 总结

平稳性检验是时间序列分析的基础步骤：

**核心概念：**
1. **严平稳：** 所有统计性质不随时间变化
2. **弱平稳：** 均值、方差、自协方差不随时间变化
3. **单位根：** 非平稳的重要特征

**检验方法：**
1. **图形检验：** 时序图、ACF图
2. **统计检验：** ADF、PP、KPSS、DF-GLS
3. **建议：** 结合多种方法进行判断

**平稳化方法：**
1. **差分：** 消除趋势
2. **变换：** 对数、Box-Cox（稳定方差）
3. **去趋势：** 减去确定性趋势
4. **组合方法：** 根据数据特点选择

**实践要点：**
1. 先进行图形检验，了解数据特征
2. 使用多种统计检验相互验证
3. 选择合适的平稳化方法
4. 避免过度差分
5. 考虑数据的经济含义和背景

理解并正确应用平稳性检验，是进行有效时间序列建模的前提。
