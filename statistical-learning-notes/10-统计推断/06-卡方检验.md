# 卡方检验（Chi-square Test）

## 定义

卡方检验（χ²检验）是基于卡方分布的一类统计假设检验方法，用于分析分类数据。卡方检验主要用于检验观察频数与期望频数之间是否存在显著差异。

## 卡方分布

### 卡方统计量

$$\chi^2 = \sum_{i=1}^{k} \frac{(O_i - E_i)^2}{E_i}$$

其中：
- $O_i$ 是第i类别的观察频数（Observed frequency）
- $E_i$ 是第i类别的期望频数（Expected frequency）
- k 是类别数

### 卡方分布的性质

1. **非负性：** 卡方统计量总是非负的（χ² ≥ 0）
2. **非对称性：** 卡方分布是右偏的，形状由自由度决定
3. **自由度：** df = k - 1 - m（m是估计的参数个数）
4. **可加性：** 独立卡方变量的和仍服从卡方分布

5. **均值和方差：**
   - E[χ²] = df
   - Var[χ²] = 2df

### 卡方分布的形状

- 当df = 1时，分布极度右偏
- 当df增大时，分布趋近于正态分布
- 当df > 30时，可以用正态分布近似

## 卡方检验的类型

### 1. 拟合优度检验（Goodness-of-fit Test）

**用途：** 检验分类数据是否符合某种理论分布。

#### 单样本拟合优度检验

**假设：**
- H₀: 观察频数与期望频数一致
- H₁: 观察频数与期望频数不一致

**步骤：**
1. 确定期望频数（基于理论分布）
2. 计算卡方统计量
3. 确定自由度：df = k - 1
4. 查卡方分布表或计算p值
5. 做出决策

**期望频数计算：**
$$E_i = n \times p_i$$

其中：
- n 是总样本量
- $p_i$ 是第i类别的理论概率

**应用实例：**
某骰子掷100次，各面出现的观察频数为：[15, 18, 12, 20, 17, 18]。检验骰子是否公平。

#### 应用条件

1. **样本量：** n ≥ 20
2. **期望频数：** 每个类别的期望频数E_i ≥ 5
3. **独立性：** 观测值相互独立

### 2. 独立性检验（Test of Independence）

**用途：** 检验两个分类变量是否独立。

#### 列联表分析

**假设：**
- H₀: 两个变量相互独立
- H₁: 两个变量不独立

**期望频数计算：**
$$E_{ij} = \frac{R_i \times C_j}{n}$$

其中：
- $R_i$ 是第i行的行合计
- $C_j$ 是第j列的列合计
- n 是总样本量

**自由度：**
$$df = (r - 1)(c - 1)$$

其中：
- r 是行数
- c 是列数

**应用实例：**
研究吸烟与肺癌的关系：

| | 肺癌 | 无肺癌 | 行合计 |
|---|-----|-------|-------|
| 吸烟 | 60 | 140 | 200 |
| 不吸烟 | 30 | 270 | 300 |
| 列合计 | 90 | 410 | 500 |

检验吸烟与肺癌是否独立。

#### 列联表的类型

| 列联表类型 | 大小 | 自由度 |
|-----------|------|--------|
| 2×2列联表 | 最简单 | 1 |
| r×c列联表 | 一般情况 | (r-1)(c-1) |
| 更高维 | 多变量交互 | 复杂 |

### 3. 齐性检验（Test of Homogeneity）

**用途：** 检验多个群体的分布是否相同。

**假设：**
- H₀: 各群体的分布相同
- H₁: 至少有一个群体的分布不同

**计算方法：** 与独立性检验相同

**区别：**
- 独立性检验：一个样本，两个变量
- 齐性检验：多个样本，一个变量

**应用实例：**
比较三个不同地区选民对某政策的态度分布是否相同。

## 卡方检验的前提条件

### 1. 分类数据
- 数据必须是定类或定序数据
- 不能直接用于连续数据

### 2. 随机抽样
- 样本是从总体中随机抽取的

### 3. 样本量要求
- 总样本量n ≥ 20

### 4. 期望频数要求
- **宽松标准：** 所有E_i ≥ 5
- **严格标准：** 超过20%的E_i < 5视为不满足

### 5. 独立性
- 观测值之间相互独立
- 每个观测值只能属于一个类别

## 小样本时的修正

### Yates连续性校正

对于2×2列联表，当样本量较小时使用：

$$\chi^2 = \sum_{i=1}^{4} \frac{(|O_i - E_i| - 0.5)^2}{E_i}$$

**适用条件：**
- 2×2列联表
- 总样本量n < 100
- 某个期望频数E_i < 10

### Fisher精确检验

当期望频数过小时使用：

- 适用于2×2列联表
- 特别适用于小样本
- 计算超几何分布的精确概率

### 合并类别

当某些类别期望频数过小时：

- 合并相邻类别
- 减少类别数量
- 使每个类别的期望频数满足要求

## 卡方检验的步骤

1. **建立假设：**
   - 明确原假设H₀和备择假设H₁

2. **整理数据：**
   - 构建列联表或列出观察频数

3. **计算期望频数：**
   - 根据零假设计算各期望频数

4. **计算卡方统计量：**
   - 使用卡方公式计算χ²值

5. **确定自由度：**
   - 根据检验类型确定df

6. **确定临界值或p值：**
   - 查卡方分布表或计算p值

7. **做出决策：**
   - 如果 χ² > χ²_{α, df} 或 p < α，拒绝H₀
   - 否则，不拒绝H₀

8. **得出结论：**
   - 根据决策解释实际意义

## 效应量：Phi系数与Cramer's V

### Phi系数（Φ）

适用于2×2列联表：

$$\Phi = \sqrt{\frac{\chi^2}{n}}$$

其中n是总样本量。

**解释：**
- 0 ≤ |Φ| ≤ 1
- |Φ| = 0.1: 小效应
- |Φ| = 0.3: 中等效应
- |Φ| = 0.5: 大效应

### Cramer's V

适用于大于2×2的列联表：

$$V = \sqrt{\frac{\chi^2}{n \times \min(r-1, c-1)}}$$

其中min(r-1, c-1)是行数-1和列数-1中的较小者。

**解释：**
- 0 ≤ V ≤ 1
- 解释与Phi系数类似

## Python实现示例

```python
import numpy as np
from scipy import stats
import pandas as pd

# 1. 拟合优度检验
# 示例：检验骰子是否公平
observed = np.array([15, 18, 12, 20, 17, 18])  # 观察频数
n = np.sum(observed)
expected = np.array([n/6] * 6)  # 公平骰子的期望频数

# 计算卡方统计量
chi2 = np.sum((observed - expected)**2 / expected)
df = len(observed) - 1
p_value = 1 - stats.chi2.cdf(chi2, df)

print(f"卡方统计量: {chi2:.4f}")
print(f"p值: {p_value:.4f}")
print(f"自由度: {df}")

# 使用scipy直接计算
chi2, p_value = stats.chisquare(f_obs=observed, f_exp=expected)
print(f"scipy计算: chi2 = {chi2:.4f}, p = {p_value:.4f}")

# 2. 独立性检验
# 示例：吸烟与肺癌的关系
# 构建列联表
contingency_table = np.array([
    [60, 140],  # 吸烟
    [30, 270]   # 不吸烟
])

# 卡方检验
chi2, p_value, df, expected = stats.chi2_contingency(contingency_table)

print(f"\n吸烟与肺癌独立性检验:")
print(f"卡方统计量: {chi2:.4f}")
print(f"p值: {p_value:.4f}")
print(f"自由度: {df}")
print("期望频数:")
print(expected)

# 3. Yates连续性校正（2×2表）
chi2_corrected, p_value, df, expected = stats.chi2_contingency(
    contingency_table, correction=True
)

print(f"\nYates校正后:")
print(f"卡方统计量: {chi2_corrected:.4f}")
print(f"p值: {p_value:.4f}")

# 4. Fisher精确检验（小样本）
oddsratio, p_value = stats.fisher_exact(contingency_table)
print(f"\nFisher精确检验:")
print(f"优势比: {oddsratio:.4f}")
print(f"p值: {p_value:.4f}")

# 5. 更大的列联表
# 示例：三个地区选民态度
contingency_table_large = np.array([
    [45, 30, 25],  # 地区A
    [55, 35, 20],  # 地区B
    [40, 40, 30]   # 地区C
])

chi2, p_value, df, expected = stats.chi2_contingency(contingency_table_large)

print(f"\n三个地区选民态度检验:")
print(f"卡方统计量: {chi2:.4f}")
print(f"p值: {p_value:.4f}")
print(f"自由度: {df}")

# 6. 效应量计算
def calculate_effect_size(chi2, n, r, c):
    """计算Cramer's V"""
    return np.sqrt(chi2 / (n * min(r-1, c-1)))

n = np.sum(contingency_table_large)
r, c = contingency_table_large.shape
cramers_v = calculate_effect_size(chi2, n, r, c)

print(f"\nCramer's V: {cramers_v:.4f}")

if cramers_v < 0.1:
    print("小效应")
elif cramers_v < 0.3:
    print("中等效应")
else:
    print("大效应")

# 7. 残差分析
# 计算标准化残差
def standardized_residuals(observed, expected):
    """计算标准化残差"""
    return (observed - expected) / np.sqrt(expected)

std_residuals = standardized_residuals(contingency_table, expected)
print(f"\n标准化残差:")
print(std_residuals)

# 标准化残差绝对值大于2表示该格子对卡方贡献显著
print(f"\n显著格子（残差绝对值 > 2）:")
for i in range(r):
    for j in range(c):
        if abs(std_residuals[i, j]) > 2:
            print(f"位置({i}, {j}): {std_residuals[i, j]:.2f}")
```

## 卡方检验的注意事项

### 1. 样本量影响

- **大样本：** 即使实际差异很小，也可能得到显著结果
- **小样本：** 即使实际差异很大，也可能得不到显著结果

**建议：** 始终检查效应量，不要仅依赖p值。

### 2. 期望频数不足

当期望频数E_i < 5时：

- 尝试合并类别
- 使用精确检验（如Fisher精确检验）
- 使用Bootstrap方法

### 3. 单元格独立性

卡方检验不能处理一个观测值出现在多个类别的情况。

### 4. 有序分类数据

对于有序分类数据，卡方检验没有利用顺序信息。

**替代方法：**
- Mann-Whitney U检验
- Kruskal-Wallis检验
- 趋势卡方检验

### 5. 小样本时的偏差

小样本时，卡方检验的I类错误率可能偏高。

**解决方案：**
- 使用Yates连续性校正
- 使用Fisher精确检验
- 使用Monte Carlo模拟

## 卡方检验与其他检验的关系

### 卡方检验与t检验

- **t检验：** 比较两组均值（连续变量）
- **卡方检验：** 比较两组分布（分类变量）

### 卡方检验与Fisher精确检验

| 特征 | 卡方检验 | Fisher精确检验 |
|------|----------|----------------|
| 大样本 | 精确 | 精确 |
| 小样本 | 近似 | 精确 |
| 计算复杂度 | 低 | 高 |
| 适用范围 | 任何列联表 | 主要是2×2表 |

## 总结

卡方检验是分析分类数据的重要工具，具有以下特点：

**主要应用：**
1. 拟合优度检验：数据是否符合理论分布
2. 独立性检验：两个分类变量是否独立
3. 齐性检验：多个群体分布是否相同

**关键要点：**
1. 基于观察频数与期望频数的比较
2. 数据必须是分类数据
3. 期望频数应满足E_i ≥ 5
4. 注意样本量对结果的影响
5. 报告效应量而非仅p值

**注意事项：**
1. 检查前提条件是否满足
2. 小样本时使用校正或精确检验
3. 有序数据考虑更合适的检验方法
4. 不要忽视实际意义

正确理解和使用卡方检验，是进行分类数据分析的基础。
