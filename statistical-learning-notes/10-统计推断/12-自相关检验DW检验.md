# 自相关检验（DW检验）

## 定义

自相关（Autocorrelation）是指时间序列数据中的误差项之间存在相关性。在时间序列回归分析中，自相关是一个常见且重要的问题，因为它违背了经典线性回归模型中误差项相互独立的假设。

### 数学定义

对于时间序列数据，误差项 $\varepsilon_t$ 的自相关可以表示为：

$$\rho_k = \frac{\text{Cov}(\varepsilon_t, \varepsilon_{t-k})}{\sqrt{\text{Var}(\varepsilon_t)\text{Var}(\varepsilon_{t-k})}} = \frac{E[(\varepsilon_t - \mu)(\varepsilon_{t-k} - \mu)]}{\sigma^2}$$

其中 $\rho_k$ 是滞后k阶的自相关系数。

### 自相关的类型

#### 1. 正自相关（Positive Autocorrelation）

正自相关意味着连续的误差项倾向于具有相同的符号。

**模式：**
```
残差
   ↑
  2│  ●    ●    ●    ●
   │   ●    ●    ●
  0│    ●    ●
   │  ●    ●
 -2│    ●    ●    ●
   └────────────────→ 时间
```

**典型场景：**
- 经济数据的惯性效应
- 季节性模式
- 遗漏的滞后变量

#### 2. 负自相关（Negative Autocorrelation）

负自相关意味着连续的误差项倾向于具有相反的符号。

**模式：**
```
残差
   ↑
  2│  ●    ●    ●
   │    ●    ●
  0│●    ●    ●    ●
   │  ●    ●
 -2│    ●    ●
   └────────────────→ 时间
```

**典型场景：**
- 采样误差
- 数据处理中的过度调整
- 罕见情况

## 自相关的影响

当回归模型存在自相关时，会产生以下问题：

### 1. OLS估计量仍然无偏但不再有效

OLS估计量 $\hat{\beta}$ 仍然是无偏的，但不再是BLUE（最佳线性无偏估计量），因为方差不再是所有线性无偏估计量中最小的。

### 2. 标准误估计有偏

OLS估计的标准误通常低估真实标准误，导致：
- t统计量被高估
- 置信区间过窄
- 第一类错误率增加（错误拒绝原假设）

### 3. 预测效果变差

自相关会影响模型的预测准确性，特别是短期预测。

### 4. 假设检验失效

由于标准误估计有偏，所有基于标准误的假设检验（t检验、F检验）都可能失效。

## Durbin-Watson检验

Durbin-Watson（DW）检验是检验一阶自相关最常用的方法。

### 检验统计量

DW统计量的计算公式为：

$$DW = \frac{\sum_{t=2}^{n}(e_t - e_{t-1})^2}{\sum_{t=1}^{n}e_t^2}$$

其中：
- $e_t$ 是第t期的残差
- n 是样本量

### DW统计量的性质

对于一阶自相关模型 $\varepsilon_t = \rho \varepsilon_{t-1} + u_t$：

- 当 $\rho = 0$（无自相关）时，DW ≈ 2
- 当 $\rho = 1$（完全正自相关）时，DW ≈ 0
- 当 $\rho = -1$（完全负自相关）时，DW ≈ 4

因此，DW与自相关系数 $\rho$ 的近似关系为：

$$DW \approx 2(1 - \rho)$$

### 检验程序

#### 检验正自相关

1. **建立假设：**
   - H₀: $\rho = 0$（无自相关）
   - H₁: $\rho > 0$（正自相关）

2. **决策规则：**
   - 如果 DW < $d_L$：拒绝H₀，存在显著正自相关
   - 如果 DW > $d_U$：不拒绝H₀，无显著正自相关
   - 如果 $d_L \leq DW \leq d_U$：检验结果不确定

#### 检验负自相关

1. **建立假设：**
   - H₀: $\rho = 0$（无自相关）
   - H₁: $\rho < 0$（负自相关）

2. **决策规则：**
   - 如果 DW > $4 - d_L$：拒绝H₀，存在显著负自相关
   - 如果 DW < $4 - d_U$：不拒绝H₀，无显著负自相关
   - 如果 $4 - d_U \leq DW \leq 4 - d_L$：检验结果不确定

#### 双侧检验

检验任何形式的自相关（正或负）：

| DW值范围 | 结论 |
|----------|------|
| 0 < DW < $d_L$ | 正自相关 |
| $d_L \leq DW \leq d_U$ | 不确定 |
| $d_U < DW < 4 - d_U$ | 无自相关 |
| $4 - d_U \leq DW \leq 4 - d_L$ | 不确定 |
| $4 - d_L < DW < 4$ | 负自相关 |

### DW检验的局限

1. **仅适用于一阶自相关：** DW检验只能检测 $\varepsilon_t = \rho \varepsilon_{t-1} + u_t$ 形式的自相关

2. **需要常数项：** 模型必须包含截距项

3. **滞后因变量问题：** 当模型包含滞后因变量（$y_{t-1}$）时，DW检验失效

4. **无法处理中间状态：** 当DW值落在不确定区域时，需要使用其他检验方法

## 其他自相关检验方法

### 1. Breusch-Godfrey检验（LM检验）

Breusch-Godfrey检验可以检测高阶自相关。

**检验步骤：**
1. 估计原模型得到残差 $e_t$
2. 将残差对原解释变量和滞后残差进行回归
3. 检验滞后残差系数的联合显著性

### 2. Ljung-Box检验

Ljung-Box检验用于检验时间序列的随机性，可以检测多阶自相关。

**检验统计量：**

$$Q = n(n+2)\sum_{k=1}^{h}\frac{\hat{\rho}_k^2}{n-k}$$

其中：
- n 是样本量
- h 是检验的滞后阶数
- $\hat{\rho}_k$ 是第k阶样本自相关系数

### 3. 游程检验（Runs Test）

游程检验是一种非参数方法，通过观察残差符号的变化来检测自相关。

## 自相关的处理方法

### 1. 广义最小二乘法（GLS）

当自相关的结构已知时，可以使用GLS获得更有效的估计。

#### 一阶自相关模型

假设误差项遵循一阶自回归过程：

$$\varepsilon_t = \rho \varepsilon_{t-1} + u_t$$

其中 $u_t$ 是白噪声（满足经典假设）。

#### 广义差分变换

对原始模型进行广义差分：

$$y_t = \beta_0(1-\rho) + \beta_1 x_t + \rho y_{t-1} - \rho \beta_1 x_{t-1} + u_t$$

变换后的模型为：

$$y_t^* = \beta_0^* + \beta_1 x_t^* + u_t$$

其中：
- $y_t^* = y_t - \rho y_{t-1}$
- $x_t^* = x_t - \rho x_{t-1}$
- $\beta_0^* = \beta_0(1-\rho)$

### 2. Cochrane-Orcutt方法

Cochrane-Orcutt方法是一种迭代方法，用于处理一阶自相关。

**算法步骤：**

1. **初始估计：** 使用OLS估计原模型，得到残差 $e_t$

2. **估计自相关系数：** 用残差估计 $\rho$
   $$\hat{\rho} = \frac{\sum_{t=2}^{n}e_t e_{t-1}}{\sum_{t=2}^{n}e_{t-1}^2}$$

3. **广义差分：** 使用 $\hat{\rho}$ 对数据进行广义差分变换

4. **重新估计：** 对变换后的数据使用OLS

5. **迭代：** 重复步骤2-4，直到 $\hat{\rho}$ 收敛

6. **调整截距：** $\hat{\beta}_0 = \hat{\beta}_0^* / (1 - \hat{\rho})$

### 3. Prais-Winsten方法

Prais-Winsten方法是对Cochrane-Orcutt的改进，保留了第一个观测值。

对于第一个观测值，变换为：
$$y_1^* = \sqrt{1-\rho^2} \cdot y_1$$
$$x_1^* = \sqrt{1-\rho^2} \cdot x_1$$

这样可以保留更多信息，提高估计效率。

### 4. Newey-West标准误

当样本量足够大时，可以使用Newey-West异方差自相关稳健标准误（HAC标准误）来修正标准误的估计。

这种方法不需要改变系数估计，只调整标准误的计算，使其在存在异方差和自相关时仍然有效。

### 5. 添加滞后项

有时自相关是因为模型遗漏了重要的滞后变量。可以通过添加滞后因变量或滞后自变量来解决。

**示例：**
$$y_t = \beta_0 + \beta_1 x_t + \beta_2 y_{t-1} + \beta_3 x_{t-1} + \varepsilon_t$$

### 6. 变量变换

对变量进行差分、对数变换等可能消除自相关。

**一阶差分：**
$$\Delta y_t = y_t - y_{t-1}$$
$$\Delta x_t = x_t - x_{t-1}$$

### 2. Cochrane-Orcutt方法

```python
def cochrane_orcutt(y, X, max_iter=100, tol=1e-6):
    """
    Cochrane-Orcutt迭代方法处理一阶自相关

    参数：
    y: 因变量
    X: 自变量矩阵（包含常数项）
    max_iter: 最大迭代次数
    tol: 收敛容忍度

    返回：
    model: 最终模型
    rho: 估计的自相关系数
    history: 迭代历史
    """
    rho_history = []
    rho = 0.0

    for iteration in range(max_iter):
        # 广义差分变换
        if iteration == 0:
            # 第一次迭代，使用原始数据
            y_transformed = y
            X_transformed = X
        else:
            # 广义差分
            y_transformed = y[1:] - rho * y[:-1]
            X_transformed = X[1:] - rho * X[:-1]

        # OLS估计
        model = sm.OLS(y_transformed, X_transformed).fit()
        residuals = model.resid

        # 估计自相关系数
        if len(residuals) > 1:
            new_rho = np.sum(residuals[1:] * residuals[:-1]) / np.sum(residuals[:-1]**2)
        else:
            new_rho = 0

        # 检查收敛
        if abs(new_rho - rho) < tol:
            break

        rho = new_rho
        rho_history.append(rho)

        print(f"迭代 {iteration + 1}: rho = {rho:.6f}")

    return model, rho, rho_history

# 应用Cochrane-Orcutt方法
print("=" * 50)
print("Cochrane-Orcutt方法")
print("=" * 50)
model_co, rho_co, rho_history = cochrane_orcutt(data['y'].values, X.values)
print(f"\n最终自相关系数: {rho_co:.4f}")
print(f"\nCochrane-Orcutt模型结果:")
print(model_co.summary())
print()

# 绘制rho收敛图
if len(rho_history) > 1:
    plt.figure(figsize=(10, 6))
    plt.plot(range(1, len(rho_history) + 1), rho_history, 'o-')
```

### 3. Prais-Winsten方法

```python
def prais_winsten(y, X, max_iter=100, tol=1e-6):
    """
    Prais-Winsten方法处理一阶自相关

    参数：
    y: 因变量
    X: 自变量矩阵（包含常数项）
    max_iter: 最大迭代次数
    tol: 收敛容忍度

    返回：
    model: 最终模型
    rho: 估计的自相关系数
    """
    rho = 0.0

    for iteration in range(max_iter):
        if iteration == 0:
            # 第一次迭代
            y_transformed = y
            X_transformed = X
        else:
            # 广义差分（保留第一个观测值）
            n = len(y)
            y_transformed = np.zeros(n)
            X_transformed = np.zeros((n, X.shape[1]))

            # 第一个观测值
            y_transformed[0] = np.sqrt(1 - rho**2) * y[0]
            X_transformed[0] = np.sqrt(1 - rho**2) * X[0]

            # 其余观测值
            y_transformed[1:] = y[1:] - rho * y[:-1]
            X_transformed[1:] = X[1:] - rho * X[:-1]

        # OLS估计
        model = sm.OLS(y_transformed, X_transformed).fit()
        residuals = model.resid

        # 估计自相关系数
        if len(residuals) > 1:
            new_rho = np.sum(residuals[1:] * residuals[:-1]) / np.sum(residuals[:-1]**2)
        else:
            new_rho = 0

        # 检查收敛
        if abs(new_rho - rho) < tol:
            break

        rho = new_rho

    return model, rho

# 应用Prais-Winsten方法

print(model_pw.summary())
print()
```

### 示例1：消费函数的时间序列分析

```python
import pandas as pd
import numpy as np
import statsmodels.api as sm
from statsmodels.stats.stattools import durbin_watson

# 假设我们有消费和收入的时间序列数据
np.random.seed(42)
n = 50

# 生成收入数据（有趋势）
income = 100 + 2 * np.arange(n) + np.random.normal(0, 5, n)

# 生成消费数据（带有自相关）
epsilon = np.zeros(n)
epsilon[0] = np.random.normal(0, 10)
for i in range(1, n):
    epsilon[i] = 0.6 * epsilon[i-1] + np.random.normal(0, 10)

consumption = 20 + 0.7 * income + epsilon

# 创建DataFrame
consumption_data = pd.DataFrame({
    'year': range(1970, 1970 + n),
    'consumption': consumption,
    'income': income
})

# 拟合消费函数
X_cons = sm.add_constant(consumption_data['income'])
model_cons = sm.OLS(consumption_data['consumption'], X_cons).fit()

print("消费函数回归结果:")
print(model_cons.summary())
print()

# 检验自相关
dw_cons = durbin_watson(model_cons.resid)
print(f"Durbin-Watson统计量: {dw_cons:.4f}")

if dw_cons < 2:
    print("检测到正自相关，需要修正")

# 使用Cochrane-Orcutt修正
model_cons_co, rho_cons, _ = cochrane_orcutt(
    consumption_data['consumption'].values,
    X_cons.values
)

print(f"\n修正后的消费函数:")
print(f"边际消费倾向: {model_cons_co.params[1]:.4f}")
print(f"截距: {model_cons_co.params[0] / (1 - rho_cons):.4f}")
print(f"自相关系数: {rho_cons:.4f}")
```

## 选择合适的方法

| 情况 | 推荐方法 | 原因 |
|------|----------|------|
| 已知自相关结构 | 广义最小二乘法(GLS) | 最有效 |
| 一阶自相关 | Cochrane-Orcutt | 标准方法 |
| 小样本 | Prais-Winsten | 保留更多信息 |
| 大样本，不确定结构 | Newey-West标准误 | 稳健方法 |
| 遗漏滞后变量 | 添加滞后项 | 解决根本原因 |
| 趋势性自相关 | 差分变换 | 消除趋势 |

## 注意事项

1. **诊断优先：** 在处理自相关之前，先诊断问题的严重程度

2. **理解原因：** 确定自相关是由于模型设定错误还是数据本身的特性

3. **比较方法：** 尝试不同的方法，比较结果

4. **报告方法：** 在报告中说明使用的修正方法

5. **检查假设：** 修正后重新检查模型假设

6. **不要过度修正：** 轻微自相关可能不需要处理

## 总结

自相关是时间序列回归分析中的重要问题：

**核心要点：**
1. 自相关违背了误差独立性的假设
2. Durbin-Watson检验是检测一阶自相关的常用方法
3. 自相关导致标准误估计有偏，影响推断
4. 多种方法可以处理自相关，各有优劣

**检验方法：**
1. Durbin-Watson检验（一阶）
2. Breusch-Godfrey检验（高阶）
3. Ljung-Box检验（多阶）
4. 残差图和ACF/PACF图

**处理方法：**
1. 广义最小二乘法（GLS）
2. Cochrane-Orcutt迭代法
3. Prais-Winsten方法
4. Newey-West稳健标准误
5. 添加滞后变量
6. 变量变换（差分）

**最佳实践：**
1. 先诊断，后处理
2. 理解自相关的来源
3. 选择合适的方法
4. 修正后重新诊断
5. 报告所用的方法

理解自相关及其处理方法，对于进行正确的时间序列回归分析至关重要。
