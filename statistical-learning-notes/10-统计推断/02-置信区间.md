# 置信区间（Confidence Interval）

## 定义

置信区间是用区间估计方法构造的，用于估计总体参数的取值范围。置信区间给出了对总体参数真实值的不确定性的量化表示。

## 基本概念

### 点估计与区间估计

- **点估计：** 用单一数值估计总体参数（如样本均值$\bar{x}$估计总体均值$\mu$）
- **区间估计：** 用一个区间估计总体参数，给出估计的精确度和可信度

### 置信水平的解释

置信水平（Confidence Level, 1-α）表示如果重复抽样多次，构造的置信区间中有$(1-α)\times100\%$的区间会包含真实的总体参数。

**重要说明：** 95%置信区间**不是**说总体参数有95%的概率落在这个区间内。总体参数是一个固定的未知值，要么在区间内，要么不在。

## 均值的置信区间

### 单总体均值的置信区间

#### 已知总体标准差σ（Z区间）

当总体标准差已知时，使用标准正态分布：

$$\bar{x} \pm z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}$$

其中：
- $\bar{x}$ 是样本均值
- $z_{\alpha/2}$ 是标准正态分布的临界值
- $\sigma$ 是总体标准差
- $n$ 是样本大小

常用$z_{\alpha/2}$值：
- 90%置信水平：1.645
- 95%置信水平：1.960
- 99%置信水平：2.576

#### 未知总体标准差σ（t区间）

当总体标准差未知时，用样本标准差s替代，使用t分布：

$$\bar{x} \pm t_{\alpha/2, df} \cdot \frac{s}{\sqrt{n}}$$

其中：
- $s$ 是样本标准差
- $t_{\alpha/2, df}$ 是t分布的临界值
- 自由度 df = n - 1

### 两总体均值差异的置信区间

#### 等方差情况

$$\bar{x}_1 - \bar{x}_2 \pm t_{\alpha/2, df} \cdot s_p\sqrt{\frac{1}{n_1} + \frac{1}{n_2}}$$

其中合并方差：
$$s_p^2 = \frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1 + n_2 - 2}$$

自由度：df = n₁ + n₂ - 2

#### 异方差情况（Welch方法）

$$(\bar{x}_1 - \bar{x}_2) \pm t_{\alpha/2, df} \cdot \sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}$$

自由度使用Welch-Satterthwaite近似计算。

## 比例的置信区间

### 单总体比例的置信区间

$$\hat{p} \pm z_{\alpha/2} \cdot \sqrt{\frac{\hat{p}(1-\hat{p})}{n}}$$

其中：
- $\hat{p} = x/n$ 是样本比例
- $x$ 是成功次数
- $n$ 是样本大小

**条件：** $n\hat{p} \geq 10$ 且 $n(1-\hat{p}) \geq 10$

### 两总体比例差异的置信区间

$$(\hat{p}_1 - \hat{p}_2) \pm z_{\alpha/2} \cdot \sqrt{\frac{\hat{p}_1(1-\hat{p}_1)}{n_1} + \frac{\hat{p}_2(1-\hat{p}_2)}{n_2}}$$

## 方差的置信区间

### 单总体方差的置信区间

$$\left[\frac{(n-1)s^2}{\chi^2_{\alpha/2, n-1}}, \frac{(n-1)s^2}{\chi^2_{1-\alpha/2, n-1}}\right]$$

其中$\chi^2_{\alpha, df}$是卡方分布的临界值。

## 置信区间的宽度

置信区间的宽度受以下因素影响：

1. **置信水平：** 置信水平越高，区间越宽
   - 99% CI 比 95% CI 更宽

2. **样本大小：** 样本越大，区间越窄
   - 区间宽度与$\frac{1}{\sqrt{n}}$成正比

3. **总体变异：** 变异越大，区间越宽
   - 用标准差s衡量

4. **数据类型：** 比例数据在$\hat{p} = 0.5$时区间最宽

**平衡精确度与置信度：**
- 增加样本量可以同时保持高置信度和窄区间
- 提高置信水平会牺牲精确度

## 不同置信水平的选择

| 置信水平 | 应用场景 |
|----------|----------|
| 90% | 初步探索性研究，风险较高可接受 |
| 95% | **最常用**，社会科学、医学研究的标准 |
| 99% | 高风险领域（安全相关）、精确度要求极高 |

**选择考虑因素：**
- 犯错的代价：后果越严重，置信水平应越高
- 研究目的：验证性研究用高置信水平，探索性研究用较低置信水平
- 资源限制：高置信水平需要更大的样本量

## 置信区间与假设检验的关系

置信区间与假设检验是等价的：

- **双侧检验：** 如果95%置信区间不包含H₀中的假设值，则在α = 0.05水平下拒绝H₀
- **单侧检验：** 可以通过置信区间的边界进行判断

**优势：** 置信区间提供了更多信息，不仅知道是否显著，还知道效应量的可能范围。

## 置信区间的误解

### 常见误区

1. **误区1：** "95%置信区间意味着参数有95%的概率在这个区间内"
   - **纠正：** 参数是固定的，区间是随机的。应该说是"构造的区间有95%的概率覆盖参数"

2. **误区2：** 置信区间包含参数值的可能性是均匀的
   - **纠正：** 参数在区间内的位置不一定是均匀分布的

3. **误区3：** 置信区间越窄越好
   - **纠正：** 过窄的区间可能样本量过小，不够可靠

4. **误区4：** 置信区间不包含零就意味着结果有实际意义
   - **纠正：** 需要考虑效应量和实际意义

## Bootstrap置信区间

当数据不符合正态分布时，可以使用Bootstrap方法构造置信区间：

### Bootstrap方法步骤

1. 从原始样本中重复抽样n次（有放回），生成Bootstrap样本
2. 计算Bootstrap样本的统计量
3. 重复步骤1-2 B次（如B = 1000）
4. 使用Bootstrap统计量的分布构造置信区间

### Bootstrap置信区间类型

1. **百分位法：** 直接使用百分位数
2. **BCa（Bias-corrected and accelerated）法：** 更精确，考虑偏差和偏度
3. **Bootstrap t法：** 模拟t统计量

## Python实现示例

```python
import numpy as np
from scipy import stats

# 均值的置信区间（未知标准差）
sample = np.array([25, 27, 24, 26, 25, 26, 28, 24, 25, 27])
n = len(sample)
x_bar = np.mean(sample)
s = np.std(sample, ddof=1)
df = n - 1
alpha = 0.05
t_crit = stats.t.ppf(1 - alpha/2, df)
margin_error = t_crit * s / np.sqrt(n)
ci = (x_bar - margin_error, x_bar + margin_error)
print(f"95% CI: ({ci[0]:.2f}, {ci[1]:.2f})")

# 使用scipy直接计算
ci = stats.t.interval(0.95, df, loc=x_bar, scale=s/np.sqrt(n))

# 比例的置信区间
x = 30  # 成功次数
n = 100
p_hat = x / n
z_crit = stats.norm.ppf(1 - alpha/2)
margin_error = z_crit * np.sqrt(p_hat * (1 - p_hat) / n)
ci = (p_hat - margin_error, p_hat + margin_error)

# 方差的置信区间
s_squared = np.var(sample, ddof=1)
ci_low = (n - 1) * s_squared / stats.chi2.ppf(1 - alpha/2, df)
ci_high = (n - 1) * s_squared / stats.chi2.ppf(alpha/2, df)
print(f"95% CI for variance: ({ci_low:.2f}, {ci_high:.2f})")

# Bootstrap置信区间
def bootstrap_ci(data, n_bootstrap=1000, alpha=0.05):
    boot_stats = []
    for _ in range(n_bootstrap):
        boot_sample = np.random.choice(data, size=len(data), replace=True)
        boot_stats.append(np.mean(boot_sample))
    boot_stats = np.array(boot_stats)
    return np.percentile(boot_stats, [alpha/2*100, (1-alpha/2)*100])

bootstrap_ci(sample)
```

## 置信区间的报告

标准报告格式：

- "均值的95%置信区间为（23.45，26.78）"
- 或："$\bar{x} = 25.11, 95\% CI [23.45, 26.78]$"

报告中应包含：
1. 点估计值
2. 置信水平
3. 区间上下限

## 总结

置信区间是统计推断中的重要工具，它：
- 提供了对总体参数不确定性的量化
- 反映了估计的精确度和可信度
- 可以用于假设检验
- 比单纯的p值提供更多信息

理解置信区间的正确解释和应用条件对于正确的统计分析至关重要。
