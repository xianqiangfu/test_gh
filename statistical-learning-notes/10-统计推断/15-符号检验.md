# 符号检验

## 问题

统计推断部分缺少符号检验的学习笔记。符号检验是最简单的非参数检验方法之一。

## 说明

符号检验用于检验配对数据的差值是否对称分布，不依赖于分布假设，适用于数据不满足正态性的情况。

## 符号检验的定义和原理

符号检验（Sign Test）是最简单的非参数检验方法之一，用于检验配对样本差值的对称性。它只关注差值的符号（正、负、零），而不考虑差值的大小。

### 基本思想

符号检验的核心思想是：
1. 计算每对数据的差值
2. 统计正号和负号的数目
3. 如果中位数为零，正负号的数目应该大致相等
4. 通过二项分布检验正号（或负号）的数量是否显著偏离期望值

### 与其他检验的比较

- **配对t检验**：假设差值服从正态分布
- **Wilcoxon符号秩检验**：考虑差值的大小和符号
- **符号检验**：只考虑符号，最简单的非参数检验

## 应用场景

符号检验适用于以下情况：

1. **配对数据**：同一对象的前后测量、双胞胎比较等
2. **中位数检验**：检验中位数是否等于某个特定值
3. **非正态分布**：当数据不满足正态性假设时
4. **序数数据**：只知数据的相对顺序，不知精确值
5. **存在异常值**：对异常值不敏感

## 统计量的计算

### 步骤1：计算差值

对于n对数据 (Xᵢ, Yᵢ)，计算差值：
```
Dᵢ = Xᵢ - Yᵢ
```

### 步骤2：统计符号

统计差值的符号：
- S⁺：正号的数目
- S⁻：负号的数目
- S⁰：零的数目（不计入检验）

有效样本量：n* = n - S⁰

### 步骤3：计算检验统计量

检验统计量T通常取较小的符号数：
```
T = min(S⁺, S⁻)
```

### 步骤4：计算p值

在H₀（差值中位数为0）下，S⁺服从二项分布B(n*, 0.5)

双侧检验的p值：
```
p = 2 × P(T ≤ t)
```

单侧检验的p值：
```
p = P(T ≤ t)
```

### 精确计算 vs 正态近似

**小样本（n ≤ 25）**：使用二项分布精确计算
```
p = Σ C(n, k) × (0.5)^n
```

**大样本（n > 25）**：使用正态近似
```
Z = (T - np₀) / √(np₀(1-p₀))
```

其中 p₀ = 0.5

## 假设检验步骤

### 步骤1：建立假设

- 原假设（H₀）：差值的中位数为0（M = 0）
- 备择假设（H₁）：
  - M ≠ 0（双侧）
  - M > 0（右侧）
  - M < 0（左侧）

### 步骤2：选择显著性水平

通常选择α = 0.05或α = 0.01

### 步骤3：计算检验统计量

按上述步骤计算T值

### 步骤4：计算p值

使用二项分布或正态近似计算p值

### 步骤5：做出决策

- 如果 p < α，拒绝H₀
- 否则，无法拒绝H₀

### 步骤6：解释结果

在实际应用背景下解释结果

## 与参数检验的比较

| 特征 | 符号检验 | 配对t检验 |
|------|---------|-----------|
| 统计量 | 符号数目 | 差值均值 |
| 分布假设 | 无 | 正态分布 |
| 检验功效 | 较低 | 较高（正态时） |
| 对异常值 | 不敏感 | 敏感 |
| 信息利用 | 只利用符号 | 利用全部信息 |

**何时使用符号检验**：
- 数据不满足正态性
- 存在严重异常值
- 数据是序数而非等距
- 只关心符号差异

## Python实现示例

### 基础符号检验

```python
import numpy as np
from scipy import stats

def sign_test(x, y=None, mu=0, alpha=0.05, alternative='two-sided'):
    """
    符号检验

    参数:
        x: 第一组数据，或差值数据（当y=None时）
        y: 第二组数据（配对样本）
        mu: 检验的中位数（单样本时）
        alpha: 显著性水平
        alternative: 备择假设类型 ('two-sided', 'greater', 'less')

    返回:
        result: 包含检验结果的字典
    """
    if y is None:
        # 单样本检验：检验中位数是否为mu
        d = x - mu
    else:
        # 配对样本检验
        d = np.array(x) - np.array(y)

    # 去除零差值
    d = d[d != 0]
    n = len(d)

    # 统计正负号
    s_plus = np.sum(d > 0)
    s_minus = np.sum(d < 0)

    # 检验统计量（取较小的符号数）
    test_stat = min(s_plus, s_minus)

    # 计算p值
    if n <= 25:
        # 小样本：使用二项分布精确计算
        if alternative == 'two-sided':
            p_value = 2 * stats.binom.cdf(test_stat, n, 0.5)
        elif alternative == 'greater':
            p_value = stats.binom.cdf(test_stat, n, 0.5)
        elif alternative == 'less':
            p_value = stats.binom.cdf(test_stat, n, 0.5)
    else:
        # 大样本：使用正态近似
        z_stat = (test_stat - n * 0.5) / np.sqrt(n * 0.25)

        if alternative == 'two-sided':
            p_value = 2 * (1 - stats.norm.cdf(abs(z_stat)))
        elif alternative == 'greater':
            p_value = 1 - stats.norm.cdf(z_stat)
        elif alternative == 'less':
            p_value = stats.norm.cdf(z_stat)

        test_stat = z_stat

    reject = p_value < alpha

    return {
        'statistic': test_stat,
        'p_value': p_value,
        'reject_null': reject,
        'n_pairs': n,
        'positive_signs': s_plus,
        'negative_signs': s_minus
    }

# 示例1：配对样本检验（某药物效果）
np.random.seed(42)
before = np.array([85, 78, 92, 88, 76, 90, 82, 79, 91, 87])
after = np.array([88, 80, 95, 86, 78, 93, 84, 82, 94, 89])

result = sign_test(before, after, alpha=0.05, alternative='two-sided')

print("配对样本符号检验结果（药物治疗效果）:")
print(f"正号数目: {result['positive_signs']}")
print(f"负号数目: {result['negative_signs']}")
print(f"有效配对数: {result['n_pairs']}")
print(f"检验统计量: {result['statistic']:.4f}")
print(f"p值: {result['p_value']:.4f}")
print(f"是否拒绝原假设: {result['reject_null']}")
```

### 与配对t检验的比较

```python
# 示例3：比较符号检验与配对t检验
np.random.seed(42)

# 正态数据
normal_data1 = np.random.normal(50, 5, 30)
normal_data2 = normal_data1 + np.random.normal(2, 3, 30)

# 非正态数据（包含异常值）
skewed_data1 = np.concatenate([np.random.exponential(10, 25), [100, 120]])
skewed_data2 = skewed_data1 + np.random.exponential(5, 27)

print("\n正态数据比较:")
print("符号检验:")
result1 = sign_test(normal_data1, normal_data2)
print(f"p值: {result1['p_value']:.4f}")

print("\n配对t检验:")
t_stat, t_p = stats.ttest_rel(normal_data1, normal_data2)
print(f"p值: {t_p:.4f}")

print("\n非正态数据比较:")
print("符号检验:")
result2 = sign_test(skewed_data1, skewed_data2)
print(f"p值: {result2['p_value']:.4f}")

print("\n配对t检验:")
t_stat2, t_p2 = stats.ttest_rel(skewed_data1, skewed_data2)
print(f"p值: {t_p2:.4f}")
```

### 使用scipy的wilcoxon检验

虽然scipy没有直接的符号检验函数，但可以通过wilcoxon函数的zero_method参数实现类似效果：

```python
# 注意：这是Wilcoxon符号秩检验，比符号检验更强大
stat, p_value = stats.wilcoxon(before, after, zero_method='pratt')

print("\nWilcoxon符号秩检验（参考）:")
print(f"统计量: {stat}")
print(f"p值: {p_value:.4f}")
```

### 可视化

```python
import matplotlib.pyplot as plt

def plot_sign_test_differences(x, y, title="符号检验：差值分布"):
    """可视化配对差值"""
    differences = np.array(x) - np.array(y)

    fig, axes = plt.subplots(1, 2, figsize=(12, 5))

    # 差值直方图
    axes[0].hist(differences, bins=10, alpha=0.7, color='skyblue', edgecolor='black')
    axes[0].axvline(x=0, color='red', linestyle='--', linewidth=2, label='零线')
    axes[0].set_xlabel('差值')
    axes[0].set_ylabel('频数')
    axes[0].set_title(f'{title} - 直方图')
    axes[0].legend()

    # 符号条形图
    positive = np.sum(differences > 0)
    negative = np.sum(differences < 0)
    zero = np.sum(differences == 0)

    axes[1].bar(['正号', '负号', '零'], [positive, negative, zero],
                color=['green', 'red', 'gray'], alpha=0.7)
    axes[1].set_ylabel('频数')
    axes[1].set_title(f'{title} - 符号分布')
    axes[1].grid(axis='y', alpha=0.3)

    plt.tight_layout()
    plt.show()

# 可视化配对样本差值
plot_sign_test_differences(before, after, "药物治疗前后对比")
```

## 符号检验的优缺点

### 优点

1. **简单直观**：计算简单，易于理解
2. **无分布假设**：不需要假设数据服从正态分布
3. **对异常值稳健**：只考虑符号，不受极端值影响
4. **适用广泛**：可用于序数数据、配对数据
5. **小样本有效**：即使样本量很小也能使用

### 缺点

1. **检验功效低**：只利用符号信息，损失信息较多
2. **忽略差异大小**：无法区分大差异和小差异
3. **效率较低**：与t检验相比，渐进相对效率约为64%
4. **双边检验保守**：当存在零差值时会减少有效样本量

## 注意事项

1. **零差值的处理**：差值为零的对不计入检验，减少有效样本量
2. **样本量要求**：小样本时使用精确二项分布，大样本时使用正态近似
3. **配对要求**：数据必须是配对的，而非独立样本
4. **独立性**：配对之间应该相互独立
