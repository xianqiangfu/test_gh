# 先验分布与后验分布（Prior and Posterior Distribution）

## 定义

在贝叶斯统计中，先验分布和后验分布是贝叶斯推断的核心概念。

- **先验分布（Prior Distribution）：** 在观测数据之前，对未知参数的主观概率分布，记为 $p(\theta)$
- **后验分布（Posterior Distribution）：** 结合观测数据后，对未知参数更新的概率分布，记为 $p(\theta|D)$

其中 $\theta$ 表示未知参数，$D$ 表示观测数据。

## 贝叶斯定理

后验分布通过贝叶斯定理计算：

$$p(\theta|D) = \frac{p(D|\theta) \cdot p(\theta)}{p(D)}$$

其中：
- $p(\theta|D)$：后验分布
- $p(D|\theta)$：似然函数（Likelihood），表示给定参数下观测数据的概率
- $p(\theta)$：先验分布
- $p(D)$：边际似然（Marginal Likelihood），作为归一化常数

边际似然的计算：

$$p(D) = \int p(D|\theta) \cdot p(\theta) \, d\theta$$

## 贝叶斯更新的直观理解

贝叶斯更新是一个不断修正认识的过程：

1. **初始状态：** 根据先验知识或主观判断设定先验分布 $p(\theta)$
2. **收集数据：** 观测到数据 $D$
3. **更新认识：** 结合先验和数据，通过贝叶斯定理得到后验分布 $p(\theta|D)$
4. **迭代更新：** 新的后验分布可以作为新的先验，继续结合新数据

这个过程体现了"学习"的本质：从先验知识出发，通过数据不断修正和完善认知。

## 先验分布的类型

### 1. 共轭先验（Conjugate Prior）

**定义：** 先验分布与似然函数属于同一分布族，使得后验分布也属于同一族。

**优点：**
- 后验分布具有解析形式
- 计算简便
- 便于理论分析

**常见共轭先验对：**

| 似然函数 | 先验分布 | 后验分布 |
|---------|---------|---------|
| 二项分布 | Beta分布 | Beta分布 |
| 泊松分布 | Gamma分布 | Gamma分布 |
| 正态分布（已知方差） | 正态分布 | 正态分布 |
| 正态分布（已知均值） | 逆Gamma分布 | 逆Gamma分布 |
| 多项分布 | Dirichlet分布 | Dirichlet分布 |

#### Beta-Binomial共轭先验示例

**似然函数（二项分布）：**
$$p(D|p) = \binom{n}{k} p^k (1-p)^{n-k}$$

**先验分布（Beta分布）：**
$$p(p) = \frac{p^{\alpha-1}(1-p)^{\beta-1}}{B(\alpha,\beta)}$$

**后验分布（Beta分布）：**
$$p(p|D) = \frac{p^{\alpha+k-1}(1-p)^{\beta+n-k-1}}{B(\alpha+k,\beta+n-k)}$$

其中 $B(\alpha,\beta)$ 是Beta函数。

**直观解释：**
- $\alpha$ 可以理解为"先验成功次数"
- $\beta$ 可以理解为"先验失败次数"
- 观测到 $k$ 次成功和 $n-k$ 次失败后，后验参数变为 $\alpha+k$ 和 $\beta+n-k$

### 2. 无信息先验（Non-informative Prior）

**目的：** 表示对参数几乎没有先验信息，让数据主导后验分布。

**类型：**

#### 2.1 均匀先验（Flat Prior）
$$p(\theta) \propto 1, \quad \theta \in [a,b]$$

#### 2.2 Jeffreys先验
基于Fisher信息矩阵的无信息先验：
$$p(\theta) \propto \sqrt{I(\theta)}$$

其中 $I(\theta)$ 是Fisher信息：
$$I(\theta) = -E\left[\frac{\partial^2 \ln p(D|\theta)}{\partial \theta^2}\right]$$

**优点：**
- 参数变换下的不变性
- 更客观的无信息表示

#### 2.3 Reference先验
使后验分布与先验分布之间的Kullback-Leibler散度最大化的先验。

### 3. 弱信息先验（Weakly Informative Prior）

**定义：** 提供一些先验信息，但不强到主导数据。

**特点：**
- 避免后验分布在不合理区域
- 允许数据主要决定后验分布
- 例如：$N(0, 10)$ 对于大多数实际问题都是弱信息先验

### 4. 有信息先验（Informative Prior）

**定义：** 基于领域知识、历史数据或专家经验的有意义先验。

**来源：**
- 文献报告的结果
- 专家的主观判断
- 历史实验数据
- 物理约束

## 后验分布的计算

### 1. 解析计算（共轭先验）

当使用共轭先验时，后验分布有解析形式，可以直接计算。

**示例：** 正态均值的估计

似然：$p(D|\mu) = \prod_{i=1}^{n} N(x_i|\mu,\sigma^2)$

先验：$p(\mu) = N(\mu_0,\tau^2)$

后验：$p(\mu|D) = N(\mu_n,\tau_n^2)$

其中：
$$\mu_n = \frac{\frac{n\bar{x}}{\sigma^2} + \frac{\mu_0}{\tau^2}}{\frac{n}{\sigma^2} + \frac{1}{\tau^2}}$$

$$\tau_n^2 = \frac{1}{\frac{n}{\sigma^2} + \frac{1}{\tau^2}}$$

### 2. 数值计算（非共轭先验）

当后验分布没有解析形式时，使用数值方法：

#### 2.1 网格近似（Grid Approximation）
在离散的参数网格上计算后验：
$$p(\theta_i|D) \approx \frac{p(D|\theta_i)p(\theta_i)}{\sum_j p(D|\theta_j)p(\theta_j)}$$

#### 2.2 拉普拉斯近似
用正态分布近似后验分布：
- 找到后验众数（最大后验估计，MAP）
- 计算众数处的Hessian矩阵
- 构造正态近似

#### 2.3 MCMC采样
使用马尔可夫链蒙特卡洛方法从后验分布采样：
- Metropolis-Hastings算法
- Gibbs采样
- Hamiltonian Monte Carlo（HMC）
- No-U-Turn Sampler（NUTS）

## 先验选择的影响

### 1. 信息量的影响

**大样本情况下：**
$$\text{似然} \gg \text{先验} \Rightarrow \text{后验} \approx \text{似然}$$

数据主导后验分布，先验选择影响较小。

**小样本情况下：**
先验分布对后验分布有较大影响。

### 2. 敏感性分析

**目的：** 评估结论对先验选择的稳健性。

**方法：**
1. 尝试不同的先验（共轭先验、无信息先验、弱信息先验）
2. 比较不同先验下的后验分布
3. 如果结论一致，说明结果稳健
4. 如果结论差异很大，说明需要更多数据或更仔细的先验选择

### 3. 先验的影响因素

**样本量 n：**
$$\tau_n^2 = \frac{1}{\frac{n}{\sigma^2} + \frac{1}{\tau^2}}$$

当 $n \to \infty$ 时，$\tau_n^2 \to \frac{\sigma^2}{n}$，先验方差 $\tau^2$ 的影响趋于0。

**先验强度：**
- 强先验（$\tau^2$ 小）：即使数据少，后验也接近先验
- 弱先验（$\tau^2$ 大）：需要更多数据才能显著改变后验

## 贝叶斯更新的实例

### 实例1：抛硬币问题

**问题：** 估计硬币正面的概率 $p$

**先验：** $Beta(1,1)$（均匀先验，表示完全无信息）

**数据：** 抛10次，7次正面

**后验：** $Beta(1+7, 1+3) = Beta(8,4)$

**后验均值：**
$$E[p|D] = \frac{8}{8+4} = \frac{2}{3}$$

**95%可信区间：**
使用`scipy.stats.beta.interval(0.95, 8, 4)`计算

### 实例2：正态均值估计

**问题：** 估计正态分布的均值 $\mu$

**先验：** $N(0, 1)$（弱信息先验）

**数据：** $x = [1.2, 1.5, 0.8, 1.1, 1.3]$，已知 $\sigma^2 = 1$

**统计量：**
- $n = 5$
- $\bar{x} = 1.18$

**后验：**
$$\mu_n = \frac{\frac{5 \times 1.18}{1} + \frac{0}{1}}{\frac{5}{1} + \frac{1}{1}} = \frac{5.9}{6} = 0.983$$

$$\tau_n^2 = \frac{1}{5 + 1} = \frac{1}{6} = 0.167$$

后验分布：$N(0.983, 0.167)$

## Python实现示例

### 1. Beta-Binomial共轭先验

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# 设置先验参数
alpha_prior = 1
beta_prior = 1

# 观测数据
n = 10  # 抛10次
k = 7   # 7次正面

# 计算后验参数
alpha_posterior = alpha_prior + k
beta_posterior = beta_prior + (n - k)

# 后验分布
posterior = stats.beta(alpha_posterior, beta_posterior)

# 后验统计量
posterior_mean = posterior.mean()
posterior_median = posterior.median()
posterior_std = posterior.std()
ci_95 = posterior.interval(0.95)

print(f"后验均值: {posterior_mean:.4f}")
print(f"后验中位数: {posterior_median:.4f}")
print(f"后验标准差: {posterior_std:.4f}")
print(f"95%可信区间: [{ci_95[0]:.4f}, {ci_95[1]:.4f}]")

# 可视化
x = np.linspace(0, 1, 1000)
plt.figure(figsize=(10, 6))
plt.plot(x, stats.beta(alpha_prior, beta_prior).pdf(x),
         label=f'先验 Beta({alpha_prior},{beta_prior})')
plt.plot(x, posterior.pdf(x),
         label=f'后验 Beta({alpha_posterior},{beta_posterior})')
plt.xlabel('p')
plt.ylabel('概率密度')
plt.title('Beta-Binomial共轭先验更新')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
```

### 2. 正态分布的贝叶斯更新

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# 设置先验参数
mu_prior = 0
sigma_prior = 1

# 观测数据
data = np.array([1.2, 1.5, 0.8, 1.1, 1.3])
sigma_known = 1  # 已知的总体标准差

# 计算后验参数
n = len(data)
x_bar = np.mean(data)

# 后验均值和方差
mu_posterior = (n * x_bar / sigma_known**2 + mu_prior / sigma_prior**2) / \n               (n / sigma_known**2 + 1 / sigma_prior**2)
sigma_posterior = np.sqrt(1 / (n / sigma_known**2 + 1 / sigma_prior**2))

# 后验分布
posterior = stats.norm(mu_posterior, sigma_posterior)

# 后验统计量
ci_95 = posterior.interval(0.95)

print(f"后验均值: {mu_posterior:.4f}")
print(f"后验标准差: {sigma_posterior:.4f}")
print(f"95%可信区间: [{ci_95[0]:.4f}, {ci_95[1]:.4f}]")

# 可视化
x = np.linspace(-1, 3, 1000)
plt.figure(figsize=(10, 6))
plt.plot(x, stats.norm(mu_prior, sigma_prior).pdf(x),
         label=f'先验 N({mu_prior},{sigma_prior}²)')
plt.plot(x, posterior.pdf(x),
         label=f'后验 N({mu_posterior:.3f},{sigma_posterior:.3f}²)')
plt.axvline(x_bar, color='red', linestyle='--', label=f'样本均值 {x_bar:.3f}')
plt.xlabel('μ')
plt.ylabel('概率密度')
plt.title('正态分布的贝叶斯更新')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
```

### 3. 使用PyMC3进行贝叶斯推断

```python
import numpy as np
import pymc3 as pm
import arviz as az

# 生成模拟数据
np.random.seed(42)
true_mu = 1.5
true_sigma = 0.5
data = np.random.normal(true_mu, true_sigma, 50)

# 定义模型
with pm.Model() as model:
    # 先验分布
    mu = pm.Normal('mu', mu=0, sigma=2)  # 弱信息先验
    sigma = pm.HalfNormal('sigma', sigma=1)  # 正的约束

    # 似然函数
    likelihood = pm.Normal('likelihood', mu=mu, sigma=sigma, observed=data)

    # 采样
    trace = pm.sample(2000, tune=1000, chains=2, return_inferencedata=True)

# 诊断和可视化
summary = az.summary(trace)
print(summary)

# 绘制后验分布
az.plot_posterior(trace)
plt.show()

# 绘制轨迹图
az.plot_trace(trace)
plt.show()

# 后验预测检验
with model:
    ppc = pm.sample_posterior_predictive(trace, samples=500)

az.plot_ppc(az.from_pymc3(trace=trace, posterior_predictive=ppc))
plt.show()
```

## 先验分布选择的准则

### 1. 实际考虑

**问题：** 参数的合理范围是什么？
**策略：** 根据实际约束设定先验

例如：
- 概率参数：使用Beta分布
- 正参数：使用Gamma分布或Half-Normal
- 任意实数：使用正态分布

### 2. 计算考虑

**问题：** 是否需要解析形式？
**策略：** 如果需要，选择共轭先验

### 3. 主观考虑

**问题：** 是否有先验知识？
**策略：** 使用有信息先验或弱信息先验

### 4. 鲁棒性考虑

**问题：** 结论是否对先验选择敏感？
**策略：** 进行敏感性分析

## 常见误区

### 1. 先验总是主观的

**误解：** 贝叶斯方法因为先验而主观。

**纠正：**
- 无信息先验可以做到几乎客观
- 频率学方法的模型选择同样主观
- 先验是透明可讨论的

### 2. 需要大量数据才能使用贝叶斯方法

**误解：** 小样本时贝叶斯方法不可靠。

**纠正：**
- 贝叶斯方法特别适合小样本
- 先验可以补充信息不足
- 大样本时先验影响自然变小

### 3. 无信息先验完全无影响

**误解：** 使用无信息先验就是完全客观。

**纠正：**
- 无信息先验仍是先验
- Jeffreys先验的构造涉及主观选择
- 应该明确声明使用的先验

## 总结

先验分布和后验分布是贝叶斯统计的核心：

1. **先验分布**：表达数据收集前的知识或信念
2. **后验分布**：结合数据后的更新认识
3. **贝叶斯更新**：通过不断积累数据修正认识的过程
4. **共轭先验**：便于计算的先验选择
5. **先验选择影响**：小样本时重要，大样本时影响减小

理解先验和后验分布是掌握贝叶斯推断的关键，它们提供了从数据中学习并将先验知识与观测数据结合的框架。
