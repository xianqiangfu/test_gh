# 皮尔逊相关系数（Pearson Correlation Coefficient）

## 定义

皮尔逊相关系数（Pearson Correlation Coefficient，记为 r）是一种衡量两个连续变量之间线性关系强度和方向的统计指标。它是最常用的相关性度量方法，由卡尔·皮尔逊（Karl Pearson）提出。

## 相关系数公式

### 总体相关系数

$$\rho_{X,Y} = \frac{Cov(X,Y)}{\sigma_X \sigma_Y} = \frac{E[(X-\mu_X)(Y-\mu_Y)]}{\sigma_X \sigma_Y}$$

### 样本相关系数

$$r = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^{n}(x_i - \bar{x})^2} \sqrt{\sum_{i=1}^{n}(y_i - \bar{y})^2}}$$

其中：
- $\bar{x}, \bar{y}$ 分别是变量 X 和 Y 的样本均值
- $n$ 是样本大小
- 分子是协方差，分母是标准差的乘积

### 等价形式

$$r = \frac{n\sum xy - \sum x \sum y}{\sqrt{[n\sum x^2 - (\sum x)^2][n\sum y^2 - (\sum y)^2]}}$$

## 相关系数的性质

### 取值范围

- **r ∈ [-1, 1]**

### 取值解释

| r值范围 | 相关强度 | 解释 |
|---------|----------|------|
| r = 1   | 完全正相关 | 所有点落在一条正斜率的直线上 |
| r ∈ [0.7, 1) | 强正相关 | 明显的正线性关系 |
| r ∈ [0.3, 0.7) | 中度正相关 | 一定的正线性关系 |
| r ∈ (0, 0.3) | 弱正相关 | 轻微的正线性关系 |
| r = 0   | 无线性相关 | 不存在线性关系 |
| r ∈ (-0.3, 0) | 弱负相关 | 轻微的负线性关系 |
| r ∈ (-0.7, -0.3) | 中度负相关 | 一定的负线性关系 |
| r ∈ (-1, -0.7] | 强负相关 | 明显的负线性关系 |
| r = -1  | 完全负相关 | 所有点落在一条负斜率的直线上 |

### 重要性质

1. **对称性：** r(X,Y) = r(Y,X)
2. **线性不变性：** 对 X 或 Y 进行线性变换不改变 r 的绝对值
   - 如果 Y' = aY + b，则 r(X,Y') = r(X,Y) × sign(a)
3. **归一化：** 相关系数已经标准化，不受单位影响
4. **只衡量线性关系：** r = 0 不一定意味着变量独立，可能存在非线性关系

### 判定系数（Coefficient of Determination）

$$R^2 = r^2$$

R² 表示 Y 的变异中可以被 X 的线性关系解释的比例。例如，r = 0.8 时，R² = 0.64，说明 64% 的变异可以被线性关系解释。

## 相关系数的假设检验

### 检验假设

- **原假设 H₀:** ρ = 0（总体相关系数为零）
- **备择假设 H₁:** ρ ≠ 0（总体相关系数不为零）

### t检验统计量

$$t = \frac{r\sqrt{n-2}}{\sqrt{1-r^2}}$$

其中：
- r 是样本相关系数
- n 是样本大小
- 自由度 df = n - 2

### 检验步骤

1. **计算样本相关系数 r**
2. **计算 t 统计量：** t = r√(n-2)/√(1-r²)
3. **确定临界值：** 查 t 分布表，自由度为 n-2
4. **做出决策：**
   - 如果 |t| > t_{α/2, n-2} 或 p < α，拒绝 H₀
   - 否则，不拒绝 H₀

|--------|------------------|------------------|

## 相关系数的置信区间

### Fisher Z变换

由于相关系数的采样分布不对称，构建置信区间需要使用 Fisher Z 变换：

$$Z = \frac{1}{2} \ln\left(\frac{1+r}{1-r}\right) = \text{arctanh}(r)$$

### Z值的置信区间

$$Z \pm z_{\alpha/2} \cdot \frac{1}{\sqrt{n-3}}$$

### 反变换回相关系数

$$r = \frac{e^{2Z} - 1}{e^{2Z} + 1} = \tanh(Z)$$

### 置信区间计算步骤

1. 计算 Fisher Z 变换： Z = arctanh(r)
2. 计算标准误： SE = 1/√(n-3)
3. 构建 Z 的置信区间： [Z_L, Z_U] = Z ± z_{α/2} · SE
4. 反变换回 r： r_L = tanh(Z_L), r_U = tanh(Z_U)

### 示例

假设 n = 100，r = 0.5，α = 0.05：

1. Z = arctanh(0.5) = 0.549
2. SE = 1/√(97) = 0.102
3. Z的95% CI = [0.549 - 1.96×0.102, 0.549 + 1.96×0.102] = [0.350, 0.749]
4. r的95% CI = [tanh(0.350), tanh(0.749)] = [0.337, 0.636]

## Python实现示例

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# 生成示例数据
np.random.seed(42)
n = 50
x = np.random.normal(0, 1, n)
y = 0.7 * x + np.random.normal(0, 0.5, n)

# 计算皮尔逊相关系数
r, p_value = stats.pearsonr(x, y)

print(f"相关系数 r = {r:.4f}")
print(f"p值 = {p_value:.6f}")
print(f"判定系数 R² = {r**2:.4f}")

# 手动计算t统计量
t_stat = r * np.sqrt(n - 2) / np.sqrt(1 - r**2)
print(f"t统计量 = {t_stat:.4f}")
print(f"自由度 = {n-2}")

# 构建置信区间
def pearson_ci(r, n, alpha=0.05):
    """计算皮尔逊相关系数的置信区间"""
    # Fisher Z变换
    z = np.arctanh(r)
    se = 1 / np.sqrt(n - 3)

    # Z的置信区间
    z_crit = stats.norm.ppf(1 - alpha/2)
    z_lower = z - z_crit * se
    z_upper = z + z_crit * se

    # 反变换
    r_lower = np.tanh(z_lower)
    r_upper = np.tanh(z_upper)

    return r_lower, r_upper

ci_lower, ci_upper = pearson_ci(r, n)
print(f"95%置信区间: [{ci_lower:.4f}, {ci_upper:.4f}]")

# 可视化
plt.figure(figsize=(10, 6))
plt.scatter(x, y, alpha=0.7)
plt.xlabel('X')
plt.ylabel('Y')
plt.title(f'散点图 (r = {r:.3f}, p = {p_value:.6f})')

# 添加趋势线
z = np.polyfit(x, y, 1)
p = np.poly1d(z)
plt.plot(x, p(x), "r--", alpha=0.8, label='趋势线')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

# 不同相关强度的可视化示例
fig, axes = plt.subplots(2, 3, figsize=(15, 10))
correlations = [-0.9, -0.5, -0.2, 0.2, 0.5, 0.9]
titles = ['强负相关', '中度负相关', '弱负相关',
          '弱正相关', '中度正相关', '强正相关']

for ax, corr, title in zip(axes.flat, correlations, titles):
    x_temp = np.random.normal(0, 1, n)
    y_temp = corr * x_temp + np.random.normal(0, np.sqrt(1-corr**2), n)
    r_temp, _ = stats.pearsonr(x_temp, y_temp)

    ax.scatter(x_temp, y_temp, alpha=0.7)
    z_temp = np.polyfit(x_temp, y_temp, 1)
    p_temp = np.poly1d(z_temp)
    ax.plot(x_temp, p_temp(x_temp), "r--", alpha=0.8)
    ax.set_title(f'{title}\nr = {r_temp:.3f}')
    ax.grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

# 检验不同样本量的显著性
sample_sizes = [10, 20, 30, 50, 100, 200]
print("\n不同样本量的显著性检验:")
print("样本量 | 相关系数 | p值 | 显著性(α=0.05)")
print("-" * 45)

for n_size in sample_sizes:
    x_temp = np.random.normal(0, 1, n_size)
    y_temp = 0.5 * x_temp + np.random.normal(0, 0.866, n_size)
    r_temp, p_temp = stats.pearsonr(x_temp, y_temp)

    significance = "是" if p_temp < 0.05 else "否"
    print(f"{n_size:6d} | {r_temp:8.4f} | {p_temp:4.6f} | {significance}")
```

## 前提条件

皮尔逊相关系数的有效性依赖于以下假设：

### 1. 连续变量

- 变量必须是连续的或至少是定序的
- 不适用于名义变量（应使用其他关联度量，如 Cramer's V）

### 2. 线性关系

- 变量之间的关系应该是线性的
- 非线性关系可能导致相关系数低估真实关联
- 建议先用散点图检查线性关系

### 3. 正态性（对于假设检验）

- 对于小样本，两个变量都应近似服从二元正态分布
- 对于大样本（n > 30），t检验对正态性偏离有一定的鲁棒性
- 可用 Shapiro-Wilk 检验检验正态性

### 4. 同方差性

- 对于给定 X，Y 的方差应该是常数
- 方差不恒定可能影响相关系数的解释

### 5. 无显著异常值

- 异常值会显著影响相关系数
- 建议检查并处理异常值

### 6. 独立性

- 观测值之间应该相互独立
- 时间序列数据可能违反独立性假设

## 注意事项和局限性

### 1. 相关不等于因果

- 相关系数只衡量关联性，不暗示因果关系
- 高相关性可能是由于第三个变量的影响（混杂变量）
- 必须结合领域知识判断因果关系

### 2. 只衡量线性关系

- r = 0 并不意味着变量无关
- 可能存在强的非线性关系（如二次、指数关系）
- 示例：Y = X² 时，即使关系完美，r 也可能接近 0

### 3. 异常值的影响

- 单个异常值可能显著改变相关系数
- 建议使用散点图识别异常值
- 考虑使用稳健的相关度量（如 Spearman 相关系数）

### 4. 范围限制（Range Restriction）

- 当变量的取值范围受限时，相关系数会被低估
- 示例：只在高分学生中研究考试分数与学习时间的相关性

### 5. 样本量的影响

- 小样本时，即使 r 很大也可能不显著
- 大样本时，很小的 r 也可能达到统计显著性
- 需要同时考虑效应量和显著性

### 6. 辛普森悖论（Simpson's Paradox）

- 分组数据的相关系数可能与合并数据的相关系数相反
- 必须考虑潜在的分组变量

### 7. 非正态分布

- 对于严重偏态的分布，相关系数可能不准确
- 考虑使用秩相关（Spearman 或 Kendall）

### 8. 测量误差

- 测量误差通常会降低相关系数
-  attenuation effect: 观测到的相关系数 ≤ 真实相关系数

## 与其他相关系数的比较

| 相关系数类型 | 变量类型 | 关系类型 | 对异常值敏感度 |
|-------------|----------|----------|----------------|
| Pearson     | 连续变量 | 线性关系 | 高 |
| Spearman    | 定序变量 | 单调关系 | 低 |
| Kendall     | 定序变量 | 单调关系 | 低 |

### Spearman 相关系数（秩相关）

- 适用于非线性但单调的关系
- 对异常值不敏感
- 基于变量的秩而非原始值

$$\rho_s = 1 - \frac{6\sum d_i^2}{n(n^2-1)}$$

其中 dᵢ 是第 i 个观测值的秩差。

## 效应量解读

除了统计显著性，还需要考虑实际意义：

| r值范围 | 效应量 | 解释 |
|---------|--------|------|
| |r| < 0.1 | 极小 | 几乎没有实际意义 |
| 0.1 ≤ |r| < 0.3 | 小 | 弱效应，实际意义有限 |
| 0.3 ≤ |r| < 0.5 | 中等 | 中等效应，有一定实际意义 |
| 0.5 ≤ |r| < 0.7 | 大 | 强效应，实际意义明显 |
| |r| ≥ 0.7 | 极大 | 极强效应 |

## 应用实例

### 1. 教育研究

研究学生数学成绩和物理成绩的相关性：

```python
math_scores = [85, 78, 92, 65, 88, 72, 95, 68, 80, 90]
physics_scores = [82, 75, 89, 62, 85, 70, 92, 65, 77, 88]

r, p = stats.pearsonr(math_scores, physics_scores)
# r = 0.989, p < 0.001
# 结论：数学和物理成绩存在极强的正相关关系
```

### 2. 医学研究

研究血压和胆固醇水平的关系：

```python
blood_pressure = [120, 135, 128, 142, 118, 130, 145, 122, 138, 125]
cholesterol = [180, 220, 195, 240, 175, 200, 250, 185, 230, 190]

r, p = stats.pearsonr(blood_pressure, cholesterol)
# r = 0.892, p < 0.001
# 结论：血压和胆固醇水平存在强正相关关系
```

### 3. 经济研究

研究GDP和失业率的关系：

```python
gdp_growth = [3.2, 2.8, 4.1, 1.5, 3.8, 2.5, 4.5, 2.0, 3.5, 2.2]
unemployment = [5.2, 5.8, 4.5, 6.8, 4.8, 6.0, 4.2, 6.5, 5.0, 5.5]

r, p = stats.pearsonr(gdp_growth, unemployment)
# r = -0.934, p < 0.001
# 结论：GDP增长与失业率存在强负相关关系（奥肯定律）
```

## 诊断检查清单

在报告相关系数之前，检查以下内容：

1. [ ] 绘制散点图检查线性关系
2. [ ] 检查异常值及其影响
3. [ ] 检验变量的正态性（特别是小样本时）
4. [ ] 考虑潜在的混杂变量
5. [ ] 检查数据的范围限制
6. [ ] 计算并报告置信区间
7. [ ] 同时报告效应量和p值
8. [ ] 考虑使用稳健相关度量（如 Spearman）进行比较
9. [ ] 明确说明相关不等于因果
10. [ ] 结合领域知识解释结果的实际意义

## 总结

皮尔逊相关系数是统计分析中最重要的工具之一，用于衡量两个连续变量之间的线性关系。正确使用相关系数需要：

1. 理解其定义、性质和取值解释
2. 掌握假设检验和置信区间的构建方法
3. 注意前提条件和局限性
4. 结合可视化和其他统计方法进行综合分析
5. 谨慎解读结果，避免过度推断

记住：相关系数提供的是一种描述性工具，而不是因果关系的证明。正确的解释需要结合研究设计、领域知识和统计严谨性。
