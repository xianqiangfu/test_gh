# 置换检验（Permutation Test）

## 定义

置换检验（Permutation Test），也称为随机化检验（Randomization Test），是一种非参数统计检验方法。它通过随机重排观测值来构建检验统计量的零分布，不需要对数据的分布做出假设。

## 基本原理

### 核心思想

置换检验的核心思想是：**如果原假设成立，那么样本中观测值的标签（如组别）应该是随机的**。通过重新分配这些标签，我们可以构建检验统计量在原假设下的分布。

### 历史背景

- **1935年**：Ronald Fisher 在《实验设计》一书中提出了随机化检验的思想
- **1930年代**：应用于农业实验设计
- **现代**：随着计算能力提升，置换检验在各个领域得到广泛应用

### 数学原理

假设我们有两组数据：
- 组A：$X_1, X_2, ..., X_n$
- 组B：$Y_1, Y_2, ..., Y_m$

原假设 $H_0$：两组数据来自同一分布

检验统计量：$T = \bar{X} - \bar{Y}$

在原假设下，将所有 $n+m$ 个观测值随机分成两组（n个和m个），计算检验统计量。重复这个过程B次，得到检验统计量的经验分布。

p值计算：
$$p = \frac{\#\{T_i \geq T_{obs}\}}{B}$$

其中 $T_{obs}$ 是观测到的检验统计量。

## 置换检验的类型

### 1. 独立双样本置换检验

用于比较两个独立样本的均值差异。

### 2. 配对样本置换检验

用于比较配对样本的差异，通常检验差值的分布是否以0为中心。

### 3. 多组置换检验

用于比较多个组的差异（类似于ANOVA）。

### 4. 相关性置换检验

用于检验变量之间的相关性是否显著。

## 与参数检验的比较

### 置换检验 vs. 参数检验

| 特征 | 置换检验 | 参数检验 |
|------|----------|----------|
| **分布假设** | 无需分布假设 | 需要假设数据分布（如正态分布） |
| **适用性** | 适用于任何样本量和分布 | 通常需要满足特定条件 |
| **统计量** | 可以使用任意检验统计量 | 通常有固定的检验统计量 |
| **计算** | 计算量大（需要大量置换） | 计算量小 |
| **精度** | 精度依赖于置换次数 | 理论上精确 |
| **p值** | 近似p值 | 精确p值（基于理论分布） |

### 何时使用置换检验

1. **样本量小**：当样本量较小，无法满足参数检验的假设时
2. **非正态数据**：当数据严重偏离正态分布时
3. **异常值**：当数据中存在异常值时，置换检验更稳健
4. **非传统统计量**：需要使用非标准检验统计量时
5. **验证参数检验结果**：验证参数检验结果的稳健性

## 置换检验的步骤

### 标准步骤

1. **建立假设**
   - 原假设 $H_0$：各组来自同一分布，或没有差异
   - 备择假设 $H_1$：存在差异

2. **选择检验统计量**
   - 根据问题选择合适的统计量
   - 常用：均值差、t统计量、中位数差等

3. **计算观测统计量**
   - 使用原始数据计算检验统计量 $T_{obs}$

4. **生成置换分布**
   - 将观测值随机重排
   - 计算每次置换后的检验统计量
   - 重复B次（通常B ≥ 1000）

5. **计算p值**
   - 比较置换统计量与观测统计量
   - 计算p值

6. **做出决策**
   - 如果 p < α（通常为0.05），拒绝 $H_0$
   - 否则，不拒绝 $H_0$

### 图示说明

```
原始数据：组A [5, 7, 8], 组B [3, 4, 6]
检验统计量：均值差 = 6.67 - 4.33 = 2.34

第1次置换：组A [5, 4, 6], 组B [3, 7, 8]
检验统计量：均值差 = 5.00 - 6.00 = -1.00

第2次置换：组A [7, 3, 8], 组B [5, 4, 6]
检验统计量：均值差 = 6.00 - 5.00 = 1.00

... 重复 B 次 ...

p值 = 超过2.34的置换统计量的比例
```

### 使用置换检验

```python
import numpy as np
from scipy import stats

def permutation_test(x, y, n_permutations=10000, statistic='mean'):
    """
    置换检验
    
    参数：
    x, y: 两组数据
    n_permutations: 置换次数
    statistic: 检验统计量类型 ('mean', 't', 'median')
    
    返回：
    observed_stat: 观测到的统计量
    p_value: p值
    perm_stats: 置换统计量数组
    """
    # 计算观测统计量
    if statistic == 'mean':
        observed_stat = np.mean(x) - np.mean(y)
    elif statistic == 't':
        observed_stat = stats.ttest_ind(x, y).statistic
    elif statistic == 'median':
        observed_stat = np.median(x) - np.median(y)
    else:
        raise ValueError("statistic must be 'mean', 't', or 'median'")
    
    # 合并数据
    combined = np.concatenate([x, y])
    n_x = len(x)
    n_y = len(y)
    n_total = n_x + n_y
    
    # 进行置换
    perm_stats = np.zeros(n_permutations)
    for i in range(n_permutations):
        # 随机重排
        perm = np.random.permutation(combined)
        
        # 分割为两组
        perm_x = perm[:n_x]
        perm_y = perm[n_x:]
        
        # 计算置换统计量
        if statistic == 'mean':
            perm_stats[i] = np.mean(perm_x) - np.mean(perm_y)
        elif statistic == 't':
            perm_stats[i] = stats.ttest_ind(perm_x, perm_y).statistic
        elif statistic == 'median':
            perm_stats[i] = np.median(perm_x) - np.median(perm_y)
    
    # 计算p值（双侧检验）
    p_value = np.mean(np.abs(perm_stats) >= np.abs(observed_stat))
    
    return observed_stat, p_value, perm_stats

# 应用置换检验
group_a = np.array([85, 82, 88, 79, 86, 84, 87, 83, 85, 81])
group_b = np.array([82, 80, 85, 78, 81, 83, 80, 82, 79, 84])

# 使用均值差
obs_mean, p_mean, perm_mean = permutation_test(group_a, group_b, n_permutations=10000, statistic='mean')
print(f"观测均值差: {obs_mean:.4f}")
print(f"置换检验p值 (均值): {p_mean:.4f}")

# 使用t统计量
obs_t, p_t, perm_t = permutation_test(group_a, group_b, n_permutations=10000, statistic='t')
print(f"观测t统计量: {obs_t:.4f}")
print(f"置换检验p值 (t): {p_t:.4f}")

# 传统t检验结果
t_stat, p_value = stats.ttest_ind(group_a, group_b)
print(f"传统t检验p值: {p_value:.4f}")
```

## 配对样本置换检验

### 应用场景

比较同一组对象在不同条件下的测量结果，如：
- 前后测设计
- 同一对象的两种处理
- 匹配样本的比较

### 实现方法

#### 方法1：随机符号法

```python
def paired_permutation_test(before, after, n_permutations=10000):
    """
    配对样本置换检验
    
    参数：
    before, after: 配对数据
    n_permutations: 置换次数
    
    返回：
    observed_mean: 观测到的均值差
    p_value: p值
    perm_stats: 置换统计量数组
    """
    # 计算观测统计量（差值的均值）
    differences = np.array(before) - np.array(after)
    observed_mean = np.mean(differences)
    
    # 置换检验：随机改变差值的符号
    n_pairs = len(differences)
    perm_stats = np.zeros(n_permutations)
    
    for i in range(n_permutations):
        # 随机选择符号
        signs = np.random.choice([-1, 1], size=n_pairs)
        
        # 计算置换后的均值
        perm_stats[i] = np.mean(signs * differences)
    
    # 计算p值（双侧检验）
    p_value = np.mean(np.abs(perm_stats) >= np.abs(observed_mean))
    
    return observed_mean, p_value, perm_stats

# 示例：减肥前后体重变化
before = np.array([70, 72, 68, 75, 71, 69, 73, 70, 72, 71])
after = np.array([68, 70, 66, 72, 69, 67, 71, 68, 70, 69])

obs_mean, p_value, perm_stats = paired_permutation_test(before, after)
print(f"观测到的平均减重: {obs_mean:.4f} kg")
print(f"置换检验p值: {p_value:.4f}")

# 传统配对t检验
t_stat, p_ttest = stats.ttest_rel(after, before)
print(f"传统配对t检验p值: {p_ttest:.4f}")
```

#### 方法2：交换配对法

```python
def paired_swap_permutation_test(x, y, n_permutations=10000):
    """
    配对样本置换检验（交换配对法）
    
    参数：
    x, y: 配对数据
    n_permutations: 置换次数
    
    返回：
    observed_stat: 观测统计量
    p_value: p值
    """
    # 计算观测统计量
    observed_stat = np.mean(np.array(x) - np.array(y))
    
    n_pairs = len(x)
    perm_stats = np.zeros(n_permutations)
    
    for i in range(n_permutations):
        # 对于每对数据，随机决定是否交换
        swap_decisions = np.random.randint(0, 2, size=n_pairs)
        
        perm_x = np.zeros(n_pairs)
        perm_y = np.zeros(n_pairs)
        
        for j in range(n_pairs):
            if swap_decisions[j] == 0:
                perm_x[j] = x[j]
                perm_y[j] = y[j]
            else:
                perm_x[j] = y[j]
                perm_y[j] = x[j]
        
        perm_stats[i] = np.mean(perm_x - perm_y)
    
    # 计算p值
    p_value = np.mean(np.abs(perm_stats) >= np.abs(observed_stat))
    
    return observed_stat, p_value, perm_stats

# 应用
obs_stat, p_swap, _ = paired_swap_permutation_test(before, after)
print(f"交换配对法p值: {p_swap:.4f}")
```

## 置换检验的优缺点

### 优点

1. **无需分布假设**
   - 不假设数据服从任何特定分布
   - 适用于正态性检验无法通过的样本

2. **稳健性强**
   - 对异常值不敏感
   - 适用于偏态分布

3. **灵活性高**
   - 可以使用任何检验统计量
   - 不局限于传统的均值或中位数

4. **精确性**
   - 对于小样本也能得到可靠的p值
   - 置换次数足够多时，p值近似精确

5. **直观性**
   - 基于重抽样的思想，易于理解
   - 便于解释给非统计专业人士

### 缺点

1. **计算成本高**
   - 需要进行大量置换（通常≥1000次）
   - 对于大数据集，计算时间可能很长

2. **近似性**
   - p值是近似的，取决于置换次数
   - 置换次数不足时，p值可能不准确

3. **设计要求**
   - 需要满足可交换性假设
   - 某些复杂设计难以应用

4. **软件支持**
   - 并非所有统计软件都内置置换检验
   - 需要自定义代码

5. **检验功效**
   - 在某些情况下，参数检验可能更有效
   - 特别是当分布假设满足时

## 计算复杂度问题

### 时间复杂度

对于独立双样本置换检验：
- 每次置换：O(n)（排序、分组、计算统计量）
- 总复杂度：O(B × n)

其中：
- B = 置换次数
- n = 总样本量

### 置换次数的选择

| 样本量 | 推荐置换次数 | 原因 |
|--------|------------|------|
| 小样本 (n < 20) | 1000-5000 | 置换成本低，可增加次数 |
| 中等样本 (20 ≤ n < 100) | 10000 | 平衡精度和效率 |
| 大样本 (n ≥ 100) | 1000-5000 | 置换成本高，适当减少次数 |

### p值的精度

p值的精度与置换次数B的关系：
- 最小可检测的p值约为 1/B
- 对于95%置信水平，需要B ≥ 1000
- 对于更精确的p值（如0.01以下），需要B ≥ 10000

### 计算优化策略

#### 1. 并行计算

```python
from joblib import Parallel, delayed
import numpy as np
from scipy import stats

def parallel_permutation_test(x, y, n_permutations=10000, n_jobs=-1):
    """
    并行置换检验
    
    参数：
    x, y: 两组数据
    n_permutations: 置换次数
    n_jobs: 并行任务数 (-1表示使用所有CPU核心)
    
    返回：
    observed_stat: 观测统计量
    p_value: p值
    """
    # 计算观测统计量
    observed_stat = np.mean(x) - np.mean(y)
    
    # 合并数据
    combined = np.concatenate([x, y])
    n_x = len(x)
    
    # 定义单次置换函数
    def single_permutation(_):
        perm = np.random.permutation(combined)
        perm_x = perm[:n_x]
        perm_y = perm[n_x:]
        return np.mean(perm_x) - np.mean(perm_y)
    
    # 并行执行
    perm_stats = Parallel(n_jobs=n_jobs)(
        delayed(single_permutation)(i) for i in range(n_permutations)
    )
    perm_stats = np.array(perm_stats)
    
    # 计算p值
    p_value = np.mean(np.abs(perm_stats) >= np.abs(observed_stat))
    
    return observed_stat, p_value, perm_stats
```

#### 2. 提前终止

```python
def early_stopping_permutation_test(x, y, alpha=0.05, max_permutations=10000, batch_size=100):
    """
    提前终止的置换检验
    
    当可以确定是否拒绝原假设时提前终止
    
    参数：
    x, y: 两组数据
    alpha: 显著性水平
    max_permutations: 最大置换次数
    batch_size: 每批处理的置换次数
    
    返回：
    observed_stat: 观测统计量
    p_value: p值
    n_used: 实际使用的置换次数
    """
    observed_stat = np.mean(x) - np.mean(y)
    combined = np.concatenate([x, y])
    n_x = len(x)
    
    # 预计算置换统计量
    ext_count = 0  # 超过观测统计量的次数
    
    for i in range(max_permutations):
        perm = np.random.permutation(combined)
        perm_x = perm[:n_x]
        perm_y = perm[n_x:]
        perm_stat = np.mean(perm_x) - np.mean(perm_y)
        
        if np.abs(perm_stat) >= np.abs(observed_stat):
            ext_count += 1
        
        # 每批检查是否可以提前终止
        if (i + 1) % batch_size == 0:
            current_p = ext_count / (i + 1)
            
            # 如果p值明显大于alpha，可以提前终止
            if current_p > alpha * 2 and (i + 1) >= 1000:
                return observed_stat, current_p, i + 1
            
            # 如果p值明显小于alpha，也可以提前终止
            if current_p < alpha / 2 and (i + 1) >= 1000:
                return observed_stat, current_p, i + 1
    
    # 计算最终p值
    final_p = ext_count / max_permutations
    return observed_stat, final_p, max_permutations
```

## Python实现示例

### 完整的置换检验工具类

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt
from joblib import Parallel, delayed

class PermutationTest:
    """
    置换检验工具类
    """
    
    def __init__(self, n_permutations=10000, random_state=None):
        """
        初始化
        
        参数：
        n_permutations: 置换次数
        random_state: 随机种子
        """
        self.n_permutations = n_permutations
        self.random_state = random_state
        
        if random_state is not None:
            np.random.seed(random_state)
        
        self.observed_stat = None
        self.p_value = None
        self.perm_stats = None
    
    def _compute_statistic(self, x, y, statistic='mean'):
        """计算检验统计量"""
        if statistic == 'mean':
            return np.mean(x) - np.mean(y)
        elif statistic == 't':
            return stats.ttest_ind(x, y).statistic
        elif statistic == 'median':
            return np.median(x) - np.median(y)
        elif statistic == 'variance':
            return np.var(x, ddof=1) - np.var(y, ddof=1)
        else:
            raise ValueError(f"Unknown statistic: {statistic}")
    
    def test(self, x, y, statistic='mean', alternative='two-sided', n_jobs=1):
        """
        执行独立双样本置换检验
        
        参数：
        x, y: 两组数据
        statistic: 检验统计量类型
        alternative: 备择假设类型 ('two-sided', 'greater', 'less')
        n_jobs: 并行任务数
        
        返回：
        self: 返回自身以便链式调用
        """
        # 计算观测统计量
        self.observed_stat = self._compute_statistic(x, y, statistic)
        
        # 合并数据
        combined = np.concatenate([x, y])
        n_x = len(x)
        
        # 定义单次置换函数
        def single_permutation(_):
            perm = np.random.permutation(combined)
            perm_x = perm[:n_x]
            perm_y = perm[n_x:]
            return self._compute_statistic(perm_x, perm_y, statistic)
        
        # 执行置换
        if n_jobs == 1:
            self.perm_stats = np.array([single_permutation(i) 
                                       for i in range(self.n_permutations)])
        else:
            self.perm_stats = np.array(Parallel(n_jobs=n_jobs)(
                delayed(single_permutation)(i) for i in range(self.n_permutations)
            ))
        
        # 计算p值
        if alternative == 'two-sided':
            self.p_value = np.mean(np.abs(self.perm_stats) >= np.abs(self.observed_stat))
        elif alternative == 'greater':
            self.p_value = np.mean(self.perm_stats >= self.observed_stat)
        elif alternative == 'less':
            self.p_value = np.mean(self.perm_stats <= self.observed_stat)
        else:
            raise ValueError(f"Unknown alternative: {alternative}")
        
        return self
    
    def paired_test(self, x, y, alternative='two-sided'):
        """
        执行配对样本置换检验
        
        参数：
        x, y: 配对数据
        alternative: 备择假设类型
        
        返回：
        self: 返回自身以便链式调用
        """
        # 计算观测统计量
        differences = np.array(x) - np.array(y)
        self.observed_stat = np.mean(differences)
        
        n_pairs = len(differences)
        self.perm_stats = np.zeros(self.n_permutations)
        
        for i in range(self.n_permutations):
            # 随机改变符号
            signs = np.random.choice([-1, 1], size=n_pairs)
            self.perm_stats[i] = np.mean(signs * differences)
        
        # 计算p值
        if alternative == 'two-sided':
            self.p_value = np.mean(np.abs(self.perm_stats) >= np.abs(self.observed_stat))
        elif alternative == 'greater':
            self.p_value = np.mean(self.perm_stats >= self.observed_stat)
        elif alternative == 'less':
            self.p_value = np.mean(self.perm_stats <= self.observed_stat)
        
        return self
    
    def summary(self):
        """打印检验结果摘要"""
        print("=" * 50)
        print("置换检验结果")
        print("=" * 50)
        print(f"观测统计量: {self.observed_stat:.4f}")
        print(f"p值: {self.p_value:.4f}")
        print(f"置换次数: {self.n_permutations}")
        
        if self.p_value < 0.01:
            print("结论: 拒绝原假设 (p < 0.01)")
        elif self.p_value < 0.05:
            print("结论: 拒绝原假设 (p < 0.05)")
        elif self.p_value < 0.1:
            print("结论: 边缘显著 (0.05 < p < 0.1)")
        else:
            print("结论: 不拒绝原假设")
        print("=" * 50)
    
    def plot_distribution(self):
        """绘制置换分布"""
        plt.figure(figsize=(10, 6))
        
        # 绘制直方图
        plt.hist(self.perm_stats, bins=50, alpha=0.7, 
                color='skyblue', edgecolor='black')
        
        # 标记观测统计量
        plt.axvline(self.observed_stat, color='red', linestyle='--', 
                   linewidth=2, label=f'观测值: {self.observed_stat:.4f}')
        
        # 标记临界区域
        if self.p_value < 0.05:
            if self.observed_stat > 0:
                plt.axvline(np.percentile(self.perm_stats, 95), color='orange', 
                           linestyle=':', label='95%临界值')
            else:
                plt.axvline(np.percentile(self.perm_stats, 5), color='orange', 
                           linestyle=':', label='5%临界值')
        
        plt.xlabel('检验统计量')
        plt.ylabel('频数')
        plt.title(f'置换分布 (p = {self.p_value:.4f})')
        plt.legend()
        plt.grid(True, alpha=0.3)
        plt.show()
    
    def get_confidence_interval(self, alpha=0.05):
        """获取置信区间"""
        lower = np.percentile(self.perm_stats, alpha/2 * 100)
        upper = np.percentile(self.perm_stats, (1 - alpha/2) * 100)
        return lower, upper
```

```python
def correlation_permutation_test(x, y, n_permutations=10000, random_state=None):
    """
    相关性置换检验
    
    参数：
    x, y: 两个变量
    n_permutations: 置换次数
    random_state: 随机种子
    
    返回：
    observed_r: 观测到的相关系数
    p_value: p值
    perm_r: 置换相关系数数组
    """
    if random_state is not None:
        np.random.seed(random_state)
    
    # 计算观测相关系数
    observed_r = np.corrcoef(x, y)[0, 1]
    
    n = len(x)
    perm_r = np.zeros(n_permutations)

        perm_y = np.random.permutation(y)

    # 计算p值
    p_value = np.mean(np.abs(perm_r) >= np.abs(observed_r))
    
    return observed_r, p_value, perm_r
```

### 1. 方差置换检验

```python
def variance_permutation_test(x, y, n_permutations=10000, random_state=None):
    """
    方差置换检验
    
    检验两组数据的方差是否相等
    """
    if random_state is not None:
        np.random.seed(random_state)
    
    # 计算观测方差比
    observed_ratio = np.var(x, ddof=1) / np.var(y, ddof=1)
    
    combined = np.concatenate([x, y])
    n_x = len(x)
    
    perm_ratios = np.zeros(n_permutations)
    
    for i in range(n_permutations):
        perm = np.random.permutation(combined)
        perm_x = perm[:n_x]
        perm_y = perm[n_x:]
        
        # 计算方差比（确保大于等于1）
        var_x = np.var(perm_x, ddof=1)
        var_y = np.var(perm_y, ddof=1)
        perm_ratios[i] = max(var_x / var_y, var_y / var_x)
    
    # 计算p值
    p_value = np.mean(perm_ratios >= observed_ratio)
    
    return observed_ratio, p_value

print(f"p值: {p_var:.4f}")
```

### 2. 多组置换检验

```python
def multi_group_permutation_test(*groups, n_permutations=10000, random_state=None):
    """
    多组置换检验（类似于ANOVA）
    
    参数：
    *groups: 多组数据
    n_permutations: 置换次数
    random_state: 随机种子
    
    返回：
    observed_f: 观测到的F统计量
    p_value: p值
    perm_f: 置换F统计量数组
    """
    if random_state is not None:
        np.random.seed(random_state)
    
    def calculate_f_stat(groups):
        """计算F统计量"""
        # 组间平方和
        grand_mean = np.mean(np.concatenate(groups))
        ss_between = sum(len(g) * (np.mean(g) - grand_mean)**2 for g in groups)
        df_between = len(groups) - 1
        
        # 组内平方和
        ss_within = sum(sum((x - np.mean(g))**2 for x in g) for g in groups)
        df_within = sum(len(g) for g in groups) - len(groups)
        
        # F统计量
        if df_within > 0:
            f_stat = (ss_between / df_between) / (ss_within / df_within)
        else:
            f_stat = 0
        
        return f_stat
    
    # 计算观测F统计量
    observed_f = calculate_f_stat(groups)
    
    # 合并所有数据
    all_data = np.concatenate(groups)
    group_sizes = [len(g) for g in groups]
    
    perm_f = np.zeros(n_permutations)
    
    for i in range(n_permutations):
        # 随机重排
        perm_data = np.random.permutation(all_data)
        
        # 分配到各组
        start = 0
        perm_groups = []
        for size in group_sizes:
            perm_groups.append(perm_data[start:start+size])
            start += size
        
        # 计算置换F统计量
        perm_f[i] = calculate_f_stat(perm_groups)
    
    # 计算p值
    p_value = np.mean(perm_f >= observed_f)
    
    return observed_f, p_value, perm_f

# 示例

group2 = np.random.normal(55, 10, 20)
group3 = np.random.normal(60, 10, 20)

                                                  n_permutations=10000)

print(f"p值: {p_multi:.4f}")
```

### 3. 自助法与置换检验的结合

```python
def bootstrap_permutation_test(x, y, n_bootstrap=1000, n_permutations=1000):
    """
    结合自助法和置换检验
    
    首先通过自助法估计置信区间，然后使用置换检验检验显著性
    """
    # 自助法估计置信区间
    boot_means = np.zeros(n_bootstrap)
    for i in range(n_bootstrap):
        boot_x = np.random.choice(x, size=len(x), replace=True)
        boot_y = np.random.choice(y, size=len(y), replace=True)
        boot_means[i] = np.mean(boot_x) - np.mean(boot_y)
    
    ci_lower = np.percentile(boot_means, 2.5)
    ci_upper = np.percentile(boot_means, 97.5)
    
    # 置换检验
    observed = np.mean(x) - np.mean(y)
    combined = np.concatenate([x, y])
    n_x = len(x)
    
    perm_stats = np.zeros(n_permutations)
    for i in range(n_permutations):
        perm = np.random.permutation(combined)
        perm_x = perm[:n_x]
        perm_y = perm[n_x:]
        perm_stats[i] = np.mean(perm_x) - np.mean(perm_y)
    
    p_value = np.mean(np.abs(perm_stats) >= np.abs(observed))
    
    return {
        'observed': observed,
        'p_value': p_value,
        'bootstrap_ci': (ci_lower, ci_upper),
        'bootstrap_means': boot_means,
        'perm_stats': perm_stats
    }

# 示例
np.random.seed(42)
x = np.random.normal(10, 2, 30)
y = np.random.normal(12, 2, 30)

results = bootstrap_permutation_test(x, y)
print(f"观测差异: {results['observed']:.4f}")
print(f"p值: {results['p_value']:.4f}")
print(f"95%自助法置信区间: [{results['bootstrap_ci'][0]:.4f}, {results['bootstrap_ci'][1]:.4f}]")
```

## 总结

置换检验是一种强大而灵活的非参数统计检验方法：

### 核心优势

1. **无需分布假设**：不依赖数据的分布假设，适用范围广
2. **稳健性强**：对异常值和非正态数据不敏感
3. **灵活多样**：可以使用任意检验统计量
4. **直观易懂**：基于重抽样的思想，易于理解和解释

### 适用场景

- 小样本情况
- 数据不满足正态性假设
- 存在异常值
- 需要使用非传统统计量
- 验证参数检验结果的稳健性

### 实践建议

1. **置换次数选择**：通常使用10000次置换以获得可靠结果
2. **并行计算**：对于大数据集，使用并行计算加速
3. **统计量选择**：根据问题选择合适的统计量（均值、中位数等）
4. **结果验证**：可以与传统检验结果进行比较验证
5. **报告详细**：报告置换次数、p值、置信区间等

### 注意事项

1. 置换检验的计算成本较高，对于大数据集需要考虑效率
2. p值是近似的，依赖于置换次数
3. 需要满足可交换性假设
4. 对于某些复杂设计，可能需要特定的置换方案

置换检验为统计推断提供了一个无需分布假设的替代方案，在现代数据分析中越来越重要。随着计算能力的提升，置换检验的应用范围和实用性都在不断扩大。
