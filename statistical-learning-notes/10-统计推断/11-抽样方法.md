# 抽样方法（Sampling Methods）

## 定义

抽样方法研究如何从总体中抽取样本的技术和方法。它是统计推断的基础，直接影响统计分析和研究结果的可靠性和有效性。

## 基本概念

### 总体与样本

- **总体（Population）：** 研究对象的全体，通常用N表示总体大小
- **样本（Sample）：** 从总体中抽取的部分个体，通常用n表示样本大小
- **抽样框（Sampling Frame）：** 包含所有抽样单位的名单或列表

### 抽样目标

1. **代表性：** 样本应能代表总体的特征
2. **准确性：** 尽量减少抽样误差
3. **效率性：** 在给定成本下获得最大精度
4. **可行性：** 考虑实际操作的限制

## 概率抽样方法

### 1. 简单随机抽样（Simple Random Sampling）

**定义：** 总体中每个个体被抽中的概率相等

**特点：**
- 每个样本被抽中的概率相同
- 适用于总体分布相对均匀的情况
- 是最基本的抽样方法

**抽样概率：**
$$P(某个个体被抽中) = \frac{n}{N}$$

**优点：**
- 操作简单，容易理解
- 无偏估计
- 方差计算简单

**缺点：**
- 需要完整的抽样框
- 当总体内部差异大时，可能精度较低
- 抽样单位分散，成本较高

**适用场景：**
- 总体规模较小
- 总体内部差异不大
- 有完整的抽样框

**Python实现：**
```python
import numpy as np
import pandas as pd
from scipy import stats

# 简单随机抽样
population = np.arange(1, 101)  # 总体：1到100
sample_size = 20
sample = np.random.choice(population, size=sample_size, replace=False)
print(f"简单随机抽样样本: {sample}")

# 使用pandas
df = pd.DataFrame({'value': np.random.normal(100, 15, 1000)})
simple_random_sample = df.sample(n=100, random_state=42)
print(f"样本均值: {simple_random_sample['value'].mean():.2f}")
print(f"总体均值: {df['value'].mean():.2f}")
```

### 2. 分层抽样（Stratified Sampling）

**定义：** 将总体按某种特征分成若干层，然后从每层中进行独立抽样

**步骤：**
1. 根据相关变量将总体分层
2. 确定各层的样本量
3. 在每层内进行抽样

**样本量分配方法：**

#### 比例分配（Proportional Allocation）
$$n_h = n \times \frac{N_h}{N}$$

其中：
- $n_h$：第h层的样本量
- $N_h$：第h层的总体量
- $N$：总体总量
- $n$：总样本量

#### 最优分配（Optimal Allocation）
$$n_h = n \times \frac{N_h \sigma_h}{\sum N_h \sigma_h}$$

其中$\sigma_h$是第h层的标准差

**优点：**
- 提高估计精度
- 能保证各层的代表性
- 便于层间比较

**缺点：**
- 需要分层信息
- 层内抽样框需要完整
- 分层变量选择很重要

**适用场景：**
- 总体有明显层次结构
- 层内差异小，层间差异大
- 需要各层的估计结果

**Python实现：**
```python
from sklearn.model_selection import train_test_split

# 分层抽样示例
data = pd.DataFrame({
    'group': ['A'] * 400 + ['B'] * 300 + ['C'] * 300,
    'value': np.concatenate([
        np.random.normal(100, 10, 400),
        np.random.normal(110, 15, 300),
        np.random.normal(95, 8, 300)
    ])
})

# 使用train_test_split进行分层抽样
stratified_sample, _ = train_test_split(
    data, 
    test_size=0.9,  # 保留10%作为样本
    stratify=data['group'],
    random_state=42
)

print("分层抽样各层样本量：")
print(stratified_sample['group'].value_counts())

# 手动实现分层抽样
def stratified_sample(df, stratify_col, sample_frac, random_state=None):
    """分层抽样函数"""
    samples = []
    for group in df[stratify_col].unique():
        group_data = df[df[stratify_col] == group]
        group_sample = group_data.sample(frac=sample_frac, random_state=random_state)
        samples.append(group_sample)
    return pd.concat(samples)

sample = stratified_sample(data, 'group', 0.1, random_state=42)
print(f"\n分层抽样样本量: {len(sample)}")
```

### 3. 系统抽样（Systematic Sampling）

**定义：** 按照一定间隔从总体中抽取样本

**步骤：**
1. 将总体单位按某种顺序排列
2. 计算抽样间隔：$k = \frac{N}{n}$
3. 在1到k中随机确定一个起始点
4. 从起始点开始，每隔k个单位抽取一个样本

**抽样公式：**
$$样本索引 = r + i \times k$$

其中：
- r：起始随机数（1 ≤ r ≤ k）
- k：抽样间隔
- i：第i个样本（i = 0, 1, 2, ..., n-1）

**优点：**
- 操作简便，易于实施
- 样本在总体中分布均匀
- 不需要完整的抽样框（仅需排列顺序）

**缺点：**
- 如果总体存在周期性，可能产生偏差
- 起始点选择影响样本代表性
- 方差估计较复杂

**适用场景：**
- 总体有自然顺序
- 总体无明显周期性
- 需要快速获得样本

**注意事项：**
- 避免抽样间隔与总体周期重合
- 检查总体是否存在隐藏模式

**Python实现：**
```python
def systematic_sampling(population, sample_size, random_state=None):
    """系统抽样"""
    if random_state:
        np.random.seed(random_state)
    
    N = len(population)
    k = N // sample_size  # 抽样间隔
    r = np.random.randint(1, k + 1)  # 随机起始点
    
    indices = [r + i * k - 1 for i in range(sample_size) if r + i * k <= N]
    sample = [population[i] for i in indices]
    
    return sample, k, r

# 使用示例
population = list(range(1, 101))
sample, k, r = systematic_sampling(population, sample_size=10, random_state=42)
print(f"系统抽样间隔 k={k}, 起始点 r={r}")
print(f"样本: {sample}")

# 使用pandas
df = pd.DataFrame({'id': range(1, 1001), 'value': np.random.randn(1000)})
def pandas_systematic_sampling(df, sample_size, random_state=None):
    """Pandas系统抽样"""
    if random_state:
        np.random.seed(random_state)
    
    N = len(df)
    k = N // sample_size
    r = np.random.randint(0, k)
    
    return df.iloc[r::k][:sample_size]

systematic_sample = pandas_systematic_sampling(df, 100, random_state=42)
print(f"\n系统抽样样本量: {len(systematic_sample)}")
```

### 4. 整群抽样（Cluster Sampling）

**定义：** 将总体分成若干群，随机抽取部分群，对抽中群的所有单位进行调查

**步骤：**
1. 将总体划分为若干群
2. 随机抽取部分群
3. 对抽中群的所有单位进行全面调查

**与分层抽样的区别：**
- 分层抽样：从每层都抽样
- 整群抽样：只对部分群调查

**优点：**
- 大幅降低抽样成本
- 实施方便，适合地理分散的总体
- 不需要每个单位的抽样框

**缺点：**
- 群内相似性高时，估计精度低
- 群间差异大时，抽样误差大
- 设计效应通常大于1

**设计效应（Deff）：**
$$Deff = 1 + (m-1)\rho$$

其中：
- m：群内平均单位数
- $\rho$：群内相关系数

**适用场景：**
- 总体地理分布广泛
- 群内差异大，群间差异小
- 成本考虑是主要因素

**Python实现：**
```python
def cluster_sampling(data, cluster_col, num_clusters, random_state=None):
    """整群抽样"""
    if random_state:
        np.random.seed(random_state)
    
    clusters = data[cluster_col].unique()
    selected_clusters = np.random.choice(clusters, size=num_clusters, replace=False)
    
    sample = data[data[cluster_col].isin(selected_clusters)]
    
    return sample, selected_clusters

# 使用示例
data = pd.DataFrame({
    'cluster': np.repeat(['A', 'B', 'C', 'D', 'E'], 200),
    'value': np.concatenate([
        np.random.normal(100, 10, 200),
        np.random.normal(105, 12, 200),
        np.random.normal(98, 9, 200),
        np.random.normal(102, 11, 200),
        np.random.normal(99, 8, 200)
    ])
})

sample, selected_clusters = cluster_sampling(data, 'cluster', num_clusters=2, random_state=42)
print(f"选中的群: {selected_clusters}")
print(f"整群抽样样本量: {len(sample)}")
print(f"样本均值: {sample['value'].mean():.2f}")
print(f"总体均值: {data['value'].mean():.2f}")
```

### 5. 多阶段抽样（Multi-stage Sampling）

**定义：** 分多个阶段进行抽样，先抽大单位，再抽小单位，依此类推

**常见形式：**

#### 两阶段抽样（Two-stage Sampling）
1. 第一阶段：抽取初级抽样单位（PSU）
2. 第二阶段：在每个PSU中抽取次级抽样单位（SSU）

#### 三阶段抽样（Three-stage Sampling）
1. 第一阶段：抽取省份
2. 第二阶段：在省份中抽取城市
3. 第三阶段：在城市中抽取家庭

**优点：**
- 灵活性高，适应复杂情况
- 平衡成本与精度
- 不需要完整的底层抽样框

**缺点：**
- 方差估计复杂
- 需要每阶段的抽样框
- 设计可能较复杂

**适用场景：**
- 大规模、多层次的总体
- 需要兼顾成本和精度
- 总体结构复杂

**Python实现：**
```python
def two_stage_sampling(data, stage1_col, stage1_units, stage2_size, random_state=None):
    """两阶段抽样"""
    if random_state:
        np.random.seed(random_state)
    
    # 第一阶段：抽取初级单位
    stage1_units_list = data[stage1_col].unique()
    selected_stage1 = np.random.choice(stage1_units_list, size=stage1_units, replace=False)
    
    # 第二阶段：在选中的初级单位中抽取次级单位
    sample_parts = []
    for unit in selected_stage1:
        unit_data = data[data[stage1_col] == unit]
        sample_part = unit_data.sample(n=min(stage2_size, len(unit_data)), random_state=random_state)
        sample_parts.append(sample_part)
    
    final_sample = pd.concat(sample_parts)
    
    return final_sample, selected_stage1

# 使用示例
data = pd.DataFrame({
    'province': np.repeat(['北京', '上海', '广东', '江苏'], 250),
    'city': np.repeat(['A', 'B', 'C', 'D'], 250),
    'value': np.random.randn(1000) * 10 + 100
})

# 假设city是初级抽样单位
sample, selected_cities = two_stage_sampling(
    data, 
    stage1_col='city', 
    stage1_units=2,  # 抽取2个城市
    stage2_size=50,  # 每个城市抽取50个样本
    random_state=42
)

print(f"选中的城市: {selected_cities}")
print(f"两阶段抽样样本量: {len(sample)}")
print(f"样本均值: {sample['value'].mean():.2f}")
```

## 非概率抽样方法

### 1. 方便抽样（Convenience Sampling）

**定义：** 选择最容易获得的样本

**特点：**
- 成本低，速度快
- 代表性差
- 存在严重选择偏差

**适用场景：**
- 探索性研究
- 试点研究
- 资源严重受限

### 2. 判断抽样（Judgmental Sampling）

**定义：** 根据研究者的判断选择样本

**特点：**
- 依赖专家知识
- 可能存在主观偏差
- 适用于特殊群体研究

### 3. 配额抽样（Quota Sampling）

**定义：** 按照一定配额从各子总体中选择样本

**特点：**
- 类似分层抽样
- 但非随机选择
- 快速获得结构化样本

### 4. 滚雪球抽样（Snowball Sampling）

**定义：** 通过已有样本推荐获得新样本

**特点：**
- 适用于难以接触的群体
- 样本间存在相关性
- 难以控制样本构成

## 抽样误差与非抽样误差

### 抽样误差（Sampling Error）

**定义：** 由于抽样本身的随机性而产生的误差

**特点：**
- 不可避免
- 可通过增加样本量减小
- 可以测量和控制

**标准误（Standard Error）：**

简单随机抽样的均值标准误：
$$SE(\bar{x}) = \frac{s}{\sqrt{n}}\sqrt{1 - \frac{n}{N}}$$

其中$\sqrt{1 - \frac{n}{N}}$是有限总体校正系数（FPC）

**置信区间：**
$$\bar{x} \pm t_{\alpha/2} \times SE(\bar{x})$$

### 非抽样误差（Non-sampling Error）

**定义：** 除抽样误差外的所有误差

**类型：**

#### 1. 覆盖误差（Coverage Error）
- 抽样框不完整
- 总体单位遗漏或重复

#### 2. 无响应误差（Non-response Error）
- 拒绝回答
- 无法联系
- 数据缺失

#### 3. 测量误差（Measurement Error）
- 问卷设计问题
- 访谈者偏差
- 记录错误

#### 4. 处理误差（Processing Error）
- 数据录入错误
- 编码错误
- 计算错误

**控制方法：**
- 完善抽样框
- 提高响应率
- 培训调查人员
- 数据质量检查
- 加权调整

## 样本量确定

### 简单随机抽样的样本量

#### 估计总体均值
$$n = \frac{Z^2 \sigma^2}{e^2}$$

其中：
- Z：标准正态分布的临界值
- $\sigma$：总体标准差（可用样本标准差估计）
- e：允许误差

#### 估计总体比例
$$n = \frac{Z^2 p(1-p)}{e^2}$$

其中p是总体比例的估计值

### 考虑有限总体
$$n' = \frac{n}{1 + \frac{n-1}{N}}$$

其中N是总体大小

### 考虑设计效应
$$n_{adjusted} = n \times Deff$$

**Python实现：**
```python
from scipy import stats

def sample_size_mean(population_std, margin_of_error, confidence_level=0.95, population_size=None):
    """计算估计总体均值所需的样本量"""
    alpha = 1 - confidence_level
    z = stats.norm.ppf(1 - alpha/2)
    
    n = (z * population_std / margin_of_error) ** 2
    n = np.ceil(n)
    
    # 有限总体校正
    if population_size is not None:
        n_corrected = n / (1 + (n - 1) / population_size)
        n = np.ceil(n_corrected)
    
    return int(n)

def sample_size_proportion(estimated_proportion, margin_of_error, confidence_level=0.95, population_size=None):
    """计算估计总体比例所需的样本量"""
    alpha = 1 - confidence_level
    z = stats.norm.ppf(1 - alpha/2)
    
    p = estimated_proportion
    n = (z * np.sqrt(p * (1 - p)) / margin_of_error) ** 2
    n = np.ceil(n)
    
    # 有限总体校正
    if population_size is not None:
        n_corrected = n / (1 + (n - 1) / population_size)
        n = np.ceil(n_corrected)
    
    return int(n)

# 使用示例
print("估计总体均值的样本量：")
n_mean = sample_size_mean(population_std=10, margin_of_error=2, confidence_level=0.95)
print(f"标准差=10, 允许误差=2, 置信水平=95%: {n_mean}")

print("\n估计总体比例的样本量：")
n_proportion = sample_size_proportion(estimated_proportion=0.5, margin_of_error=0.05, confidence_level=0.95)
print(f"比例=0.5, 允许误差=0.05, 置信水平=95%: {n_proportion}")

# 考虑有限总体
n_finite = sample_size_mean(population_std=10, margin_of_error=2, confidence_level=0.95, population_size=500)
print(f"\n考虑有限总体(N=500): {n_finite}")
```

## 抽样方法比较

| 方法 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 简单随机抽样 | 无偏、简单、易分析 | 需要完整抽样框、成本高 | 总体小、分布均匀 |
| 分层抽样 | 精度高、保证代表性 | 需要分层信息 | 总体有层次结构 |
| 系统抽样 | 简便、分布均匀 | 周期性风险 | 有自然顺序 |
| 整群抽样 | 成本低、易实施 | 精度可能低 | 地理分散总体 |
| 多阶段抽样 | 灵活、平衡成本精度 | 复杂、方差难估计 | 大规模复杂总体 |

    print(f"  收入误差: {income_error:.2f}")
    print(f"  年龄误差: {age_error:.2f}")

# 计算置信区间
def calculate_confidence_interval(sample, column, confidence_level=0.95):
    """计算置信区间"""
    mean = sample[column].mean()
    std = sample[column].std()
    n = len(sample)
    se = std / np.sqrt(n)
    
    alpha = 1 - confidence_level
    t_critical = stats.t.ppf(1 - alpha/2, df=n-1)
    
    margin_of_error = t_critical * se
    ci_lower = mean - margin_of_error
    ci_upper = mean + margin_of_error
    
    return ci_lower, ci_upper, margin_of_error

print(f"\n=== 简单随机抽样收入均值的95%置信区间 ===")
ci_lower, ci_upper, moe = calculate_confidence_interval(simple_sample, 'income')
print(f"置信区间: [{ci_lower:.2f}, {ci_upper:.2f}]")
print(f"总体真实均值: {data['income'].mean():.2f}")
print(f"区间是否包含总体均值: {ci_lower <= data['income'].mean() <= ci_upper}")

## 最佳实践

### 抽样设计原则

1. **明确研究目标：** 清楚定义需要估计的参数
2. **了解总体特征：** 收集总体相关信息
3. **选择合适方法：** 根据研究目的和总体特征选择
4. **平衡成本与精度：** 在资源约束下优化设计
5. **控制非抽样误差：** 重视数据质量控制

### 抽样实施要点

1. **准备抽样框：** 确保抽样框的完整性和准确性
2. **制定详细计划：** 明确抽样流程和质量控制
3. **培训调查人员：** 提高数据收集质量
4. **监控抽样过程：** 及时发现和解决问题
5. **记录抽样信息：** 便于后续分析和复现

### 数据分析注意事项

1. **考虑抽样设计：** 使用加权方法调整
2. **计算正确的标准误：** 考虑设计效应
3. **检验假设条件：** 验证统计方法的适用性

## 总结

抽样方法是统计学的核心内容，正确的抽样设计是获得可靠统计推断的基础。选择合适的抽样方法需要综合考虑研究目的、总体特征、成本约束和精度要求。

**关键要点：**
- 概率抽样提供无偏估计，非概率抽样适用于特殊场景
- 分层抽样通常能提高估计精度
- 整群抽样和多阶段抽样降低实施成本
- 抽样误差不可避免但可控制，非抽样误差需要严格管理
- 样本量确定需考虑置信水平、允许误差和总体变异

掌握抽样方法的理论和实践，对于进行高质量的统计研究和数据分析至关重要。
