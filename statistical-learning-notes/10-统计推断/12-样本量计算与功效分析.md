# 样本量计算与功效分析

## 定义

样本量计算与功效分析是研究设计中的关键步骤，用于确定研究需要的样本量，以确保研究能够检测到预期的效应大小。功效分析帮助我们平衡统计功效、样本量、效应大小和显著性水平之间的关系。

## 第一类错误和第二类错误

### 第一类错误（Type I Error）

- **定义：** 原假设H₀为真时，错误地拒绝H₀
- **概率：** α（显著性水平）
- **解释：** "假阳性"、"虚惊"
- **常见设置：** α = 0.05（表示容忍5%的假阳性率）

### 第二类错误（Type II Error）

- **定义：** 原假设H₀为假时，错误地不拒绝H₀
- **概率：** β
- **解释：** "假阴性"、"漏报"
- **常见设置：** β = 0.20（对应80%的功效）

### 统计功效（Statistical Power）

- **定义：** 当H₀为假时，正确拒绝H₀的概率
- **公式：** Power = 1 - β
- **意义：** 检测真实效应的能力
- **常见设置：** Power = 0.80（80%的功效）

### 决策矩阵

| 真实情况 | 不拒绝H₀ | 拒绝H₀ |
|----------|----------|--------|
| H₀为真 | 正确决策（1-α） | 第一类错误（α） |
| H₀为假 | 第二类错误（β） | 正确决策（1-β=Power） |

## 功效分析的四要素

功效分析涉及四个相互关联的要素，给定其中三个，可以计算第四个：

### 1. 效应大小（Effect Size）

**定义：** 统计检验中感兴趣的效应的量级。

**常见效应量：**

- **Cohen's d（t检验）：**
  $$d = \frac{\mu_1 - \mu_2}{\sigma}$$
  - 小效应：d = 0.2
  - 中等效应：d = 0.5
  - 大效应：d = 0.8

- **相关系数r：**
  - 小效应：r = 0.1
  - 中等效应：r = 0.3
  - 大效应：r = 0.5

- **Odds Ratio（OR）：**
  $$OR = \frac{p_1/(1-p_1)}{p_2/(1-p_2)}$$

### 2. 样本量（Sample Size）

**定义：** 研究中需要的观测数量。

**影响因素：**
- 效应大小：效应越小，需要的样本量越大
- 功效：功效要求越高，需要的样本量越大
- 显著性水平：α越小，需要的样本量越大
- 检验类型：不同检验需要不同的样本量

### 3. 显著性水平（Significance Level）

**定义：** 拒绝H₀的概率阈值。

**常见值：**
- α = 0.05（最常用）
- α = 0.01（更严格）
- α = 0.10（探索性研究）

**注意：** α和功效之间是权衡关系。

### 4. 统计功效（Power）

**定义：** 检测到真实效应的概率。

**常见设置：**
- Power = 0.80（80%，最常用）
- Power = 0.90（90%，更严格）
- Power = 0.95（95%，非常严格）

### 四要素之间的关系

$$Power = f(n, \alpha, effect\_size)$$

$$n = f(Power, \alpha, effect\_size)$$

$$effect\_size = f(n, Power, \alpha)$$

$$\alpha = f(n, Power, effect\_size)$$

## 常用检验的功效分析

### 1. t检验的功效分析

#### 单样本t检验

$$t = \frac{\bar{x} - \mu_0}{s/\sqrt{n}}$$

#### 双样本t检验

$$t = \frac{\bar{x}_1 - \bar{x}_2}{s_p\sqrt{1/n_1 + 1/n_2}}$$

其中 $s_p$ 是合并标准差。

#### 样本量公式（近似）

对于双样本t检验：

$$n = \frac{2(z_{\alpha/2} + z_{\beta})^2\sigma^2}{(\mu_1 - \mu_2)^2}$$

其中：
- $z_{\alpha/2}$ 是标准正态分布的α/2临界值
- $z_{\beta}$ 是功效对应的临界值
- σ是标准差

### 2. ANOVA的功效分析

#### 单因素ANOVA

$$F = \frac{MS_{between}}{MS_{within}}$$

#### 效应量：f（Cohen's f）

$$f = \sqrt{\frac{\sum_{i=1}^{k}(\mu_i - \mu)^2/n}{\sigma^2}}$$

其中：
- $\mu_i$ 是第i组的均值
- $\mu$ 是总体均值
- k是组数

#### f的解释
- 小效应：f = 0.1
- 中等效应：f = 0.25
- 大效应：f = 0.4

### 3. 相关检验的功效分析

#### Pearson相关检验

$$t = \frac{r\sqrt{n-2}}{\sqrt{1-r^2}}$$

#### 样本量公式（近似）

$$n = \left(\frac{z_{\alpha/2} + z_{\beta}}{0.5 \ln\left(\frac{1+r}{1-r}\right)}\right)^2 + 3$$

其中r是期望的相关系数。

### 4. 比例检验的功效分析

#### 单样本比例检验

$$z = \frac{\hat{p} - p_0}{\sqrt{p_0(1-p_0)/n}}$$

#### 双样本比例检验

$$z = \frac{\hat{p}_1 - \hat{p}_2}{\sqrt{\hat{p}(1-\hat{p})(1/n_1 + 1/n_2)}}$$

#### 样本量公式（双样本）

$$n = \frac{(z_{\alpha/2}\sqrt{2\bar{p}(1-\bar{p})} + z_{\beta}\sqrt{p_1(1-p_1) + p_2(1-p_2)})^2}{(p_1 - p_2)^2}$$

其中 $\bar{p} = (p_1 + p_2)/2$

### 5. 卡方检验的功效分析

#### 卡方检验

$$\chi^2 = \sum_{i=1}^{r}\sum_{j=1}^{c}\frac{(O_{ij} - E_{ij})^2}{E_{ij}}$$

#### 效应量：w

$$w = \sqrt{\sum_{i=1}^{r}\sum_{j=1}^{c}\frac{(P_{ij} - P_{0ij})^2}{P_{0ij}}}$$

其中：
- $P_{ij}$ 是备择假设下的期望概率
- $P_{0ij}$ 是原假设下的期望概率

## 样本量计算方法

### 1. 理论公式法

使用统计公式计算样本量。

**优点：**
- 计算快速
- 不需要软件

**缺点：**
- 假设条件严格
- 只适用于简单设计

### 2. 软件计算法

使用统计软件进行样本量计算。

**常用软件：**
- **G*Power：** 免费开源，界面友好
- **R：** `pwr`包
- **Python：** `statsmodels.stats.power`
- **SAS：** `PROC POWER`
- **PASS：** 商业软件

### 3. 模拟法

使用蒙特卡洛模拟估计样本量。

**优点：**
- 适用于复杂设计
- 可以考虑违反假设的情况
- 灵活性高

**缺点：**
- 计算耗时
- 需要编程能力

### 4. 实证数据法

基于先前研究的数据估计样本量。

**方法：**
- 查阅相关文献
- 使用元分析结果
- 进行预实验

```python
import numpy as np
from scipy import stats
from statsmodels.stats.power import TTestIndPower, TTestPower, NormalIndPower
import matplotlib.pyplot as plt

# 1. 双样本t检验的功效分析
def ttest_two_sample_power(effect_size, n1, n2=None, alpha=0.05):
    """
    计算双样本t检验的功效
    
    参数:
        effect_size: Cohen's d
        n1: 第一组样本量
        n2: 第二组样本量（如果为None，则n2=n1）
        alpha: 显著性水平
    
    返回:
        power: 统计功效
    """
    if n2 is None:
        n2 = n1
    
    # 计算功效
    power_analysis = TTestIndPower()
    power = power_analysis.power(effect_size, nobs1=n1, alpha=alpha, ratio=n2/n1)
    
    return power

# 示例：中等效应（d=0.5），每组50人
effect_size = 0.5
n = 50
power = ttest_two_sample_power(effect_size, n)
print(f"双样本t检验功效: d={effect_size}, n={n}, Power={power:.3f}")

# 2. 计算需要的样本量
def ttest_two_sample_sample_size(effect_size, power=0.8, alpha=0.05, ratio=1.0):
    """
    计算双样本t检验需要的样本量
    
    参数:
        effect_size: Cohen's d
        power: 期望的功效
        alpha: 显著性水平
        ratio: n2/n1的比值
    
    返回:
        n1: 第一组需要的样本量
        n2: 第二组需要的样本量
    """
    power_analysis = TTestIndPower()
    n1 = power_analysis.solve_power(effect_size=effect_size, power=power, 
                                     alpha=alpha, ratio=ratio)
    n2 = n1 * ratio
    
    return int(np.ceil(n1)), int(np.ceil(n2))

# 示例：检测中等效应（d=0.5），期望80%功效
effect_size = 0.5
n1, n2 = ttest_two_sample_sample_size(effect_size, power=0.8)
print(f"需要样本量: n1={n1}, n2={n2}, 总计={n1+n2}")

# 3. 不同效应大小需要的样本量
print("\n不同效应大小需要的样本量（Power=0.8, alpha=0.05）:")
effect_sizes = [0.2, 0.5, 0.8]  # 小、中、大效应
for es in effect_sizes:
    n1, n2 = ttest_two_sample_sample_size(es, power=0.8)
    print(f"效应大小 d={es}: n1={n1}, n2={n2}")
```

```python
# 绘制功效曲线
def plot_power_curve(effect_size, alpha=0.05, max_n=200):
    """
    绘制功效曲线
    
    参数:
        effect_size: Cohen's d
        alpha: 显著性水平
        max_n: 最大样本量
    """
    power_analysis = TTestIndPower()
    sample_sizes = np.arange(10, max_n + 1, 5)
    powers = power_analysis.power(effect_size, nobs1=sample_sizes, alpha=alpha)
    
    plt.figure(figsize=(10, 6))
    plt.plot(sample_sizes, powers, 'b-', linewidth=2, label=f'效应大小 d={effect_size}')
    plt.axhline(y=0.8, color='r', linestyle='--', label='目标功效=0.8')
    plt.axvline(x=sample_sizes[np.argmin(np.abs(powers - 0.8))], 
                color='g', linestyle=':', label=f'需要样本量={sample_sizes[np.argmin(np.abs(powers - 0.8))]}')
    
    plt.xlabel('每组样本量', fontsize=12)
    plt.ylabel('统计功效', fontsize=12)
    plt.title(f'双样本t检验功效曲线（α={alpha}）', fontsize=14)
    plt.legend(fontsize=10)
    plt.grid(True, alpha=0.3)
    plt.ylim([0, 1])
    plt.show()

# 绘制不同效应大小的功效曲线
def plot_multiple_power_curves(effect_sizes, alpha=0.05, max_n=200):

        alpha: 显著性水平
```

```python
from statsmodels.stats.power import FTestAnovaPower

# ANOVA功效分析
def anova_power(f_effect_size, n_groups, n_per_group, alpha=0.05):
    """
    计算单因素ANOVA的功效
    
    参数:
        f_effect_size: Cohen's f效应大小
        n_groups: 组数
        n_per_group: 每组样本量
        alpha: 显著性水平
    
    返回:
        power: 统计功效
    """
    power_analysis = FTestAnovaPower()
    n_total = n_groups * n_per_group
    power = power_analysis.power(f_effect_size, nobs=n_total, alpha=alpha, 
                                 k_groups=n_groups)
    return power

# 示例：3组，中等效应（f=0.25），每组30人
f_effect_size = 0.25
n_groups = 3
n_per_group = 30
power = anova_power(f_effect_size, n_groups, n_per_group)
print(f"ANOVA功效: f={f_effect_size}, 每组{n_per_group}人, Power={power:.3f}")

# 计算需要的样本量
def anova_sample_size(f_effect_size, n_groups, power=0.8, alpha=0.05):
    """
    计算ANOVA需要的样本量
    
    参数:
        f_effect_size: Cohen's f效应大小
        n_groups: 组数
        power: 期望的功效
        alpha: 显著性水平
    
    返回:
        n_per_group: 每组需要的样本量
        n_total: 总样本量
    """
    power_analysis = FTestAnovaPower()
    n_total = power_analysis.solve_power(f_effect_size, power=power, alpha=alpha, 
                                         k_groups=n_groups)
    n_per_group = int(np.ceil(n_total / n_groups))
    
    return n_per_group, int(np.ceil(n_total))

# 示例：3组，中等效应（f=0.25），期望80%功效
f_effect_size = 0.25
n_groups = 3
n_per_group, n_total = anova_sample_size(f_effect_size, n_groups, power=0.8)
print(f"需要样本量: 每组{n_per_group}人, 总计{n_total}人")
```

```python
# 相关系数的功效分析
def correlation_power(r, n, alpha=0.05):
    """
    计算相关检验的功效
    
    参数:
        r: 相关系数
        n: 样本量
        alpha: 显著性水平
    
    返回:
        power: 统计功效
    """
    # 将相关系数转换为Fisher z
    z = np.arctanh(r)
    
    # 计算功效
    power_analysis = NormalIndPower()
    effect_size = z
    power = power_analysis.power(effect_size, nobs1=n, alpha=alpha)
    
    return power

# 计算需要的样本量
def correlation_sample_size(r, power=0.8, alpha=0.05):
    """
    计算相关检验需要的样本量
    
    参数:
        r: 期望的相关系数
        power: 期望的功效
        alpha: 显著性水平
    
    返回:
        n: 需要的样本量
    """
    # 将相关系数转换为Fisher z
    z = np.arctanh(r)
    
    # 计算样本量
    power_analysis = NormalIndPower()
    effect_size = z
    n = power_analysis.solve_power(effect_size=effect_size, power=power, alpha=alpha)
    
    return int(np.ceil(n))
```

```python
from statsmodels.stats.power import NormalIndPower, zt_ind_solve_power

# 双样本比例检验的功效分析
def proportion_test_power(p1, p2, n1, n2=None, alpha=0.05):
    """
    计算双样本比例检验的功效
    
    参数:
        p1: 第一组比例
        p2: 第二组比例
        n1: 第一组样本量
        n2: 第二组样本量（如果为None，则n2=n1）
        alpha: 显著性水平
    
    返回:
        power: 统计功效
    """
    if n2 is None:
        n2 = n1
    
    # 计算效应大小（Cohen's h）
    # h = 2 * arcsin(sqrt(p1)) - 2 * arcsin(sqrt(p2))
    from math import asin, sqrt
    h = 2 * (asin(sqrt(p1)) - asin(sqrt(p2)))
    
    # 计算功效
    power_analysis = NormalIndPower()
    power = power_analysis.power(h, nobs1=n1, alpha=alpha, ratio=n2/n1)
    
    return power

# 计算需要的样本量
def proportion_test_sample_size(p1, p2, power=0.8, alpha=0.05, ratio=1.0):
    """
    计算双样本比例检验需要的样本量
    
    参数:
        p1: 第一组比例
        p2: 第二组比例
        power: 期望的功效
        alpha: 显著性水平
        ratio: n2/n1的比值
    
    返回:
        n1: 第一组需要的样本量
        n2: 第二组需要的样本量
    """
    # 计算效应大小（Cohen's h）
    from math import asin, sqrt
    h = 2 * (asin(sqrt(p1)) - asin(sqrt(p2)))
    
    # 计算样本量
    power_analysis = NormalIndPower()
    n1 = power_analysis.solve_power(effect_size=h, power=power, 
                                     alpha=alpha, ratio=ratio)
    n2 = n1 * ratio
    
    return int(np.ceil(n1)), int(np.ceil(n2))

# 示例：对照组50%响应率，实验组60%响应率
p1 = 0.50  # 对照组
p2 = 0.60  # 实验组
n1, n2 = proportion_test_sample_size(p1, p2, power=0.8)
print(f"比例检验需要样本量: n1={n1}, n2={n2}, 总计={n1+n2}")

# 不同比例差异需要的样本量
print("\n不同比例差异需要的样本量（Power=0.8, alpha=0.05）:")
p1 = 0.50
for delta in [0.05, 0.10, 0.15, 0.20]:
    p2 = p1 + delta
    n1, n2 = proportion_test_sample_size(p1, p2, power=0.8)
    print(f"比例差异 {delta:.2f}: n1={n1}, n2={n2}")
```

### 6. 功效分析函数总结

```python
# 创建一个综合的功效分析工具
class PowerAnalyzer:
    """功效分析综合工具"""
    
    def __init__(self):
        self.ttest_power = TTestIndPower()
        self.anova_power = FTestAnovaPower()
        self.normal_power = NormalIndPower()
    
    def analyze_ttest(self, effect_size, n, alpha=0.05, test_type='two-sample'):
        """t检验功效分析"""
        if test_type == 'two-sample':
            power = self.ttest_power.power(effect_size, nobs1=n, alpha=alpha)
            required_n = self.ttest_power.solve_power(effect_size=effect_size, 
                                                     power=0.8, alpha=alpha)
            required_n = int(np.ceil(required_n))
        elif test_type == 'one-sample':
            power_analysis = TTestPower()
            power = power_analysis.power(effect_size, nobs=n, alpha=alpha)
            required_n = power_analysis.solve_power(effect_size=effect_size, 
                                                   power=0.8, alpha=alpha)
            required_n = int(np.ceil(required_n))
        
        return {
            'power': power,
            'required_n': required_n,
            'effect_size': effect_size,
            'n': n,
            'alpha': alpha
        }
    
    def analyze_anova(self, f_effect_size, n_groups, n_per_group, alpha=0.05):
        """ANOVA功效分析"""
        n_total = n_groups * n_per_group
        power = self.anova_power.power(f_effect_size, nobs=n_total, 
                                       alpha=alpha, k_groups=n_groups)
        required_total = self.anova_power.solve_power(f_effect_size, 
                                                       power=0.8, alpha=alpha, 
                                                       k_groups=n_groups)
        required_per_group = int(np.ceil(required_total / n_groups))
        
        return {
            'power': power,
            'required_n_per_group': required_per_group,
            'required_n_total': int(np.ceil(required_total)),
            'effect_size': f_effect_size,
            'n_per_group': n_per_group,
            'n_groups': n_groups,
            'alpha': alpha
        }
    
    def analyze_correlation(self, r, n, alpha=0.05):
        """相关检验功效分析"""
        z = np.arctanh(r)
        power = self.normal_power.power(z, nobs1=n, alpha=alpha)
        required_n = self.normal_power.solve_power(effect_size=z, 
                                                   power=0.8, alpha=alpha)
        required_n = int(np.ceil(required_n))
        
        return {
            'power': power,
            'required_n': required_n,
            'correlation': r,
            'n': n,
            'alpha': alpha
        }

# 使用示例
analyzer = PowerAnalyzer()

# t检验分析
result = analyzer.analyze_ttest(effect_size=0.5, n=50)
```

## 功效分析的注意事项

### 1. 效应大小的选择

**问题：** 如何确定期望的效应大小？

**解决方法：**
- 查阅文献
- 进行预实验
- 使用保守估计（小效应）
- 考虑实际意义

### 2. 样本量与成本

**权衡：**
- 样本量越大，功效越高
- 但成本也越高

**策略：**
- 计算多个功效水平下的样本量
- 评估成本效益
- 考虑分阶段研究

### 3. 假设违反

**常见问题：**
- 非正态数据
- 方差不齐
- 缺失数据

**解决方法：**
- 使用非参数检验的功效分析
- 进行敏感性分析
- 考虑数据转换

### 4. 多重比较

**问题：** 进行多次检验时，第一类错误率增加。

**解决方法：**
- 使用更严格的α值
- 调整功效要求
- 考虑多重比较校正

### 5. 纵向研究

**特点：**
- 重复测量
- 相关观测
- 需要考虑相关性

**方法：**
- 使用重复测量的功效分析
- 考虑失访率（通常增加20-30%）

### 1. 蒙特卡洛模拟

当理论公式不适用时，可以使用模拟估计功效。

```python
# 蒙特卡洛模拟示例
def monte_carlo_power(effect_size, n, n_sim=10000, alpha=0.05):
    """
    使用蒙特卡洛模拟估计功效
    
    参数:
        effect_size: Cohen's d
        n: 每组样本量
        n_sim: 模拟次数
        alpha: 显著性水平
    
    返回:
        estimated_power: 估计的功效
    """
    significant_count = 0
    
    for _ in range(n_sim):
        # 生成数据
        group1 = np.random.normal(0, 1, n)
        group2 = np.random.normal(effect_size, 1, n)
        
        # 进行t检验
        _, p_value = stats.ttest_ind(group1, group2)
        
        # 判断是否显著
        if p_value < alpha:
            significant_count += 1
    
    estimated_power = significant_count / n_sim
    return estimated_power

# 示例
effect_size = 0.5
n = 50
power = monte_carlo_power(effect_size, n)
print(f"蒙特卡洛模拟估计功效: {power:.3f}")
```

### 2. 生存分析的功效分析

```python
from statsmodels.stats.power import NormalIndPower

# 生存分析需要考虑随访时间和事件发生率
def survival_analysis_power(hr, n, event_rate=0.5, alpha=0.05):
    """
    生存分析功效分析
    
    参数:
        hr: 风险比（Hazard Ratio）
        n: 总样本量
        event_rate: 事件发生率
        alpha: 显著性水平
    
    返回:
        power: 统计功效
    """
    # 将风险比转换为效应大小
    # 简化：使用log(hazard_ratio)
    effect_size = np.log(hr)
    
    # 调整样本量（考虑事件率）
    n_effective = n * event_rate
    
    # 计算功效
    power_analysis = NormalIndPower()
    power = power_analysis.power(effect_size, nobs1=n_effective, alpha=alpha)
    
    return power

hr = 0.7
```

### 3. 嵌套设计

对于嵌套数据结构（如学生嵌套在班级中），需要考虑组内相关系数（ICC）。

```python
def cluster_randomized_power(effect_size, n_clusters, n_per_cluster, icc, alpha=0.05):
    """
    整群随机化试验功效分析
    
    参数:
        effect_size: Cohen's d
        n_clusters: 整群数量
        n_per_cluster: 每个整群的个体数
        icc: 组内相关系数
        alpha: 显著性水平
    
    返回:
        power: 统计功效
    """
    # 计算设计效应
    deff = 1 + (n_per_cluster - 1) * icc
    
    # 计算有效样本量
    effective_n = n_clusters * n_per_cluster / deff
    
    # 计算功效
    power_analysis = TTestIndPower()
    power = power_analysis.power(effect_size, nobs1=effective_n/2, alpha=alpha)
    
    return power

# 示例：ICC=0.05，20个班级，每班30人
effect_size = 0.5
n_clusters = 20
n_per_cluster = 30
icc = 0.05
power = cluster_randomized_power(effect_size, n_clusters, n_per_cluster, icc)
print(f"整群随机化功效: {power:.3f}")
```

## 总结

样本量计算与功效分析是研究设计中的关键步骤。

**核心概念：**
1. **第一类错误（α）：** 假阳性率，通常设为0.05
2. **第二类错误（β）：** 假阴性率
3. **统计功效（1-β）：** 检测真实效应的能力，通常要求0.80
4. **效应大小：** 感兴趣的效应量级

**功效分析四要素：**
- 效应大小
- 样本量
- 显著性水平
- 统计功效

**关键要点：**
1. 给定三个要素，可以计算第四个
2. 效应越小，需要的样本量越大
3. 功效要求越高，需要的样本量越大
4. α越小，需要的样本量越大

**实际应用：**
1. 在研究开始前进行功效分析
2. 考虑实际意义和成本
3. 使用适当的软件进行计算
4. 考虑研究设计的复杂性

**最佳实践：**
1. 基于文献或预实验估计效应大小
2. 选择适当的功效水平（通常0.80）
3. 考虑失访率（增加20-30%样本）
4. 进行敏感性分析
5. 报告功效分析的假设和结果

理解样本量计算与功效分析的原理，对于设计高效、可行的研究至关重要。这有助于避免资源浪费，提高研究的科学价值。
