# Z检验

## 问题

统计推断部分缺少Z检验的学习笔记，但已有t检验的笔记。Z检验是统计学中重要的假设检验方法，应该在笔记中涵盖。

## 说明

Z检验用于总体标准差已知或样本量较大（通常n > 30）的情况，是t检验的特例。

## Z检验的定义和应用场景

Z检验是一种基于正态分布的假设检验方法，用于检验样本统计量与总体参数之间的差异。Z检验适用于以下情况：

1. **总体标准差已知**：当知道总体的标准差σ时
2. **大样本情况**：当样本量较大（通常n ≥ 30）时，根据中心极限定理，样本均值的抽样分布近似正态分布
3. **比例检验**：检验总体比例时，当np和n(1-p)都大于等于5时

## Z统计量的计算

### 单样本Z检验

用于检验样本均值是否与总体均值有显著差异：

```
Z = (x̄ - μ₀) / (σ/√n)
```

其中：
- x̄：样本均值
- μ₀：假设的总体均值
- σ：总体标准差
- n：样本量

### 双样本Z检验（独立样本）

用于检验两个独立样本的均值是否有显著差异：

```
Z = (x̄₁ - x̄₂) / √(σ₁²/n₁ + σ₂²/n₂)
```

其中：
- x̄₁, x̄₂：两个样本的均值
- σ₁, σ₂：两个总体标准差
- n₁, n₂：两个样本的样本量

### 比例Z检验

用于检验样本比例与总体比例的差异：

```
Z = (p̂ - p₀) / √(p₀(1-p₀)/n)
```

其中：
- p̂：样本比例
- p₀：假设的总体比例
- n：样本量

### 双样本比例Z检验

用于检验两个独立样本比例的差异：

```
Z = (p̂₁ - p̂₂) / √(p̂(1-p̂)(1/n₁ + 1/n₂))
```

其中：
- p̂₁, p̂₂：两个样本的比例
- p̂ = (x₁ + x₂)/(n₁ + n₂)：合并比例
- n₁, n₂：两个样本的样本量

## Z检验的前提条件

使用Z检验需要满足以下前提条件：

1. **正态性或大样本**：总体分布正态，或样本量足够大（n ≥ 30）
2. **独立性**：样本观测值相互独立
3. **已知标准差**：对于均值检验，需要知道总体标准差σ
4. **随机抽样**：样本是随机抽取的

## Z检验与t检验的区别和联系

| 特征 | Z检验 | t检验 |
|------|-------|-------|
| 标准差 | 已知总体标准差 | 未知，用样本标准差估计 |
| 样本量 | 通常大样本（n ≥ 30） | 小样本 |
| 分布 | 标准正态分布 | t分布 |
| 自由度 | 无 | n-1或(n₁+n₂-2) |
| 保守性 | 较不保守 | 较保守 |

**联系**：当样本量很大时，t分布趋近于标准正态分布，此时t检验和Z检验的结果几乎相同。

## 假设检验步骤

### 步骤1：建立假设

- 原假设（H₀）：μ = μ₀（或μ₁ = μ₂）
- 备择假设（H₁）：μ ≠ μ₀（双侧），μ > μ₀（右侧），或μ < μ₀（左侧）

### 步骤2：选择显著性水平

通常选择α = 0.05或α = 0.01

### 步骤3：计算Z统计量

根据上述公式计算Z值

### 步骤4：确定临界值或p值

- 临界值法：查标准正态分布表得到Zα/2
- p值法：计算P(Z > \|Zobs\|)

### 步骤5：做出决策

- 如果 \|Z\| > Zα/2 或 p < α，拒绝H₀
- 否则，无法拒绝H₀

### 步骤6：解释结果

在统计显著性的语境下解释结果的实际意义

## Z统计量的解释

Z统计量表示样本统计量与假设值之间的差异，以标准差为单位：

- Z = 0：样本统计量与假设值完全相同
- Z > 0：样本统计量大于假设值
- Z < 0：样本统计量小于假设值
- \|Z\| 越大：差异越显著

## Python实现示例

### 单样本Z检验

```python
import numpy as np
from scipy import stats

def one_sample_z_test(sample, population_mean, population_std, alpha=0.05, alternative='two-sided'):
    """
    单样本Z检验

    参数:
        sample: 样本数据
        population_mean: 假设的总体均值
        population_std: 总体标准差
        alpha: 显著性水平
        alternative: 备择假设类型 ('two-sided', 'greater', 'less')

    返回:
        z_stat: Z统计量
        p_value: p值
        reject: 是否拒绝原假设
    """
    sample_mean = np.mean(sample)
    n = len(sample)
    standard_error = population_std / np.sqrt(n)

    z_stat = (sample_mean - population_mean) / standard_error

    if alternative == 'two-sided':
        p_value = 2 * (1 - stats.norm.cdf(abs(z_stat)))
    elif alternative == 'greater':
        p_value = 1 - stats.norm.cdf(z_stat)
    elif alternative == 'less':
        p_value = stats.norm.cdf(z_stat)

    reject = p_value < alpha

    return {
        'z_statistic': z_stat,
        'p_value': p_value,
        'reject_null': reject,
        'sample_mean': sample_mean,
        'critical_value': stats.norm.ppf(1 - alpha/2) if alternative == 'two-sided' else stats.norm.ppf(1 - alpha)
    }

# 示例：检验某班级平均成绩是否显著高于80分
np.random.seed(42)
scores = np.random.normal(loc=82, scale=5, size=50)

result = one_sample_z_test(scores, population_mean=80, population_std=5, alpha=0.05, alternative='greater')

print(f"样本均值: {result['sample_mean']:.2f}")
print(f"Z统计量: {result['z_statistic']:.4f}")
print(f"p值: {result['p_value']:.4f}")
print(f"临界值: {result['critical_value']:.4f}")
print(f"是否拒绝原假设: {result['reject_null']}")
```

### 双样本Z检验

```python
def two_sample_z_test(sample1, sample2, std1, std2, alpha=0.05, alternative='two-sided'):
    """
    双样本Z检验（独立样本）

    参数:
        sample1, sample2: 两个样本数据
        std1, std2: 两个总体的标准差
        alpha: 显著性水平
        alternative: 备择假设类型

    返回:
        z_stat: Z统计量
        p_value: p值
        reject: 是否拒绝原假设
    """
    mean1, mean2 = np.mean(sample1), np.mean(sample2)
    n1, n2 = len(sample1), len(sample2)

    z_stat = (mean1 - mean2) / np.sqrt(std1**2/n1 + std2**2/n2)

    if alternative == 'two-sided':
        p_value = 2 * (1 - stats.norm.cdf(abs(z_stat)))
    elif alternative == 'greater':
        p_value = 1 - stats.norm.cdf(z_stat)
    elif alternative == 'less':
        p_value = stats.norm.cdf(z_stat)

    reject = p_value < alpha

    return {
        'z_statistic': z_stat,
        'p_value': p_value,
        'reject_null': reject,
        'mean_diff': mean1 - mean2,
        'critical_value': stats.norm.ppf(1 - alpha/2) if alternative == 'two-sided' else stats.norm.ppf(1 - alpha)
    }

# 示例：比较两个班级的平均成绩
np.random.seed(42)
class_a = np.random.normal(loc=82, scale=5, size=45)
class_b = np.random.normal(loc=78, scale=5, size=50)

result = two_sample_z_test(class_a, class_b, std1=5, std2=5, alpha=0.05, alternative='greater')

print(f"班级A平均分: {np.mean(class_a):.2f}")
print(f"班级B平均分: {np.mean(class_b):.2f}")
print(f"平均分差异: {result['mean_diff']:.2f}")
print(f"Z统计量: {result['z_statistic']:.4f}")
print(f"p值: {result['p_value']:.4f}")
print(f"是否拒绝原假设: {result['reject_null']}")
```

### 比例Z检验

```python
def proportion_z_test(x, n, p0, alpha=0.05, alternative='two-sided'):
    """
    单样本比例Z检验

    参数:
        x: 成功次数
        n: 样本量
        p0: 假设的总体比例
        alpha: 显著性水平
        alternative: 备择假设类型

    返回:
        z_stat: Z统计量
        p_value: p值
        reject: 是否拒绝原假设
    """
    p_hat = x / n
    standard_error = np.sqrt(p0 * (1 - p0) / n)

    z_stat = (p_hat - p0) / standard_error

    if alternative == 'two-sided':
        p_value = 2 * (1 - stats.norm.cdf(abs(z_stat)))
    elif alternative == 'greater':
        p_value = 1 - stats.norm.cdf(z_stat)
    elif alternative == 'less':
        p_value = stats.norm.cdf(z_stat)

    reject = p_value < alpha

    return {
        'z_statistic': z_stat,
        'p_value': p_value,
        'reject_null': reject,
        'sample_proportion': p_hat
    }

# 示例：检验某产品合格率是否显著高于90%
result = proportion_z_test(x=95, n=100, p0=0.90, alpha=0.05, alternative='greater')

print(f"样本比例: {result['sample_proportion']:.4f}")
print(f"Z统计量: {result['z_statistic']:.4f}")
print(f"p值: {result['p_value']:.4f}")
print(f"是否拒绝原假设: {result['reject_null']}")
```


## Z检验的局限性

1. **需要已知总体标准差**：这在实际应用中往往难以满足
2. **正态性假设**：小样本时对正态性要求严格
3. **独立性假设**：样本观测值必须相互独立
4. **对异常值敏感**：异常值会影响均值和标准差的估计
