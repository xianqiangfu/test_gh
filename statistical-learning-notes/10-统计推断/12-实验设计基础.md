# 实验设计基础（Experimental Design）

## 定义

实验设计是研究如何合理安排实验，以最大限度地获取信息、控制混杂因素、提高研究效率的科学方法。它是统计学的重要分支，也是科学研究的基础。

## 实验设计的基本原则

### 1. 随机化（Randomization）

**目的：** 控制未知混杂因素的影响，消除系统性偏差。

**方法：**
- 完全随机化：所有实验单位随机分配到各处理组
- 限制随机化：在约束条件下进行随机（如区组内随机）

**优势：**
- 平衡各组间的已知和未知因素
- 保证统计推断的有效性
- 防止选择偏差

**Python示例：**
```python
import numpy as np
from sklearn.model_selection import train_test_split

# 完全随机化
subjects = np.arange(100)
groups = np.random.choice(['A', 'B', 'C'], size=100, p=[1/3, 1/3, 1/3])

# 均衡随机化（各组样本量相等）
groups_balanced = np.random.permutation(['A']*33 + ['B']*33 + ['C']*34)
```

### 2. 重复（Replication）

**目的：** 估计实验误差，提高检验功效。

**类型：**
- 真重复（True Replication）：独立的实验单位
- 伪重复（Pseudoreplication）：同一实验单位的多次测量

**重复次数的选择：**
- 基于功效分析确定样本量
- 考虑效应大小、变异程度、显著性水平
- 一般每组至少10-20个重复

**Python示例：**
```python
# 功效分析确定样本量
from statsmodels.stats.power import FTestAnovaPower

power_analysis = FTestAnovaPower()
sample_size = power_analysis.solve_power(
    effect_size=0.5,  # 中等效应
    alpha=0.05,       # 显著性水平
    power=0.8,        # 检验功效
    k_groups=3        # 组数
)
print(f"每组需要的样本量: {sample_size:.1f}")
```

### 3. 区组（Blocking）

**目的：** 控制已知混杂因素的影响，提高实验精度。

**原理：** 将相似的实验单位分成区组，区组内进行比较，消除区组间的变异。

**应用场景：**
- 不同时间进行的实验
- 不同实验者操作
- 不同设备/批次
- 地理位置

**Python示例：**
```python
import pandas as pd
import numpy as np

# 创建区组设计
blocks = ['上午', '下午', '晚上']
treatments = ['A', 'B', 'C']

design = []
for block in blocks:
    # 区组内随机分配处理
    block_treatments = np.random.permutation(treatments)
    for treatment in block_treatments:
        design.append({'block': block, 'treatment': treatment})

df_design = pd.DataFrame(design)
print("随机区组设计:")
print(df_design)
```

### 三原则的关系

```
随机化 + 重复 + 区组 → 最优实验设计
```

- **随机化：** 消除系统性偏差
- **重复：** 估计误差、提高功效
- **区组：** 控制已知变异、提高精度

## 完全随机设计（Completely Randomized Design, CRD）

### 定义

所有实验单位随机分配到各个处理组，不考虑任何区组因素。

### 模型

$$y_{ij} = \mu + \tau_i + \varepsilon_{ij}$$

其中：
- $y_{ij}$ 是第i处理第j个观测值
- $\mu$ 是总体均值
- $\tau_i$ 是第i处理的效应
- $\varepsilon_{ij} \sim N(0, \sigma^2)$ 是随机误差

### 适用场景

- 实验单位同质性好
- 没有明显的区组因素
- 简单实验设计

### 优点

- 设计简单
- 分析容易
- 自由度损失最少
- 适合同质性好的实验单位

### 缺点

- 效率可能较低（存在变异时）
- 无法控制已知混杂因素

### 方差分析表

| 变异来源 | 平方和(SS) | 自由度(df) | 均方(MS) | F值 |
|----------|-----------|-----------|----------|-----|
| 处理 | $SS_{treatment}$ | k-1 | $MS_{treatment}$ | $MS_{treatment}/MS_{error}$ |
| 误差 | $SS_{error}$ | N-k | $MS_{error}$ | |
| 总计 | $SS_{total}$ | N-1 | | |

### Python实现

```python
import numpy as np
import pandas as pd
from scipy import stats
import statsmodels.api as sm
from statsmodels.formula.api import ols
import matplotlib.pyplot as plt
import seaborn as sns

# 完全随机设计示例：测试三种肥料的效果
np.random.seed(42)
n_per_group = 10

# 生成数据（假设肥料B效果最好）
data = pd.DataFrame({
    'yield': np.concatenate([
        np.random.normal(50, 5, n_per_group),      # 肥料A
        np.random.normal(60, 5, n_per_group),      # 肥料B
        np.random.normal(52, 5, n_per_group)       # 肥料C
    ]),
    'fertilizer': ['A']*n_per_group + ['B']*n_per_group + ['C']*n_per_group
})

# 方差分析
model = ols('yield ~ C(fertilizer)', data=data).fit()
anova_table = sm.stats.anova_lm(model, typ=2)
print("完全随机设计 ANOVA表:")
print(anova_table)

# 可视化
plt.figure(figsize=(10, 6))
sns.boxplot(x='fertilizer', y='yield', data=data)
plt.title('完全随机设计：不同肥料对产量的影响')
plt.xlabel('肥料类型')
plt.ylabel('产量')
plt.show()
```

## 随机区组设计（Randomized Block Design, RBD）

### 定义

将实验单位按某个因素（区组）分组，每个区组内包含所有处理，在区组内随机分配处理。

### 模型

$$y_{ij} = \mu + \tau_i + \beta_j + \varepsilon_{ij}$$

其中：
- $\tau_i$ 是第i处理的效应
- $\beta_j$ 是第j区组的效应
- $\varepsilon_{ij} \sim N(0, \sigma^2)$ 是随机误差

### 适用场景

- 存在已知的区组因素
- 实验单位间有系统性差异
- 可以识别和分离区组变异

### 优点

- 控制区组变异
- 提高实验精度
- 减少误差项的变异

### 缺点

- 自由度损失（区组效应）
- 设计复杂度增加
- 区组划分可能不完美

### 方差分析表

| 变异来源 | 平方和(SS) | 自由度(df) | 均方(MS) | F值 |
|----------|-----------|-----------|----------|-----|
| 区组 | $SS_{block}$ | b-1 | $MS_{block}$ | $MS_{block}/MS_{error}$ |
| 处理 | $SS_{treatment}$ | k-1 | $MS_{treatment}$ | $MS_{treatment}/MS_{error}$ |
| 误差 | $SS_{error}$ | (k-1)(b-1) | $MS_{error}$ | |
| 总计 | $SS_{total}$ | kb-1 | | |

其中：
- k 是处理数
- b 是区组数

### Python实现

```python
# 随机区组设计示例：不同时间点测试三种教学方法
np.random.seed(42)
n_blocks = 4  # 4个时间区组
n_treatments = 3  # 3种教学方法

data_rbd = []
for block in range(n_blocks):
    block_effect = block * 2  # 每个区组有系统性差异
    for treatment in ['A', 'B', 'C']:
        if treatment == 'A':
            base_yield = 70
        elif treatment == 'B':
            base_yield = 75
        else:
            base_yield = 72
        
        # 添加区组效应和随机误差
        yield_value = base_yield + block_effect + np.random.normal(0, 3)
        data_rbd.append({
            'score': yield_value,
            'method': treatment,
            'block': f'时间{block+1}'
        })

df_rbd = pd.DataFrame(data_rbd)

# 随机区组设计的方差分析
model_rbd = ols('score ~ C(block) + C(method)', data=df_rbd).fit()
anova_table_rbd = sm.stats.anova_lm(model_rbd, typ=2)
print("随机区组设计 ANOVA表:")
print(anova_table_rbd)

# 计算区组效率（相对CRD）
ms_error_rbd = anova_table_rbd.loc['Residual', 'sum_sq'] / anova_table_rbd.loc['Residual', 'df']

# CRD的误差（忽略区组效应）
model_crd = ols('score ~ C(method)', data=df_rbd).fit()
anova_table_crd = sm.stats.anova_lm(model_crd, typ=2)
ms_error_crd = anova_table_crd.loc['Residual', 'sum_sq'] / anova_table_crd.loc['Residual', 'df']

efficiency = (ms_error_crd / ms_error_rbd) * 100
print(f"\n区组效率: {efficiency:.1f}%")

# 可视化
plt.figure(figsize=(12, 6))
sns.pointplot(x='method', y='score', hue='block', data=df_rbd, 
              ci=None, markers=['o', 's', '^', 'D'], linestyles=['-', '--', ':', '-.'])
plt.title('随机区组设计：不同方法和区组的分数')
plt.xlabel('教学方法')
plt.ylabel('分数')
plt.legend(title='区组')
plt.grid(True, alpha=0.3)
plt.show()
```

## 因子设计（Factorial Design）

### 定义

同时研究两个或多个因素的效应，可以分析主效应和交互效应。

### 两因子完全随机设计

#### 模型

$$y_{ijk} = \mu + \alpha_i + \beta_j + (\alpha\beta)_{ij} + \varepsilon_{ijk}$$

其中：
- $\alpha_i$ 是因子A第i水平的主效应
- $\beta_j$ 是因子B第j水平的主效应
- $(\alpha\beta)_{ij}$ 是A和B的交互效应
- $\varepsilon_{ijk} \sim N(0, \sigma^2)$ 是随机误差

#### 方差分析表

| 变异来源 | 平方和(SS) | 自由度(df) | 均方(MS) | F值 |
|----------|-----------|-----------|----------|-----|
| 因子A | $SS_A$ | a-1 | $MS_A$ | $MS_A/MS_{error}$ |
| 因子B | $SS_B$ | b-1 | $MS_B$ | $MS_B/MS_{error}$ |
| 交互AB | $SS_{AB}$ | (a-1)(b-1) | $MS_{AB}$ | $MS_{AB}/MS_{error}$ |
| 误差 | $SS_{error}$ | ab(n-1) | $MS_{error}$ | |
| 总计 | $SS_{total}$ | abn-1 | | |

#### 交互效应

**定义：** 一个因素的效应依赖于另一个因素的水平。

**类型：**
- 无交互：各因素效应独立
- 有序交互：交互效应有规律
- 无序交互：交互效应无规律

**解读：**
- 交互显著：不能单独解释主效应
- 交互不显著：可以单独解释主效应

### Python实现：两因子设计

```python
# 两因子设计示例：教学方法 × 学生水平
np.random.seed(42)
n_levels_A = 3  # 3种教学方法
n_levels_B = 2  # 2种学生水平
n_replicates = 8

data_factorial = []
for method in ['讲授', '讨论', '实验']:
    for level in ['初级', '高级']:
        for _ in range(n_replicates):
            # 基础分数
            if method == '讲授' and level == '初级':
                base = 70
            elif method == '讲授' and level == '高级':
                base = 75
            elif method == '讨论' and level == '初级':
                base = 72
            elif method == '讨论' and level == '高级':
                base = 78
            elif method == '实验' and level == '初级':
                base = 65
            else:  # 实验 × 高级
                base = 85  # 交互效应：实验对高级学生特别有效
            
            score = base + np.random.normal(0, 3)
            data_factorial.append({
                'score': score,
                'method': method,
                'level': level
            })

df_factorial = pd.DataFrame(data_factorial)

# 两因子方差分析
model_factorial = ols('score ~ C(method) * C(level)', data=df_factorial).fit()
anova_table_factorial = sm.stats.anova_lm(model_factorial, typ=2)
print("两因子设计 ANOVA表:")
print(anova_table_factorial)

# 交互效应可视化
plt.figure(figsize=(14, 6))

# 交互作用图
plt.subplot(1, 2, 1)
sns.pointplot(x='method', y='score', hue='level', data=df_factorial, 
              ci=95, markers=['o', 's'], linestyles=['-', '--'])
plt.title('交互作用图')
plt.xlabel('教学方法')
plt.ylabel('分数')
plt.legend(title='学生水平')
plt.grid(True, alpha=0.3)

# 热力图
plt.subplot(1, 2, 2)
pivot_table = df_factorial.pivot_table(values='score', index='method', columns='level', aggfunc='mean')
sns.heatmap(pivot_table, annot=True, fmt='.1f', cmap='RdYlGn', center=75)
plt.title('各组合均值热力图')
plt.tight_layout()
plt.show()
```

### 全因子设计 vs 部分因子设计

| 特征 | 全因子设计 | 部分因子设计 |
|------|-----------|------------|
| 组合数 | 完整的所有组合 | 部分组合 |
| 效应估计 | 所有效应可估计 | 部分效应混杂 |
| 实验次数 | 多 | 少 |
| 成本 | 高 | 低 |
| 适用性 | 因子数少时 | 因子数多时 |

### 部分因子设计

**目的：** 在较少实验次数中估计主要效应。

**方法：**
- 分式因子设计
- Plackett-Burman设计
- Taguchi设计

**混杂：** 部分设计效应之间可能混杂，需要仔细选择设计。

### Python实现：部分因子设计

```python

# pip install pyDOE2

# 创建2^(5-1)部分因子设计

# 转换为数据框

print("\n部分因子设计:")
print(df_design)
```

## 拉丁方设计（Latin Square Design）

### 定义

处理数 = 行数 = 列数，每个处理在每行每列恰好出现一次。

### 结构

对于3×3拉丁方（处理A, B, C）：

```
   C1 C2 C3
R1  A  B  C
R2  B  C  A
R3  C  A  B
```

### 模型

$$y_{ijk} = \mu + \rho_i + \kappa_j + \tau_k + \varepsilon_{ijk}$$

其中：
- $\rho_i$ 是第i行（区组1）的效应
- $\kappa_j$ 是第j列（区组2）的效应
- $\tau_k$ 是第k个处理的效应
- $\varepsilon_{ijk} \sim N(0, \sigma^2)$ 是随机误差

### 适用场景

- 有两个区组因素
- 处理数等于区组数
- 每个区组水平与处理数相同

### 优点

- 控制两个方向的变异
- 实验次数少（每个处理只出现n次）

### 缺点

- 要求处理数 = 行数 = 列数
- 假设行、列、处理效应无交互
- 自由度损失较多

### 方差分析表

| 变异来源 | 平方和(SS) | 自由度(df) | 均方(MS) | F值 |
|----------|-----------|-----------|----------|-----|
| 行 | $SS_{row}$ | n-1 | $MS_{row}$ | $MS_{row}/MS_{error}$ |
| 列 | $SS_{col}$ | n-1 | $MS_{col}$ | $MS_{col}/MS_{error}$ |
| 处理 | $SS_{treatment}$ | n-1 | $MS_{treatment}$ | $MS_{treatment}/MS_{error}$ |
| 误差 | $SS_{error}$ | (n-1)(n-2) | $MS_{error}$ | |
| 总计 | $SS_{total}$ | n²-1 | | |

### Python实现

```python
# 拉丁方设计示例：4种饲料在4个时间、4种动物上的测试
np.random.seed(42)
n = 4  # 4×4拉丁方

# 标准拉丁方
latin_square = [
    ['A', 'B', 'C', 'D'],
    ['B', 'C', 'D', 'A'],
    ['C', 'D', 'A', 'B'],
    ['D', 'A', 'B', 'C']
]

# 生成数据
data_latin = []
for i in range(n):
    for j in range(n):
        treatment = latin_square[i][j]
        
        # 不同饲料的基础效果
        if treatment == 'A':
            base_yield = 70
        elif treatment == 'B':
            base_yield = 75
        elif treatment == 'C':
            base_yield = 72
        else:
            base_yield = 68
        
        # 添加行（时间）和列（动物）效应
        row_effect = i * 1.5
        col_effect = j * 1.0
        
        yield_value = base_yield + row_effect + col_effect + np.random.normal(0, 2)
        data_latin.append({
            'yield': yield_value,
            'treatment': treatment,
            'row': f'时间{i+1}',
            'col': f'动物{j+1}'
        })

df_latin = pd.DataFrame(data_latin)

# 拉丁方设计的方差分析
model_latin = ols('yield ~ C(row) + C(col) + C(treatment)', data=df_latin).fit()
anova_table_latin = sm.stats.anova_lm(model_latin, typ=2)
print("拉丁方设计 ANOVA表:")
print(anova_table_latin)

# 可视化拉丁方结构
plt.figure(figsize=(12, 5))

# 拉丁方矩阵
plt.subplot(1, 2, 1)
square_matrix = [[treatment for treatment in row] for row in latin_square]
colors = {'A': '#ff6b6b', 'B': '#4ecdc4', 'C': '#45b7d1', 'D': '#f9ca24'}
for i in range(n):
    for j in range(n):
        plt.text(j+0.5, n-0.5-i, square_matrix[i][j], 
                ha='center', va='center', fontsize=16, fontweight='bold',
                bbox=dict(boxstyle='round', facecolor=colors[square_matrix[i][j]], alpha=0.7))
plt.xlim(0, n)
plt.ylim(0, n)
plt.grid(False)
plt.title('拉丁方结构')
plt.xlabel('列')
plt.ylabel('行')

# 产量热力图
plt.subplot(1, 2, 2)
pivot_latin = df_latin.pivot_table(values='yield', index='row', columns='col', aggfunc='mean')
sns.heatmap(pivot_latin, annot=True, fmt='.1f', cmap='RdYlGn', center=72)
plt.title('各位置的产量')
plt.tight_layout()
plt.show()
```

## 交叉设计（Crossover Design）

### 定义

每个被试接受所有处理，但顺序不同。常用于医学研究。

### 类型

1. **2×2交叉设计：** 两个处理，两个时期
2. **多处理交叉设计：** 多个处理

### 2×2交叉设计

#### 结构

```
序列1: 时期1(T1) → 洗脱期 → 时期2(T2)
序列2: 时期1(T2) → 洗脱期 → 时期2(T1)
```

#### 模型

$$y_{ijk} = \mu + \pi_i + \lambda_j + \tau_k + \varepsilon_{ijk}$$

其中：
- $\pi_i$ 是序列i的效应
- $\lambda_j$ 是时期j的效应
- $\tau_k$ 是处理k的效应
- $\varepsilon_{ijk} \sim N(0, \sigma^2)$ 是随机误差

#### 适用场景

- 慢性病研究
- 被试间变异大
- 处理效应短暂（可被洗脱）

#### 优点

- 控制被试间变异
- 所需样本量少
- 每个被试作为自身对照

#### 缺点

- 需要洗脱期
- 可能存在残留效应
- 不适合永久性改变

### Python实现

```python
# 2×2交叉设计示例：测试两种药物的效果
np.random.seed(42)
n_per_sequence = 15

# 序列1: 药物A → 药物B
data_sequence1 = []
for subject in range(n_per_sequence):
    # 时期1：药物A
    y1 = 70 + np.random.normal(0, 4)
    # 时期2：药物B（假设有5单位的残留效应）
    y2 = 75 + 5 + np.random.normal(0, 4)  # 75是B的基础效果，5是残留效应
    
    data_sequence1.append({'subject': subject, 'sequence': 'AB', 
                           'period': 1, 'treatment': 'A', 'response': y1})
    data_sequence1.append({'subject': subject, 'sequence': 'AB', 
                           'period': 2, 'treatment': 'B', 'response': y2})

# 序列2: 药物B → 药物A
data_sequence2 = []
for subject in range(n_per_sequence, 2*n_per_sequence):
    # 时期1：药物B
    y1 = 75 + np.random.normal(0, 4)
    # 时期2：药物A（假设有3单位的残留效应）
    y2 = 70 + 3 + np.random.normal(0, 4)
    
    data_sequence2.append({'subject': subject, 'sequence': 'BA', 
                           'period': 1, 'treatment': 'B', 'response': y1})
    data_sequence2.append({'subject': subject, 'sequence': 'BA', 
                           'period': 2, 'treatment': 'A', 'response': y2})

df_crossover = pd.DataFrame(data_sequence1 + data_sequence2)

# 交叉设计的方差分析
# 注意：这里简化分析，实际需要考虑序列效应和残留效应
model_crossover = ols('response ~ C(sequence) + C(period) + C(treatment)', 
                      data=df_crossover).fit()
anova_table_crossover = sm.stats.anova_lm(model_crossover, typ=2)
print("交叉设计 ANOVA表:")
print(anova_table_crossover)

# 可视化
plt.figure(figsize=(14, 6))

# 按序列和时期绘制
plt.subplot(1, 2, 1)
sns.pointplot(x='period', y='response', hue='treatment',
```

## 平衡与混杂（Balancing and Confounding）

### 平衡设计

**定义：** 各处理组合的重复次数相等。

**优势：**
- 方差分析简单
- 估计量性质优良
- 检验功效最大化

**类型：**
- 完全平衡：所有组合重复次数相同
- 部分平衡：某些组合重复次数相同

### 混杂

**定义：** 两个或多个效应无法区分估计。

**类型：**

1. **完全混杂：** 两个效应完全无法区分
2. **部分混杂：** 效应部分可区分

**例子：**
- 在2^(3-1)部分因子设计中，ABC主效应与截距混杂
- 在某些设计中，主效应可能与交互效应混杂

### 混杂结构示例

```python
# 展示2^(3-1)设计中的混杂关系
import itertools

# 生成全因子设计（8个组合）
factors = ['A', 'B', 'C']
full_factorial = list(itertools.product([0, 1], repeat=3))

# 定义生成器（这里用C = AB）
def confounded_relation(x1, x2, x3):
    """检查是否满足C = AB的关系（模2）"""
    return (x3 == (x1 * x2) % 2)

# 筛选满足关系的组合（得到2^(3-1)设计）
fractional_design = [combo for combo in full_factorial if confounded_relation(*combo)]

print("2^(3-1)部分因子设计（生成器：C = AB）:")
for combo in fractional_design:
    print(f"{''.join([factors[i] if combo[i]==1 else f'not {factors[i]}' for i in range(3)])}")

# 混杂关系
print("\n混杂关系:")
print("- 主效应A 与 交互效应BC 混杂")
print("- 主效应B 与 交互效应AC 混杂")
print("- 主效应C 与 交互效应AB 混杂")
print("- 交互效应ABC 与 截距混杂")
```

### 最小化混杂的设计原则

1. **避免主效应混杂：** 主效应不与交互效应混杂
2. **低阶效应优先：** 保护低阶效应
3. **选择合适的生成器：** 仔细选择部分因子设计的生成器
4. **使用设计表：** 参考标准设计表

## 实验设计的选择

### 设计比较表

| 设计类型 | 适用场景 | 优势 | 局限性 | 实验次数 |
|---------|---------|------|-------|---------|
| 完全随机 | 同质实验单位 | 简单 | 无法控制变异 | N = kn |
| 随机区组 | 有一个区组因素 | 控制区组变异 | 自由度损失 | N = kn |
| 拉丁方 | 有两个区组因素 | 控制两个方向变异 | 限制多 | N = n² |
| 全因子 | 因子少 | 估计所有效应 | 实验次数多 | N = abn |
| 部分因子 | 因子多 | 节省实验次数 | 效应混杂 | N = 2^(k-p)n |
| 交叉设计 | 慢性病研究 | 自身对照 | 需要洗脱期 | N = 2n |

### 选择流程图

```
开始
  ↓
有几个区组因素？
  ├─ 0个 → 完全随机设计
  ├─ 1个 → 随机区组设计
  └─ 2个 → 拉丁方设计
```

```
有几个研究因子？
  ├─ 1个 → 单因素设计
  ├─ 2个 → 两因子设计
  └─ >2个 → 
      ├─ 因子少 → 全因子设计
      └─ 因子多 → 部分因子设计
```

```
是否需要同一被试多次测量？
  ├─ 是 → 交叉设计 / 重复测量设计
  └─ 否 → 组间设计
```

## 实验设计的常见错误

### 1. 伪重复

**问题：** 将同一实验单位的多次测量当作独立重复。

**示例：** 同一棵树上的多次测量当作不同树。

**解决：** 确保每个观测来自独立的实验单位。

### 2. 忽略区组因素

**问题：** 存在明显的区组变异但未控制。

**示例：** 不同天进行实验但不考虑日期效应。

**解决：** 识别并控制区组因素。

### 3. 样本量不足

**问题：** 实验次数不足以检测实际效应。

**解决：** 进行功效分析确定合适的样本量。

### 4. 随机化不当

**问题：** 分配不随机或随机化方案不当。

**解决：** 使用适当的随机化方法。

### 5. 违反假设

**问题：** 数据不满足方差分析的假设。

**解决：** 检查假设，必要时使用非参数方法。

```python
from statsmodels.formula.api import ols
```

## 总结

实验设计是科学研究的基础，正确的实验设计可以：

**核心原则：**
1. **随机化：** 消除系统性偏差
2. **重复：** 估计误差、提高功效
3. **区组：** 控制已知变异、提高精度

**常用设计：**
1. **完全随机设计：** 简单、适用于同质实验单位
2. **随机区组设计：** 控制一个区组因素
3. **拉丁方设计：** 控制两个区组因素
4. **因子设计：** 研究多个因素的主效应和交互效应
5. **交叉设计：** 自身对照，适用于医学研究

**选择依据：**
- 因素数量和水平
- 区组因素的存在和数量
- 实验单位的同质性
- 资源限制
- 研究问题的性质

**注意事项：**
1. 进行功效分析确定样本量
2. 检查方差分析的假设条件
3. 避免常见错误（伪重复、忽略区组等）
4. 报告效应量和置信区间
5. 考虑实际意义而非仅统计显著性

良好的实验设计是可靠研究结论的前提，值得在研究开始时投入充分的时间和精力进行规划和设计。
