## 决策边界函数

```python
def plot_decision_boundary(model, X, y, title='Decision Boundary'):
    """绘制决策边界"""

    # 定义网格
    x_min, x_max = X[:, 0].min() - 1, X[:, 0].max() + 1
    y_min, y_max = X[:, 1].min() - 1, X[:, 1].max() + 1
    xx, yy = np.meshgrid(np.arange(x_min, x_max, 0.02),
                         np.arange(y_min, y_max, 0.02))

    # 预测网格
    Z = model.predict(np.c_[xx.ravel(), yy.ravel()])
    Z = Z.reshape(xx.shape)

    # 绘制
    plt.figure(figsize=(8, 6))

    # 决策边界
    plt.contourf(xx, yy, Z, alpha=0.3, cmap='viridis')

    # 训练数据
    plt.scatter(X[:, 0], X[:, 1], c=y, cmap='viridis', edgecolors='k', alpha=0.7)

    plt.xlabel('Feature 1')
    plt.ylabel('Feature 2')
    plt.title(title)
    plt.show()
```

## 比较多个模型

```python
from sklearn.linear_model import LogisticRegression
from sklearn.svm import SVC
from sklearn.tree import DecisionTreeClassifier
from sklearn.ensemble import RandomForestClassifier
from sklearn.neighbors import KNeighborsClassifier

# 定义模型
models = [
    ('Logistic Regression', LogisticRegression(random_state=42)),
    ('Linear SVM', LinearSVC(random_state=42)),
    ('SVM (RBF)', SVC(kernel='rbf', random_state=42)),
    ('Decision Tree', DecisionTreeClassifier(max_depth=3, random_state=42)),
    ('Random Forest', RandomForestClassifier(n_estimators=50,
                                               max_depth=3,
                                               random_state=42)),
    ('KNN', KNeighborsClassifier(n_neighbors=5))
]

# 绘制所有模型
fig, axes = plt.subplots(2, 3, figsize=(18, 10))
axes = axes.flatten()

for idx, (name, model) in enumerate(models):
    model.fit(X, y)

    # 网格
    x_min, x_max = X[:, 0].min() - 1, X[:, 0].max() + 1
    y_min, y_max = X[:, 1].min() - 1, X[:, 1].max() + 1
    xx, yy = np.meshgrid(np.arange(x_min, x_max, 0.02),
                         np.arange(y_min, y_max, 0.02))

    Z = model.predict(np.c_[xx.ravel(), yy.ravel()])
    Z = Z.reshape(xx.shape)

    # 绘制
    axes[idx].contourf(xx, yy, Z, alpha=0.3, cmap='viridis')
    axes[idx].scatter(X[:, 0], X[:, 1], c=y, cmap='viridis',
                    edgecolors='k', alpha=0.7)
    axes[idx].set_title(name)
    axes[idx].set_xlabel('Feature 1')
    axes[idx].set_ylabel('Feature 2')

plt.tight_layout()
plt.show()
```

## 不同数据集上的表现

```python
# 三个数据集
datasets = [

    ('Circles', X2, y2),

                             np.arange(y_min, y_max, 0.02))

        Z = Z.reshape(xx.shape)

        axes[i, j].contourf(xx, yy, Z, alpha=0.3, cmap='viridis')
        axes[i, j].scatter(X[:, 0], X[:, 1], c=y, cmap='viridis',
                          edgecolors='k', alpha=0.7)

        if i == 0:
            axes[i, j].set_title(model_name)
        if j == 0:
```

## 决策边界的理解

### 线性边界

```
决策边界是直线

      ●●●
     ╱     ╲
●●●─┼─────●●●
    ╲     ╱
     ●●●
```

### 非线性边界

```
决策边界是曲线

      ●●●
     ╱   ╲
●●●─┤     ├●●●
    ╲     ╱
     ●●●
```

### 复杂边界

```
决策边界很复杂

    ●●●
   ╱ ╲ ╱ ╲
●●─┤   ├─●●
   ╲ ╱ ╲ ╱
    ●●●
```

## 实践要点

### 1. 理解模型

- 线性模型：直线边界
- 核SVM：非线性边界
- 决策树：分段线性
- KNN：不规则边界

### 2. 观察过拟合

- 边界太复杂：可能过拟合
- 边界太简单：可能欠拟合

### 3. 选择模型

- 线性数据：线性模型
- 非线性数据：非线性模型
- 需要解释：简单模型

---

## 常见问题

### Q: 为什么有些模型边界很复杂？

A:
- 模型复杂度高
- 训练了太久
- 没有正则化

### Q: 如何简化决策边界？

A:
- 降低模型复杂度
- 增加正则化
- 用更简单的模型

### Q: 决策边界能告诉我们什么？

A:
- 模型的决策方式
- 是否过拟合
- 模型的鲁棒性
