## 从零实现

### 1. 模型定义

```python
def predict(X, w, b):
    """预测函数"""
    return np.dot(X, w) + b
```

### 2. 损失函数（MSE）

```python
def compute_loss(X, y, w, b):
    """计算均方误差"""
    n = len(y)
    predictions = predict(X, w, b)
    loss = np.sum((predictions - y) ** 2) / (2 * n)
    return loss
```

### 3. 梯度计算

```python
def compute_gradients(X, y, w, b):
    """计算梯度"""
    n = len(y)
    predictions = predict(X, w, b)
    dw = np.dot(X.T, (predictions - y)) / n
    db = np.sum(predictions - y) / n
    return dw, db
```

### 4. 训练函数

```python
def train(X, y, learning_rate=0.1, epochs=1000):
    """训练线性回归模型"""
    # 初始化参数
    w = np.random.randn(1, 1)
    b = np.random.randn(1)

    # 记录损失
    losses = []

    for epoch in range(epochs):
        # 计算梯度
        dw, db = compute_gradients(X, y, w, b)

        # 更新参数
        w -= learning_rate * dw
        b -= learning_rate * db

        # 计算损失
        loss = compute_loss(X, y, w, b)
        losses.append(loss)

        # 每100轮打印一次
        if epoch % 100 == 0:
            print(f'Epoch {epoch}, Loss: {loss:.4f}')

    return w, b, losses
```

### 5. 训练模型

```python
# 训练
w, b, losses = train(X, y, learning_rate=0.1, epochs=1000)

print(f'\n训练结果:')
print(f'权重 w: {w[0][0]:.3f} (真实值: 3.0)')
print(f'偏置 b: {b[0]:.3f} (真实值: 4.0)')
```

## 实践要点

### 1. 学习率选择

```python
# 尝试不同的学习率
learning_rates = [0.01, 0.1, 1.0]

for lr in learning_rates:
    w, b, losses = train(X, y, learning_rate=lr, epochs=1000)
    print(f'学习率 {lr}: 最终损失 {losses[-1]:.4f}')
```

### 2. 早停

```python
def train_with_early_stopping(X, y, learning_rate=0.1, max_epochs=1000, patience=50):
    """带早停的训练"""
    w = np.random.randn(1, 1)
    b = np.random.randn(1)
    losses = []

    best_loss = float('inf')
    patience_counter = 0

    for epoch in range(max_epochs):
        dw, db = compute_gradients(X, y, w, b)
        w -= learning_rate * dw
        b -= learning_rate * db

        loss = compute_loss(X, y, w, b)
        losses.append(loss)

        # 早停
        if loss < best_loss:
            best_loss = loss
            patience_counter = 0
        else:
            patience_counter += 1
            if patience_counter >= patience:
                print(f'Early stopping at epoch {epoch}')
                break

    return w, b, losses
```

## 常见问题

### Q: 为什么我的模型不收敛？

A:
- 学习率太大：减小学习率
- 学习率太小：增加学习率或训练轮数
- 数据未标准化：标准化数据

### Q: 如何判断模型是否训练好了？

A:
- 损失不再下降
- 损失很小
- 在验证集上表现好

### Q: 预测效果不好怎么办？

A:
- 检查数据质量
- 增加特征
- 尝试更复杂的模型
- 调整参数
