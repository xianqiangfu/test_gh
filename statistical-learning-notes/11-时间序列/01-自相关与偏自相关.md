# 自相关与偏自相关

## 定义

### 自相关函数

自相关函数（Autocorrelation Function, ACF）用于衡量时间序列与其滞后版本之间的相关性。它是时间序列分析中识别模型、诊断模型适配性的核心工具。

**数学定义：**

对于时间序列 {X_t}，滞后 k 的自相关函数定义为：

$$\rho_k = \frac{Cov(X_t, X_{t-k})}{\sqrt{Var(X_t)Var(X_{t-k})}} = \frac{\gamma_k}{\gamma_0}$$

其中：
- $\gamma_k = Cov(X_t, X_{t-k})$ 是自协方差函数
- $\gamma_0 = Var(X_t)$ 是方差
- $k$ 是滞后阶数

**性质：**
1. $\rho_0 = 1$（序列与自身的相关系数为1）
2. $\rho_k = \rho_{-k}$（对称性）
3. $|\rho_k| \leq 1$（相关系数范围）

### 偏自相关函数

偏自相关函数（Partial Autocorrelation Function, PACF）是指在去除中间滞后期影响后，$X_t$ 与 $X_{t-k}$ 之间的相关性。

**数学定义：**

PACF在滞后 k 的值定义为线性回归中 $X_{t-k}$ 的系数：

$$X_t = \phi_{k1}X_{t-1} + \phi_{k2}X_{t-2} + \cdots + \phi_{kk}X_{t-k} + \varepsilon_t$$

其中 $\phi_{kk}$ 就是第 k 阶偏自相关系数，记为 $\pi_k$。

**直观理解：**
- ACF：直接相关性，包含所有间接路径的影响
- PACF：直接相关性，排除中间变量的影响

### 样本ACF和PACF

在实际应用中，我们使用样本自相关函数和样本偏自相关函数来估计理论值。

**样本自相关函数：**

$$\hat{\rho}_k = \frac{\sum_{t=k+1}^{n}(X_t - \bar{X})(X_{t-k} - \bar{X})}{\sum_{t=1}^{n}(X_t - \bar{X})^2}$$

其中 $\bar{X}$ 是样本均值。

**样本偏自相关函数：**

可以使用Yule-Walker方程递推计算：

$$\hat{\pi}_k = \frac{\hat{\rho}_k - \sum_{j=1}^{k-1}\hat{\phi}_{k-1,j}\hat{\rho}_{k-j}}{1 - \sum_{j=1}^{k-1}\hat{\phi}_{k-1,j}\hat{\rho}_j}$$

## ACF和PACF的性质

### 平稳时间序列的性质

对于平稳时间序列，ACF和PACF具有以下性质：

1. **ACF性质：**
   - 当滞后增大时，ACF趋于0
   - ACF图可以帮助识别数据是否平稳
   - ACF衰减速度反映序列的记忆性

2. **PACF性质：**
   - PACF在理论模型阶数后截断
   - PACF图可以识别AR模型的阶数
   - PACF不受季节性模式影响

### 不同模型的ACF/PACF特征

| 模型 | ACF特征 | PACF特征 |
|------|---------|----------|
| AR(p) | 拖尾（指数或正弦衰减） | 截尾（p阶后为0） |
| MA(q) | 截尾（q阶后为0） | 拖尾（指数或正弦衰减） |
| ARMA(p,q) | 拖尾 | 拖尾 |
| ARIMA(p,d,q) | 拖尾 | 拖尾 |
| 白噪声 | 拖尾（全部接近0） | 拖尾（全部接近0） |

### 拖尾与截尾

**拖尾（Tailing）：**
- ACF或PACF逐渐衰减到0
- 通常呈现指数衰减或阻尼正弦波形式
- 表示序列具有长记忆特征

**截尾（Cut-off）：**
- ACF或PACF在某个滞后后突然变为0
- 通常用于识别模型的阶数
- MA模型的ACF、AR模型的PACF通常截尾

## ACF/PACF图解读

### 标准ACF图解读

ACF图是自相关系数与滞后阶数的散点图，通常包含置信区间。

**关键元素：**
1. **蓝线：** 自相关系数值
2. **阴影区域：** 95%置信区间
3. **截断点：** 超出置信区间的第一个滞后

**解读步骤：**

1. **检查截断：**
   - 如果ACF在某个滞后后截断（全部在置信区间内），可能是MA模型
   - 记录截断位置，这是MA的阶数

2. **检查拖尾模式：**
   - 指数衰减：一个或几个AR项
   - 阻尼正弦：季节性或高阶AR项
   - 慢速衰减：非平稳序列

3. **检查季节性：**
   - 在季节性滞后（如12）处显著值
   - 可能需要季节差分

### 标准PACF图解读

PACF图用于识别AR模型的阶数。

**解读步骤：**

1. **确定AR阶数：**
   - 找到最后一个显著的PACF值（超出置信区间）
   - 该滞后阶数就是AR(p)的p值

2. **检查拖尾：**
   - 如果PACF拖尾，可能是ARMA模型或MA模型

### 综合分析

**AR(p)模型识别：**
- ACF拖尾
- PACF在滞后p处截断

**MA(q)模型识别：**
- ACF在滞后q处截断
- PACF拖尾

**ARMA(p,q)模型识别：**
- ACF和PACF都拖尾
- 需要通过其他方法（如信息准则）确定阶数

## 与ARMA模型的关系

### 自回归模型 AR(p)

**定义：**
$$X_t = \phi_1 X_{t-1} + \phi_2 X_{t-2} + \cdots + \phi_p X_{t-p} + \varepsilon_t$$

**ACF特征：**
- 拖尾，遵循与自回归系数相关的差分方程
- 可以表现为指数衰减或阻尼正弦波

**PACF特征：**
- 在滞后p处截断（$\pi_p = \phi_p$）
- 对于k > p，$\pi_k = 0$

**示例：AR(1)模型**
- $X_t = \phi X_{t-1} + \varepsilon_t$
- ACF：$\rho_k = \phi^k$（指数衰减）
- PACF：$\pi_1 = \phi$，$\pi_k = 0$（k > 1）

### 移动平均模型 MA(q)

**定义：**
$$X_t = \varepsilon_t + \theta_1 \varepsilon_{t-1} + \theta_2 \varepsilon_{t-2} + \cdots + \theta_q \varepsilon_{t-q}$$

**ACF特征：**
- 在滞后q处截断
- $\rho_k = 0$（k > q）

**PACF特征：**
- 拖尾，表现更复杂
- 通常为阻尼正弦或指数衰减

**示例：MA(1)模型**
- $X_t = \varepsilon_t + \theta \varepsilon_{t-1}$
- ACF：$\rho_1 = \frac{\theta}{1+\theta^2}$，$\rho_k = 0$（k > 1）
- PACF：拖尾，$\pi_k = (-\theta)^k$

### ARMA(p,q)模型

**定义：**
$$X_t = \sum_{i=1}^p \phi_i X_{t-i} + \varepsilon_t + \sum_{j=1}^q \theta_j \varepsilon_{t-j}$$

**ACF和PACF特征：**
- 都拖尾
- 模式更复杂
- 需要结合其他方法（如AIC、BIC）确定阶数

### 模型选择流程图

```
开始

```

## 样本ACF/PACF的计算

### Yule-Walker方程

Yule-Walker方程提供了理论ACF与模型系数之间的关系：

对于AR(p)模型：
$$\rho_k = \sum_{j=1}^p \phi_j \rho_{k-j}, \quad k = 1, 2, \ldots, p$$

矩阵形式：
$$
\begin{bmatrix}
1 & \rho_1 & \cdots & \rho_{p-1} \\
\rho_1 & 1 & \cdots & \rho_{p-2} \\
\vdots & \vdots & \ddots & \vdots \\
\rho_{p-1} & \rho_{p-2} & \cdots & 1
\end{bmatrix}
\begin{bmatrix}
\phi_1 \\
\phi_2 \\
\vdots \\
\phi_p
\end{bmatrix}
=
\begin{bmatrix}
\rho_1 \\
\rho_2 \\
\vdots \\
\rho_p
\end{bmatrix}
$$

### 递推计算PACF

使用Levinson-Durbin算法递推计算PACF：

1. **初始化：**
   $\phi_{11} = \rho_1$

2. **递推步骤：**
   对于k = 2, 3, ..., p：
   $$\phi_{kk} = \frac{\rho_k - \sum_{j=1}^{k-1} \phi_{k-1,j}\rho_{k-j}}{1 - \sum_{j=1}^{k-1} \phi_{k-1,j}\rho_j}$$
   $$\phi_{kj} = \phi_{k-1,j} - \phi_{kk}\phi_{k-1,k-j}, \quad j = 1, 2, \ldots, k-1$$

其中 $\phi_{kk}$ 就是第k阶偏自相关系数。

### 置信区间

对于白噪声序列，样本自相关系数的近似分布：

$$\sqrt{n} \hat{\rho}_k \approx N(0, 1)$$

95%置信区间：
$$\pm \frac{1.96}{\sqrt{n}}$$

如果样本自相关系数超出此区间，认为该滞后显著。

### 手动计算ACF和PACF

```python
def calculate_acf(series, max_lag):
    """计算自相关函数"""
    n = len(series)
    mean = np.mean(series)
    var = np.var(series, ddof=0)
    
    acf = []
    for k in range(max_lag + 1):
        if k == 0:
            acf.append(1.0)
        else:
            numerator = np.sum((series[k:] - mean) * (series[:-k] - mean))
            acf.append(numerator / (n * var))
    
    return np.array(acf)

def calculate_pacf(acf):
    """使用Yule-Walker方程计算偏自相关函数"""
    max_lag = len(acf) - 1
    pacf = np.zeros(max_lag + 1)
    pacf[0] = 1.0
    
    if max_lag >= 1:
        pacf[1] = acf[1]
    
    # 使用递推算法
    for k in range(2, max_lag + 1):
        # 计算当前阶数的PACF
        numerator = acf[k] - np.sum(pacf[1:k] * acf[k-1:0:-1])
        denominator = 1 - np.sum(pacf[1:k] * acf[1:k])
        pacf[k] = numerator / denominator
    
    return pacf

# 示例：分析AR(1)过程
np.random.seed(42)
ar_data = ar1_data  # 使用之前生成的AR(1)数据

# 计算ACF和PACF
max_lag = 10
my_acf = calculate_acf(ar_data, max_lag)
my_pacf = calculate_pacf(my_acf)

# 与statsmodels结果比较
from statsmodels.tsa.stattools import acf, pacf
sm_acf = acf(ar_data, nlags=max_lag, fft=False)
sm_pacf = pacf(ar_data, nlags=max_lag, method='ywm')

print("手动计算与statsmodels比较：")
print(f"ACF - 前5个滞后: {my_acf[:5]}")
print(f"ACF - statsmodels: {sm_acf[:5]}")
print(f"PACF - 前5个滞后: {my_pacf[:5]}")
print(f"PACF - statsmodels: {sm_pacf[:5]}")
```

### 模型选择和验证

```python
from statsmodels.tsa.arima.model import ARIMA
from sklearn.metrics import mean_squared_error

def select_arima_model(series, max_p=3, max_q=3, d=0):
    """基于ACF/PACF和AIC选择ARIMA模型"""
    
    # 绘制ACF和PACF以获得初始判断
    fig, axes = plt.subplots(1, 2, figsize=(12, 4))
    plot_acf(series, ax=axes[0], lags=20)
    plot_pacf(series, ax=axes[1], lags=20, method='ywm')
    plt.show()
    
    # 网格搜索最佳模型
    best_aic = float('inf')
    best_order = None
    results = []
    
    for p in range(max_p + 1):
        for q in range(max_q + 1):
            try:
                model = ARIMA(series, order=(p, d, q))
                fitted = model.fit()
                
                if fitted.aic < best_aic:
                    best_aic = fitted.aic
                    best_order = (p, d, q)
                
                results.append({
                    'order': (p, d, q),
                    'AIC': fitted.aic,
                    'BIC': fitted.bic,
                    'LLF': fitted.llf
                })
            except:
                continue
    
    results_df = pd.DataFrame(results)
    results_df = results_df.sort_values('AIC').reset_index(drop=True)
    
    print("模型比较结果（按AIC排序）：")
    print(results_df.head())
    print(f"\n最佳模型: ARIMA{best_order}")
    print(f"AIC: {best_aic:.2f}")
    
    # 拟合最佳模型
    best_model = ARIMA(series, order=best_order)
    best_fitted = best_model.fit()
    
    # 模型诊断
    print("\n模型摘要:")
    print(best_fitted.summary())
    
    # 残差诊断
    residuals = best_fitted.resid
    fig, axes = plt.subplots(2, 2, figsize=(12, 8))
    
    # 残差时间序列
    axes[0, 0].plot(residuals)
    axes[0, 0].set_title('残差时间序列')
    axes[0, 0].set_xlabel('时间')
    axes[0, 0].set_ylabel('残差')
    
    # 残差ACF
    plot_acf(residuals, ax=axes[0, 1], lags=20, title='残差ACF')
    
    # 残差QQ图
    from scipy import stats
    stats.probplot(residuals, dist="norm", plot=axes[1, 0])
    axes[1, 0].set_title('残差QQ图')
    
    # 残差直方图
    axes[1, 1].hist(residuals, bins=30, density=True)
    x = np.linspace(min(residuals), max(residuals), 100)
    axes[1, 1].plot(x, stats.norm.pdf(x, np.mean(residuals), np.std(residuals)), 'r-')
    axes[1, 1].set_title('残差分布')
    
    plt.tight_layout()
    plt.show()
    
    # Ljung-Box检验（白噪声检验）
    from statsmodels.stats.diagnostic import acorr_ljungbox
    lb_test = acorr_ljungbox(residuals, lags=10)
    print("\nLjung-Box检验结果:")
    print(lb_test)
    
    return best_fitted

# 示例：对ARMA(1,1)数据选择模型
print("为ARMA(1,1)过程选择模型")
model_results = select_arima_model(arma11_data[:500], max_p=3, max_q=3)
```

import yfinance as yf

# 获取股票价格数据
def analyze_stock_data(symbol='AAPL', start_date='2020-01-01'):

    # 获取数据
    stock = yf.download(symbol, start=start_date)

    axes[0, 0].tick_params(axis='x', rotation=45)
    
    # 收益率时间序列
    axes[0, 1].plot(returns.index, returns.values)
    axes[0, 1].set_title(f'{symbol} 收益率')
    axes[0, 1].set_xlabel('日期')
    axes[0, 1].set_ylabel('收益率')
    axes[0, 1].tick_params(axis='x', rotation=45)
    
    # 收益率ACF
    plot_acf(returns, ax=axes[1, 0], lags=30, title='收益率ACF')
    
    # 收益率PACF
    plot_pacf(returns, ax=axes[1, 1], lags=30, title='收益率PACF', method='ywm')
    
    plt.tight_layout()
    plt.show()
    
    # 计算并显示关键统计量
    print(f"{symbol} 股票收益统计分析:")
    print(f"平均收益率: {returns.mean():.6f}")
    print(f"收益率标准差: {returns.std():.6f}")
    print(f"收益率偏度: {returns.skew():.4f}")
    print(f"收益率峰度: {returns.kurtosis():.4f}")
    
    # 检查收益率的一阶自相关
    first_lag_acf = returns.autocorr(lag=1)
    print(f"一阶自相关系数: {first_lag_acf:.4f}")
    
    # 检验收益率是否为白噪声
    from statsmodels.stats.diagnostic import acorr_ljungbox
    lb_test = acorr_ljungbox(returns, lags=10)
    print("\nLjung-Box检验（收益率是否为白噪声）:")
    print(f"前10个滞后的p值: {lb_test['lb_pvalue'].values}")
    
    if any(lb_test['lb_pvalue'] < 0.05):
        print("拒绝白噪声假设，收益率序列存在自相关")
    else:
        print("不能拒绝白噪声假设，收益率序列可能为白噪声")

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from statsmodels.graphics.tsaplots import plot_acf, plot_pacf

# 生成模拟气温数据（有季节性和自相关）
np.random.seed(42)

## 常见问题和注意事项

### 1. 非平稳序列的处理

**问题：** ACF缓慢衰减，难以识别模式

**解决方法：**
- 进行差分：$\Delta X_t = X_t - X_{t-1}$
- 检查差分后的ACF/PACF
- 可能需要进行多次差分

### 2. 季节性模式

**问题：** 在季节性滞后处（如12）出现显著峰值

**解决方法：**
- 季节差分：$\Delta_{12} X_t = X_t - X_{t-12}$
- 使用季节性ARIMA模型
- 结合去季节化方法

### 3. 异常值影响

**问题：** 异常值可能导致ACF/PACF估计不准确

**解决方法：**
- 识别和处理异常值
- 使用鲁棒的自相关估计方法
- 对数据进行适当的变换

### 4. 样本量限制

**问题：** 小样本下ACF/PACF估计不稳定

**解决方法：**
- 增加样本量
- 使用更少的滞后阶数
- 谨慎解释结果

### 5. 多重假设检验

**问题：** 同时检验多个滞后的显著性会增加第一类错误

**解决方法：**
- 使用Ljung-Box等综合检验
- 调整显著性水平
- 关注整体模式而非单个滞后

## 总结

自相关和偏自相关函数是时间序列分析的核心工具：

**核心概念：**
1. ACF衡量序列与滞后版本的总相关性
2. PACF衡量去除中间影响后的直接相关性
3. ACF/PACF图用于识别ARMA模型
4. 样本ACF/PACF需要足够大的样本量

**关键要点：**
1. ACF截断 → MA模型
2. PACF截断 → AR模型
3. 都拖尾 → ARMA模型
4. 缓慢衰减 → 可能需要差分

**实际应用：**
1. 模型识别和选择
2. 模型诊断和验证
3. 季节性模式识别
4. 白噪声检验

**注意事项：**
1. 确保序列平稳后再分析
2. 注意季节性效应
3. 考虑异常值影响
4. 综合多种方法判断

正确理解和应用ACF/PACF是成功进行时间序列建模的关键第一步。
