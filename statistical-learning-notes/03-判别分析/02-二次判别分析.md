# 二次判别分析

## 什么是QDA？

二次判别分析（Quadratic Discriminant Analysis，QDA）是LDA的**扩展**。

简单说：**允许不同类别有不同的协方差矩阵，决策边界是二次的。**

---

## LDA vs QDA

### LDA 的假设

- 所有类别共享相同的协方差矩阵
- 决策边界是线性的

### QDA 的假设

- 每个类别有自己的协方差矩阵
- 决策边界是二次的

| 特点 | LDA | QDA |
|------|-----|-----|
| 协方差矩阵 | 共享 | 各自不同 |
| 决策边界 | 线性 | 二次 |
| 参数量 | 少 | 多 |
| 灵活性 | 低 | 高 |
| 过拟合风险 | 低 | 高 |

---

## 数学原理

### 判别函数

对于类别 k，QDA 的判别函数为：

```
δₖ(x) = -1/2 log|Σₖ| - 1/2 (x - μₖ)ᵀ Σₖ⁻¹ (x - μₖ) + log(πₖ)
```

- x：输入样本
- Σₖ：类别 k 的协方差矩阵
- μₖ：类别 k 的均值
- πₖ：类别 k 的先验概率

### 和 LDA 的区别

LDA 的判别函数（共享 Σ）：

```
δₖ(x) = xᵀ Σ⁻¹ μₖ - 1/2 μₖᵀ Σ⁻¹ μₖ + log(πₖ)
```

QDA 多了 Σₖ 相关的项。

---

## 决策边界

### 二分类情况

决策边界是二次的：

```
δ₁(x) = δ₂(x)
```

展开后得到：

```
xᵀ A x + bᵀ x + c = 0
```

这是二次曲线：
- 二维：椭圆、双曲线、抛物线
- 高维：二次曲面

### 为什么叫"二次"？

因为决策边界是二次的。

## 可视化

### LDA 的决策边界

```
类别1 ●●●●●●●●●●
     ╱          ╲
    ╱            ╲
━━━━╳──────────────╳━━━━ ← 线性边界
    ╲            ╱

```

### QDA 的决策边界

```
类别1 ●●●●●●●●●●
     ╱          ╲
    ╱  ●●●     ╲
    ●      ●    ╲
    ●     ╳─────╳ ← 二次边界
    ●      ●    ╱
     ╲  ●●●    ╱
     ╲          ╱
类别2 ●●●●●●●●●●
```

QDA 可以拟合更复杂的边界。

## 参数数量

### LDA

- 每个类别的均值：K × p
- 共享的协方差矩阵：p × p

总计：Kp + p²

### QDA

- 每个个类别的均值：K × p
- 每个类别的协方差矩阵：K × p × p

总计：Kp + Kp² = K(p² + p)

当 K 很大时，QDA 的参数量比 LDA 多很多。

---

## 偏差-方差权衡

### LDA

- 参数少，方差低
- 假设强，偏差可能高
- 样本少时表现好

### QDA

- 参数多，方差高
- 假设弱，偏差低
- 样本多时表现好

### 什么时候选哪个？

| 情况 | 推荐 |
|------|------|
| 样本少 | LDA |
| 样本多 | QDA |
| 协方差相似 | LDA |
| 协方差差异大 | QDA |
| 需要简单模型 | LDA |
| 需要灵活模型 | QDA |

---

## 优点

✅ **更灵活**
- 可以适应不同形状的类别分布
- 决策边界更复杂

✅ **更准确（在某些情况下）**
- 当协方差矩阵差异大时，效果更好

✅ **不强求线性边界**
- 可以拟合非线性关系

---

## 缺点

❌ **参数多**
- 容易过拟合
- 需要更多数据

❌ **计算复杂**
- 每个类别要估计协方差矩阵
- 预测时计算量更大

❌ **对异常值敏感**
- 像其他判别分析一样

---

## 何时使用QDA？

### 适合的情况

✅ 不同类别的分布形状差异大
✅ 决策边界明显非线性
✅ 有足够的训练数据
✅ 需要更高的分类精度

### 不适合的情况

❌ 数据量小
❌ 各类别协方差相似
❌ 需要简单可解释的模型

---

## 实际例子

### 例子：两个类别的分布

类别1：协方差矩阵是"瘦长"的
类别2：协方差矩阵是"扁圆"的

### LDA 的表现

假设两者协方差相同，决策边界是直的，效果不好。

### QDA 的表现

分别考虑各自的协方差，决策边界是曲线，效果更好。

---

## 与其他分类器的比较

| 特点 | LDA | QDA | 逻辑回归 | 决策树 |
|------|-----|-----|---------|--------|
| 决策边界 | 线性 | 二次 | 线性 | 非线性 |
| 参数量 | 少 | 多 | 中等 | 中等 |
| 灵活性 | 低 | 高 | 低 | 高 |
| 过拟合风险 | 低 | 高 | 低 | 中等 |
| 计算速度 | 快 | 中等 | 快 | 中等 |

---

## 实践技巧

### 1. 检查协方差

比较各类的协方差矩阵，差异大时用 QDA。

### 2. 交叉验证

用交叉验证比较 LDA 和 QDA 的效果。

### 3. 正则化

可以用正则化防止过拟合。

### 4. 特征选择

减少特征数量，降低参数量。

## 代码示例

### Scikit-learn

```python
from sklearn.discriminant_analysis import QuadraticDiscriminantAnalysis
from sklearn.model_selection import cross_val_score

# QDA 分类器
qda = QuadraticDiscriminantAnalysis()

# 交叉验证
scores = cross_val_score(qda, X, y, cv=5)

print(f"平均准确率: {scores.mean():.3f}")
print(f"标准差: {scores.std():.3f}")

# 拟合模型
qda.fit(X_train, y_train)

# 预测
y_pred = qda.predict(X_test)

# 查看各类的协方差矩阵
for i, cov in enumerate(qda.covariance_):
    print(f"类别 {i} 的协方差矩阵:")
    print(cov)
```

## 正则化 QDA

### 问题

当特征多样本少时，协方差矩阵可能奇异（不可逆）。

### 解决方法

对协方差矩阵正则化：

```
Σₖ = (1 - α) Σₖ + α I
```

- α：正则化参数
- I：单位矩阵

### Scikit-learn

```python
# 正则化 QDA
qda_reg = QuadraticDiscriminantAnalysis(reg_param=0.1)
```

## 常见问题

### Q: 什么时候用 QDA 而不是 LDA？

A:
- 不同类别的分布形状差异大
- 有足够的数据
- 决策边界明显非线性

### Q: QDA 的二次边界一定是曲线吗？

A:
- 如果协方差矩阵成比例，边界退化成直线（等于 LDA）
- 一般情况是二次曲线

### Q: QDA 比 LDA 好吗？

A:
- 不一定，取决于数据
- 数据多且协方差差异大时 QDA 好
- 数据少或协方差相似时 LDA 好

## 总结

**二次判别分析 (QDA)**：

1. 允许各类有不同协方差矩阵
2. 决策边界是二次的
3. 更灵活，但参数更多
4. 适合协方差差异大的情况
5. 需要更多数据避免过拟合

**核心思想**：
- 每个类别用自己的协方差
- 可以适应不同形状的分布
- 决策边界更复杂

**关键要点**：
- 比 LDA 灵活但参数多
- 样本多时效果好
- 用交叉验证选择 LDA 或 QDA
- 可以正则化防止过拟合

**记住**：QDA 就像"灵活的艺术家"，能画出更复杂的决策边界，但需要更多"颜料"（数据）！
