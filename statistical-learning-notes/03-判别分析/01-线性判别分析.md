# 线性判别分析

## 什么是LDA？

线性判别分析（Linear Discriminant Analysis，LDA）是一种**分类算法**和**降维技术**。

简单说：**找到一个线性组合，让不同类别尽量分开，同一类别尽量聚集。**

---

## 核心思想

### 目标

最大化**类间距离**，最小化**类内距离**。

```
类间距离越大越好   类内距离越小越好
    ↓                    ↓
  ●●●                 ●●●
●     ●             ●   ●
●     ●             ●   ●
  ●●●                 ●●●
```

### 几何理解

把数据投影到一条直线上，使得：

- 不同类别的中心相距远
- 同一类别的点紧凑

---

## 数学原理

### 假设

LDA 假设：

1. 每个类别的数据服从正态分布
2. 所有类别有相同的协方差矩阵
3. 类先验概率相等（可以调整）

### 判别函数

对于类别 k，判别函数为：

```
δₖ(x) = xᵀ Σ⁻¹ μₖ - 1/2 μₖᵀ Σ⁻¹ μₖ + log(πₖ)
```

- x：输入样本
- Σ：共享的协方差矩阵
- μₖ：类别 k 的均值
- πₖ：类别 k 的先验概率

### 分类规则

```
预测类别 = argmax δₖ(x)
         k
```

选择使判别函数最大的类别。

---

## 决策边界

### 二分类情况

决策边界是线性的：

```
δ₁(x) = δ₂(x)
```

展开后得到：

```
wᵀx + b = 0
```

这是一条直线（高维是超平面）。

### 为什么叫"线性"？

因为决策边界是线性的。

---

## 投影角度

### 找最佳投影方向

LDA 找一个方向 w，投影后：

- 类间方差最大
- 类内方差最小

### 目标函数

```
最大化：wᵀ S_B w / (wᵀ S_W w)
```

- S_B：类间散度矩阵
- S_W：类内散度矩阵

### 解

最优方向 w 是 S_W⁻¹ S_B 的特征向量。

---

## LDA vs PCA

### PCA（主成分分析）

- 无监督
- 最大化总方差
- 不考虑类别信息

### LDA

- 有监督
- 最大化类间差异
- 考虑类别信息

| 特点 | PCA | LDA |
|------|-----|-----|
| 监督 | 无监督 | 有监督 |
| 目标 | 最大化方差 | 最大化类间差异 |
| 分类 | 不考虑类别 | 考虑类别 |
| 降维 | 可以 | 可以 |
| 分类性能 | 不是为分类设计 | 专门为分类 |

---

## LDA 用于降维

### 降维到 d 维

LDA 可以把数据降到 d 维：

```
d ≤ K - 1
```

K 是类别数。

### 例子

- 2 类：降到 1 维
- 3 类：最多到 2 维
- 4 类：最多到 3 维

---

## 优点

✅ **考虑类别信息**
- 专门为分类设计
- 通常分类效果好

✅ **可解释性强**
- 可以看出哪些特征重要

✅ **计算高效**
- 有闭式解，计算快

✅ **适合小样本**
- 在样本较少时表现好

✅ **可以降维**
- 既能分类又能降维

---

## 缺点

❌ **假设强**
- 假设正态分布
- 假设相同协方差矩阵

❌ **对异常值敏感**
- 像线性模型一样

❌ **类别不能太多**
- 降维维度 ≤ K-1

❌ **线性决策边界**
- 对于复杂的非线性问题表现差

---

## 何时使用LDA？

### 适合的情况

✅ 数据类别明确
✅ 类别分布接近正态
✅ 需要分类和降维
✅ 特征数多于样本数

### 不适合的情况

❌ 决策边界是非线性的
❌ 数据不满足正态假设
❌ 类别协方差差异大

## 实际例子

### 例子：鸢尾花分类

数据：150朵鸢尾花，3个品种，4个特征

特征：
- 花萼长度、花萼宽度
- 花瓣长度、花瓣宽度

### LDA 的过程

1. 计算每个品种的均值
2. 计算类内和类间散度
3. 找最佳投影方向
4. 投影数据
5. 分类

### 结果

LDA 可以很好地把3个品种分开。

## 与其他分类器的比较

| 特点 | LDA | 逻辑回归 | 决策树 |
|------|-----|---------|--------|
| 决策边界 | 线性 | 线性 | 非线性 |
| 假设 | 正态分布 | 无 | 无 |
| 可解释性 | 强 | 强 | 强 |
| 计算速度 | 快 | 快 | 中等 |
| 对异常值 | 敏感 | 敏感 | 不敏感 |
| 特征相关 | 能处理 | 需要处理 | 能处理 |

## 实践技巧

### 1. 检查假设

- 正态性：画图或检验
- 协方差矩阵：比较各类的协方差

### 2. 特征缩放

标准化特征，效果更好。

### 3. 处理异常值

检测并处理异常值。

### 4. 类别平衡

类别不平衡时，可以调整先验概率。

## 代码示例

### Scikit-learn

```python
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# 分割数据
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# LDA 分类器
lda = LinearDiscriminantAnalysis()

# 训练
lda.fit(X_train, y_train)

# 预测
y_pred = lda.predict(X_test)

# 评估
print(accuracy_score(y_test, y_pred))

# LDA 降维
lda_2d = LinearDiscriminantAnalysis(n_components=2)
```

用核技巧处理非线性问题。

### 增量 LDA

适合大数据集，可以增量学习。

## 常见问题

### Q: LDA 和逻辑回归的区别？

A:
- LDA：基于概率分布假设，需要正态分布
- 逻辑回归：直接建模条件概率，假设少

### Q: 什么时候用 LDA 而不是逻辑回归？

A:
- 需要降维时用 LDA
- 数据接近正态分布时 LDA 效果好
- 需要考虑类别先验时用 LDA

### Q: LDA 的决策边界为什么是线性的？

A:
- 因为假设各类有相同的协方差矩阵
- 判别函数的差是线性的

## 总结

**线性判别分析 (LDA)**：

1. 最大化类间距离，最小化类内距离
2. 假设正态分布和相同协方差矩阵
3. 线性决策边界
4. 既能分类又能降维
5. 计算高效，可解释性强

**核心思想**：
- 找一个投影方向
- 让不同类别尽量分开
- 让同一类别尽量聚集

**关键要点**：
- 考虑类别信息，适合分类
- 可以降维，但维度有限
- 假设较强，需要检查
- 和 PCA 是互补的方法

**记住**：LDA 就像"分类专家"，专门为分类而设计，同时还能帮你降维！
