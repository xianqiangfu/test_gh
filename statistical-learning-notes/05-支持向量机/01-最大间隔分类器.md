# 最大间隔分类器

## 什么是最大间隔分类器？

最大间隔分类器是**支持向量机（SVM）的基础**。

简单说：**找一个分类边界，让两类数据离边界都尽可能远。**

---

## 核心思想

### 最大间隔

想象你要在两组人之间画一条线：

```
●●●●●●              ○○○○○○
   │                    │
   │←──── 间隔 ────→│
   │                    │
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

你要画一条线，让两边的人离线都尽可能远。

### 为什么最大间隔？

- 间隔越大，模型越"安全"
- 对新数据更鲁棒
- 泛化能力更强

---

## 决策边界

### 线性可分

如果两类数据可以用一条直线完全分开，称为**线性可分**。

```
   ●●●●●●

●●●●  ○○○○

   ○○○○○○
```

### 决策边界

中间的线就是决策边界。

```
   ●●●●●●

●●●●━━━━○○○○

   ○○○○○○
```

---

## 支持向量

### 什么是支持向量？

**支持向量**是离决策边界最近的那些点。

```
   ●●●●●●
    ★    ★
●●●●━━━━○○○○
    ★    ★
   ○○○○○○
```

★ 标记的点就是支持向量。

### 重要性

- 只有支持向量决定边界
- 其他点不影响边界
- 间隔由支持向量决定

---

## 数学表示

### 决策边界

```
wᵀx + b = 0
```

### 间隔边界

```
wᵀx + b = 1   （类别1的边界）
wᵀx + b = -1  （类别0的边界）
```

### 间隔大小

```
间隔 = 2 / ||w||
```

最大化间隔等价于最小化 ||w||。

---

## 优化问题

### 原始问题

```
最小化：1/2 × ||w||²
约束条件：yᵢ(wᵀxᵢ + b) ≥ 1，对所有 i
```

### 对偶问题

```
最大化：Σαᵢ - 1/2 ΣΣαᵢαⱼyᵢyⱼxᵢᵀxⱼ
约束条件：Σαᵢyᵢ = 0，αᵢ ≥ 0
```

- αᵢ：拉格朗日乘子
- 只有支持向量的 αᵢ > 0

---

## 几何解释

### 权重向量 w

w 垂直于决策边界：

```
决策边界
━━━━━━━━━━━━
      ↑ w
      │
      │
```

### 预测

```
如果 wᵀx + b > 0，预测为类别1
如果 wᵀx + b < 0，预测为类别0
```

---

## 优点

✅ **理论基础扎实**
- 有凸优化保证
- 有全局最优解

✅ **间隔最大化**
- 泛化能力强
- 对新数据鲁棒

✅ **只依赖支持向量**
- 计算效率高
- 可以处理大量数据

✅ **可以扩展**
- 用核函数处理非线性

---

## 缺点

❌ **只适用于线性可分**
- 实际数据很难线性可分
- 需要扩展到软间隔或核函数

❌ **对数据量敏感**
- 数据量大时计算慢
- 内存占用高

❌ **对噪声敏感**
- 异常值影响大
- 需要软间隔

---

## 实际例子

### 简单二分类

数据：
- 类别0：(1, 1), (2, 2)
- 类别1：(4, 4), (5, 5)

决策边界可能是：
```
x + y = 6
```

支持向量是 (2, 2) 和 (4, 4)。

## 代码示例

### Scikit-learn

```python
from sklearn.svm import LinearSVC
from sklearn.datasets import make_blobs
import matplotlib.pyplot as plt

# 生成数据
X, y = make_blobs(n_samples=50, centers=2, random_state=42)

# 创建模型
clf = LinearSVC(C=1000)  # 大C接近硬间隔

# 训练
clf.fit(X, y)

# 查看参数
print(f"权重: {clf.coef_}")
print(f"偏置: {clf.intercept_}")

# 查看支持向量
print(f"支持向量数量: {len(clf.support_vectors_)}")
```

### 可视化

```python
import numpy as np

# 创建网格
xx, yy = np.meshgrid(np.linspace(X[:, 0].min()-1, X[:, 0].max()+1, 100),
                     np.linspace(X[:, 1].min()-1, X[:, 1].max()+1, 100))

# 预测
Z = clf.decision_function(np.c_[xx.ravel(), yy.ravel()])
Z = Z.reshape(xx.shape)

# 绘制
plt.contourf(xx, yy, Z, alpha=0.3)
plt.contour(xx, yy, Z, colors='k', levels=[-1, 0, 1], linestyles=['--', '-', '--'])
plt.scatter(X[:, 0], X[:, 1], c=y)
plt.show()
```

---

## 参数

### C 参数

C 是正则化参数：

- C 大：接近硬间隔，严格分离
- C 小：允许一些错误，更灵活

### 关系

```
C 大 → 硬间隔 → 可能过拟合
C 小 → 软间隔 → 更鲁棒
```

---

## 最大间隔分类器的局限

### 硬间隔问题

如果数据不是线性可分，硬间隔无法工作：

```
   ●●●●●●
        ●
●●●●    ○○○○
    ○○○○○○
```

这个数据集不是线性可分的。

### 解决方法

1. **软间隔**：允许一些分类错误
2. **核函数**：映射到高维空间

---

## 与其他分类器的比较

| 特点 | 最大间隔分类器 | 逻辑回归 | 感知机 |
|------|---------------|---------|--------|
| 目标 | 最大化间隔 | 最大化似然 | 最小化错误 |
| 决策边界 | 线性 | 线性 | 线性 |
| 全局最优 | 有 | 有 | 不一定 |
| 支持向量 | 有 | 无 | 无 |

## 总结

**最大间隔分类器**：

1. 找到让两类离边界都尽可能远的边界
2. 间隔由支持向量决定
3. 有全局最优解
4. 是SVM的基础
5. 只适用于线性可分数据

**核心思想**：
- 最大化间隔
- 只依赖支持向量
- 理论基础扎实

**关键要点**：
- 间隔越大，泛化越强
- 只用支持向量训练
- 是硬间隔SVM
- 需要扩展处理非线性

**记住**：最大间隔分类器就像"安全驾驶"，离两边都远一点，更不容易出问题！
