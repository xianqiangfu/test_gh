# 核函数

## 什么是核函数？

核函数是**SVM处理非线性问题的核心技术**。

简单说：**把数据映射到高维空间，让原来不可分的数据变得可分。**

---

## 为什么需要核函数？

### 线性不可分问题

有些数据在低维空间中无法用直线分开：

```
   ●●●
 ●   ●
●  ○  ●
 ●   ●
   ●●●
```

这是一类数据围绕另一类，一条直线无法分开。

### 映射到高维

如果映射到3D空间，可能可以用一个平面分开：

```
   3D空间
    ↑
    │      ●●●
    │    ●   ●
    │   ●  ○  ●
    │    ●   ●
    │      ●●●
    └────────────→
```

---

## 核技巧

### 直接映射的问题

直接映射到高维空间：
- 计算复杂
- 存储开销大
- 甚至可能无限维

### 核技巧

不显式映射，直接计算高维空间的内积：

```
K(xᵢ, xⱼ) = φ(xᵢ)ᵀ φ(xⱼ)
```

- φ(x)：映射函数
- K(xᵢ, xⱼ)：核函数

### 优势

- 不需要知道 φ(x)
- 计算量小
- 可以处理无限维

---

## 常见核函数

### 1. 线性核

```
K(x, y) = xᵀy
```

- 等价于没有映射
- 用于线性可分问题
- 计算最快

### 2. 多项式核

```
K(x, y) = (xᵀy + c)ᵈ
```

- c：偏置项
- d：多项式次数
- 可以拟合多项式边界

### 3. 高斯核（RBF核）

```
K(x, y) = exp(-γ × ||x - y||²)
```

- γ：控制影响范围
- 最常用
- 可以拟合任意复杂边界

### 4. Sigmoid核

```
K(x, y) = tanh(α × xᵀy + c)
```

- 类似神经网络
- 不常用

---

## 高斯核（RBF）

### 为什么最常用？

✅ **灵活**
- 可以拟合任意复杂边界
- 理论上可以逼近任何函数

✅ **数值稳定**
- 计算稳定
- 不容易数值问题

✅ **参数少**
- 只有 γ 需要调

### 参数 γ

```
K(x, y) = exp(-γ × ||x - y||²)
```

- γ 大：影响范围小，边界复杂
- γ 小：影响范围大，边界简单

```
γ 大 → 高方差 → 可能过拟合
γ 小 → 高偏差 → 可能欠拟合
```

---

## 核函数的选择

| 核函数 | 适用场景 | 特点 |
|--------|---------|------|
| 线性核 | 特征多、线性可分 | 快、简单 |
| 多项式核 | 多项式关系 | 可解释性强 |
| RBF核 | 复杂非线性关系 | 最灵活、常用 |
| Sigmoid核 | 神经网络类似 | 不常用 |

### 选择策略

1. **先用线性核**
   - 简单快速
   - 建立基线

2. **如果效果不好，用RBF核**
   - 通常效果最好
   - 需要调参

3. **有先验知识用多项式核**
   - 知道数据是多项式关系

---

## 核函数的数学

### 理论条件

要成为有效的核函数，必须满足**Mercer条件**：

1. 对称性：K(x, y) = K(y, x)
2. 正定性：Gram矩阵是半正定的

### 常用核都满足

线性、多项式、RBF、Sigmoid等常用核都满足Mercer条件。

## 代码示例

### Scikit-learn

```python
from sklearn.svm import SVC
from sklearn.datasets import make_circles
import matplotlib.pyplot as plt

# 生成环形数据
X, y = make_circles(n_samples=100, factor=0.3, noise=0.05)

# 不同核函数
kernels = ['linear', 'poly', 'rbf', 'sigmoid']

fig, axes = plt.subplots(2, 2, figsize=(10, 10))

for ax, kernel in zip(axes.flat, kernels):
    # 创建模型
    clf = SVC(kernel=kernel, C=1.0)
    clf.fit(X, y)

    # 绘制决策边界
    x_min, x_max = X[:, 0].min() - 0.5, X[:, 0].max() + 0.5
    y_min, y_max = X[:, 1].min() - 0.5, X[:, 1].max() + 0.5
    xx, yy = np.meshgrid(np.arange(x_min, x_max, 0.02),
                         np.arange(y_min, y_max, 0.02))
    Z = clf.predict(np.c_[xx.ravel(), yy.ravel()])
    Z = Z.reshape(xx.shape)

    ax.contourf(xx, yy, Z, alpha=0.3)
    ax.scatter(X[:, 0], X[:, 1], c=y)
    ax.set_title(f'{kernel} 核')

plt.tight_layout()
plt.show()
```

## 参数调优

### RBF核的参数

| 参数 | 说明 | 推荐范围 |
|------|------|----------|
| C | 正则化强度 | 0.1, 1, 10, 100 |
| γ | 核宽度 | 0.001, 0.01, 0.1, 1 |

### 调参策略

1. 固定 C = 1
2. 用网格搜索找 γ
3. 然后调 C
4. 用交叉验证评估

```python
from sklearn.model_selection import GridSearchCV

param_grid = {
    'C': [0.1, 1, 10, 100],
    'gamma': [0.001, 0.01, 0.1, 1]
}

grid = GridSearchCV(SVC(kernel='rbf'), param_grid, cv=5)
grid.fit(X, y)

print(grid.best_params_)
```

## 核函数的优缺点

### 优点

✅ **处理非线性**
- 可以处理复杂非线性关系

✅ **理论基础**
- 有完整的数学理论

✅ **计算高效**
- 不需要显式映射

### 缺点

❌ **调参复杂**
- C、γ 等参数需要仔细调

❌ **解释性差**
- 难以解释为什么这么分类

❌ **计算量大**
- 数据量大时慢
- 需要存储核矩阵

## 常见问题

### Q: 如何选择核函数？

A:
- 默认RBF核
- 数据特征多时考虑线性核
- 有先验知识用多项式核

### Q: RBF核的 γ 怎么选？

A:
- 用网格搜索
- γ 在 [0.001, 1] 范围尝试

### Q: 核函数会过拟合吗？

A:
- 会，尤其是RBF核
- 用交叉验证和正则化防止

## 总结

**核函数**：

1. 把数据映射到高维空间
2. 使非线性可分数据线性可分
3. 不需要显式映射，计算高效
4. RBF核最常用
5. 是SVM处理非线性的关键

**核心思想**：
- 核技巧：直接计算高维内积
- Mercer条件：保证有效性
- 灵活选择不同核

**关键要点**：
- RBF核通常最好
- 需要调 C 和 γ
- 可以处理任意复杂边界
- 理论基础完善

**记住**：核函数就像"魔法眼镜"，让你在高维空间看到原来看不到的东西！
