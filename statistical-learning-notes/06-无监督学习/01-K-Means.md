# K-Means

## 什么是K-Means？

K-Means是最经典、最常用的**聚类算法**。

简单说：**把数据分成 K 组，让每组内部的点尽量接近，组之间的点尽量远离。**

---

## 核心思想

### K 是什么？

K 是要分成的**簇（cluster）数量**。

### 目标

- 同一组内的点尽可能相似
- 不同组的点尽可能不同

---

## 算法步骤

### 步骤1：初始化

随机选择 K 个点作为初始中心：

```
    ★    ★    ★
    ↓    ↓    ↓
  中心1 中心2 中心3
```

### 步骤2：分配簇

每个点分配到最近的中心：

```
●   ●   ●   ●   ●   ●
│   │   │   │   │   │
└───┴───┴───┴───┴───┴───
簇1  簇1  簇2  簇2  簇3  簇3
```

### 步骤3：更新中心

计算每个簇的新中心（均值）：

```
簇1的均值 → 新中心1
簇2的均值 → 新中心2
簇3的均值 → 新中心3
```

### 步骤4：重复

重复步骤2-3，直到中心不再变化。

## K 的选择

### 如何选择 K？

没有标准答案，但有一些方法：

#### 1. 肘部法则

```
SSE
  │
  │      ╱╲
  │     ╱  ╲
  │    ╱    ╲
  │   ╱      ╲
  │  ╱        ╲
  │ ╱          ╲
  └────────────────→ K
      肘部
```

- SSE：簇内误差平方和
- 肘部点是最佳 K

#### 2. 轮廓系数

衡量每个点与同簇和最近异簇的距离：

- 接近 1：分得好
- 接近 0：模糊
- 接近 -1：分错了

#### 3. 业务知识

根据实际需求选择。

## 距离度量

### 欧几里得距离

最常用的距离：

```
d(x, y) = √Σ(xᵢ - yᵢ)²
```

### 曼哈顿距离

```
d(x, y) = Σ|xᵢ - yᵢ|
```

### 余弦相似度

```
相似度 = (x·y) / (||x|| × ||y||)
```

---

## 目标函数

### 簇内误差平方和（SSE）

```
J = Σ Σ ||xᵢ - μⱼ||²
    j  xᵢ∈Cⱼ
```

- μⱼ：簇 j 的中心
- Cⱼ：簇 j 的所有点

### 目标

最小化 SSE。

---

## 优点

✅ **简单易懂**
- 算法直观
- 容易实现

✅ **高效**
- 时间复杂度 O(nkt)
- n：样本数，k：簇数，t：迭代次数

✅ **可扩展**
- 可以处理大数据

✅ **结果可解释**
- 每个簇有明确含义

---

## 缺点

❌ **需要指定 K**
- K 的选择影响结果
- 没有标准方法

❌ **对初始值敏感**
- 不同的初始结果可能不同
- 需要多次运行

❌ **对异常值敏感**
- 一个异常点会影响中心

❌ **假设簇是球形**
- 不适合非球形簇

---

## 初始化方法

### 随机初始化

随机选 K 个点作为中心。

### K-Means++

改进的初始化方法：

1. 随机选一个中心
2. 选择离已有中心最远的点作为新中心
3. 重复 K 次

效果通常更好。

## 代码示例

### Scikit-learn

```python
from sklearn.cluster import KMeans
from sklearn.datasets import make_blobs
import matplotlib.pyplot as plt

# 生成数据
X, y = make_blobs(n_samples=300, centers=3, random_state=42)

# 创建K-Means
kmeans = KMeans(n_clusters=3, random_state=42)

# 训练
kmeans.fit(X)

# 预测
labels = kmeans.labels_

# 中心
centers = kmeans.cluster_centers_

# 绘制
plt.scatter(X[:, 0], X[:, 1], c=labels)
plt.scatter(centers[:, 0], centers[:, 1], c='red', marker='x', s=200)
plt.show()
```

### 肘部法则

```python
from sklearn.cluster import KMeans

SSE = []
K_range = range(1, 10)

for k in K_range:
    kmeans = KMeans(n_clusters=k, random_state=42)
    kmeans.fit(X)
    SSE.append(kmeans.inertia_)

plt.plot(K_range, SSE, 'bo-')
plt.xlabel('K')
plt.ylabel('SSE')
plt.title('Elbow Method')
plt.show()
```

## 常见问题

### Q: 如何选择 K？

A:
- 肘部法则
- 轮廓系数
- 业务知识

### Q: K-Means 一定会收敛吗？

A:
- 是的，一定会收敛
- 但可能收敛到局部最优

### Q: 为什么对初始值敏感？

A:
- K-Means 是贪婪算法
- 不同的初始导致不同的局部最优
- 用K-Means++改善

## 扩展

### Mini-Batch K-Means

### K-Medoids

用实际点作为中心，更鲁棒。

### 分层K-Means

先粗聚类，再细聚类。

## 总结

**K-Means**：

1. 把数据分成 K 组
2. 迭代分配和更新中心
3. 最小化簇内距离
4. 简单高效，但需指定 K
5. 对初始值敏感

**核心思想**：
- 同组内的点尽可能接近
- 不同组的点尽可能远离
- 迭代优化直到收敛

**关键要点**：
- K 的选择很重要
- 用肘部法则和轮廓系数
- K-Means++ 改善初始化
- 对异常值和初始值敏感

**记住**：K-Means 就像"物以类聚"，把相似的东西放在一起！
