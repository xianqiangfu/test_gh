# 评估指标（回归）

## 什么是回归评估指标？

回归评估指标是**衡量回归模型预测准确性的标准**。

简单说：**用数值告诉我们预测值和真实值有多接近。**

---

## 常见指标

### MSE（均方误差）

### 定义

```
MSE = 平均((预测值 - 真实值)²)
```

### 特点

- 单位：原单位的平方
- 对大误差敏感
- 范围：[0, ∞)
- 越小越好

### 示例

```
真实值: [100, 200, 150]
预测值: [95, 205, 145]

误差: [5, -5, -5]
误差²: [25, 25, 25]
MSE = (25 + 25 + 25) / 3 = 25
```

---

### RMSE（均方根误差）

### 定义

```
RMSE = √MSE = √平均((预测值 - 真实值)²)
```

### 特点

- 单位：与原单位相同
- 容易解释
- 对大误差敏感
- 范围：[0, ∞)
- 越小越好

### 示例

```
MSE = 25
RMSE = √25 = 5
```

含义：平均误差约为5个单位。

---

### MAE（平均绝对误差）

### 定义

```
MAE = 平均|预测值 - 真实值|
```

### 特点

- 单位：与原单位相同
- 对异常值不敏感
- 容易解释
- 范围：[0, ∞)
- 越小越好

### 示例

```
真实值: [100, 200, 150]
预测值: [95, 205, 145]

误差: [5, -5, -5]
|误差|: [5, 5, 5]
MAE = (5 + 5 + 5) / 3 = 5
```

---

### R²（决定系数）

### 定义

```
R² = 1 - SSE / SST
```

其中：
- SSE = Σ(预测值 - 真实值)²
- SST = Σ(真实值 - 平均值)²

### 特点

- 无单位
- 范围：(-∞, 1]
- 越接近1越好
- R²=1：完美拟合
- R²=0：等于预测平均值
- R²<0：比预测平均值还差

### 示例

```
真实值: [1, 2, 3, 4, 5]
预测值: [1.1, 2.2, 2.9, 3.8, 5.1]

SSE = (0.1² + 0.2² + 0.1² + 0.2² + 0.1²) = 0.11
SST = (4² + 3² + 2² + 1² + 0²) = 30
R² = 1 - 0.11/30 = 0.996
```

解释：模型解释了99.6%的方差。

---

## 指标比较

| 指标 | 单位 | 异常值敏感 | 解释性 | 范围 |
|------|------|-----------|--------|------|
| MSE | 原单位² | 非常 | 差 | [0, ∞) |
| RMSE | 原单位 | 非常 | 好 | [0, ∞) |
| MAE | 原单位 | 不敏感 | 好 | [0, ∞) |
| R² | 无 | 中等 | 中等 | (-∞, 1] |

---

## 什么时候用哪个？

### MSE

- 训练时（可导）
- 关注大误差

### RMSE

- 需要报告结果
- 需要原单位

### MAE

- 有异常值
- 需要鲁棒性

### R²

- 需要解释方差
- 模型比较

---

## 代码示例

### Scikit-learn

```python
from sklearn.metrics import mean_squared_error, mean_absolute_error, r2_score
import numpy as np

# 真实值和预测值
y_true = [100, 200, 150]
y_pred = [95, 205, 145]

# MSE
mse = mean_squared_error(y_true, y_pred)
print(f"MSE: {mse}")

# RMSE
rmse = np.sqrt(mse)
print(f"RMSE: {rmse}")

# MAE
mae = mean_absolute_error(y_true, y_pred)
print(f"MAE: {mae}")

# R²
r2 = r2_score(y_true, y_pred)
print(f"R²: {r2}")
```

---

## 指标关系

### RMSE vs MAE

```
RMSE ≥ MAE
```

RMSE ≥ MAE，因为平方放大了大误差。

### R² 和其他指标

```
R² = 1 - MSE / 方差(真实值)
```

R² 与 MSE 相关。

---

## 残差分析

### 什么是残差？

```
残差 = 预测值 - 真实值
```

### 残差图

```
残差
  ↑
  │  ●   ●   ●
  │ ● ● ● ● ●
  │●●●●●●●●●
  └─────────→ 真实值
```

### 检查点

- 残差应该随机分布
- 不应该有模式
- 方差应该一致

## 常见问题

### Q: MSE 和 RMSE 哪个好？

A:
- RMSE 更容易解释
- MSE 训练时用（可导）
- 两者单调相关

### Q: R² 什么时候不好用？

A:
- 非线性模型
- 小样本
- 异常值多

### Q: 如何判断模型好不好？

A:
- 对比基线模型
- 看业务需求
- 多指标综合

## 调整R²

### 定义

```
调整R² = 1 - (1 - R²) × (n - 1) / (n - p - 1)
```

其中：
- n：样本数
- p：特征数

### 为什么需要调整？

普通R²会随特征数增加而增加，调整R²考虑了特征数。

### 代码

```python
from sklearn.metrics import r2_score

n = len(y_true)
p = X.shape[1]  # 特征数

r2 = r2_score(y_true, y_pred)
adjusted_r2 = 1 - (1 - r2) * (n - 1) / (n - p - 1)
```

## 总结

**回归评估指标**：

1. MSE：均方误差，训练用
2. RMSE：均方根误差，结果报告用
3. MAE：平均绝对误差，鲁棒
4. R²：决定系数，解释方差
5. 调整R²：考虑特征数

**核心思想**：
- 衡量预测误差
- 不同指标不同特点
- 根据需求选择

**关键要点**：
- RMSE容易解释
- MAE对异常值不敏感
- R²解释方差
- 多指标综合评估

**记住**：回归指标告诉你"差多少"，分类指标告诉你"对多少"！
