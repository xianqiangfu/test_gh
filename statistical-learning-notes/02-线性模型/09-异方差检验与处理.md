# 异方差检验与处理

## 什么是异方差？

**异方差（Heteroscedasticity）**是指回归模型中误差项的方差不是常数，而是随自变量的变化而变化。

简单来说：**误差的"波动程度"在不同情况下不同**。

---

## 同方差 vs 异方差

### 同方差（理想情况）

```
对于所有 x，Var(ε|x) = σ²（常数）
```

**图形理解：**
```
残差
   ↑
  2│  ●   ●   ●
   │  ●   ●   ●
  0│●●●●●●●●●●●●
   │  ●   ●   ●
 -2│  ●   ●   ●
   └────────────→ 拟合值
    小    中    大
```

散点均匀分布，"宽度"一致。

---

### 异方差（实际问题）

```
Var(ε|x) 随 x 的变化而变化
```

**图形理解：**
```
残差
   ↑
  2│  ●     ●     ●
   │  ●     ●     ●
  0│●●●   ●●●   ●●●
   │  ●     ●     ●
 -2│  ●     ●     ●
   └────────────→ 拟合值
    小    中    大
```

散点呈漏斗形，"宽度"随拟合值变化。

---

## 异方差的类型

### 1. 递增型异方差

误差方差随预测值增大而增大。

**常见例子：**
- 收入 vs 教育年限：高收入人群收入差异更大
- 消费 vs 收入：高收入人群消费选择更多样

**图形：**
```
   ↑
  3│      ●●●
  2│     ●●●●●
  1│    ●●●●●●●
  0│   ●●●●●●●●●
 -1│  ●●●●●●●●●●●
 -2│ ●●●●●●●●●●●●●
 -3│●●●●●●●●●●●●●●●
   └────────────→ 拟合值
```

---

### 2. 递减型异方差

误差方差随预测值增大而减小。

**常见例子：**
- 小型公司的利润波动可能更大

**图形：**
```
   ↑
  3│●●●●●●●●●●●●●●●
  2│ ●●●●●●●●●●●●●●
  1│  ●●●●●●●●●●●●
  0│   ●●●●●●●●●●
 -1│    ●●●●●●●●
 -2│     ●●●●●●
 -3│      ●●●
   └────────────→ 拟合值
```

---

### 3. 复杂型异方差

误差方差呈非单调变化。

**常见例子：**
- 价格 vs 需求：中间价格可能波动最大

---

## 异方差的影响

### 1. 系数估计仍然无偏

**好消息：** 异方差不会影响系数的点估计。

```
E(β̂) = β（仍然无偏）
```

---

### 2. 标准误估计不准确

**坏消息：** 标准误的计算会出错。

- **低估标准误** → t统计量虚高 → 虚假显著
- **高估标准误** → t统计量过低 → 遗漏真实效应

---

### 3. 假设检验失效

由于标准误不准确：
- t检验结果不可靠
- F检验结果不可靠
- 置信区间不准确

---

### 4. 预测区间不准确

预测区间依赖于误差方差的估计，异方差会导致预测区间：
- 在某些区域过宽
- 在某些区域过窄

---

## 异方差的诊断方法

### 方法1：残差图诊断（最直观）

#### 残差 vs 拟合值图

```python
import matplotlib.pyplot as plt

# 残差 vs 拟合值图
plt.scatter(model.fittedvalues, model.resid)
plt.axhline(y=0, color='r', linestyle='--')
plt.xlabel('Fitted Values')
plt.ylabel('Residuals')
plt.title('Residuals vs Fitted Values')
plt.show()
```

**如何判断：**
- ✅ **随机散布** → 同方差
- ❌ **漏斗形** → 异方差
- ❌ **其他模式** → 异方差或非线性

---

#### Scale-Location图

更敏感的异方差检测方法。

```python
# Scale-Location图
sqrt_abs_resid = np.sqrt(np.abs(model.resid))
plt.scatter(model.fittedvalues, sqrt_abs_resid)
plt.xlabel('Fitted Values')
plt.ylabel('√|Standardized Residuals|')
plt.title('Scale-Location Plot')
plt.show()
```

**如何判断：**
- ✅ **水平线** → 同方差
- ❌ **斜线或曲线** → 异方差

---

#### 残差 vs 自变量图

检查哪个变量导致异方差。

```python
# 对每个自变量绘制残差图
for i, var in enumerate(['x1', 'x2', 'x3']):
    plt.subplot(2, 2, i+1)
    plt.scatter(df[var], model.resid)
    plt.axhline(y=0, color='r', linestyle='--')
    plt.xlabel(var)
    plt.ylabel('Residuals')
    plt.title(f'Residuals vs {var}')

plt.tight_layout()
plt.show()
```

---

### 方法2：Breusch-Pagan检验

最常用的异方差检验方法。

**原理：** 检验残差平方是否可以用自变量解释。

**假设：**
- H₀：同方差（误差方差是常数）
- H₁：异方差（误差方差随自变量变化）

**Python实现：**

```python
from statsmodels.stats.diagnostic import het_breuschpagan

# Breusch-Pagan检验
bp_test = het_breuschpagan(model.resid, model.model.exog)

print(f"LM统计量: {bp_test[0]:.4f}")
print(f"LM检验p值: {bp_test[1]:.4f}")
print(f"F统计量: {bp_test[2]:.4f}")
print(f"F检验p值: {bp_test[3]:.4f}")

# 判断
if bp_test[1] < 0.05:
    print("拒绝原假设，存在异方差")
else:
    print("不能拒绝原假设，同方差假设成立")
```

**优点：**
- 计算简单
- 理论基础好
- 适用于多种情况

**缺点：**
- 对正态性假设敏感
- 只能检测特定形式的异方差

---

### 方法3：White检验

更一般化的异方差检验。

**原理：** 检验残差平方是否可以用自变量、自变量平方和交叉项解释。

**假设：**
- H₀：同方差
- H₁：异方差（任意形式）

**Python实现：**

```python
from statsmodels.stats.diagnostic import het_white

# White检验
white_test = het_white(model.resid, model.model.exog)

print(f"LM统计量: {white_test[0]:.4f}")
print(f"LM检验p值: {white_test[1]:.4f}")
print(f"F统计量: {white_test[2]:.4f}")
print(f"F检验p值: {white_test[3]:.4f}")

# 判断
if white_test[1] < 0.05:
    print("拒绝原假设，存在异方差")
else:
    print("不能拒绝原假设，同方差假设成立")
```

**优点：**
- 检验更全面
- 不假设异方差的具体形式

**缺点：**
- 计算量较大
- 在小样本下功效较低
- 可能对无关异方差也显著

---

### 方法4：Goldfeld-Quandt检验

适用于误差方差随某个变量单调变化的情况。

**原理：** 比较数据两端的方差。

**假设：**
- H₀：同方差
- H₁：递增或递减异方差

**Python实现：**

```python
from statsmodels.stats.diagnostic import het_goldfeldquandt

# Goldfeld-Quandt检验
gq_test = het_goldfeldquandt(df['y'], model.model.exog)

print(f"F统计量: {gq_test[0]:.4f}")
print(f"p值: {gq_test[1]:.4f}")

# 判断
if gq_test[1] < 0.05:
    print("拒绝原假设，存在异方差")
else:
    print("不能拒绝原假设，同方差假设成立")
```

---

### 检验方法比较

| 检验方法 | 适用场景 | 优点 | 缺点 |
|---------|---------|------|------|
| **残差图** | 初步诊断 | 直观、发现模式 | 主观、无统计推断 |
| **Breusch-Pagan** | 一般异方差 | 简单、常用 | 对正态性敏感 |
| **White** | 任意异方差 | 全面、通用 | 小样本功效低 |
| **Goldfeld-Quandt** | 单调异方差 | 简单直接 | 只适用单调情况 |

**建议：**
1. 先看残差图
2. 再用Breusch-Pagan检验
3. 如有需要，用White检验确认

## 异方差的处理方法

### 方法1：变量变换（最简单）

#### 对数变换

最常用的变换方法。

**适用情况：** 递增型异方差

**Python实现：**

```python
import numpy as np

# 对因变量取对数
df['log_y'] = np.log(df['y'])

# 重新拟合模型
X = sm.add_constant(df[['x1', 'x2']])
model_log = sm.OLS(df['log_y'], X).fit()

print(model_log.summary())
```

**优点：**
- 简单易行
- 常能有效解决异方差
- 结果容易解释（百分比变化）

**缺点：**
- 只适用于正值
- 改变了模型解释

---

#### 其他变换

```python
# 平方根变换
df['sqrt_y'] = np.sqrt(df['y'])

# 倒数变换
df['inv_y'] = 1 / df['y']

# Box-Cox变换（自动选择）
from scipy import stats
df['boxcox_y'], lambda_ = stats.boxcox(df['y'])
print(f"Box-Cox λ = {lambda_:.4f}")
```

---

### 方法2：加权最小二乘法（WLS）

**思想：** 给方差小的观测值更大的权重。

**权重设置：**
- 权重与误差方差成反比
- wᵢ = 1/σᵢ²

**Python实现：**

```python
# 步骤1：先用OLS估计残差
model_ols = sm.OLS(df['y'], X).fit()
residuals = model_ols.resid

# 步骤2：估计权重
# 假设方差与x1成正比
weights = 1 / df['x1']**2

# 步骤3：使用WLS
model_wls = sm.WLS(df['y'], X, weights=weights).fit()

print(model_wls.summary())
```

**自动权重选择：**

```python
# 使用残差绝对值的倒数作为权重
weights = 1 / np.abs(residuals)
model_wls = sm.WLS(df['y'], X, weights=weights).fit()
```

**优点：**
- 理论上最优
- 有效利用方差信息

**缺点：**
- 需要知道或估计权重
- 权重估计可能不准确

---

### 方法3：稳健标准误（HC标准误）

**思想：** 不改变系数估计，只调整标准误的计算。

**Python实现：**

```python
# HC0标准误（White稳健标准误）
model_hc0 = model.get_robustcov_results(cov_type='HC0')

# HC1标准误（小样本调整）
model_hc1 = model.get_robustcov_results(cov_type='HC1')

# HC2标准误（进一步小样本调整）
model_hc2 = model.get_robustcov_results(cov_type='HC2')

# HC3标准误（Jackknife方法，推荐）
model_hc3 = model.get_robustcov_results(cov_type='HC3')

# 比较结果
print("OLS标准误:")
print(model.summary().tables[1])

print("\nHC3稳健标准误:")
print(model_hc3.summary().tables[1])
```

**HC系列比较：**

| 类型 | 特点 | 推荐场景 |
|------|------|---------|
| HC0 | 原始White标准误 | 大样本 |
| HC1 | HC0的小样本调整 | 一般情况 |
| HC2 | HC1的进一步调整 | 中等样本 |
| HC3 | Jackknife方法 | 小样本，最保守 |

**优点：**
- 简单易用
- 不改变系数估计
- 对未知异方差形式有效

**缺点：**
- 只解决推断问题，不解决预测问题
- 小样本下可能不准确

---

### 方法4：广义最小二乘法（GLS）

**思想：** 对数据进行变换，使变换后的误差满足同方差。

**适用情况：** 知道异方差的具体形式

**Python实现：**

```python
# 假设误差方差与x的平方成正比
sigma = df['x1']**2

# 构建权重矩阵
weights = 1 / sigma

# 使用GLS
model_gls = sm.GLS(df['y'], X, sigma=sigma).fit()

print(model_gls.summary())
```

---

### 处理方法选择指南

| 情况 | 推荐方法 | 理由 |
|------|---------|------|
| **轻微异方差** | 稳健标准误 | 简单有效 |
| **递增异方差，y>0** | 对数变换 | 常有效，易解释 |
| **知道异方差形式** | WLS/GLS | 理论最优 |
| **不知道异方差形式** | 稳健标准误 | 通用方法 |
| **预测重要** | 变换或WLS | 提高预测精度 |
| **推断重要** | 稳健标准误 | 保证假设检验有效 |

## 完整示例：从诊断到处理

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import statsmodels.api as sm
from statsmodels.stats.diagnostic import het_breuschpagan, het_white

# ========== 1. 生成具有异方差的模拟数据 ==========
np.random.seed(42)
n = 200
x = np.linspace(1, 10, n)
y = 2 * x + 3 + np.random.normal(0, 0.5 * x, n)  # 误差标准差随x增大

df = pd.DataFrame({'x': x, 'y': y})

# ========== 2. 拟合普通OLS模型 ==========
X = sm.add_constant(df['x'])
model_ols = sm.OLS(df['y'], X).fit()

print("=" * 50)
print("OLS模型结果")
print("=" * 50)
print(model_ols.summary())

# ========== 3. 诊断异方差 ==========

# 3.1 残差图
fig, axes = plt.subplots(2, 2, figsize=(12, 10))

# 残差 vs 拟合值
axes[0, 0].scatter(model_ols.fittedvalues, model_ols.resid)
axes[0, 0].axhline(y=0, color='r', linestyle='--')
axes[0, 0].set_xlabel('Fitted Values')
axes[0, 0].set_ylabel('Residuals')
axes[0, 0].set_title('Residuals vs Fitted Values')

# Scale-Location图
sqrt_abs_resid = np.sqrt(np.abs(model_ols.resid))
axes[0, 1].scatter(model_ols.fittedvalues, sqrt_abs_resid)
axes[0, 1].set_xlabel('Fitted Values')
axes[0, 1].set_ylabel('√|Standardized Residuals|')
axes[0, 1].set_title('Scale-Location Plot')

# 残差 vs 自变量
axes[1, 0].scatter(df['x'], model_ols.resid)
axes[1, 0].axhline(y=0, color='r', linestyle='--')
axes[1, 0].set_xlabel('x')
axes[1, 0].set_ylabel('Residuals')
axes[1, 0].set_title('Residuals vs x')

# Q-Q图
from scipy import stats
stats.probplot(model_ols.resid, dist="norm", plot=axes[1, 1])
axes[1, 1].set_title('Normal Q-Q Plot')

plt.tight_layout()
plt.show()

# 3.2 统计检验
print("\n" + "=" * 50)
print("异方差检验")
print("=" * 50)

# Breusch-Pagan检验
bp_test = het_breuschpagan(model_ols.resid, model_ols.model.exog)
print(f"\nBreusch-Pagan检验:")
print(f"  LM统计量: {bp_test[0]:.4f}, p值: {bp_test[1]:.4f}")
print(f"  {'存在异方差' if bp_test[1] < 0.05 else '无异方差'}")

# White检验
white_test = het_white(model_ols.resid, model_ols.model.exog)
print(f"\nWhite检验:")
print(f"  LM统计量: {white_test[0]:.4f}, p值: {white_test[1]:.4f}")
print(f"  {'存在异方差' if white_test[1] < 0.05 else '无异方差'}")

# ========== 4. 处理异方差 ==========

# 方法1：对数变换
print("\n" + "=" * 50)
print("方法1：对数变换")
print("=" * 50)
df['log_y'] = np.log(df['y'])
model_log = sm.OLS(df['log_y'], X).fit()
print(model_log.summary())

# 检验对数变换后的异方差
bp_test_log = het_breuschpagan(model_log.resid, model_log.model.exog)
print(f"\n对数变换后Breusch-Pagan检验: p值 = {bp_test_log[1]:.4f}")
print(f"  {'存在异方差' if bp_test_log[1] < 0.05 else '无异方差'}")

# 方法2：加权最小二乘
print("\n" + "=" * 50)
print("方法2：加权最小二乘法")
print("=" * 50)
# 使用x的倒数平方作为权重
weights = 1 / df['x']**2
model_wls = sm.WLS(df['y'], X, weights=weights).fit()
print(model_wls.summary())

# 检验WLS后的异方差
bp_test_wls = het_breuschpagan(model_wls.resid, model_wls.model.exog)
print(f"\nWLS后Breusch-Pagan检验: p值 = {bp_test_wls[1]:.4f}")
print(f"  {'存在异方差' if bp_test_wls[1] < 0.05 else '无异方差'}")

# 方法3：稳健标准误
print("\n" + "=" * 50)
print("方法3：稳健标准误（HC3）")
print("=" * 50)
model_hc3 = model_ols.get_robustcov_results(cov_type='HC3')
print(model_hc3.summary())

# ========== 5. 比较不同方法 ==========
print("\n" + "=" * 50)
print("方法比较")
print("=" * 50)

results = pd.DataFrame({
    '方法': ['OLS', '对数变换', 'WLS', 'OLS+HC3'],
    '截距': [
        model_ols.params['const'],
        model_log.params['const'],
        model_wls.params['const'],
        model_ols.params['const']
    ],
    '截距标准误': [
        model_ols.bse['const'],
        model_log.bse['const'],
        model_wls.bse['const'],
        model_hc3.bse['const']
    ],
    '斜率': [
        model_ols.params['x'],
        model_log.params['x'],
        model_wls.params['x'],
        model_ols.params['x']
    ],
    '斜率标准误': [
        model_ols.bse['x'],
        model_log.bse['x'],
        model_wls.bse['x'],
        model_hc3.bse['x']
    ],
    'R²': [
        model_ols.rsquared,
        model_log.rsquared,
        model_wls.rsquared,
        model_ols.rsquared
    ]
})

print(results)
```

## 实际应用案例

### 案例1：收入与教育

**问题：** 研究教育年限对收入的影响

**数据特点：** 高收入人群收入差异更大（递增异方差）

**处理方案：**

```python
# 对收入取对数
df['log_income'] = np.log(df['income'])

# 使用对数收入作为因变量
X = sm.add_constant(df[['education', 'experience']])
model = sm.OLS(df['log_income'], X).fit(cov_type='HC3')

# 解释：教育年限增加1年，收入增加exp(β)倍
print(f"教育回报率: {np.exp(model.params['education']) - 1:.2%}")
```

### 案例2：房价预测

**问题：** 预测房屋价格

**数据特点：** 高价房价格差异更大

**处理方案：**

```python
# 方法1：对数变换
df['log_price'] = np.log(df['price'])
model_log = sm.OLS(df['log_price'], X).fit()

# 方法2：稳健标准误
model_robust = sm.OLS(df['price'], X).fit(cov_type='HC3')
```

### 案例3：消费函数

**问题：** 研究收入对消费的影响

**数据特点：** 高收入家庭消费更灵活

**处理方案：**

```python
# 使用WLS，权重与收入平方成反比
weights = 1 / df['income']**2
model_wls = sm.WLS(df['consumption'], X, weights=weights).fit()
```

## 常见误区

### 误区1：忽略异方差检查

**错误：** 直接使用OLS结果，不检查异方差

**正确：** 始终检查残差图和进行统计检验

---

### 误区2：过度依赖统计检验

**错误：** 只看p值，不看残差图

**正确：** 先看图形，再结合统计检验

---

### 误区3：盲目删除数据

**错误：** 删除"异常"的残差来消除异方差

**正确：** 异方差可能是数据特征，不是错误

---

### 误区4：只关注系数，忽略标准误

**错误：** 只看系数估计，不管标准误是否准确

**正确：** 关注推断的有效性，使用稳健标准误

---

### 误区5：对数变换万能

**错误：** 对任何异方差都取对数

**正确：** 根据异方差类型选择合适方法

## 总结

**异方差的核心要点：**

1. **定义**：误差方差随自变量变化
2. **影响**：标准误不准确，假设检验失效
3. **诊断**：残差图 + 统计检验
4. **处理**：变换、WLS、稳健标准误

**处理方法选择：**

- **简单情况**：对数变换
- **已知形式**：加权最小二乘
- **未知形式**：稳健标准误
- **预测重要**：变换或WLS
- **推断重要**：稳健标准误

**记住：**

- 异方差很常见，不要忽视
- 残差图是最直观的诊断工具
- 稳健标准误是简单有效的解决方案
- 根据具体情况选择处理方法

好的回归分析 = 正确的模型 + 充分的诊断 + 合适的处理
