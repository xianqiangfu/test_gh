# Sigmoid函数

## 什么是Sigmoid函数？

Sigmoid函数（也叫Logistic函数）是一个**S形的函数**，能把任意实数映射到(0,1)区间。

它是逻辑回归的核心组件。

---

## 数学定义

```
σ(x) = 1 / (1 + e^(-x))
```

- x：任意实数
- e：自然对数的底（约2.718）
- σ(x)：输出值，范围在(0,1)之间

---

### 关键点

- 当 x = 0 时，σ(0) = 0.5
- 当 x → ∞ 时，σ(x) → 1
- 当 x → -∞ 时，σ(x) → 0

---

## 函数性质

### 1. 值域

```
σ(x) ∈ (0, 1)
```

输出始终在0和1之间，正好表示概率。

### 2. 对称性

```
σ(-x) = 1 - σ(x)
```

关于点(0, 0.5)对称。

### 3. 导数

```
σ'(x) = σ(x) × (1 - σ(x))
```

这个性质非常重要，使得梯度计算很简单。

### 4. 单调性

函数单调递增，值越大，输出越接近1。

---

## 导数的重要性

### 为什么导数重要？

在训练逻辑回归时，需要计算梯度。

### 导数公式

```
σ'(x) = σ(x) × (1 - σ(x))
```

### 推导过程

```
σ(x) = 1 / (1 + e^(-x))

σ'(x) = d/dx [1 / (1 + e^(-x))]
      = (0 - (-e^(-x))) / (1 + e^(-x))²
      = e^(-x) / (1 + e^(-x))²
      = 1/(1+e^(-x)) × e^(-x)/(1+e^(-x))
      = σ(x) × (1 - σ(x))
```

### 实际意义

- 当 σ(x) 接近 0 或 1 时，导数接近 0
- 当 σ(x) = 0.5 时，导数最大（0.25）

这说明：当预测很"确定"时，梯度小；当预测"不确定"时，梯度大。

---

## 为什么要用Sigmoid？

### 1. 概率映射

把任意数值映射到(0,1)，正好表示概率。

### 2. 可导

函数处处可导，可以计算梯度。

### 3. 单调递增

输入越大，概率越大。

### 4. 导数简单

梯度计算很方便。

---

## 计算例子

### 例子1：x = 0

```
σ(0) = 1 / (1 + e^(-0))
     = 1 / (1 + 1)
     = 1 / 2
     = 0.5
```

### 例子2：x = 1

```
σ(1) = 1 / (1 + e^(-1))
     = 1 / (1 + 0.368)
     ≈ 0.731
```

### 例子3：x = -1

```
σ(-1) = 1 / (1 + e^(-(-1)))
       = 1 / (1 + e^1)
       = 1 / (1 + 2.718)
       ≈ 0.269
```

注意：σ(-1) = 1 - σ(1) ≈ 1 - 0.731 = 0.269

### 例子4：x = 5（很大）

```
σ(5) = 1 / (1 + e^(-5))
     ≈ 1 / (1 + 0.0067)
     ≈ 0.993
```

接近1。

### 例子5：x = -5（很小）

```
σ(-5) = 1 / (1 + e^5)
       ≈ 1 / (1 + 148.4)
       ≈ 0.007
```

接近0。

## 在机器学习中的应用

### 1. 逻辑回归

把线性模型的输出转换为概率：

```
z = wx + b
p = σ(z)
```

### 2. 神经网络

作为激活函数：

```
输出 = σ(输入 × 权重 + 偏置)
```

注意：现代深度学习中，sigmoid用得较少，更多用ReLU。

### 3. 二分类

设定阈值（通常是0.5）：

```
如果 p ≥ 0.5，预测为类别1
如果 p < 0.5，预测为类别0
```

---

## Sigmoid的局限

### 1. 梯度消失

当输入很大或很小时，梯度接近0。

```
x 很大 → σ(x) 接近1 → σ'(x) 接近0
x 很小 → σ(x) 接近0 → σ'(x) 接近0
```

这会导致深层神经网络难以训练。

### 2. 输出不中心化

输出在(0,1)之间，不是以0为中心。

这会影响梯度下降的效率。

### 3. 计算成本

包含指数运算，相对较慢。

---

## 其他激活函数

| 函数 | 表达式 | 优点 | 缺点 |
|------|--------|------|------|
| Sigmoid | 1/(1+e^(-x)) | 输出概率 | 梯度消失 |
| Tanh | (e^x-e^(-x))/(e^x+e^(-x)) | 以0为中心 | 梯度消失 |
| ReLU | max(0, x) | 计算快，无梯度消失 | Dead ReLU |
| Leaky ReLU | max(0.01x, x) | 解决Dead ReLU | - |
| Softmax | e^xi/Σe^xj | 多分类 | - |

## 实际应用

### 例子：邮件分类

特征：
- 包含"免费"的次数：x₁
- 包含"点击"的次数：x₂
- 邮件长度：x₃

模型：
```
z = 2 × x₁ + 1.5 × x₂ - 0.01 × x₃ - 2
p = σ(z)
```

如果：
- "免费"出现3次
- "点击"出现2次
- 邮件长度1000字

计算：
```
z = 2×3 + 1.5×2 - 0.01×1000 - 2
  = 6 + 3 - 10 - 2
  = -3

p = σ(-3) ≈ 0.047
```

预测：只有4.7%是垃圾邮件的概率。

## 实现技巧

### 1. 数值稳定性

当 x 很大或很小时，直接计算可能溢出。

**改进方法**：
```
如果 x ≥ 0：
    σ(x) = 1 / (1 + e^(-x))
如果 x < 0：
    σ(x) = e^x / (1 + e^x)
```

### 2. 预计算

可以预先计算一些常用值，提高速度。

---

## 总结

**Sigmoid函数**：

1. 数学形式：σ(x) = 1/(1+e^(-x))
2. 值域：(0, 1)，适合表示概率
3. 导数：σ'(x) = σ(x)×(1-σ(x))，计算简单
4. 对称性：σ(-x) = 1 - σ(x)
5. 应用：逻辑回归、神经网络激活

**关键理解**：
- 把任意数值映射到(0,1)
- 导数形式简单，方便梯度计算
- 是逻辑回归的核心
- 在深度学习中用得较少

**记住**：Sigmoid就像一个"开关"，平滑地把数值转换成概率！
