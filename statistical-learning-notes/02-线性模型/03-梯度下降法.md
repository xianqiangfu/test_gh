# 梯度下降法

## 什么是梯度下降？

**梯度下降**是一种**优化算法**，用来找到函数的最小值。

简单理解：**下山的过程**。

---

## 生活类比：下山

想象你在山上迷路了，想下山：

1. 你看不到山脚在哪里
2. 但你可以看脚下：哪边是下坡
3. 就朝哪个方向走一步
4. 重复这个过程，直到到达山脚

```
        ↑
       / \  ← 山顶
      /   \n     /     \n    /       \n   /         \  ← 下山路径
  /           \n /_____________\n```

这就是梯度下降的核心思想！

---

## 数学概念

### 梯度是什么？

**梯度**是一个向量，指向函数**增长最快**的方向。

**负梯度**指向函数**下降最快**的方向。

```
梯度 ∇f(x) = [∂f/∂x₁, ∂f/∂x₂, ...]
```

### 梯度下降的步骤

1. 计算当前点的梯度
2. 沿负梯度方向走一步
3. 重复直到收敛

```
x_new = x_old - 学习率 × 梯度
```

---

## 在线性回归中的应用

### 目标

找到使损失函数最小的 w 和 b。

### 损失函数（MSE）

```
L(w,b) = 平均[(wx + b - y)²]
```

### 梯度

对 w 和 b 分别求偏导：

```
∂L/∂w = 2 × 平均[(wx + b - y) × x]
∂L/∂b = 2 × 平均[(wx + b - y)]
```

### 更新规则

```
w = w - 学习率 × ∂L/∂w
b = b - 学习率 × ∂L/∂b
```

---

## 学习率 (Learning Rate)

**学习率**控制每一步走多大。

```
x_new = x_old - 学习率 × 梯度
```

### 学习率太大

- 走得太大，可能"跳过"最小值
- 甚至会越来越远

### 学习率太小

- 走得很慢，需要很多步
- 训练时间很长

### 如何选择学习率？

```
尝试不同的值：0.001, 0.01, 0.1, 1.0
```

通常从0.01或0.1开始尝试。

---

## 可视化

### 损失函数的曲面

```
损失值
  ↑
  │        ● 最小值
  │       ╱ ╲
  │      ╱   ╲
  │     ╱     ╲
  │    ●       ●
  │   ╱         ╲
  │  ●           ●
  │ ╱             ╲
  └───────────────→ w
```

梯度下降就是从某个点开始，沿着曲面"滚"到最低点。

## 梯度下降的类型

### 1. 批量梯度下降

**每次用所有数据计算梯度**。

**优点**：
- 收敛稳定
- 理论保证

**缺点**：
- 计算量大
- 大数据集很慢

### 2. 随机梯度下降 (SGD)

**每次用1个样本计算梯度**。

**优点**：
- 计算快
- 可以在线学习

**缺点**：
- 路径波动大
- 不太稳定

### 3. 小批量梯度下降

**每次用一小批样本计算梯度**（常用）。

**优点**：
- 兼顾两者
- 效果好且快

**默认选择**：小批量梯度下降

---

## 收敛问题

### 什么时候停止？

1. **损失变化很小**
   ```
   |新损失 - 旧损失| < 阈值
   ```

2. **达到最大迭代次数**
   ```
   迭代次数 > 最大轮数
   ```

3. **梯度接近0**
   ```
   ||梯度|| < 阈值
   ```

---

## 局部最优 vs 全局最优

### 对于凸函数（如线性回归）

```
损失值
  ↑
  │    ╱╲
  │   ╱  ╲
  │  ╱    ╲
  │ ╱      ╲
  │╱________╲
  └──────────→ w
```

只有一个最小值，一定会找到全局最优。

### 对于非凸函数（如神经网络）

```
损失值
  ↑
  │    ╱╲      ╱╲
  │   ╱  ╲    ╱  ╲
  │  ╱    ╲  ╱    ╲
  │ ╱      ╲╱      ╲
  │╱________________╲
  └────────────────→ w
```

有多个局部最优，可能陷入局部最优。

## 常见技巧

### 1. 学习率衰减

逐步降低学习率：
```
学习率 = 初始学习率 / (1 + 衰减率 × 迭代次数)
```

### 2. 动量

记录之前的梯度方向，加速收敛：
```
v = 动量 × v - 学习率 × 梯度
x = x + v
```

### 3. 自适应学习率

根据梯度大小自动调整学习率。

常用算法：
- Adam
- RMSprop
- Adagrad

---

## 梯度下降 vs 解析解

| 特点 | 梯度下降 | 解析解（最小二乘） |
|------|----------|------------------|
| 计算方式 | 迭代 | 直接计算 |
| 数据量大时 | 较快 | 较慢 |
| 理论保证 | 收敛到局部最优 | 全局最优 |
| 适用范围 | 广泛 | 只适用于特定模型 |

---

## 实际应用

### 训练神经网络

梯度下降是神经网络训练的核心方法。

### 深度学习

几乎所有深度学习模型都用梯度下降训练。

### 逻辑回归

虽然逻辑回归有解析解，但通常也用梯度下降。

---

## 常见陷阱

### 1. 学习率不当

太大：发散或震荡
太小：收敛太慢

### 2. 梯度消失/爆炸

深度神经网络常见问题。

### 3. 鞍点

在某些方向是极小值，某些方向是极大值。

---

## 总结

**梯度下降**：

1. 核心思想：沿负梯度方向迭代优化
2. 关键参数：学习率
3. 常用变体：小批量梯度下降
4. 适用广泛：很多机器学习算法的基础

**关键理解**：
- 梯度 = 增长最快的方向
- 负梯度 = 下降最快的方向
- 学习率 = 步长
- 迭代直到收敛

梯度下降是机器学习的"引擎"。理解它，就理解了大部分模型是如何训练的！
