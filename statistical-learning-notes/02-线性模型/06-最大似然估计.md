# 最大似然估计

## 什么是最大似然估计？

**最大似然估计 (Maximum Likelihood Estimation, MLE)** 是一种**统计方法**，用来找到"最可能"产生观测数据的参数。

简单理解：**在所有可能的参数中，找到最符合观测数据的那一个。**

---

## 生活中的例子

### 例子1：猜硬币

假设你不知道硬币是不是均匀的。

你抛了10次，得到了7次正面、3次反面。

问：这个硬币出现正面的概率是多少？

**直觉**：最可能的是 0.7

这就是最大似然估计：假设正面概率为 p，那么观测到7正3反的概率最大时，p=0.7。

### 例子2：猜身高

你要估计某地成年男性的平均身高。

你随机测量了100个人，平均身高175cm。

**最大似然估计**：该地成年男性平均身高是175cm。

---

## 数学原理

### 似然函数

给定参数 θ，观测到数据 X 的概率：

```
L(θ|X) = P(X|θ)
```

读作：给定数据 X，参数 θ 的似然。

### 目标

找到让似然函数最大的参数：

```
θ̂ = argmax L(θ|X)
      θ
```

---

## 伯努利分布的MLE

### 问题

抛硬币，出现正面的概率是 p。

抛了 n 次，有 k 次正面。

估计 p。

### 似然函数

```
L(p) = P(k次正面) = C(n,k) × p^k × (1-p)^(n-k)
```

- C(n,k) 是组合数，是常数，可以忽略
- p^k 是正面出现的概率
- (1-p)^(n-k) 是反面出现的概率

### 求最大值

取对数（把乘法变成加法）：

```
log L(p) = k × log(p) + (n-k) × log(1-p)
```

对 p 求导，令导数为0：

```
d/dp [log L(p)] = k/p - (n-k)/(1-p) = 0

k/p = (n-k)/(1-p)

k(1-p) = (n-k)p

k - kp = np - kp

k = np

p = k/n
```

### 结论

**最大似然估计：p̂ = k/n**

就是：正面次数除以总次数。

## 正态分布的MLE

### 问题

数据服从正态分布 N(μ, σ²)，估计 μ 和 σ²。

### 似然函数

```
L(μ, σ²) = ∏[1/(√(2π)σ) × exp(-(xᵢ-μ)²/(2σ²))]
```

### 取对数

```
log L(μ, σ²) = -n/2 × log(2π) - n × log(σ) - Σ(xᵢ-μ)²/(2σ²)
```

### 对 μ 求导

```
∂/∂μ [log L] = Σ(xᵢ-μ)/σ² = 0
Σxᵢ - nμ = 0
μ̂ = Σxᵢ/n = 样本均值
```

### 对 σ² 求导

```
∂/∂σ² [log L] = -n/(2σ²) + Σ(xᵢ-μ)²/(2σ⁴) = 0
σ̂² = Σ(xᵢ-μ̂)²/n = 样本方差
```

### 结论

- μ 的 MLE = 样本均值
- σ² 的 MLE = 样本方差

## 逻辑回归中的MLE

### 问题

给定数据 (xᵢ, yᵢ)，估计逻辑回归的参数 w 和 b。

### 似然函数

```
L(w,b) = ∏P(yᵢ|xᵢ)

P(yᵢ|xᵢ) = pᵢ^yᵢ × (1-pᵢ)^(1-yᵢ)
         = σ(zᵢ)^yᵢ × (1-σ(zᵢ))^(1-yᵢ)
```

其中：
- zᵢ = wxᵢ + b
- pᵢ = σ(zᵢ)

### 对数似然

```
log L(w,b) = Σ[yᵢ × log(pᵢ) + (1-yᵢ) × log(1-pᵢ)]
```

### 最大化

我们最大化对数似然，等价于最小化负对数似然：

```
最小化：-Σ[yᵢ × log(pᵢ) + (1-yᵢ) × log(1-pᵢ)]
```

这就是**交叉熵损失**！

---

## 为什么用对数似然？

### 1. 数值稳定性

似然函数是很多小数相乘，可能接近0。

取对数后变成加法，避免数值问题。

### 2. 计算方便

对数把乘法变成加法，求导更容易。

### 3. 单调性

对数函数单调递增，最大化的位置不变。

```
max L(θ) 等价于 max log L(θ)
```

---

## MLE 的性质

### 1. 一致性

样本量趋于无穷时，MLE 收敛到真实值。

### 2. 渐近正态性

在大样本下，MLE 服从正态分布。

### 3. 有效性

在所有无偏估计中，MLE 的方差最小。

### 4. 不变性

如果 θ̂ 是 θ 的 MLE，那么 g(θ̂) 是 g(θ) 的 MLE。

---

## MLE vs 最小二乘

### 最小二乘法

假设误差服从正态分布时，等价于 MLE。

### MLE

更通用，适用于各种分布。

## 实际例子

### 例子：学生考试

观察数据：
```
学生A：学习3小时，通过
学生B：学习1小时，失败
学生C：学习4小时，通过
学生D：学习2小时，失败
```

### 逻辑回归模型

```
p = σ(wx + b)
```

### 似然函数

```
L(w,b) = σ(3w+b) × (1-σ(w+b)) × σ(4w+b) × (1-σ(2w+b))
```

### 对数似然

```
log L = log[σ(3w+b)] + log[1-σ(w+b)] + log[σ(4w+b)] + log[1-σ(2w+b)]
```

### 梯度下降

最大化对数似然（或最小化负对数似然），得到最优的 w 和 b。

---

## 常见误区

### 1. MLE 是概率？

不，MLE 是参数估计，不是概率。

### 2. MLE 一定最优？

不一定，可能会过拟合。可以用正则化解决。

### 3. 似然是概率？

不是。概率是已知参数求事件概率，似然是已知事件求参数。

---

## MLE vs MAP

### 最大后验估计 (MAP)

结合了先验知识：

```
MAP: 最大化 P(θ|X) = P(X|θ) × P(θ) / P(X)
    = 最大化 L(θ|X) × P(θ)
```

- L(θ|X)：似然
- P(θ)：先验

### MLE vs MAP

| 特点 | MLE | MAP |
|------|-----|-----|
| 先验 | 无 | 有 |
| 风险 | 可能过拟合 | 更稳定 |
| 计算量 | 较小 | 较大 |

当先验是均匀分布时，MAP = MLE。

---

## 总结

**最大似然估计 (MLE)**：

1. 目标：找到最可能产生观测数据的参数
2. 方法：最大化似然函数
3. 工具：通常用对数似然，方便计算
4. 性质：一致、渐近正态、有效
5. 应用：逻辑回归的损失函数就是负对数似然

**核心思想**：
- 假设数据服从某个分布
- 观测到数据后，估计分布的参数
- 找到最"合理"的参数

**记住**：MLE 就是在说"这个参数最可能是对的"！
