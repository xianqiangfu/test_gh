# 逻辑回归

## 什么是逻辑回归？

虽然名字里有"回归"，但逻辑回归其实是**分类算法**。

它的核心任务：**预测一个样本属于某一类别的概率**。

---

## 为什么叫"逻辑"？

因为使用了**逻辑函数 (Sigmoid函数)** 来进行变换。

这个函数把任意实数映射到 (0,1) 区间，正好表示概率。

---

## 线性回归的问题

假设我们想预测邮件是否是垃圾邮件：

- 线性回归的输出可能是：2.5, -1.3, 0.7...
- 但概率应该在 0 到 1 之间

**问题**：线性回归的输出范围不对！

---

## Sigmoid 函数

### 定义

```
σ(z) = 1 / (1 + e^(-z))
```

### 图像

```
σ(z)
1.0 │      ╭──────
    │     ╱
0.5 │    ╱
    │   ╱
0.0 │  ╱
    │ ╱
    └──────────→ z
```

### 性质

- 输入：任意实数
- 输出：0 到 1 之间
- z = 0 时，σ(z) = 0.5
- z → ∞ 时，σ(z) → 1
- z → -∞ 时，σ(z) → 0

---

## 逻辑回归的模型

### 完整公式

```
z = w₁x₁ + w₂x₂ + ... + wₙxₙ + b
p = σ(z) = 1 / (1 + e^(-z))
```

- 先计算线性组合 z
- 再通过 sigmoid 函数得到概率 p

### 分类规则

```
如果 p ≥ 0.5，预测为类别 1
如果 p < 0.5，预测为类别 0
```

---

## 直观理解

### 例子：预测学生是否通过考试

特征：
- 学习时间：x₁
- 作业完成率：x₂

模型：
```
z = 2 × 学习时间 + 1.5 × 作业完成率 - 3
p = σ(z)
```

如果学生：
- 学习了5小时
- 作业完成率80%

计算：
```
z = 2 × 5 + 1.5 × 0.8 - 3 = 8.2
p = 1 / (1 + e^(-8.2)) ≈ 0.9997
```

预测：几乎肯定会通过（概率99.97%）

---

## 决策边界

### 二分类情况

当 p = 0.5 时，z = 0

```
w₁x₁ + w₂x₂ + b = 0
```

这就是一条**决策边界**，把两类分开。

```
类别1区域
    │
    │   ●●●
    │  ●   ●
    │●     ●
━━━━┼━━━━━━━━━━ 决策边界
    │●     ●
    │  ●   ●
    │   ●●●
    │
  类别0区域
```

## 损失函数

### 交叉熵损失

逻辑回归使用**交叉熵损失**，而不是均方误差。

对于单个样本：

```
L = -[y × log(p) + (1-y) × log(1-p)]
```

- y = 1 时：L = -log(p)
- y = 0 时：L = -log(1-p)

### 直观理解

- 当真实标签是1，预测概率接近1时，损失小
- 当真实标签是1，预测概率接近0时，损失大

---

## 训练过程

### 步骤

1. **初始化参数**
   - 随机设置 w 和 b

2. **前向传播**
   - 计算 z 和 p

3. **计算损失**
   - 用交叉熵损失

4. **反向传播**
   - 计算梯度

5. **更新参数**
   - 用梯度下降

6. **重复**
   - 直到收敛

---

## 多分类逻辑回归

### Softmax 函数

对于多分类问题，使用 Softmax 函数：

```
pᵢ = e^zᵢ / (e^z₁ + e^z₂ + ... + e^zₖ)
```

保证所有概率之和为1。

---

## 逻辑回归 vs 线性回归

| 特点 | 线性回归 | 逻辑回归 |
|------|---------|----------|
| 任务 | 回归 | 分类 |
| 输出范围 | (-∞, ∞) | (0, 1) |
| 激活函数 | 无 | Sigmoid |
| 损失函数 | 均方误差 | 交叉熵 |
| 可解释性 | 可解释 | 可解释 |

---

## 优点

✅ **简单高效**
- 训练快
- 计算量小

✅ **可解释性强**
- 可以看出每个特征的影响
- 权重表示特征的重要性

✅ **输出概率**
- 不仅给出分类，还给出概率
- 可以设定不同的阈值

✅ **理论成熟**
- 有完善的统计学基础
- 容易理解和解释

---

## 缺点

❌ **只能处理线性决策边界**
- 对于复杂的非线性问题表现差

❌ **对异常值敏感**
- 像线性回归一样

❌ **特征需要独立**
- 特征之间高度相关时会出问题

---

## 正则化

### L2 正则化（岭回归）

```
损失 = 交叉熵损失 + α × Σw²
```

防止过拟合，让权重不太大。

### L1 正则化（Lasso）

```
损失 = 交叉熵损失 + α × Σ|w|
```

可以做特征选择，让一些权重变为0。

| 应用 | 输入特征 | 输出类别 |
|------|---------|---------|
| 风险评估 | 信用记录、收入、年龄 | 高风险/低风险 |

## 实践技巧

### 1. 特征缩放

标准化或归一化特征，让梯度下降更快收敛。

### 2. 特征工程

创造新的特征，提高模型能力。

### 3. 处理类别不平衡

使用加权、过采样等方法。

### 4. 调整阈值

默认是0.5，但可以根据需求调整：
- 阈值低：召回率高，精确率低
- 阈值高：召回率低，精确率高

---

## 常见面试题

### Q: 为什么逻辑回归不用均方误差？

A: 均方误差会导致损失函数非凸，有多个局部最优，而交叉熵损失是凸函数，有全局最优。

### Q: 逻辑回归和线性模型的关系？

A: 逻辑回归是在线性模型基础上加了 sigmoid 函数，用于分类。

### Q: 逻辑回归怎么处理多分类？

A: 使用 softmax 函数，或者用一对多策略。

---

## 总结

**逻辑回归**：

1. 虽然叫"回归"，实际是分类算法
2. 使用 sigmoid 函数输出概率
3. 用交叉熵损失训练
4. 简单、快速、可解释
5. 是很多复杂模型的基础

**核心要点**：
- sigmoid 把线性输出映射到概率
- 交叉熵是合适的损失函数
- 可以设置阈值调整分类策略
- 正则化防止过拟合

逻辑回归是机器学习的"瑞士军刀"——简单但强大，在很多场景下都能用！
