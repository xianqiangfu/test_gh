# Lasso回归

## 什么是Lasso回归？

Lasso回归（Least Absolute Shrinkage and Selection Operator）是**带 L1 正则化的线性回归**。

简单说：**在线性回归基础上，加一个约束，不仅能防止过拟合，还能做特征选择。**

---

## L1 vs L2 正则化

### L2 正则化（岭回归）

```
惩罚项 = α × Σw²
```

- 权重平方和
- 权重减小但通常不为0

### L1 正则化（Lasso）

```
惩罚项 = α × Σ|w|
```

- 权重绝对值和
- 权重可以为0（稀疏解）

---

## Lasso的数学

### 原始损失（均方误差）

```
L = Σ(yᵢ - (wxᵢ + b))² / n
```

### Lasso的损失

```
L = Σ(yᵢ - (wxᵢ + b))² / n + α × Σ|w|
```

- 第一项：均方误差
- 第二项：L1 正则化（权重的绝对值和）
- α：正则化强度

### 简化表示

```
L = MSE + α × ||w||₁
```

其中 ||w||₁ 是权重的 L1 范数。

---

## 为什么Lasso能做特征选择？

### L1 的几何性质

L1 正则化的约束区域是"菱形"：

```
     │
     │  ╱│╲
     │ ╱ │ ╲
─────┼───┼───────
     │╱  │  ╲
     │   │
     │╲  │  ╱
     │ ╲ │ ╱
     │  ╲│╱
     │
```

### 拐点效应

损失函数的等高线通常也是圆/椭圆形状。

当等高线碰到 L1 的"角"时，最优解在角上，这时一些权重为0。

```
L1 约束区域（菱形）
     ╱│╲
    ╱ │ ╲
   ╱  │  ╲
  ╱   │   ╲
 ╱    │    ╲
●─────┼─────●  等高线（椭圆）
 ╲    │    ╱
  ╲   │   ╱
   ╲  │  ╱
    ╲ │ ╱
     ╲│╱

最优解可能在角上，此时一些权重为0
```

### L2 的情况

L2 的约束区域是"圆形"：

```
     ╱│╲
    ╱ │ ╲
   ╱  │  ╲
  ╱   │   ╲
 ╱    │    ╲
╱─────┼─────╲  等高线（圆形）
 ╲    │    ╱
  ╲   │   ╱
   ╲  │  ╱
    ╲ │ ╱
     ╲│╱
```

等高线可能碰到任何位置，很少正好让权重为0。

---

## Lasso的效果

### 稀疏解

随着 α 增大，越来越多的权重变为0：

```
α = 0:     [5, 3, 2, 1, 0.5]
α = 1:     [4, 2.5, 1.5, 0.3, 0]
α = 10:    [2, 1, 0, 0, 0]
α = 100:   [0.5, 0, 0, 0, 0]
α = ∞:     [0, 0, 0, 0, 0]
```

注意：Lasso的权重可以精确变为0！

### 特征选择

权重为0的特征可以被"剔除"，只保留重要特征。

---

## 求解方法

### 无解析解

Lasso没有像岭回归那样的解析解。

### 数值优化

常用方法：

1. **坐标下降法**
   - 每次优化一个权重
   - 其他权重固定

2. **最小角回归法 (LARS)**
   - 专为Lasso设计
   - 高效计算整个路径

3. **近端梯度法**
   - 结合梯度下降和近端算子

---

## 正则化强度 α 的影响

### α = 0

普通线性回归，没有正则化。

### α 较小

轻微约束，少量权重为0。

### α 适中

适度约束，一些权重为0。

### α 很大

强约束，大部分权重为0。

---

## 偏差-方差权衡

### α 太小

- 低偏差
- 高方差
- 可能过拟合
- 特征选择效果弱

### α 太大

- 高偏差
- 低方差
- 可能欠拟合
- 过多特征被剔除

### α 适中

- 平衡偏差和方差
- 适度选择特征
- 通常泛化最好

---

## 与其他方法的比较

### Lasso vs 岭回归

| 特点 | Lasso (L1) | 岭回归 (L2) |
|------|-----------|------------|
| 正则化 | 绝对值和 | 平方和 |
| 特征选择 | 可得稀疏解 | 无权重为0 |
| 计算方式 | 数值优化 | 解析解 |
| 适合场景 | 特征选择 | 特征相关 |

### Lasso vs Elastic Net

Elastic Net = L1 + L2 的组合：

```
L = MSE + α₁ × ||w||₁ + α₂ × ||w||²
```

- 结合了两者的优点
- 既能特征选择，又能处理相关特征

---

## 何时使用Lasso？

### 适合的情况

✅ 需要特征选择
✅ 特征很多（高维数据）
✅ 认为只有少数特征重要
✅ 需要可解释的模型

### 不适合的情况

❌ 特征之间高度相关（用岭回归或Elastic Net更好）
❌ 所有特征都重要
❌ 不需要特征选择

---

## 实际例子

### 例子：基因表达数据

有1000个基因（特征），只有10个真正与疾病相关。

### 线性回归

可能会给很多基因非零权重，过拟合。

### 岭回归

约束权重，但所有基因都有权重，难解释。

### Lasso

```
α = 0.1: [0.5, 0, 0.3, 0, 0, 0, 0.2, 0, 0, 0, ...]
α = 1:   [0.2, 0, 0.1, 0, 0, 0, 0.05, 0, 0, 0, ...]
```

只有少数基因有非零权重，模型简洁、可解释。

---

## 特征相关的处理

### Lasso的问题

当特征高度相关时，Lasso会随机选择一个，其他设为0。

### 解决方法

1. **用Elastic Net**
   - L1 + L2 的组合

2. **先用岭回归筛选**
   - 岭回归处理相关特征更好

3. **稳定性选择**
   - 多次运行Lasso，看哪些特征经常被选中

---

## 超参数调优

### 如何选择 α？

1. **交叉验证**
   - 尝试不同的 α 值
   - 评估验证集误差

2. **信息准则**
   - AIC、BIC等

3. **路径追踪**
   - 观察权重随 α 的变化

### 交叉验证曲线

```
验证误差
  ↑
  │      ╱╲
  │     ╱  ╲
  │    ╱    ╲
  │   ╱      ╲
  │  ╱        ╲
  └──────────────→ α
      最佳 α
```

## 数据预处理

### 标准化

像岭回归一样，Lasso也需要标准化：

```
x' = (x - 均值) / 标准差
```

**原因**：
- L1 正则化约束权重大小
- 不同尺度的特征影响不同

---

## 缩减路径

### 权重变化

随着 α 增大，权重逐渐为0：

```
特征1:  5 ────── 3 ──── 1 ──── 0 ── 0
特征2:  3 ──── 2 ──── 0 ── 0 ── 0
特征3:  2 ──── 1 ──── 0 ── 0 ── 0
特征4:  1 ──── 0 ── 0 ── 0 ── 0
特征5:  0.5 ── 0 ── 0 ── 0 ── 0

        α=0   α=1   α=10  α=100
```

可以画出完整的"缩减路径"。

## 实践技巧

### 1. 标准化

必须对特征标准化。

### 2. 交叉验证

用交叉验证找最佳 α。

### 3. 稳定性

多次运行，检查特征选择的稳定性。

### 4. 解释性

观察哪些特征被选中，是否符合领域知识。

### Scikit-learn

from sklearn.preprocessing import StandardScaler

# 标准化
scaler = StandardScaler()

## 总结

**Lasso回归**：

1. 是线性回归 + L1 正则化
2. 约束权重大小，防止过拟合
3. 可以得到稀疏解，做特征选择
4. 无解析解，需要数值优化
5. 适合高维数据和特征选择

**核心思想**：
- 加惩罚项：α × ||w||₁
- 权重可以为0
- 自动做特征选择

**关键要点**：
- 标准化很重要
- 用交叉验证调 α
- 特征相关时可能不稳定
- 结果可解释性强

**记住**：Lasso就像"精简大师"，不仅让模型稳定，还能帮你找到真正重要的特征！
