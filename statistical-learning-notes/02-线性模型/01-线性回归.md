# 线性回归

## 什么是线性回归？

线性回归是**最简单、最基础的机器学习算法**之一。

它的核心思想：**用一条直线来拟合数据**，然后用这条直线来预测。

---

## 为什么叫"线性"？

因为假设预测值和输入之间是**线性关系**。

数学表达式：
```
y = w₁x₁ + w₂x₂ + ... + wₙxₙ + b
```

- y：预测值
- x₁, x₂, ..., xₙ：输入特征
- w₁, w₂, ..., wₙ：权重（系数）
- b：偏置（截距）

**一元线性回归**（一个特征）：
```
y = wx + b
```
这是一条直线。

**多元线性回归**（多个特征）：
```
y = w₁x₁ + w₂x₂ + w₃x₃ + b
```
这是一个平面或超平面。

## 生活中的例子

### 例子1：房价预测

假设房价只和面积有关：
```
价格 = 10000 × 面积 + 50000
```

- 10000是权重（每平米1万元）
- 50000是偏置（基础价格50万）

如果房子是100平米：
```
价格 = 10000 × 100 + 50000 = 150万元
```

### 例子2：收入预测

```
月薪 = 500 × 工作年限 + 3000
```

工作5年的人：
```
月薪 = 500 × 5 + 3000 = 5500元
```

---

## 直观理解

想象你有一组数据点：
```
工作年限：1  2  3  4  5  6
月  薪：4000 5000 5500 7000 7500 8000
```

画出来就是一些散点。线性回归就是**找一条直线**，让这些点尽量靠近这条线。

```
月薪
  ↑
8000│       ●
    │      ╱
7000│     ● ╱
    │    ╱ ●
6000│   ╱  ●
    │  ●
5000│ ●
    │╱
4000└────────────→ 工作年限
   0   2   4   6
```

## 目标是什么？

线性回归的目标是**找到最好的 w 和 b**，使得预测值和真实值的误差最小。

**误差**通常用**均方误差 (MSE)** 来衡量：
```
MSE = 平均((预测值 - 真实值)²)
```

我们希望 MSE 越小越好。

---

## 训练过程

### 步骤1：准备数据

收集数据，整理成：
- 特征 X：输入变量
- 标签 y：要预测的值

### 步骤2：初始化参数

随机设置 w 和 b 的初始值，比如：
```
w = 1, b = 0
```

### 步骤3：计算预测

用当前参数计算预测值：
```
y_pred = w × x + b
```

### 步骤4：计算误差

比较预测值和真实值，计算误差。

### 步骤5：更新参数

根据误差调整 w 和 b，减小误差。

### 步骤6：重复

重复步骤3-5，直到误差足够小或不再变化。

---

## 线性回归的假设

线性回归有一些假设，理解这些很重要：

### 1. 线性关系
假设特征和目标之间是线性关系。

### 2. 独立性
每个样本是独立的。

### 3. 同方差性
误差的方差在所有特征值下是相同的。

### 4. 正态性
误差服从正态分布。

**如果这些假设不成立**，线性回归可能不是最好的选择。

---

## 优点和缺点

### 优点

✅ **简单易懂**
- 数学基础清晰
- 容易解释

✅ **计算高效**
- 训练速度快
- 不需要太多计算资源

✅ **可解释性强**
- 可以看出每个特征的影响

✅ **基准模型**
- 可以作为其他模型的基准

### 缺点

❌ **只能处理线性关系**
- 对于非线性关系表现差

❌ **对异常值敏感**
- 一个异常点可能严重影响结果

❌ **假设较多**
- 如果假设不成立，效果不好

---

## 何时使用线性回归？

### 适合的情况

✅ 问题相对简单
✅ 特征和目标有线性关系
✅ 需要可解释性
✅ 作为基准模型

### 不适合的情况

❌ 有复杂的非线性关系
❌ 数据有大量异常值
❌ 特征之间存在复杂交互

---

## 和其他算法的关系

线性回归是很多算法的基础：

- **逻辑回归**：在线性回归基础上加了sigmoid函数
- **岭回归**：线性回归加L2正则化
- **Lasso回归**：线性回归加L1正则化
- **多项式回归**：特征转换后用线性回归

---

## 总结

**线性回归**：

1. 用直线/平面拟合数据
2. 找到使误差最小的权重和偏置
3. 简单、快速、可解释
4. 是机器学习的入门算法

**记住**：
- 适合线性关系
- 对异常值敏感
- 是理解更复杂算法的基础

线性回归虽然简单，但它的思想贯穿整个机器学习。理解它，就理解了机器学习的核心！
