# 回归假设检验与诊断

## 为什么需要诊断？

线性回归模型看似简单，但要得到可靠的结论，必须满足一些**基本假设**。

就像盖房子需要检查地基是否牢固，使用回归模型之前也需要诊断假设是否成立。

**如果假设不满足**，回归结果可能：
- 系数估计有偏
- 置信区间不准确
- 预测效果差
- 得出错误结论

---

## LINE 假设

线性回归有四个核心假设，可以用记忆法 **LINE** 来记忆：

### L - Linearity（线性关系）

假设：自变量和因变量之间存在线性关系。

**如何检验：**
- 绘制散点图
- 观察残差图

**线性关系的例子：**
```
房价 vs 面积 → 大致线性关系
薪资 vs 工作年限 → 大致线性关系
```

**违反假设的情况：**
```
销售额 vs 时间 → 可能有季节性（非线性）
```

**如何修复：**
- 添加多项式项
- 使用变量变换（如对数）
- 考虑非线性模型

---

### I - Independence（独立性）

假设：每个观测值之间是独立的。

**如何检验：**
- 检查数据收集方式
- 时间序列数据：使用Durbin-Watson检验
- 绘制残差的时间序列图

**违反假设的情况：**
- 时间序列数据（今天的值可能受昨天影响）
- 空间数据（相邻地区相互影响）
- 重复测量（同一个体多次测量）

**如何修复：**
- 使用时间序列模型
- 添加随机效应（混合模型）
- 使用聚类稳健标准误

---

### N - Normality（正态性）

假设：误差项服从正态分布。

**重要说明：** 这是关于误差的假设，不是关于原始数据的假设！

**如何检验：**

#### 1. Q-Q图（分位数-分位数图）

最常用的诊断工具：

```
正态分位数
   ↑
  3│       ●
   │      ╱
  2│     ● ╱
   │    ╱  ●
  1│   ●   ●
   │  ●
  0│ ●
   │╱
 -1└────────→ 样本分位数
   -1  0   1
```

- 如果点大致在直线上 → 正态性满足
- 如果点明显偏离 → 可能不满足

#### 2. Shapiro-Wilk检验

统计检验方法：
- H₀：数据来自正态分布
- p < 0.05 → 拒绝正态性假设

#### 3. 直方图

观察误差的分布形状。

**如何修复：**
- 大样本下，正态性假设不那么严格（中心极限定理）
- 变量变换（对数、平方根）
- 使用非参数方法

---

### E - Equal Variance（同方差性）

假设：误差的方差在所有预测值水平下都相同。

**什么是同方差？**
```
对于所有 x，Var(ε|x) = σ²（常数）
```

**什么是异方差？**
```
Var(ε|x) 随 x 变化而变化
```

**如何检验：**

#### 1. 残差 vs 拟合值图

```
残差
   ↑
  2│  ●   ●
   │  ●   ●
  0│●●●●●●●●●●●●●●
   │  ●   ●
 -2│  ●   ●
   └────────→ 拟合值
```

- 散点均匀分布 → 同方差
- 散点呈漏斗形 → 异方差

#### 2. Breusch-Pagan检验

统计检验异方差性。

**违反假设的例子：**
```
收入 vs 教育年限
→ 高收入人群的收入差异更大（异方差）
```

**如何修复：**
- 对因变量进行变换（如对数变换）
- 使用加权最小二乘法（WLS）
- 使用稳健标准误

---

## 残差分析

残差是**观测值与预测值的差**：
```
残差 = 实际值 - 预测值
```

残差分析是诊断回归模型的核心工具。

### 残差图的类型

#### 1. 残差 vs 拟合值图

```
残差
   ↑
  2│    ●   ●
   │   ● ● ●
  0│ ● ● ● ● ●
   │  ● ● ●
 -2│   ●   ●
   └────────→ 拟合值
```

**理想情况：** 随机散布，无明显模式

**问题模式：**
- U形或倒U形 → 非线性关系
- 漏斗形 → 异方差
- 趋势 → 遗漏重要变量

#### 2. 残差 vs 自变量图

检查每个自变量是否被正确建模。

#### 3. 残差的时间序列图

对于时间数据，检查是否存在自相关。

```
残差
   ↑
  2│  ●    ●
   │    ●    ●
  0│●     ●     ●
   │  ●    ●
 -2│    ●
   └────────→ 时间
```

---

## 异常值和影响点

### 异常值（Outliers）

远离其他数据点的观测值。

**如何识别：**
- 标准化残差绝对值 > 2 或 3
- 学生化残差（考虑异常值本身的影响）

**例子：**
```
房价数据中出现一个 1000 万的房子
（而其他房子都在 100-500 万之间）
```

### 高杠杆点（High Leverage Points）

自变量值异常的点，对回归线有很大"杠杆"作用。

**如何识别：**
- 杠杆值（Leverage） > 2*(p+1)/n
  - p：自变量个数
  - n：样本量

### 影响点（Influential Points）

对回归系数估计有重大影响的点。

**Cook距离**
```
衡量一个点对整体模型的影响
```

**判断标准：**
- Cook距离 > 0.5 → 值得关注
- Cook距离 > 1 → 强影响点

**处理异常值的步骤：**

1. **核实数据**
   - 是录入错误吗？
   - 数据是否合理？

2. **理解原因**
   - 这个点为什么异常？
   - 代表真实现象吗？

3. **决定处理方式**
   - 如果是错误 → 修正或删除
   - 如果是真实值 → 考虑鲁棒回归
   - 报告两种结果（含/不含异常值）

## 多重共线性

当自变量之间存在高度相关时，会出现多重共线性。

### 问题表现

- 系数估计不稳定
- 方差膨胀（标准误变大）
- 难以解释各个变量的独立影响

### 如何检测

#### 1. 相关系数矩阵

```
        X1    X2    X3
X1   1.00  0.95  0.30
X2   0.95  1.00  0.25
X3   0.30  0.25  1.00
```

X1和X2高度相关（0.95）

#### 2. 方差膨胀因子（VIF）

```
VIF > 10  → 严重多重共线性
VIF > 5   → 中度多重共线性
VIF = 1   → 无共线性
```

### 如何解决

- 删除高度相关的变量之一
- 使用主成分分析（PCA）
- 使用岭回归或Lasso
- 收集更多数据

## Python 实现示例

### 基础回归和诊断

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
import statsmodels.api as sm
from scipy import stats

# 生成示例数据
np.random.seed(42)
n = 100
x1 = np.random.normal(0, 1, n)
x2 = np.random.normal(0, 1, n)
y = 2 * x1 + 3 * x2 + np.random.normal(0, 1, n)

# 创建DataFrame
df = pd.DataFrame({'x1': x1, 'x2': x2, 'y': y})

# 添加常数项（截距）
X = sm.add_constant(df[['x1', 'x2']])

# 拟合线性回归模型
model = sm.OLS(df['y'], X).fit()

# 输出回归结果
print(model.summary())
```

### 残差图

```python
# 获取残差和拟合值
residuals = model.resid
fitted = model.fittedvalues

# 残差 vs 拟合值图
plt.figure(figsize=(10, 6))
plt.scatter(fitted, residuals)
plt.axhline(y=0, color='r', linestyle='--')
plt.xlabel('Fitted Values')
plt.ylabel('Residuals')
plt.title('Residuals vs Fitted Values')
plt.show()
```

### Q-Q图（检验正态性）

```python
# Q-Q图
plt.figure(figsize=(10, 6))
stats.probplot(residuals, dist="norm", plot=plt)
plt.title('Q-Q Plot')
plt.show()

# Shapiro-Wilk检验
stat, p_value = stats.shapiro(residuals)
print(f"Shapiro-Wilk test: statistic={stat:.4f}, p-value={p_value:.4f}")
```

### 检验同方差性

```python
# Breusch-Pagan检验
from statsmodels.stats.diagnostic import het_breuschpagan

bp_test = het_breuschpagan(residuals, model.model.exog)
print(f"Breusch-Pagan test: statistic={bp_test[0]:.4f}, p-value={bp_test[1]:.4f}")
```

### 检验自相关（Durbin-Watson）

```python
# Durbin-Watson检验
from statsmodels.stats.stattools import durbin_watson

dw = durbin_watson(residuals)
print(f"Durbin-Watson statistic: {dw:.4f}")
# 值接近2表示无自相关
# 值<0表示正自相关，>4表示负自相关
```

### 识别异常值

```python
# 计算标准化残差
standardized_residuals = residuals / np.std(residuals)

# 计算Cook距离
influence = model.get_influence()
cooks_d = influence.cooks_distance[0]

# 识别异常值
outliers = np.abs(standardized_residuals) > 2
influential = cooks_d > 0.5

print(f"异常值数量: {np.sum(outliers)}")
print(f"强影响点数量: {np.sum(influential)}")

# 可视化Cook距离
plt.figure(figsize=(10, 6))
plt.stem(np.arange(len(cooks_d)), cooks_d)
plt.axhline(y=0.5, color='r', linestyle='--')
plt.xlabel('Observation Index')
plt.ylabel("Cook's Distance")
plt.title("Cook's Distance")
plt.show()
```

### 检验多重共线性

```python
# 计算VIF
from statsmodels.stats.outliers_influence import variance_inflation_factor

vif_data = pd.DataFrame()
vif_data["Variable"] = X.columns
vif_data["VIF"] = [variance_inflation_factor(X.values, i)
                   for i in range(X.shape[1])]

print(vif_data)
```

### 综合诊断图

```python
# 综合诊断图（类似R的plot.lm）
fig, axes = plt.subplots(2, 2, figsize=(12, 10))

# 1. 残差 vs 拟合值
axes[0, 0].scatter(fitted, residuals)
axes[0, 0].axhline(y=0, color='r', linestyle='--')
axes[0, 0].set_xlabel('Fitted Values')
axes[0, 0].set_ylabel('Residuals')
axes[0, 0].set_title('Residuals vs Fitted')

# 2. Q-Q图
stats.probplot(residuals, dist="norm", plot=axes[0, 1])
axes[0, 1].set_title('Normal Q-Q Plot')

# 3. Scale-Location图
sqrt_abs_resid = np.sqrt(np.abs(residuals))
axes[1, 0].scatter(fitted, sqrt_abs_resid)
axes[1, 0].set_xlabel('Fitted Values')
axes[1, 0].set_ylabel('√|Standardized Residuals|')
axes[1, 0].set_title('Scale-Location')

# 4. Cook距离
axes[1, 1].stem(np.arange(len(cooks_d)), cooks_d)
axes[1, 1].axhline(y=0.5, color='r', linestyle='--')
axes[1, 1].set_xlabel('Observation Index')
axes[1, 1].set_ylabel("Cook's Distance")
axes[1, 1].set_title("Cook's Distance")

plt.tight_layout()
plt.show()
```

## 诊断清单

使用回归模型前，检查以下项目：

### 必须检查

- [ ] **线性关系**：残差图无模式
- [ ] **独立性**：Durbin-Watson接近2
- [ ] **正态性**：Q-Q图大致为直线
- [ ] **同方差性**：残差图无漏斗形

### 建议检查

- [ ] **异常值**：Cook距离和标准化残差
- [ ] **多重共线性**：VIF值
- [ ] **模型拟合度**：R²和调整R²

### 如果假设不满足

1. **线性假设不满足**
   - 考虑变量变换
   - 添加交互项或多项式
   - 使用非线性模型

2. **正态性不满足**
   - 大样本问题不大
   - 变量变换
   - 非参数方法

3. **异方差性**
   - 变量变换（对数）
   - 加权最小二乘
   - 稳健标准误

4. **多重共线性**
   - 删除变量
   - 正则化（岭/Lasso）
   - 主成分分析

---

## 常见误区

### 误区1：检查原始数据的正态性

**错误：** 用Shapiro-Wilk检验自变量或因变量

**正确：** 应该检验**残差**的正态性

### 误区2：异常值一定要删除

**错误：** 看到异常值就删除

**正确：**
- 先核实是否为数据错误
- 理解异常原因
- 考虑使用鲁棒方法

### 误区3：完美满足所有假设

**错误：** 追求所有检验都完美

**正确：**
- 轻微偏离可以接受
- 关注实际影响
- 根据数据特征选择模型

---

## 总结

**回归诊断的核心要点：**

1. **LINE假设**是基础
   - 线性关系（Linearity）
   - 独立性（Independence）
   - 正态性（Normality）
   - 同方差性（Equal Variance）

2. **残差分析**是主要工具
   - 残差图检查各种假设
   - 可视化比纯统计检验更有用

3. **异常值处理要谨慎**
   - 先理解再决定
   - 报告处理方式

4. **Python诊断工具丰富**
   - statsmodels提供完整诊断功能
   - matplotlib/seaborn可视化

**记住：**
- 诊断是回归分析的重要组成部分
- 不要跳过诊断直接解读结果
- 诊断的目的是理解模型，不是追求完美

好的回归分析 = 好的模型 + 好的诊断 + 合理的解读
