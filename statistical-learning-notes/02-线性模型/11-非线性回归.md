# 非线性回归

## 什么是非线性回归？

**非线性回归**用于建模因变量与自变量之间的**非线性关系**。

现实生活中，很多关系都不是线性的：

- **房价增长**：越大的房子单价可能越低
- **学习曲线**：刚开始进步快，后期变慢
- **细菌生长**：指数增长
- **物体冷却**：按指数衰减

**线性回归的局限**：
- 只能拟合直线/平面
- 无法捕捉弯曲、指数、对数等关系

**非线性回归的优势**：
- 可以拟合任意形状的曲线
- 更符合实际问题的复杂性
- 提供更准确的预测

## 非线性关系的识别

### 1. 散点图观察

最直观的方法是绘制散点图：

```
y
│      ●●●
│    ●●   ●●
│   ●       ●
│  ●         ●
│ ●           ●
│●             ●
└──────────────→ x
```

观察数据点的形状：
- **直线**：线性关系
- **曲线**：非线性关系
- **S形**：可能需要逻辑函数
- **指数形状**：指数或对数关系

### 2. 残差分析

拟合线性模型后，检查残差：

```python
# 线性模型的残差
residuals = y_true - y_pred

# 绘制残差图
plt.scatter(X, residuals)
plt.axhline(y=0, color='r', linestyle='--')
```

**残差模式**：
- 随机分布：线性关系合适
- 有规律（如U形）：存在非线性关系

### 3. 相关性变换

对变量进行变换，看是否能提高线性度：

- **对数变换**：log(y) 或 log(x)
- **平方根变换**：√y 或 √x
- **倒数变换**：1/y 或 1/x
- **Box-Cox变换**：自动寻找最佳变换

### 4. 统计检验

- **残差的Durbin-Watson检验**：检验自相关
- **White检验**：检验异方差性
- **RESET检验**：检验模型设定是否正确

## 多项式回归

### 什么是多项式回归？

多项式回归是**最常用的非线性回归方法**，通过添加高次项来拟合曲线。

**一阶（线性）**：
```
y = w₁x + w₀
```

**二阶（抛物线）**：
```
y = w₂x² + w₁x + w₀
```

**三阶（三次曲线）**：
```
y = w₃x³ + w₂x² + w₁x + w₀
```

**一般形式**：
```
y = wₙxⁿ + wₙ₋₁xⁿ⁻¹ + ... + w₁x + w₀
```

### 多项式回归的实现

**方法1：手动构造特征**

```python
import numpy as np
from sklearn.linear_model import LinearRegression

# 原始数据
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([1, 4, 9, 16, 25])  # y = x²

# 构造多项式特征
X_poly = np.column_stack([
    X,           # x
    X**2,        # x²
    X**3         # x³
])

# 拟合线性模型（但实际上是多项式）
model = LinearRegression()
model.fit(X_poly, y)
```

**方法2：使用PolynomialFeatures**

```python
from sklearn.preprocessing import PolynomialFeatures
from sklearn.pipeline import make_pipeline

# 创建多项式回归管道
degree = 2  # 二次多项式
model = make_pipeline(
    PolynomialFeatures(degree=degree),
    LinearRegression()
)

# 拟合模型
model.fit(X, y)

# 预测
y_pred = model.predict(X_new)
```

### 多项式阶数的选择

**阶数过低**：
- 欠拟合
- 无法捕捉数据的真实模式

**阶数过高**：
- 过拟合
- 模型过于复杂，泛化能力差

**选择方法**：

1. **交叉验证**：
```python
from sklearn.model_selection import cross_val_score

degrees = [1, 2, 3, 4, 5]
scores = []

for degree in degrees:
    model = make_pipeline(
        PolynomialFeatures(degree=degree),
        LinearRegression()
    )
    cv_scores = cross_val_score(model, X, y, cv=5, scoring='neg_mse')
    scores.append(-cv_scores.mean())

# 选择误差最小的阶数
best_degree = degrees[np.argmin(scores)]
```

2. **AIC/BIC准则**：
```python
from sklearn.metrics import mean_squared_error

def calculate_aic(n, mse, num_params):
    return n * np.log(mse) + 2 * num_params

def calculate_bic(n, mse, num_params):
    return n * np.log(mse) + np.log(n) * num_params

# 计算不同阶数的AIC/BIC
aic_values = []
bic_values = []

for degree in degrees:
    model = make_pipeline(
        PolynomialFeatures(degree=degree),
        LinearRegression()
    )
    model.fit(X, y)
    y_pred = model.predict(X)
    mse = mean_squared_error(y, y_pred)
    num_params = degree + 1  # 阶数+1（截距项）
    
    aic_values.append(calculate_aic(len(y), mse, num_params))
    bic_values.append(calculate_bic(len(y), mse, num_params))

# 选择AIC/BIC最小的阶数
```

### 多项式回归的例子

**例子：拟合二次关系**

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.preprocessing import PolynomialFeatures
from sklearn.linear_model import LinearRegression
from sklearn.pipeline import make_pipeline

# 生成数据（带噪声的二次关系）
np.random.seed(42)
X = np.linspace(-3, 3, 100).reshape(-1, 1)
y = 2 * X**2 - 3 * X + 5 + np.random.normal(0, 2, X.shape)

# 拟合不同阶数的多项式
fig, axes = plt.subplots(2, 2, figsize=(12, 10))

degrees = [1, 2, 3, 8]
for ax, degree in zip(axes.flat, degrees):
    model = make_pipeline(
        PolynomialFeatures(degree=degree),
        LinearRegression()
    )
    model.fit(X, y)
    y_pred = model.predict(X)
    
    ax.scatter(X, y, alpha=0.5, label='数据')
    ax.plot(X, y_pred, 'r-', linewidth=2, label=f'阶数={degree}')
    ax.set_title(f'多项式回归（阶数={degree}）')
    ax.legend()

plt.tight_layout()
plt.show()
```

**结果分析**：
- 阶数=1：直线，欠拟合
- 阶数=2：抛物线，拟合良好
- 阶数=3：曲线，稍有过度
- 阶数=8：复杂曲线，严重过拟合

## 其他常见非线性模型

### 1. 指数模型

**模型形式**：
```
y = a * e^(b * x)
```

**应用场景**：
- 人口增长
- 细菌繁殖
- 复利计算
- 放射性衰变

**转化为线性模型**：

两边取对数：
```
ln(y) = ln(a) + b * x
```

令 Y = ln(y)，A = ln(a)：
```
Y = A + b * x
```

这是一个线性回归问题！

**实现**：

```python
import numpy as np
from scipy.optimize import curve_fit

# 指数函数
def exponential_func(x, a, b):
    return a * np.exp(b * x)

# 生成数据
x_data = np.linspace(0, 4, 20)
y_data = 2 * np.exp(0.5 * x_data) + np.random.normal(0, 0.5, len(x_data))

# 方法1：线性化后用线性回归
log_y = np.log(y_data)
b, log_a = np.polyfit(x_data, log_y, 1)
a_linear = np.exp(log_a)

# 方法2：使用curve_fit（非线性最小二乘）
popt, pcov = curve_fit(exponential_func, x_data, y_data, p0=[1, 0.5])
a_nonlinear, b_nonlinear = popt

print(f"线性化方法: a={a_linear:.2f}, b={b:.2f}")
print(f"非线性方法: a={a_nonlinear:.2f}, b={b_nonlinear:.2f}")
```

### 2. 对数模型

**模型形式**：
```
y = a + b * ln(x)
```

**应用场景**：
- 学习曲线
- 心理物理学中的感觉强度
- 收入与经验的关系

**实现**：

```python
from scipy.optimize import curve_fit

# 对数函数
def logarithmic_func(x, a, b):
    return a + b * np.log(x)

# 生成数据
x_data = np.linspace(1, 10, 30)
y_data = 5 + 2 * np.log(x_data) + np.random.normal(0, 0.3, len(x_data))

# 拟合对数模型
popt, pcov = curve_fit(logarithmic_func, x_data, y_data, p0=[1, 1])
a, b = popt

print(f"对数模型: y = {a:.2f} + {b:.2f} * ln(x)")
```

### 3. 幂函数模型

**模型形式**：
```
y = a * x^b
```

**应用场景**：
- 生物学中的异速生长
- 经济学中的生产函数
- 物理学中的幂律关系

**转化为线性模型**：

两边取对数：
```
ln(y) = ln(a) + b * ln(x)
```

令 Y = ln(y)，X = ln(x)，A = ln(a)：
```
Y = A + b * X
```

**实现**：

```python
# 幂函数
def power_func(x, a, b):
    return a * x**b

# 生成数据
x_data = np.linspace(1, 10, 30)
y_data = 3 * x_data**0.7 + np.random.normal(0, 0.5, len(x_data))

# 方法1：线性化
log_x = np.log(x_data)
log_y = np.log(y_data)
b, log_a = np.polyfit(log_x, log_y, 1)
a_linear = np.exp(log_a)

# 方法2：非线性拟合
popt, pcov = curve_fit(power_func, x_data, y_data, p0=[1, 1])
a_nonlinear, b_nonlinear = popt

print(f"线性化方法: y = {a_linear:.2f} * x^{b:.2f}")
print(f"非线性方法: y = {a_nonlinear:.2f} * x^{b_nonlinear:.2f}")
```

### 4. 逻辑增长模型（S形曲线）

**模型形式**：
```
y = L / (1 + e^(-k*(x - x0)))
```

**参数**：
- L：最大值（上限）
- k：增长率（控制曲线陡峭程度）
- x0：拐点（中点）

**应用场景**：
- 人口增长
- 产品市场渗透
- 疾病传播
- 学习进度

**实现**：

```python
# 逻辑函数
def logistic_func(x, L, k, x0):
    return L / (1 + np.exp(-k * (x - x0)))

# 生成数据
x_data = np.linspace(-5, 5, 50)
y_data = 10 / (1 + np.exp(-1 * (x_data - 0))) + np.random.normal(0, 0.3, len(x_data))

# 拟合逻辑模型
popt, pcov = curve_fit(logistic_func, x_data, y_data, p0=[10, 1, 0])
L, k, x0 = popt

print(f"逻辑模型: L={L:.2f}, k={k:.2f}, x0={x0:.2f}")
```

### 5. 高斯模型

**模型形式**：
```
y = a * exp(-(x - b)² / (2c²))
```

**应用场景**：
- 光谱分析
- 信号处理
- 统计分布

**实现**：

```python
# 高斯函数
def gaussian_func(x, a, b, c):
    return a * np.exp(-(x - b)**2 / (2 * c**2))

# 生成数据
x_data = np.linspace(-5, 5, 50)
y_data = 5 * np.exp(-(x_data - 0)**2 / (2 * 1**2)) + np.random.normal(0, 0.2, len(x_data))

# 拟合高斯模型
popt, pcov = curve_fit(gaussian_func, x_data, y_data, p0=[5, 0, 1])
a, b, c = popt

print(f"高斯模型: a={a:.2f}, b={b:.2f}, c={c:.2f}")
```

## 参数估计方法（非线性最小二乘）

### 非线性最小二乘法

**目标**：找到参数 θ，使得残差平方和最小：

```
S(θ) = Σ[yi - f(xi, θ)]²
```

其中：
- yi：观测值
- f(xi, θ)：模型预测值
- θ：待估参数

### 求解方法

#### 1. 高斯-牛顿法（Gauss-Newton）

**思想**：用泰勒展开线性化模型，迭代求解。

**步骤**：
1. 给定初始参数 θ₀
2. 用泰勒展开近似模型：f(x, θ) ≈ f(x, θ₀) + J(θ₀)(θ - θ₀)
3. 求解线性化的最小二乘问题
4. 更新参数，重复直到收敛

**优点**：收敛快（如果初始值好）
**缺点**：可能不收敛（如果初始值差）

#### 2. Levenberg-Marquardt算法

**思想**：结合高斯-牛顿法和梯度下降法。

**特点**：
- 残差小时，像高斯-牛顿法（收敛快）
- 残差大时，像梯度下降法（稳定）
- 是最常用的非线性最小二乘算法

#### 3. 梯度下降法

**思想**：沿着梯度反方向迭代更新参数。

**更新规则**：
```
θ_new = θ_old - α * ∇S(θ_old)
```

其中：
- α：学习率
- ∇S(θ)：梯度

**优点**：简单，保证收敛
**缺点**：收敛慢，需要调学习率

### Python实现：scipy.optimize

**使用curve_fit**：

```python
import numpy as np
from scipy.optimize import curve_fit

# 定义模型函数
def model_func(x, a, b, c):
    return a * x**2 + b * x + c

# 生成数据
x_data = np.linspace(0, 10, 20)
y_data = 2 * x_data**2 - 3 * x_data + 5 + np.random.normal(0, 1, len(x_data))

# 拟合模型
popt, pcov = curve_fit(model_func, x_data, y_data, p0=[1, 1, 1])

# 提取参数
a, b, c = popt
perr = np.sqrt(np.diag(pcov))  # 参数的标准误差

print(f"参数: a={a:.2f}±{perr[0]:.2f}, b={b:.2f}±{perr[1]:.2f}, c={c:.2f}±{perr[2]:.2f}")
```

**使用least_squares**（更灵活）：

```python
from scipy.optimize import least_squares

# 定义残差函数
def residuals(params, x, y):
    a, b, c = params
    return y - (a * x**2 + b * x + c)

# 初始参数
initial_params = [1, 1, 1]

# 拟合
result = least_squares(residuals, initial_params, args=(x_data, y_data))

# 获取结果
a, b, c = result.x
print(f"参数: a={a:.2f}, b={b:.2f}, c={c:.2f}")
print(f"成功: {result.success}")
print(f"消息: {result.message}")
```

### 初始值的选择

**重要性**：非线性最小二乘对初始值敏感！

**选择方法**：

1. **根据物理意义**：
   - 指数函数：a≈y[0]，b≈(log(y[-1]/y[0]))/(x[-1]-x[0])
   - 高斯函数：a≈max(y)，b≈x[argmax(y)]，c≈宽度估计

2. **线性化**：
   - 指数、对数、幂函数可以先线性化估计

3. **网格搜索**：
   - 在合理范围内尝试多个初始值

4. **经验**：
   - 画图观察，手动估计

## 模型选择和评估

### 评估指标

#### 1. R²（决定系数）

**含义**：模型解释的方差比例。

```python
from sklearn.metrics import r2_score

r2 = r2_score(y_true, y_pred)
```

**解释**：
- R² = 1：完美拟合
- R² = 0：模型不比均值好
- R² < 0：模型比均值还差（过拟合严重）

#### 2. RMSE（均方根误差）

```python
from sklearn.metrics import mean_squared_error

rmse = np.sqrt(mean_squared_error(y_true, y_pred))
```

#### 3. MAE（平均绝对误差）

```python
from sklearn.metrics import mean_absolute_error

mae = mean_absolute_error(y_true, y_pred)
```

#### 4. MAPE（平均绝对百分比误差）

```python
mape = np.mean(np.abs((y_true - y_pred) / y_true)) * 100
```

### 信息准则

#### AIC（赤池信息准则）

**目的**：平衡拟合优度和模型复杂度。

```
AIC = n * ln(σ²) + 2 * k
```

其中：
- n：样本数
- σ²：残差方差
- k：参数个数

**选择**：AIC越小越好

#### BIC（贝叶斯信息准则）

```
BIC = n * ln(σ²) + k * ln(n)
```

**选择**：BIC越小越好（比AIC惩罚复杂模型更重）

```python
def calculate_aic(n, mse, num_params):
    return n * np.log(mse) + 2 * num_params

def calculate_bic(n, mse, num_params):
    return n * np.log(mse) + np.log(n) * num_params

# 使用
aic = calculate_aic(len(y), mse, num_params)
bic = calculate_bic(len(y), mse, num_params)
```

### 模型比较流程

```python
def compare_models(X, y, models_dict):
    """比较多个模型的性能"""
    results = []
    
    for name, model in models_dict.items():
        # 训练
        model.fit(X, y)
        y_pred = model.predict(X)
        
        # 计算指标
        r2 = r2_score(y, y_pred)
        rmse = np.sqrt(mean_squared_error(y, y_pred))
        mae = mean_absolute_error(y, y_pred)
        
        # 交叉验证
        cv_scores = cross_val_score(model, X, y, cv=5, scoring='neg_mse')
        cv_rmse = np.sqrt(-cv_scores.mean())
        
        # 计算AIC/BIC
        n = len(y)
        mse = mean_squared_error(y, y_pred)
        # 估计参数个数（对于scikit-learn模型不太直接）
        num_params = len(model.get_params())  # 近似
        
        aic = calculate_aic(n, mse, num_params)
        bic = calculate_bic(n, mse, num_params)
        
        results.append({
            '模型': name,
            'R²': r2,
            'RMSE': rmse,
            'MAE': mae,
            'CV_RMSE': cv_rmse,
            'AIC': aic,
            'BIC': bic
        })
    
    return pd.DataFrame(results)
```

### 示例2：自动选择最佳非线性模型

```python
import numpy as np
import pandas as pd
from scipy.optimize import curve_fit
from sklearn.metrics import r2_score, mean_squared_error

# 定义各种非线性模型
def linear(x, a, b):
    return a + b * x

def quadratic(x, a, b, c):
    return a + b * x + c * x**2

def cubic(x, a, b, c, d):
    return a + b * x + c * x**2 + d * x**3

def exponential(x, a, b):
    return a * np.exp(b * x)

def logarithmic(x, a, b):
    return a + b * np.log(x + 1)

def power(x, a, b):
    return a * x**b

def logistic(x, L, k, x0):
    return L / (1 + np.exp(-k * (x - x0)))

def gaussian(x, a, b, c):
    return a * np.exp(-(x - b)**2 / (2 * c**2))

# 模型字典
models = {
    '线性': (linear, [1, 1]),
    '二次': (quadratic, [1, 1, 1]),
    '三次': (cubic, [1, 1, 1, 1]),
    '指数': (exponential, [1, 0.5]),
    '对数': (logarithmic, [1, 1]),
    '幂函数': (power, [1, 1]),
    '逻辑函数': (logistic, [1, 1, 0]),
    '高斯': (gaussian, [1, 0, 1])
}

def fit_model(x, y, model_name):
    """拟合单个模型"""
    func, p0 = models[model_name]
    
    try:
        # 拟合
        popt, pcov = curve_fit(func, x, y, p0=p0, maxfev=10000)
        
        # 预测
        y_pred = func(x, *popt)
        
        # 计算指标
        r2 = r2_score(y, y_pred)
        rmse = np.sqrt(mean_squared_error(y, y_pred))
        
        # 计算AIC
        n = len(y)
        mse = mean_squared_error(y, y_pred)
        k = len(popt)
        aic = n * np.log(mse) + 2 * k
        
        return {
            '成功': True,
            '参数': popt,
            'R²': r2,
            'RMSE': rmse,
            'AIC': aic
        }
    except Exception as e:
        return {
            '成功': False,
            '错误': str(e)
        }

def auto_select_model(x, y):
    """自动选择最佳模型"""
    results = []
    
    for model_name in models.keys():
        result = fit_model(x, y, model_name)
        result['模型'] = model_name
        results.append(result)
    
    # 转换为DataFrame
    df = pd.DataFrame(results)
    
    # 只保留成功的模型
    df_success = df[df['成功']].copy()
    
    # 按AIC排序
    df_success = df_success.sort_values('AIC')
    
    return df, df_success

# 使用示例
np.random.seed(42)
x = np.linspace(0, 5, 50)
y = 3 * np.exp(0.5 * x) + np.random.normal(0, 0.5, len(x))

# 自动选择
all_results, success_results = auto_select_model(x, y)

print("=" * 60)
print("所有模型拟合结果:")
print("=" * 60)
print(all_results[['模型', '成功']].to_string(index=False))

print("\n" + "=" * 60)
print("成功模型比较（按AIC排序）:")
print("=" * 60)
print(success_results[['模型', 'R²', 'RMSE', 'AIC']].to_string(index=False))

# 选择最佳模型
best_model = success_results.iloc[0]
print(f"\n最佳模型: {best_model['模型']}")
print(f"R² = {best_model['R²']:.4f}")
print(f"RMSE = {best_model['RMSE']:.4f}")
print(f"AIC = {best_model['AIC']:.4f}")
print(f"参数: {best_model['参数']}")
```

```python
import numpy as np

    test_r2 = r2_score(y_test, y_test_pred)

    test_rmse = np.sqrt(mean_squared_error(y_test, y_test_pred))

    cv_scores = cross_val_score(model, X_train, y_train, cv=5, scoring='neg_mse')

    results.append({
        '模型': name,
        '训练R²': train_r2,
        '测试R²': test_r2,
        '训练RMSE': train_rmse,

        'CV_RMSE': cv_rmse
```

```python
import numpy as np
```

## 常见问题和注意事项

### 1. 过拟合

**问题**：模型太复杂，记住了训练数据的噪声。

**症状**：
- 训练集误差很小，测试集误差很大
- R²接近1，但预测新数据时表现差

**解决方法**：
- 减少模型复杂度（降低多项式阶数）
- 增加训练数据
- 使用正则化（岭回归、Lasso）
- 使用交叉验证选择模型

### 2. 初始值敏感性

**问题**：非线性最小二乘对初始值敏感，可能收敛到局部最优。

**解决方法**：
- 尝试多个初始值
- 使用物理意义或线性化估计初始值
- 使用全局优化算法（如差分进化、模拟退火）

### 3. 多重共线性

**问题**：在多项式回归中，x、x²、x³等高度相关。

**解决方法**：
- 中心化变量（x - mean(x)）
- 使用正交多项式
- 使用主成分分析

### 4. 异方差性

**问题**：误差的方差不是常数，随x变化。

**解决方法**：
- 对y进行变换（如log变换）
- 使用加权最小二乘
- 使用稳健回归

### 5. 模型选择偏差

**问题**：用同一数据选择和评估模型，导致过于乐观的估计。

**解决方法**：
- 使用独立的验证集
- 使用嵌套交叉验证
- 预留测试集，只在最后使用

## 总结

**非线性回归**：

1. **适用场景**：
   - 变量间存在非线性关系
   - 线性模型无法充分拟合数据
   - 需要更灵活的建模能力

2. **主要方法**：
   - **多项式回归**：最常用，通过添加高次项实现
   - **指数/对数/幂函数**：可线性化后用线性回归
   - **逻辑/高斯函数**：需要非线性最小二乘

3. **参数估计**：
   - **非线性最小二乘**：Levenberg-Marquardt算法最常用
   - **初始值**：很重要，影响收敛性
   - **梯度下降**：简单但慢

4. **模型选择**：
   - **交叉验证**：最可靠
   - **AIC/BIC**：平衡拟合和复杂度
   - **残差分析**：检查模型假设

5. **实践建议**：
   - 先画散点图，观察关系
   - 从简单模型开始
   - 使用交叉验证防止过拟合
   - 检查残差，验证模型假设
   - 选择可解释的模型（如果有多种模型表现相近）

**记住**：
- 非线性回归是线性回归的扩展
- 多项式回归是最常用的方法
- 模型复杂度要适度
- 评估时关注泛化能力，不只是训练误差

非线性回归让我们能够捕捉现实世界中复杂的非线性关系，是数据分析中的重要工具！
