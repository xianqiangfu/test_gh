# 回归系数的置信区间与假设检验

## 什么是回归系数的统计推断？

在回归分析中，除了估计回归系数的值，我们还需要回答以下关键问题：

- **估计的系数有多准确？**（置信区间）
- **系数是否显著不为零？**（假设检验）
- **自变量对因变量是否有真实影响？**

这就是**回归系数的统计推断**的核心内容。

---

## 回归系数的抽样分布

### 为什么需要抽样分布？

当我们从总体中抽取样本拟合回归模型时，得到的回归系数（β̂）是随机变量，会因样本不同而变化。

### OLS估计量的性质

对于经典线性回归模型：
```
Y = Xβ + ε, ε ~ N(0, σ²I)
```

最小二乘估计量 β̂ 具有以下性质：

1. **无偏性**
```
E(β̂) = β
```
估计量的期望等于真实值。

2. **正态分布**
```
β̂ ~ N(β, σ²(XᵀX)⁻¹)
```
估计量服从多元正态分布。

3. **协方差矩阵**
```
Cov(β̂) = σ²(XᵀX)⁻¹
```

### 单个系数的分布

对于第 j 个系数 β̂ⱼ：
```
β̂ⱼ ~ N(βⱼ, σ²vⱼ)
```

其中 vⱼ 是 (XᵀX)⁻¹ 的第 j 个对角元素。

**标准误差**：
```
SE(β̂ⱼ) = σ̂ √vⱼ
```

其中 σ̂ 是误差标准差的估计。

---

## 回归系数的置信区间

### 置信区间的作用

置信区间告诉我们：**有 95% 的把握，真实的系数值在这个范围内**。

### 单个系数的置信区间

对于置信水平 1-α（通常 α = 0.05）：

```
β̂ⱼ ± t_(n-p-1, α/2) × SE(β̂ⱼ)
```

其中：
- β̂ⱼ：估计的回归系数
- t_(n-p-1, α/2)：t分布的临界值
- SE(β̂ⱼ)：系数的标准误差
- n：样本量
- p：自变量个数

### 解读置信区间

**例1**：
```
房价 = 50000 + 10000×面积 + [8500, 11500]
                      ↑        ↑
                    估计值    95%置信区间
```

解读：我们有95%的把握，每平米价格的真实值在8500到11500元之间。

**例2**：
```
系数 = 0.5, 95% CI = [-0.2, 1.2]
```

解读：置信区间包含0，说明该系数可能不显著。

### 与假设检验的关系

- **置信区间不包含0**：系数显著
- **置信区间包含0**：系数不显著
- **置信区间越窄**：估计越精确
- **置信区间越宽**：估计越不确定

---

## 单个系数的t检验

### t检验的目的

检验**单个回归系数是否显著不为零**。

### 假设设置

**原假设（H₀）**：βⱼ = 0（系数为0，变量无影响）

**备择假设（H₁）**：βⱼ ≠ 0（系数不为0，变量有影响）

### t统计量

```
t = β̂ⱼ / SE(β̂ⱼ)
```

这个 t 值服从自由度为 n-p-1 的 t 分布。

### 检验步骤

1. **计算 t 值**
```
t = 估计值 / 标准误差
```

2. **找到临界值**
```
t_critical = t_(n-p-1, α/2)
```

3. **比较并决策**
   - |t| > t_critical：拒绝 H₀，系数显著
   - |t| ≤ t_critical：不拒绝 H₀，系数不显著

4. **计算 p 值**
   - p < α：拒绝 H₀
   - p ≥ α：不拒绝 H₀

### 例子说明

假设房价模型结果：
```
系数      估计值    标准误差    t值     p值
面积      10000     500       20.0    <0.001
卧室数    20000     15000     1.33    0.185
房龄     -1000     500       -2.0     0.046
截距     50000     10000      5.0    <0.001
```

解读：
- **面积**：t=20.0, p<0.001 → 极显著
- **卧室数**：t=1.33, p=0.185 → 不显著
- **房龄**：t=-2.0, p=0.046 → 显著（α=0.05）
- **截距**：t=5.0, p<0.001 → 显著

## 多个系数的联合检验（F检验）

### 为什么需要F检验？

t检验只能检验单个系数，但有时我们需要检验**多个系数同时是否为零**。

### 常见应用场景

1. **检验一组变量的整体显著性**
   - 例如：检验所有教育相关变量是否都不显著

2. **检验模型之间的差异**
   - 比较完整模型和简化模型

3. **检验线性约束**
   - 例如：β₁ = β₂（两个系数相等）

### F统计量的一般形式

```
F = (SSE_R - SSE_U) / (df_R - df_U) 
    ---------------------------------
                SSE_U / df_U
```

其中：
- SSE_R：简化模型（受限模型）的误差平方和
- SSE_U：完整模型（无受限模型）的误差平方和
- df_R, df_U：相应的自由度

### 模型比较的F检验

比较两个嵌套模型：

**完整模型（U）**：
```
y = β₀ + β₁x₁ + β₂x₂ + ... + βₚxₚ + ε
```

**简化模型（R）**：
```
y = β₀ + β₁x₁ + ... + βₖxₖ + ε
```

检验最后 q = p-k 个系数是否都为零。

**假设**：
- H₀: βₖ₊₁ = βₖ₊₂ = ... = βₚ = 0
- H₁: 至少一个系数不为零

**F统计量**：
```
F = [(SSE_R - SSE_U) / q] / [SSE_U / (n - p - 1)]
```

**判定**：如果 F > F_(q, n-p-1, α)，拒绝 H₀

---

## 模型的整体显著性检验

### F检验的特殊情况

当简化模型只有截距时，F检验变为**模型整体显著性检验**。

**原假设**：
```
H₀: β₁ = β₂ = ... = βₚ = 0
```

**备择假设**：
```
H₁: 至少有一个 βⱼ ≠ 0
```

### F统计量的另一种表达

```
F = (SSR / p) / (SSE / (n - p - 1))
  = MSR / MSE
```

其中：
- SSR：回归平方和
- SSE：误差平方和
- MSR = SSR/p：回归均方
- MSE = SSE/(n-p-1)：误差均方

### R²与F值的关系

```
F = [R² / p] / [(1 - R²) / (n - p - 1)]
```

因此，F检验本质上是在检验 R² 是否显著大于0。

### 解读示例

假设模型输出：
```
                            OLS Regression Results                            
==============================================================================
Dep. Variable:                    price   R-squared:                       0.854
Model:                            OLS   Adj. R-squared:                  0.850
Method:                 Least Squares   F-statistic:                     214.3
Date:                ...                Prob (F-statistic):           1.23e-45
==============================================================================
                 coef    std err          t      P>|t|      [0.025      0.975]
------------------------------------------------------------------------------
const       5.023e+04   1.02e+04      4.926      0.000    3.01e+04    7.04e+04
area           0.985      0.049     20.102      0.000       0.889       1.081
bedrooms   2.134e+04   1.51e+04      1.412      0.160    -8.5e+03    5.12e+04
age        -987.432    493.721     -2.000      0.047   -1963.354     -11.509
==============================================================================
F-statistic: 214.3, p-value: 1.23e-45
```

解读：

1. **模型整体**：
   - F = 214.3, p < 0.001
   - 模型整体高度显著

2. **单个变量**：
   - 面积：t=20.1, p<0.001 → 极显著
   - 卧室数：t=1.41, p=0.16 → 不显著
   - 房龄：t=-2.0, p=0.047 → 显著

3. **置信区间**：
   - 面积系数：95% CI = [0.889, 1.081]
   - 卧室数系数：95% CI = [-8500, 51200]（包含0，不显著）

## Python实现示例

### 使用statsmodels进行完整的统计推断

```python
import numpy as np
import pandas as pd
import statsmodels.api as sm
from scipy import stats

# 设置随机种子
np.random.seed(42)

# 创建示例数据
n_samples = 100
area = np.random.normal(100, 20, n_samples)  # 面积
bedrooms = np.random.randint(1, 5, n_samples)  # 卧室数
age = np.random.randint(0, 20, n_samples)  # 房龄

# 生成因变量（真实系数：10000, 20000, -1000）
price = 50000 + 10000 * area + 20000 * bedrooms - 1000 * age + np.random.normal(0, 20000, n_samples)

# 创建DataFrame
X = pd.DataFrame({
    'area': area,
    'bedrooms': bedrooms,
    'age': age
})
y = price

# 添加常数项
X_with_const = sm.add_constant(X)

# 拟合模型
model = sm.OLS(y, X_with_const).fit()

# 输出完整结果
print(model.summary())

# 提取关键信息
print("\n" + "="*60)
print("回归系数统计推断")
print("="*60)

# 1. 系数估计和置信区间
print("\n1. 系数估计和95%置信区间:")
print(model.conf_int(alpha=0.05))

# 2. 单个系数的t检验
print("\n2. 单个系数的t检验:")
print(model.summary().tables[1])

# 3. 模型整体F检验
print("\n3. 模型整体F检验:")
print(f"F-statistic: {model.fvalue:.4f}")
print(f"p-value: {model.f_pvalue:.6f}")
print(f"模型整体显著: {model.f_pvalue < 0.05}")

# 4. 手动计算置信区间
print("\n4. 手动计算95%置信区间:")
alpha = 0.05
df_resid = model.df_resid
t_critical = stats.t.ppf(1 - alpha/2, df_resid)

print(f"t临界值 (df={df_resid}, α/2={alpha/2}): {t_critical:.4f}")

ci_manual = pd.DataFrame({
    '估计值': model.params,
    '标准误差': model.bse,
    't值': model.tvalues,
    '下限': model.params - t_critical * model.bse,
    '上限': model.params + t_critical * model.bse
})
print(ci_manual)

# 5. 预测区间
print("\n5. 预测区间:")
new_data = pd.DataFrame({
    'const': [1],
    'area': [100],
    'bedrooms': [2],
    'age': [5]
})

pred = model.get_prediction(new_data)
pred_summary = pred.summary_frame(alpha=0.05)

print(f"预测值: {pred_summary['mean'][0]:.2f}")
print(f"置信区间: [{pred_summary['mean_ci_lower'][0]:.2f}, {pred_summary['mean_ci_upper'][0]:.2f}]")
print(f"预测区间: [{pred_summary['obs_ci_lower'][0]:.2f}, {pred_summary['obs_ci_upper'][0]:.2f}]")

# 6. 多个系数的联合检验
print("\n6. 多个系数的联合检验:")
# 检验 bedrooms 和 age 的系数是否都为0
hypothesis = 'bedrooms = 0, age = 0'
f_test = model.f_test(hypothesis)
print(f"检验假设: {hypothesis}")
print(f"F-statistic: {f_test.fvalue[0][0]:.4f}")
print(f"p-value: {f_test.pvalue:.6f}")
print(f"拒绝原假设: {f_test.pvalue < 0.05}")

# 7. 线性约束检验
print("\n7. 线性约束检验:")
# 检验 area 和 bedrooms 的系数是否相等
constraint = 'area = bedrooms/10000'  # 单位统一
linear_test = model.t_test(constraint)
print(f"检验约束: {constraint}")
print(f"t-statistic: {linear_test.tvalue[0][0]:.4f}")
print(f"p-value: {linear_test.pvalue:.6f}")
```

### 使用sklearn进行系数检验

```python
from sklearn.linear_model import LinearRegression
from sklearn.utils import resample
import numpy as np

# 使用sklearn拟合模型
model_sklearn = LinearRegression()
model_sklearn.fit(X, y)

print("sklearn系数估计:")
print(f"截距: {model_sklearn.intercept_:.2f}")
for feature, coef in zip(X.columns, model_sklearn.coef_):
    print(f"{feature}: {coef:.2f}")

# 使用Bootstrap方法计算置信区间
print("\nBootstrap 95%置信区间:")
n_bootstrap = 1000
bootstrap_coefs = []

for i in range(n_bootstrap):
    # 重采样
    X_sample, y_sample = resample(X, y, random_state=i)
    
    # 拟合模型
    model_sample = LinearRegression()
    model_sample.fit(X_sample, y_sample)
    
    # 保存系数
    bootstrap_coefs.append([
        model_sample.intercept_,
        *model_sample.coef_
    ])

bootstrap_coefs = np.array(bootstrap_coefs)

# 计算置信区间
ci_lower = np.percentile(bootstrap_coefs, 2.5, axis=0)
ci_upper = np.percentile(bootstrap_coefs, 97.5, axis=0)

names = ['截距'] + list(X.columns)
for name, coef, lower, upper in zip(names, model_sklearn.intercept_, ci_lower, ci_upper):
    print(f"{name}: [{lower:.2f}, {upper:.2f}]")
```

## 检验结果的实际应用

### 1. 变量选择

根据检验结果选择变量：

```python
# 只保留显著变量
significant_vars = model.pvalues[model.pvalues < 0.05].index
print("显著的变量:", significant_vars.tolist())

X_significant = X[list(significant_vars.drop('const'))]
X_significant = sm.add_constant(X_significant)

model_reduced = sm.OLS(y, X_significant).fit()
print("\n简化后的模型:")
print(model_reduced.summary())
```

### 2. 模型诊断

```python
# 计算VIF（方差膨胀因子）
from statsmodels.stats.outliers_influence import variance_inflation_factor

vif_data = pd.DataFrame()
vif_data["变量"] = X.columns
vif_data["VIF"] = [variance_inflation_factor(X.values, i) 
                   for i in range(X.shape[1])]

print("\n方差膨胀因子（VIF）:")
print(vif_data)

# VIF > 10 表示存在多重共线性
high_vif = vif_data[vif_data["VIF"] > 10]
if len(high_vif) > 0:
    print(f"\n警告：以下变量存在严重多重共线性: {high_vif['变量'].tolist()}")
```

### 3. 影响诊断

```python
# 计算影响力指标
influence = model.get_influence()

# Cook's distance
cooks_d = influence.cooks_distance[0]
print("\nCook's距离（检测强影响点）:")
print(f"最大值: {cooks_d.max():.4f}")
print(f"平均值: {cooks_d.mean():.4f}")

# 通常 4/(n-p) 作为阈值
threshold = 4 / (n_samples - X.shape[1] - 1)
influential_points = np.where(cooks_d > threshold)[0]
print(f"Cook's距离 > {threshold:.4f} 的点: {len(influential_points)} 个")
print(f"索引: {influential_points}")

# 杠杆值
leverage = influence.hat_matrix_diag
print(f"\n杠杆值:")
print(f"最大值: {leverage.max():.4f}")
print(f"平均值: {2 * (X.shape[1] + 1) / n_samples:.4f}")
```

## 常见问题和注意事项

### 1. 多重共线性问题

**现象**：
- 标准误差过大
- t值小但模型整体F值大
- 系数符号与预期相反

**解决方法**：
- 删除相关变量
- 使用岭回归
- 主成分分析

### 2. 异方差问题

**现象**：
- 残差方差随预测值变化
- 标准误估计不准确

**解决方法**：
- 对变量进行变换
- 使用加权最小二乘法
- 使用稳健标准误

### 3. 样本量问题

**经验法则**：
- 每个预测变量至少10个样本
- 最少需要 n > p + 10

**小样本的影响**：
- 置信区间变宽
- 检验功效降低
- 结果不稳定

### 4. 多重比较问题

当进行多次检验时，需要调整显著性水平：

```python
# Bonferroni校正
n_tests = len(model.params)
alpha_corrected = 0.05 / n_tests

print(f"原始α: 0.05")
print(f"Bonferroni校正α: {alpha_corrected:.4f}")
print(f"显著变量数(α=0.05): {(model.pvalues < 0.05).sum()}")
print(f"显著变量数(α={alpha_corrected:.4f}): {(model.pvalues < alpha_corrected).sum()}")
```

## 总结

**回归系数的统计推断**：

### 核心概念

1. **抽样分布**：β̂ ~ N(β, σ²(XᵀX)⁻¹)
2. **置信区间**：β̂ ± t × SE(β̂)
3. **t检验**：检验单个系数
4. **F检验**：检验多个系数
5. **模型检验**：检验整体显著性

- 注意多重比较问题
