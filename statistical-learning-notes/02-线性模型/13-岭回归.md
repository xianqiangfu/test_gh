# 岭回归

## 什么是岭回归？

岭回归是**带 L2 正则化的线性回归**。

简单说：**在线性回归基础上，加一个约束，不让参数太大。**

---

## 为什么要正则化？

### 过拟合问题

当模型太复杂时，容易过拟合：

- 训练集误差很小
- 测试集误差很大
- 参数变得很大，不稳定

### 正则化的作用

约束参数的大小，让模型更稳定，避免过拟合。

---

## 岭回归的数学

### 原始损失（均方误差）

```
L = Σ(yᵢ - (wxᵢ + b))² / n
```

### 岭回归的损失

```
L = Σ(yᵢ - (wxᵢ + b))² / n + α × Σw²
```

- 第一项：均方误差
- 第二项：L2 正则化（权重的平方和）
- α：正则化强度（超参数）

### 简化表示

```
L = MSE + α × ||w||²
```

其中 ||w||² 是权重的 L2 范数。

---

## 直观理解

### 为什么约束权重？

权重太大意味着模型对某些特征"过度敏感"。

约束权重，让模型对每个特征都"温和"一些。

### 图像理解

想象损失函数的等高线：

```
损失值
  ↑
  │    ╭───── 最小值
  │   ╱
  │  ╱
  │ ╱
  │●
  │ 最小化 MSE+α||w||²
  └────────────→ w
```

正则化后，最小值会被"拉"向原点（权重更小）。

---

## 求解方法

### 解析解

岭回归有解析解：

```
ŵ = (XᵀX + αI)⁻¹Xᵀy
```

与普通最小二乘的区别：
- XᵀX 变成了 XᵀX + αI
- I 是单位矩阵

### 为什么能求解？

普通线性回归可能 XᵀX 不可逆。

加上 αI 后，矩阵一定可逆。

---

## 正则化强度 α 的影响

### α = 0

```
L = MSE + 0 × ||w||² = MSE
```

就是普通线性回归，没有正则化。

### α 很小

轻微约束权重，参数略有减小。

### α 适中

适度约束，平衡偏差和方差。

### α 很大

强约束，权重接近0，接近欠拟合。

---

## 偏差-方差权衡

### α 太小

- 低偏差
- 高方差
- 可能过拟合

### α 太大

- 高偏差
- 低方差
- 可能欠拟合

### α 适中

- 平衡偏差和方差
- 通常有最好的泛化性能

---

## 与其他方法的比较

### 岭回归 vs 线性回归

| 特点 | 线性回归 | 岭回归 |
|------|---------|--------|
| 正则化 | 无 | L2 |
| 稳定性 | 不稳定 | 稳定 |
| 特征选择 | 无 | 无 |
| 适合场景 | 简单问题 | 复杂问题、特征相关 |

### 岭回归 vs Lasso

| 特点 | 岭回归 (L2) | Lasso (L1) |
|------|------------|------------|
| 正则化 | 平方和 | 绝对值和 |
| 特征选择 | 无权重为0 | 可得稀疏解 |
| 计算方式 | 有解析解 | 无解析解 |
| 适合场景 | 特征相关 | 特征选择 |

---

## 何时使用岭回归？

### 适合的情况

✅ 特征之间存在相关性
✅ 特征数量多于样本数
✅ 需要稳定的模型
✅ 不需要特征选择

### 不适合的情况

❌ 需要特征选择（用Lasso更好）
❌ 数据非常简单，不需要正则化

---

## 实际例子

### 例子：房价预测

假设有10个特征：面积、位置、房龄、楼层、朝向、学区、周边、装修、车位、绿化。

这些特征可能相关（如位置和学区）。

### 线性回归

可能会给某些特征很大的权重，导致过拟合。

### 岭回归

约束权重，让模型更稳定：

```
面积权重：从 5000 → 3000
位置权重：从 8000 → 4000
房龄权重：从 -200 → -150
...
```

结果：训练集误差略增，但测试集误差降低。

---

## 超参数调优

### 如何选择 α？

1. **交叉验证**
   - 尝试不同的 α 值
   - 用验证集评估

2. **常用范围**

   ```
   α = 10^(-4), 10^(-3), ..., 10^0, 10^1, ...
   ```

3. **对数尺度**

   在对数尺度上尝试，如 0.01, 0.1, 1, 10, 100

### 交叉验证曲线

```
验证误差
  ↑
  │      ╱╲
  │     ╱  ╲
  │    ╱    ╲
  │   ╱      ╲
  │  ╱        ╲
  │ ╱          ╲
  └──────────────→ α
      最佳 α
```

找到使验证误差最小的 α。

---

## 数据预处理

### 标准化很重要

岭回归需要对特征进行**标准化**。

**原因**：
- 正则化是"约束权重大小"
- 如果特征尺度不同，影响不均匀

**方法**：
```
x' = (x - 均值) / 标准差
```

### 不标准化的后果

- 大尺度特征的正则化更强
- 结果不可靠

---

## 缩减路径

### 缩减系数

随着 α 从 0 到 ∞，权重逐渐缩减到0。

```
α = 0:     [5, 3, 2, 1, 0.5]
α = 1:     [3, 2, 1.5, 0.8, 0.3]
α = 10:    [1, 0.6, 0.5, 0.3, 0.1]
α = 100:   [0.3, 0.2, 0.15, 0.1, 0.03]
α = ∞:     [0, 0, 0, 0, 0]
```

注意：岭回归的权重不会精确变为0，只是接近。

---

## 实践技巧

### 1. 先标准化

必须对特征进行标准化。

### 2. 交叉验证

用交叉验证找最佳 α。

### 3. 对数尺度

在对数尺度上尝试 α 值。

### 4. 监控权重

观察权重如何随 α 变化。

from sklearn.preprocessing import StandardScaler

## 总结

**岭回归**：

1. 是线性回归 + L2 正则化
2. 约束权重大小，防止过拟合
3. 有解析解，计算高效
4. 适合特征相关的情况
5. 权重不会精确为0

**核心思想**：
- 加一个惩罚项：α × ||w||²
- 约束权重大小
- 平衡偏差和方差

**关键要点**：
- 标准化很重要
- 用交叉验证调 α
- 特征相关时效果好
- 比Lasso更稳定

**记住**：岭回归就像给模型套上"缰绳"，不让它跑得太远！
