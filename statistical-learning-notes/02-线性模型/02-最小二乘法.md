# 最小二乘法

## 什么是最小二乘法？

最小二乘法（Ordinary Least Squares，简称OLS）是**线性回归的求解方法**。

简单说：**找到一条直线，让所有点到这条线的距离平方和最小。**

---

## 为什么要用"二乘"？

"二乘"就是"平方"的意思。

我们计算误差时，把误差**平方**后再相加，所以叫"最小二乘法"。

## 为什么是平方？

不用绝对值，而用平方，有几个原因：

1. **数学性质好**
   - 平方函数处处可导
   - 求导方便，容易计算

2. **强调大误差**
   - 平方会放大大误差
   - 让模型更关注那些预测得很差的点

3. **计算简单**
   - 可以推导出闭式解（直接用公式算）
   - 不需要迭代

---

## 直观理解

想象你有一组数据点，想找一条直线拟合：

```
y
│    ●
│   ╱ ╲
│  ●   ●
│ ╱     ╲
│●       ●
└────────→ x
```

最小二乘法就是要找到一条直线，使得：

- 每个点到直线的垂直距离
- 把这些距离平方后相加
- 这个和是最小的

```
距离²的总和 = (d₁² + d₂² + d₃² + ...) 最小
```

---

## 数学表达

### 误差的计算

对于每个数据点：
```
误差 = 预测值 - 真实值
误差 = (wx + b) - y
```

### 最小二乘目标

```
最小化：Σ(预测值 - 真实值)²
最小化：Σ[(wx + b) - y]²
```

### 展开来看

对于n个数据点：
```
损失函数 = [(wx₁ + b - y₁)² +
           (wx₂ + b - y₂)² +
           ...
           (wxₙ + b - yₙ)²] / n
```

这就是**均方误差 (MSE)**。

---

## 如何求解？

### 方法1：解析解（闭式解）

最小二乘法有一个**闭式解**，可以直接用公式算出最优的w和b。

**简单情况（一元）**：
```
w = Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)²
b = ȳ - w × x̄
```

其中：
- x̄, ȳ 是均值
- Σ 表示求和

**向量形式（多元）**：
```
w = (XᵀX)⁻¹Xᵀy
```

这是著名的**正规方程**。

### 方法2：梯度下降

当数据量很大或特征很多时，用梯度下降迭代求解。

---

## 解析解 vs 梯度下降

| 特点 | 解析解 | 梯度下降 |
|------|--------|----------|
| 计算方式 | 一次计算得出 | 迭代求解 |
| 计算复杂度 | O(n³)，数据量大时慢 | O(n²) |
| 是否需要调参 | 不需要 | 需要学习率等参数 |
| 适用场景 | 数据量小 | 数据量大 |

---

## 为什么最小化平方和？

### 对比其他损失函数

| 损失函数 | 公式 | 特点 |
|---------|------|------|
| 平方损失 | (预测 - 真实)² | 对大误差敏感，可导 |
| 绝对损失 | |预测 - 真实| | 对大误差不那么敏感，不可导 |
| Huber损失 | 结合两者 | 兼顾两者优点 |

**平方损失最常用**，因为数学性质好。

---

## 几何意义

从几何角度看，最小二乘法是：

**在特征空间中找到一个点，使得预测向量尽可能接近真实向量。**

用向量的语言说：

```
预测向量 = Xw
目标向量 = y

最小化 ||Xw - y||²
```

这就是说：在X的列空间中，找最接近y的向量。

---

## 最小二乘法的性质

### 1. 无偏性

如果满足某些假设，最小二乘估计是无偏的：
```
E[ŵ] = w (真实值)
```

意思是：平均来说，估计值等于真实值。

### 2. 最佳线性无偏估计 (BLUE)

在所有线性无偏估计中，最小二乘估计方差最小。

这是著名的**高斯-马尔可夫定理**。

### 3. 正态性

如果误差服从正态分布，最小二乘估计也是正态分布的。

## 实际例子

### 简单例子

数据：
```
x: 1, 2, 3, 4, 5
y: 2, 4, 5, 4, 5
```

计算：
```
x̄ = (1+2+3+4+5)/5 = 3
ȳ = (2+4+5+4+5)/5 = 4

w = Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)²
w = [(1-3)(2-4) + (2-3)(4-4) + ...] / [(1-3)² + ...]
w = 0.6

b = ȳ - w × x̄ = 4 - 0.6 × 3 = 2.2

所以：y = 0.6x + 2.2
```

## 常见问题

### 问题1：不可逆怎么办？

如果 XᵀX 不可逆（满秩），无法求逆。

**解决方法**：
- 删除相关特征
- 使用正则化（岭回归）

### 问题2：多重共线性

如果特征之间高度相关，最小二乘估计不稳定。

**解决方法**：
- 删除相关特征
- 使用正则化
- 使用PCA降维

### 问题3：异常值

最小二乘对异常值敏感。

**解决方法**：
- 清理数据
- 使用鲁棒回归

---

## 总结

**最小二乘法**：

1. 目标：最小化预测误差的平方和
2. 方法：有解析解，可以直接计算
3. 性质：无偏、BLUE
4. 局限：对异常值敏感、需要满秩

**关键理解**：
- "二乘" = 平方
- 最小化平方误差和
- 是线性回归的标准求解方法
- 数学基础扎实，理解它对理解其他算法有帮助

最小二乘法是统计学和机器学习的基石。掌握它，就掌握了理解更复杂算法的钥匙！
